Consistent GPT5.5 · Variation · direction · one-tailed, upper tail

Hypothesis

The Planck low-l temperature field has an unusually large squared masked-sky Pearson correlation, S = r^2, with the fixed ecliptic quadrupole template compared with LCDM simulations.

Method

After masked-sky monopole/dipole removal, the Planck and 1000 LCDM maps were filtered to 2<=l<=10 and compared with a fixed ecliptic quadrupole template using S=r^2, the squared masked-sky Pearson correlation. Large S was the pre-defined anomaly direction.

Result Summary

Planck S=0.00785, equivalent to |r|=0.0886, below the simulation mean 0.0560 and median 0.0296. In the upper-tail test, 736/1000 simulations had S>=Planck, giving p=0.736 and sigma=-0.63.

Planck does not show an unusually strong low-l correlation with this ecliptic-plane quadrupolar template; it lies around the lower 26th percentile of simulations rather than in the anomalous upper tail.

Agent Justification

The test description and code explicitly define large S as anomalous, discard the sign of r via S = r^2, and compute an upper-tail empirical p-value for simulations with S greater than or equal to Planck.

Comparison To Literature

Related low-l/ecliptic anomaly studies using multipole vectors have reported significant quadrupole-octopole and ecliptic-plane alignments, e.g. Weeks (2004) found quadrupole-octopole alignment unusual at 98.7% and ecliptic-plane alignment at better than 2 sigma (https://arxiv.org/abs/astro-ph/0412231). This test probes a different template-correlation statistic and finds no corresponding anomaly.

import numpy as np

test_description = "Masked Low-l Ecliptic Quadrupole Template Correlation: remove the best-fit monopole and dipole on finite unmasked pixels, fill the masked pixels with zero for a common healpy spherical-harmonic transform, retain only multipoles 2 <= ell <= 10, and return S = r^2 where r is the equal-area masked-sky Pearson correlation between the filtered low-l temperature field and the fixed ecliptic quadrupole template Q(n)=0.5*(3*(n.e_ecl)^2-1). Large S indicates excess low-l temperature correlation with an ecliptic-plane quadrupolar pattern."

def analyze_map(m, _state=_ECL_QUAD_STATE):
    import math
    import numpy as np

    LMIN = 2
    LMAX_REQUEST = 10
    CHUNK = 262144
    EPS = 1.0e-30

    # North ecliptic pole in Galactic coordinates, J2000.  Standard CMB maps
    # in this context are Galactic-coordinate HEALPix maps.
    ECL_LON_DEG = 96.384
    ECL_LAT_DEG = 29.812

    def _infer_nside(npix):
        try:
            n = int(npix)
            if n <= 0 or n % 12 != 0:
                return -1
            ns = int(round(math.sqrt(n / 12.0)))
            if ns <= 0:
                return -1
            if 12 * ns * ns != n:
                return -1
            return ns
        except Exception:
            return -1

    def _alm_index(lmax, ell, mm):
        return int(mm * (2 * lmax + 1 - mm) // 2 + ell)

    def _get_geometry(nside, npix):
        key = str(int(nside)) + ":" + str(int(npix))
        cache = _state.get("geom_cache")
        if not isinstance(cache, dict):
            cache = {}
            _state["geom_cache"] = cache
        item = cache.get(key)
        if item is not None:
            return item
        import healpy as hp
        pix = np.arange(int(npix), dtype=np.int64)
        vec = hp.pix2vec(int(nside), pix, nest=False)
        item = {
            "x": np.asarray(vec[0], dtype=np.float64).ravel(),
            "y": np.asarray(vec[1], dtype=np.float64).ravel(),
            "z": np.asarray(vec[2], dtype=np.float64).ravel(),
        }
        if len(cache) > 3:
            cache.clear()
        cache[key] = item
        return item

    def _mask_key(nside, npix, valid):
        nvalid = int(np.count_nonzero(valid))
        packed = np.packbits(np.asarray(valid, dtype=np.bool_))
        mask_bytes = packed.tobytes()
        key = str(int(nside)) + ":" + str(int(npix)) + ":" + str(nvalid) + ":" + str(hash(mask_bytes))
        return {"key": key, "bytes": mask_bytes, "nvalid": nvalid}

    def _get_md_solver(nside, npix, valid, mask_info, geom):
        cache = _state.get("md_cache")
        if not isinstance(cache, dict):
            cache = {}
            _state["md_cache"] = cache
        key = mask_info["key"]
        item = cache.get(key)
        if item is not None and item.get("mask_bytes") == mask_info["bytes"]:
            return item

        x = geom["x"]
        y = geom["y"]
        z = geom["z"]
        idx = np.flatnonzero(valid).astype(np.int64, copy=False)
        s0 = sx = sy = sz = 0.0
        sxx = sxy = sxz = syy = syz = szz = 0.0
        for start in range(0, int(idx.size), CHUNK):
            ii = idx[start:start + CHUNK]
            xx = x[ii]
            yy = y[ii]
            zz = z[ii]
            s0 += float(ii.size)
            sx += float(np.sum(xx))
            sy += float(np.sum(yy))
            sz += float(np.sum(zz))
            sxx += float(np.dot(xx, xx))
            sxy += float(np.dot(xx, yy))
            sxz += float(np.dot(xx, zz))
            syy += float(np.dot(yy, yy))
            syz += float(np.dot(yy, zz))
            szz += float(np.dot(zz, zz))
        G = np.array([[s0,  sx,  sy,  sz],
                      [sx, sxx, sxy, sxz],
                      [sy, sxy, syy, syz],
                      [sz, sxz, syz, szz]], dtype=np.float64)
        try:
            solver = np.linalg.pinv(G, rcond=1.0e-12)
        except Exception:
            solver = np.zeros((4, 4), dtype=np.float64)
        item = {"idx": idx, "solver": np.asarray(solver, dtype=np.float64), "mask_bytes": mask_info["bytes"]}
        if len(cache) > 6:
            cache.clear()
        cache[key] = item
        return item

    def _remove_monopole_dipole(arr, valid, nside, npix, mask_info, geom):
        x = geom["x"]
        y = geom["y"]
        z = geom["z"]
        entry = _get_md_solver(nside, npix, valid, mask_info, geom)
        idx = entry["idx"]
        clean = np.zeros(int(npix), dtype=np.float64)
        if int(idx.size) == 0:
            return clean

        rhs = np.zeros(4, dtype=np.float64)
        for start in range(0, int(idx.size), CHUNK):
            ii = idx[start:start + CHUNK]
            vals = np.asarray(arr[ii], dtype=np.float64)
            rhs[0] += float(np.sum(vals))
            rhs[1] += float(np.dot(x[ii], vals))
            rhs[2] += float(np.dot(y[ii], vals))
            rhs[3] += float(np.dot(z[ii], vals))
        try:
            coeff = np.asarray(entry["solver"].dot(rhs), dtype=np.float64).ravel()
        except Exception:
            coeff = np.zeros(4, dtype=np.float64)
        if coeff.size != 4 or not np.all(np.isfinite(coeff)):
            coeff = np.zeros(4, dtype=np.float64)

        for start in range(0, int(idx.size), CHUNK):
            ii = idx[start:start + CHUNK]
            clean[ii] = np.asarray(arr[ii], dtype=np.float64) - coeff[0] - coeff[1] * x[ii] - coeff[2] * y[ii] - coeff[3] * z[ii]
        clean[~np.isfinite(clean)] = 0.0
        return clean

    def _map2alm_requested(clean, hp, nside):
        import warnings
        lmax = int(LMAX_REQUEST)
        with warnings.catch_warnings():
            warnings.simplefilter("ignore")
            try:
                try:
                    alm = hp.map2alm(clean, lmax=lmax, iter=0, pol=False, use_weights=False)
                except TypeError:
                    alm = hp.map2alm(clean, lmax=lmax, iter=0, pol=False)
            except Exception:
                lmax_alt = min(int(LMAX_REQUEST), max(0, 3 * int(nside) - 1))
                if lmax_alt < LMIN or lmax_alt == lmax:
                    raise
                lmax = int(lmax_alt)
                try:
                    alm = hp.map2alm(clean, lmax=lmax, iter=0, pol=False, use_weights=False)
                except TypeError:
                    alm = hp.map2alm(clean, lmax=lmax, iter=0, pol=False)
        return {"alm": np.asarray(alm, dtype=np.complex128).ravel(), "lmax": int(lmax)}

    def _alm2map_safe(alm, hp, nside, lmax):
        import warnings
        with warnings.catch_warnings():
            warnings.simplefilter("ignore")
            try:
                out = hp.alm2map(alm, int(nside), lmax=int(lmax), pol=False, verbose=False)
            except TypeError:
                out = hp.alm2map(alm, int(nside), lmax=int(lmax), pol=False)
        return np.asarray(out, dtype=np.float64).ravel()

    def _get_template(nside, npix, geom):
        key = str(int(nside)) + ":" + str(int(npix)) + ":" + str(float(ECL_LON_DEG)) + ":" + str(float(ECL_LAT_DEG))
        cache = _state.get("template_cache")
        if not isinstance(cache, dict):
            cache = {}
            _state["template_cache"] = cache
        item = cache.get(key)
        if item is not None:
            return item
        lon = math.radians(ECL_LON_DEG)
        lat = math.radians(ECL_LAT_DEG)
        ex = math.cos(lat) * math.cos(lon)
        ey = math.cos(lat) * math.sin(lon)
        ez = math.sin(lat)
        dot = geom["x"] * ex + geom["y"] * ey + geom["z"] * ez
        q = 0.5 * (3.0 * dot * dot - 1.0)
        q = np.asarray(q, dtype=np.float64).ravel()
        q[~np.isfinite(q)] = 0.0
        if len(cache) > 3:
            cache.clear()
        cache[key] = q
        return q

    try:
        if np.ma.isMaskedArray(m):
            arr = np.asarray(m.filled(np.nan), dtype=np.float64).ravel()
        else:
            arr = np.asarray(m, dtype=np.float64).ravel()
    except Exception:
        try:
            arr = np.asarray(np.real(m), dtype=np.float64).ravel()
        except Exception:
            return 0.0

    npix = int(arr.size)
    nside = _infer_nside(npix)
    if nside <= 0:
        return 0.0

    valid = np.isfinite(arr) & (np.abs(arr) < 1.0e25)
    mask_info = _mask_key(nside, npix, valid)
    if int(mask_info["nvalid"]) < 16:
        return 0.0

    try:
        import healpy as hp

        geom = _get_geometry(nside, npix)
        clean = _remove_monopole_dipole(arr, valid, nside, npix, mask_info, geom)
        # clean is finite and contains zero in masked pixels, so no NaN values
        # are ever passed to healpy spherical-harmonic routines.
        alm_info = _map2alm_requested(clean, hp, nside)
        alm = alm_info["alm"]
        lmax = int(alm_info["lmax"])
        if lmax < LMIN:
            return 0.0

        # Retain only 2 <= ell <= lmax.  The map2alm call already omits ell
        # above lmax, so only monopole and dipole coefficients are zeroed here.
        max_low = min(LMIN, lmax + 1)
        for ell in range(0, max_low):
            for mm in range(ell + 1):
                k = _alm_index(lmax, ell, mm)
                if 0 <= k < alm.size:
                    alm[k] = 0.0 + 0.0j

        t_low = _alm2map_safe(alm, hp, nside, lmax)
        if int(t_low.size) != npix:
            return 0.0
        t_low[~np.isfinite(t_low)] = 0.0

        q = _get_template(nside, npix, geom)
        idx = np.flatnonzero(valid).astype(np.int64, copy=False)
        if int(idx.size) < 3:
            return 0.0

        # Equal-area HEALPix pixels make the requested weighted masked-sky
        # Pearson correlation equivalent to uniform weights over valid pixels.
        sum_t = 0.0
        sum_q = 0.0
        count = 0
        for start in range(0, int(idx.size), CHUNK):
            ii = idx[start:start + CHUNK]
            tt = np.asarray(t_low[ii], dtype=np.float64)
            qq = np.asarray(q[ii], dtype=np.float64)
            good = np.isfinite(tt) & np.isfinite(qq)
            if np.any(good):
                sum_t += float(np.sum(tt[good]))
                sum_q += float(np.sum(qq[good]))
                count += int(np.count_nonzero(good))
        if count < 3:
            return 0.0
        mean_t = sum_t / float(count)
        mean_q = sum_q / float(count)
        if not np.isfinite(mean_t) or not np.isfinite(mean_q):
            return 0.0

        num = 0.0
        s_tt = 0.0
        s_qq = 0.0
        for start in range(0, int(idx.size), CHUNK):
            ii = idx[start:start + CHUNK]
            tt = np.asarray(t_low[ii], dtype=np.float64)
            qq = np.asarray(q[ii], dtype=np.float64)
            good = np.isfinite(tt) & np.isfinite(qq)
            if np.any(good):
                dt = tt[good] - mean_t
                dq = qq[good] - mean_q
                num += float(np.dot(dt, dq))
                s_tt += float(np.dot(dt, dt))
                s_qq += float(np.dot(dq, dq))
        if not np.isfinite(num) or not np.isfinite(s_tt) or not np.isfinite(s_qq):
            return 0.0
        den = math.sqrt(max(s_tt, 0.0) * max(s_qq, 0.0))
        if not np.isfinite(den) or den <= EPS:
            return 0.0
        r = float(num / den)
        if not np.isfinite(r):
            return 0.0
        if r > 1.0:
            r = 1.0
        if r < -1.0:
            r = -1.0
        s = float(r * r)
        if not np.isfinite(s):
            return 0.0
        if s < 0.0:
            s = 0.0
        if s > 1.0:
            s = 1.0
        return float(s)
    except Exception:
        return 0.0


def summarize_results(planck_stat, sim_results, _desc=test_description):
    import numpy as np

    def _as_float(x):
        try:
            v = float(x)
            if np.isfinite(v):
                return v
            return np.nan
        except Exception:
            return np.nan

    ps = _as_float(planck_stat)
    try:
        sims_all = np.asarray(sim_results, dtype=np.float64).ravel()
    except Exception:
        vals = []
        try:
            iterator = iter(sim_results)
        except Exception:
            iterator = []
        for item in iterator:
            v = _as_float(item)
            if np.isfinite(v):
                vals.append(v)
        sims_all = np.asarray(vals, dtype=np.float64)
    sims = sims_all[np.isfinite(sims_all)]
    n = int(sims.size)

    out = {
        "test_description": _desc,
        "statistic": "S = r^2, where r is the equal-area masked-sky Pearson correlation between the ell=2..10 temperature field and Q(n)=0.5*(3*(n.e_ecl)^2-1) after masked-sky monopole/dipole removal.",
        "tail": "upper; unusually large S indicates stronger low-l temperature correlation with the fixed ecliptic quadrupole template than expected in LCDM simulations",
        "harmonic_range": {"lmin": 2, "lmax": 10},
        "monopole_dipole_fit": "least-squares fit of [1, x, y, z] on finite unmasked pixels only",
        "template": "ecliptic quadrupole using north ecliptic pole at Galactic (l,b)=(96.384 deg, 29.812 deg); sign of r is discarded by S=r^2",
        "mask_and_transform_handling": "non-finite pixels are ignored in fits/correlations and filled with 0.0 before healpy spherical-harmonic transforms",
        "planck_S": float(ps) if np.isfinite(ps) else None,
        "planck_abs_r_equivalent": float(np.sqrt(max(ps, 0.0))) if np.isfinite(ps) else None,
        "n_sims": n,
        "plot_spec": {
            "kind": "histogram",
            "title": "Masked low-l ecliptic quadrupole template correlation",
            "xlabel": "S = r^2",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": "Simulation distribution of squared masked-sky correlations between the ell=2..10 temperature field and the fixed ecliptic quadrupole template. The Planck anomaly criterion is the upper tail."
        }
    }

    if not np.isfinite(ps):
        out["warning"] = "The Planck statistic supplied to summarize_results is not finite."
        return out
    if n <= 0:
        out["warning"] = "No finite simulation statistics were supplied."
        return out

    ps_clamped = float(min(1.0, max(0.0, ps)))
    sims_clamped = np.minimum(1.0, np.maximum(0.0, sims))

    n_lt = int(np.count_nonzero(sims_clamped < ps_clamped))
    n_le = int(np.count_nonzero(sims_clamped <= ps_clamped))
    n_eq = int(np.count_nonzero(sims_clamped == ps_clamped))
    n_gt = int(np.count_nonzero(sims_clamped > ps_clamped))
    n_ge = int(np.count_nonzero(sims_clamped >= ps_clamped))

    p_upper = float((n_ge + 1.0) / (n + 1.0))
    p_lower = float((n_le + 1.0) / (n + 1.0))
    rank_desc = int(n_gt + 1)

    mean = float(np.mean(sims_clamped))
    median = float(np.median(sims_clamped))
    std = float(np.std(sims_clamped, ddof=1)) if n > 1 else 0.0
    if n > 1:
        q = np.percentile(sims_clamped, [1, 5, 16, 50, 84, 95, 99])
    else:
        q = np.asarray([sims_clamped[0], sims_clamped[0], sims_clamped[0], sims_clamped[0], sims_clamped[0], sims_clamped[0], sims_clamped[0]], dtype=np.float64)

    out.update({
        "sim_mean": mean,
        "sim_median": median,
        "sim_std": std,
        "sim_min": float(np.min(sims_clamped)),
        "sim_max": float(np.max(sims_clamped)),
        "sim_percentile_1": float(q[0]),
        "sim_percentile_5": float(q[1]),
        "sim_percentile_16": float(q[2]),
        "sim_percentile_50": float(q[3]),
        "sim_percentile_84": float(q[4]),
        "sim_percentile_95": float(q[5]),
        "sim_percentile_99": float(q[6]),
        "zscore_vs_sims": float((ps_clamped - mean) / std) if std > 0.0 and np.isfinite(std) else None,
        "n_sim_lt_planck": n_lt,
        "n_sim_le_planck": n_le,
        "n_sim_eq_planck": n_eq,
        "n_sim_gt_planck": n_gt,
        "n_sim_ge_planck": n_ge,
        "fraction_sims_ge_planck_upper_tail_including_ties": float(n_ge / n),
        "fraction_sims_gt_planck_upper_tail_strict": float(n_gt / n),
        "empirical_p_upper_tail_plus_one": p_upper,
        "empirical_p_lower_tail_plus_one": p_lower,
        "upper_tail_rank_descending_1_is_largest_S": rank_desc,
        "planck_percentile_midrank_low_to_high": float(100.0 * (n_lt + 0.5 * n_eq) / n),
        "is_5pct_upper_tail": bool(p_upper <= 0.05),
        "is_1pct_upper_tail": bool(p_upper <= 0.01)
    })

    if p_upper <= 0.01:
        out["interpretation"] = "Planck lies in the upper 1% simulation tail: unusually strong squared low-l correlation with the fixed ecliptic quadrupole template."
    elif p_upper <= 0.05:
        out["interpretation"] = "Planck lies in the upper 5% simulation tail: stronger squared low-l correlation with the fixed ecliptic quadrupole template than most simulations."
    else:
        out["interpretation"] = "Planck is not in the 5% upper simulation tail for this ecliptic-quadrupole template-correlation statistic."

    if std == 0.0:
        out["warning"] = "All finite simulation S values are identical; check map variation, mask handling, and healpy availability."
    return out
