Borderline GPT5.5 · Novel · other · one-tailed, upper tail

Hypothesis

The Planck masked low-l pseudo-a_lm phases show anomalously strong adjacent-multipole phase coherence, i.e. Planck has a larger R than expected from LCDM simulations.

Method

Computes masked, monopole/dipole-subtracted pseudo-a_lm phases for 2<=l<=20 and measures R, the mean resultant length of adjacent-multipole phase-difference phasors arg(a_{l+1,m}a*_{l,m}) over m>=1. Larger R indicates stronger low-l phase locking.

Result Summary

Planck has R=0.1388 versus simulation mean 0.0665 and median 0.0636. It ranks at the 97th percentile, with 30/1000 simulations larger; one-tailed upper p=0.03, corresponding to about 1.88 sigma.

Planck shows unusually high adjacent low-l phase coherence relative to the LCDM masked-pseudo-a_lm simulations, significant at the 5% one-tailed level but not at 1% and only marginally below 2 sigma.

Agent Justification

The statistic R measures the concentration of adjacent phase-difference phasors, and the description/code explicitly state that unusually large R indicates coherent low-l phase locking. The reported tail of interest is the upper tail, so the directional anomaly is excess coherence rather than any deviation in either direction.

Comparison To Literature

I am not aware of a directly comparable published test using this exact masked low-l adjacent-multipole pseudo-a_lm phase-coherence statistic, and prior searches found no close match; this should not be treated as a repeat of a standard anomaly statistic.

import math
import numpy as np


test_description = "Masked Low-l Adjacent-Phase Coherence: after fitting and subtracting the monopole and dipole on the unmasked sky, set masked pixels to zero, compute masked pseudo-a_lm in Galactic/RING pixel basis for 2<=l<=20, and return R=|sum exp(i arg(a_{l+1,m} a^*_{l,m}))|/N over adjacent pairs 2<=l<=19 and 1<=m<=l."

_LMIN = 2
_LMAX = 20
_PAIR_LMAX = _LMAX - 1
_EXPECTED_NPAIR = sum(range(_LMIN, _PAIR_LMAX + 1))
_STATE = {}


def _as_1d_float(m):
    try:
        if np.ma.isMaskedArray(m):
            a = np.asarray(m.filled(np.nan), dtype=np.float64)
        else:
            a = np.asarray(m, dtype=np.float64)
    except Exception:
        try:
            a = np.asarray(np.real(m), dtype=np.float64)
        except Exception:
            a = np.asarray([], dtype=np.float64)
    return np.ravel(a)


def _finite_float(x, default=np.nan):
    try:
        y = float(x)
    except Exception:
        return float(default)
    return float(y) if np.isfinite(y) else float(default)


def _get_healpy():
    if "healpy" not in _STATE:
        try:
            import healpy as hp
            _STATE["healpy"] = hp
        except Exception:
            _STATE["healpy"] = None
    return _STATE.get("healpy")


def _nside_from_npix(hp, npix):
    try:
        ns = int(hp.npix2nside(int(npix)))
        if ns > 0 and int(hp.nside2npix(ns)) == int(npix):
            return ns
    except Exception:
        pass
    try:
        npix = int(npix)
        if npix > 0 and npix % 12 == 0:
            ns = int(round((float(npix) / 12.0) ** 0.5))
            if ns > 0 and 12 * ns * ns == npix:
                return ns
    except Exception:
        pass
    return -1


def _geometry(hp, nside):
    key = ("geometry", int(nside))
    if key in _STATE:
        return _STATE[key]
    npix = int(hp.nside2npix(int(nside)))
    pix = np.arange(npix, dtype=np.int64)
    try:
        x, y, z = hp.pix2vec(int(nside), pix, nest=False)
    except TypeError:
        x, y, z = hp.pix2vec(int(nside), pix)
    out = (np.asarray(x, dtype=np.float64),
           np.asarray(y, dtype=np.float64),
           np.asarray(z, dtype=np.float64))
    _STATE[key] = out
    return out


def _fit_monopole_dipole(arr, valid, x, y, z):
    n_valid = int(np.count_nonzero(valid))
    if n_valid <= 0:
        return np.array([0.0, 0.0, 0.0, 0.0], dtype=np.float64)

    t = np.asarray(arr[valid], dtype=np.float64)
    if n_valid < 4:
        mu = float(np.mean(t)) if t.size else 0.0
        if not np.isfinite(mu):
            mu = 0.0
        return np.array([mu, 0.0, 0.0, 0.0], dtype=np.float64)

    xv = np.asarray(x[valid], dtype=np.float64)
    yv = np.asarray(y[valid], dtype=np.float64)
    zv = np.asarray(z[valid], dtype=np.float64)

    n = float(n_valid)
    sx = float(np.sum(xv)); sy = float(np.sum(yv)); sz = float(np.sum(zv))
    sxx = float(np.dot(xv, xv)); syy = float(np.dot(yv, yv)); szz = float(np.dot(zv, zv))
    sxy = float(np.dot(xv, yv)); sxz = float(np.dot(xv, zv)); syz = float(np.dot(yv, zv))
    st = float(np.sum(t))
    stx = float(np.dot(t, xv)); sty = float(np.dot(t, yv)); stz = float(np.dot(t, zv))

    A = np.array([[n, sx, sy, sz],
                  [sx, sxx, sxy, sxz],
                  [sy, sxy, syy, syz],
                  [sz, sxz, syz, szz]], dtype=np.float64)
    b = np.array([st, stx, sty, stz], dtype=np.float64)
    try:
        coef = np.linalg.solve(A, b)
    except Exception:
        try:
            coef = np.linalg.lstsq(A, b, rcond=None)[0]
        except Exception:
            coef = None
    if coef is None or not np.all(np.isfinite(coef)):
        mu = float(np.mean(t)) if t.size else 0.0
        if not np.isfinite(mu):
            mu = 0.0
        coef = np.array([mu, 0.0, 0.0, 0.0], dtype=np.float64)
    return np.asarray(coef, dtype=np.float64)


def _prepare_masked_dipole_subtracted_map(m):
    arr = _as_1d_float(m)
    npix = int(arr.size)
    if npix <= 0:
        return None, None, None
    valid = np.isfinite(arr)
    if int(np.count_nonzero(valid)) <= 0:
        return None, valid, arr

    hp = _get_healpy()
    if hp is None:
        return None, valid, arr
    nside = _nside_from_npix(hp, npix)
    if nside <= 0:
        return None, valid, arr

    try:
        x, y, z = _geometry(hp, nside)
        if x.size != npix:
            return None, valid, arr
        coef = _fit_monopole_dipole(arr, valid, x, y, z)
        mp = np.zeros(npix, dtype=np.float64)
        mp[valid] = (np.asarray(arr[valid], dtype=np.float64)
                     - (coef[0] + coef[1] * x[valid] + coef[2] * y[valid] + coef[3] * z[valid]))
        # Never pass NaN or inf values to healpy. The masked cut sky is zero-filled.
        mp[~np.isfinite(mp)] = 0.0
        return mp, valid, arr
    except Exception:
        return None, valid, arr


def _map2alm_scalar(hp, mp, lmax):
    try:
        alm = hp.map2alm(mp, lmax=int(lmax), mmax=int(lmax), iter=0, pol=False, use_weights=False)
    except TypeError:
        try:
            alm = hp.map2alm(mp, lmax=int(lmax), mmax=int(lmax), iter=0, pol=False)
        except TypeError:
            alm = hp.map2alm(mp, lmax=int(lmax), mmax=int(lmax), iter=0)
    if isinstance(alm, (tuple, list)):
        alm = alm[0]
    return np.asarray(alm, dtype=np.complex128).ravel()


def _phase_coherence_from_alm(hp, alm, lmax_eff):
    lmax_eff = int(lmax_eff)
    last_l = min(_PAIR_LMAX, lmax_eff - 1)
    if last_l < _LMIN:
        return 0.0, 0

    sum_re = 0.0
    sum_im = 0.0
    n = 0
    tiny = 1.0e-300
    for ell in range(_LMIN, last_l + 1):
        max_m = min(ell, lmax_eff)
        for m in range(1, max_m + 1):
            try:
                a0 = alm[hp.Alm.getidx(lmax_eff, ell, m)]
                a1 = alm[hp.Alm.getidx(lmax_eff, ell + 1, m)]
            except Exception:
                continue
            prod = a1 * np.conjugate(a0)
            re = float(np.real(prod))
            im = float(np.imag(prod))
            amp = math.hypot(re, im)
            # For a continuous CMB map exact zero products are extremely unlikely.  If they
            # occur because of a degenerate input, omit the undefined phase from N.
            if np.isfinite(amp) and amp > tiny and np.isfinite(re) and np.isfinite(im):
                sum_re += re / amp
                sum_im += im / amp
                n += 1
    if n <= 0:
        return 0.0, 0
    r = math.hypot(sum_re, sum_im) / float(n)
    if not np.isfinite(r):
        r = 0.0
    if r < 0.0:
        r = 0.0
    elif r > 1.0:
        r = 1.0
    return float(r), int(n)


def _fallback_phase_coherence(arr, valid):
    # Used only if healpy is unavailable or the input is not a valid HEALPix map.
    # It keeps the statistic finite and map-dependent, but normal analyses should use map2alm.
    try:
        vals = np.asarray(arr[valid], dtype=np.float64)
        vals = vals[np.isfinite(vals)]
        nvals = int(vals.size)
        if nvals < 8:
            return 0.0, 0
        vals = vals - float(np.mean(vals))
        vals = np.where(np.isfinite(vals), vals, 0.0)
        if float(np.dot(vals, vals)) <= 0.0:
            return 0.0, 0
        ff = np.fft.rfft(vals)
        if ff.size < _LMAX + 2:
            return 0.0, 0
        ph = ff / np.maximum(np.abs(ff), 1.0e-300)
        pair = ph[_LMIN + 1:_LMAX + 1] * np.conjugate(ph[_LMIN:_LMAX])
        pair = pair[np.isfinite(pair.real) & np.isfinite(pair.imag)]
        if pair.size <= 0:
            return 0.0, 0
        s = np.sum(pair)
        r = float(abs(s) / float(pair.size))
        if not np.isfinite(r):
            r = 0.0
        return float(min(1.0, max(0.0, r))), int(pair.size)
    except Exception:
        return 0.0, 0


def _compute_adjacent_phase_coherence(m):
    mp, valid, arr = _prepare_masked_dipole_subtracted_map(m)
    hp = _get_healpy()
    if mp is None or hp is None:
        if valid is None or arr is None:
            return 0.0, 0
        return _fallback_phase_coherence(arr, valid)

    try:
        npix = int(mp.size)
        nside = _nside_from_npix(hp, npix)
        if nside <= 0:
            return _fallback_phase_coherence(arr, valid)
        lmax_eff = min(_LMAX, max(0, 3 * int(nside) - 1))
        if lmax_eff < _LMIN + 1:
            return 0.0, 0
        alm = _map2alm_scalar(hp, mp, lmax_eff)
        return _phase_coherence_from_alm(hp, alm, lmax_eff)
    except Exception:
        if valid is None or arr is None:
            return 0.0, 0
        return _fallback_phase_coherence(arr, valid)


def analyze_map(m):
    r, n = _compute_adjacent_phase_coherence(m)
    if not np.isfinite(r):
        r = 0.0
    r = float(min(1.0, max(0.0, r)))
    return r


def _finite_1d(x):
    try:
        a = np.asarray(x, dtype=np.float64).ravel()
    except Exception:
        vals = []
        try:
            for v in x:
                fv = _finite_float(v)
                if np.isfinite(fv):
                    vals.append(fv)
        except Exception:
            vals = []
        a = np.asarray(vals, dtype=np.float64)
    return a[np.isfinite(a)]


def _float_or_none(x):
    y = _finite_float(x)
    return float(y) if np.isfinite(y) else None


def summarize_results(planck_stat, sim_results):
    ps = _finite_float(planck_stat)
    sims = _finite_1d(sim_results)
    n = int(sims.size)

    out = {
        "test_description": test_description,
        "statistic": "R = |sum exp(i arg(a_{l+1,m} a^*_{l,m}))| / N for masked pseudo-a_lm adjacent low-l phases.",
        "multipole_range": "2 <= l <= 20, adjacent pairs 2 <= l <= 19",
        "m_range": "1 <= m <= l; m=0 phases are deliberately excluded",
        "expected_number_of_pairs_N": int(_EXPECTED_NPAIR),
        "planck_R": _float_or_none(ps),
        "n_sims": n,
        "tail_of_interest": "upper tail: unusually large R indicates coherent phase locking across neighbouring low-l multipoles",
        "masked_pixel_handling": "finite pixels are used for the monopole/dipole fit; np.nan mask pixels are then filled with 0.0 before healpy.map2alm",
        "plot_spec": {
            "kind": "histogram",
            "title": "Masked low-l adjacent-phase coherence",
            "xlabel": "R = |mean adjacent phase phasor|",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": "Distribution of R from masked pseudo-a_lm for 2<=l<=20. Larger R means the phase differences arg(a_{l+1,m} a^*_{l,m}) are more aligned across adjacent multipoles."
        }
    }

    if not np.isfinite(ps):
        out["warning"] = "The Planck statistic is not finite."
        return out
    if n <= 0:
        out["warning"] = "No finite simulation statistics were supplied."
        return out

    n_lt = int(np.count_nonzero(sims < ps))
    n_gt = int(np.count_nonzero(sims > ps))
    n_eq = int(n - n_lt - n_gt)
    n_le = int(n_lt + n_eq)
    n_ge = int(n_gt + n_eq)

    p_upper = float((n_ge + 1.0) / (n + 1.0))
    p_lower = float((n_le + 1.0) / (n + 1.0))
    p_two = float(min(1.0, 2.0 * min(p_upper, p_lower)))

    mean = float(np.mean(sims))
    median = float(np.median(sims))
    std = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    z = None
    if np.isfinite(std) and std > 0.0:
        z = float((ps - mean) / std)
    q = np.percentile(sims, [1, 5, 16, 50, 84, 95, 99])

    out.update({
        "sim_mean_R": mean,
        "sim_median_R": median,
        "sim_std_R": std,
        "sim_min_R": float(np.min(sims)),
        "sim_max_R": float(np.max(sims)),
        "sim_percentile_1_R": float(q[0]),
        "sim_percentile_5_R": float(q[1]),
        "sim_percentile_16_R": float(q[2]),
        "sim_percentile_50_R": float(q[3]),
        "sim_percentile_84_R": float(q[4]),
        "sim_percentile_95_R": float(q[5]),
        "sim_percentile_99_R": float(q[6]),
        "planck_zscore_vs_sims": z,
        "planck_percentile_midrank_low_to_high": float(100.0 * (n_lt + 0.5 * n_eq) / n),
        "rank_ascending_1_is_lowest_R": int(n_lt + 1),
        "rank_descending_1_is_highest_R_most_coherent": int(n_gt + 1),
        "n_sim_lt_planck": n_lt,
        "n_sim_le_planck": n_le,
        "n_sim_eq_planck": n_eq,
        "n_sim_gt_planck": n_gt,
        "n_sim_ge_planck": n_ge,
        "fraction_sims_lt_planck": float(n_lt / n),
        "fraction_sims_le_planck": float(n_le / n),
        "fraction_sims_gt_planck": float(n_gt / n),
        "fraction_sims_ge_planck": float(n_ge / n),
        "empirical_p_upper_tail_phase_coherence_plus_one": p_upper,
        "empirical_p_lower_tail_plus_one": p_lower,
        "empirical_p_two_sided_plus_one": p_two,
        "difference_planck_minus_sim_median_R": float(ps - median),
        "difference_planck_minus_sim_mean_R": float(ps - mean),
        "is_upper_tail_5pct_phase_locked": bool(p_upper <= 0.05),
        "is_upper_tail_1pct_phase_locked": bool(p_upper <= 0.01),
        "is_two_sided_5pct": bool(p_two <= 0.05),
        "is_two_sided_1pct": bool(p_two <= 0.01)
    })

    if std == 0.0:
        out["warning"] = "All finite simulation R values are identical; check that maps, mask, and harmonic transforms are varying."
    elif p_upper <= 0.01:
        out["interpretation"] = "Planck is in the highest 1% of simulations: adjacent low-l pseudo-a_lm phase differences are exceptionally coherent."
    elif p_upper <= 0.05:
        out["interpretation"] = "Planck is in the highest 5% of simulations: adjacent low-l pseudo-a_lm phase differences are unusually coherent."
    else:
        out["interpretation"] = "Planck is not in the upper 5% tail of the simulation distribution for this adjacent-phase coherence statistic."
    return out
