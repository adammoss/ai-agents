Hypothesis

The Planck masked low-ℓ temperature field has an anomalous hot/cold tail asymmetry relative to ΛCDM simulations, meaning its log-ratio statistic S is unusually far from the simulation expectation in either direction.

Method

Low-ℓ temperature maps filtered to 2≤ℓ≤64 were evaluated on unmasked pixels after median subtraction using S = ln[(q95−q50)/(q50−q05)], where negative S indicates a relatively extended cold tail and positive S an extended hot tail.

Result Summary

Planck SMICA gives S = -0.05498, on the cold-tail side. The 1000 ΛCDM simulations have mean S = 0.00043 and standard deviation 0.05128. Planck is about -1.08σ from the simulation mean, at the 14.3rd percentile, with a two-tailed p-value of 0.286.

The Planck value is mildly negative but well within the ΛCDM simulation distribution. It is not close to the 5% two-tailed anomaly threshold, so this test provides no evidence for anomalous low-ℓ hot/cold tail asymmetry.

Agent Justification

The statistic is signed, with negative S indicating an extended cold tail and positive S indicating an extended hot tail, but the test description and code do not specify an a priori preference for either direction. Since the anomaly could be either a cold-tail or hot-tail excess, the appropriate default is a two-tailed test.

Comparison To Literature

Related one-point/non-Gaussianity tests, including skewness, kurtosis, peak statistics, N-point functions and Minkowski functionals, are discussed in Planck Collaboration 2015, A&A 594, A16 and Planck Collaboration 2020, A&A 641, A7. Those works generally found one-point statistics consistent with Gaussian ΛCDM, while confirming other large-scale anomalies such as low variance, the Cold Spot and hemispherical asymmetry. This result is consistent with the non-detection in those related distributional tests.

def analyze_map(m):
    import math
    import numpy as np

    LMIN = 2
    LMAX = 64

    try:
        if np.ma.isMaskedArray(m):
            a = np.asarray(m.filled(np.nan), dtype=np.float64).ravel()
        else:
            a = np.asarray(m, dtype=np.float64).ravel()
    except Exception:
        try:
            a = np.asarray(np.real(m), dtype=np.float64).ravel()
        except Exception:
            return 0.0

    if a.size <= 0:
        return 0.0

    valid = np.isfinite(a)
    if int(np.count_nonzero(valid)) < 3:
        return 0.0

    def tail_log_ratio(values):
        vals = np.asarray(values, dtype=np.float64).ravel()
        vals = vals[np.isfinite(vals)]
        if vals.size < 3:
            return 0.0
        try:
            med = float(np.median(vals))
            centered = vals - med
            q05, q50, q95 = np.percentile(centered, [5.0, 50.0, 95.0])
            q05 = float(q05)
            q50 = float(q50)
            q95 = float(q95)
        except Exception:
            return 0.0
        if not (np.isfinite(q05) and np.isfinite(q50) and np.isfinite(q95)):
            return 0.0
        hot_width = q95 - q50
        cold_width = q50 - q05
        scale = max(1.0, abs(q05), abs(q50), abs(q95))
        eps = np.finfo(np.float64).eps * scale
        if hot_width <= eps and cold_width <= eps:
            return 0.0
        if hot_width <= eps:
            hot_width = eps
        if cold_width <= eps:
            cold_width = eps
        try:
            s = math.log(float(hot_width) / float(cold_width))
        except Exception:
            return 0.0
        return float(s) if np.isfinite(s) else 0.0

    low = None

    # Determine whether the input length is a valid HEALPix map.  The harmonic
    # branch is the intended production path; the direct percentile fallback is
    # only for non-HEALPix diagnostic inputs or unavailable healpy.
    nside = -1
    try:
        npix = int(a.size)
        if npix > 0 and npix % 12 == 0:
            ns = int(round(math.sqrt(npix / 12.0)))
            if ns > 0 and 12 * ns * ns == npix:
                nside = ns
    except Exception:
        nside = -1

    if nside > 0:
        try:
            import healpy as hp
            import warnings

            filled = np.array(a, dtype=np.float64, copy=True)
            filled[~np.isfinite(filled)] = 0.0

            candidate_lmax = [int(LMAX)]
            safe_lmax = min(int(LMAX), max(0, 3 * int(nside) - 1))
            if safe_lmax >= LMIN and safe_lmax not in candidate_lmax:
                candidate_lmax.append(int(safe_lmax))

            for lmax in candidate_lmax:
                if lmax < LMIN:
                    continue
                try:
                    with warnings.catch_warnings():
                        warnings.simplefilter("ignore")
                        try:
                            alm = hp.map2alm(filled, lmax=int(lmax), iter=0, pol=False)
                        except TypeError:
                            alm = hp.map2alm(filled, lmax=int(lmax), iter=0)

                    fl = np.zeros(int(lmax) + 1, dtype=np.float64)
                    fl[LMIN:int(lmax) + 1] = 1.0
                    try:
                        alm = hp.almxfl(alm, fl, inplace=False)
                    except TypeError:
                        alm = hp.almxfl(alm, fl)

                    with warnings.catch_warnings():
                        warnings.simplefilter("ignore")
                        try:
                            tmp = hp.alm2map(alm, nside=int(nside), lmax=int(lmax), pol=False, verbose=False)
                        except TypeError:
                            try:
                                tmp = hp.alm2map(alm, int(nside), lmax=int(lmax), verbose=False)
                            except TypeError:
                                try:
                                    tmp = hp.alm2map(alm, nside=int(nside), lmax=int(lmax), pol=False)
                                except TypeError:
                                    tmp = hp.alm2map(alm, int(nside), lmax=int(lmax))

                    tmp = np.asarray(tmp, dtype=np.float64).ravel()
                    if tmp.size == a.size:
                        tmp[~np.isfinite(tmp)] = 0.0
                        low = tmp
                        break
                except Exception:
                    low = None
        except Exception:
            low = None

    if low is not None and low.size == a.size:
        result = tail_log_ratio(low[valid])
    else:
        result = tail_log_ratio(a[valid])

    try:
        result = float(result)
    except Exception:
        result = 0.0
    return result if np.isfinite(result) else 0.0


def summarize_results(planck_stat, sim_results):
    import numpy as np

    desc = "Masked Low-l Tail-Asymmetry Log-Ratio: retain multipoles 2<=ell<=64, evaluate only originally unmasked pixels, subtract the median, and return S = ln[(q95-q50)/(q50-q05)]. Negative S indicates an extended cold tail; positive S indicates an extended hot tail."

    try:
        ps = float(planck_stat)
    except Exception:
        ps = np.nan
    if not np.isfinite(ps):
        ps = np.nan

    try:
        sims = np.asarray(sim_results, dtype=np.float64).ravel()
    except Exception:
        vals = []
        try:
            for x in sim_results:
                try:
                    y = float(x)
                    if np.isfinite(y):
                        vals.append(y)
                except Exception:
                    pass
        except Exception:
            pass
        sims = np.asarray(vals, dtype=np.float64)
    sims = sims[np.isfinite(sims)]
    n = int(sims.size)

    out = {
        "test_description": desc,
        "statistic": "S = ln[(q95 - q50)/(q50 - q05)] on originally unmasked pixels of the 2<=ell<=64 filtered temperature field after median subtraction.",
        "multipole_range": "2 <= ell <= 64",
        "mask_handling": "Pixels with non-finite input values are excluded from the percentile calculation. They are filled with 0.0 only for the healpy spherical-harmonic transform.",
        "sign_convention": "S < 0 means a wider cold tail; S > 0 means a wider hot tail.",
        "planck_S": float(ps) if np.isfinite(ps) else None,
        "n_sims": n,
        "plot_spec": {
            "kind": "histogram",
            "title": "Masked low-l tail-asymmetry log-ratio",
            "xlabel": "S = ln[(q95 - q50)/(q50 - q05)]",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": "Simulation distribution of the signed low-l tail-asymmetry statistic; the Planck value is compared to this distribution."
        }
    }

    if not np.isfinite(ps):
        out["warning"] = "The Planck statistic is not finite."
        return out
    if n <= 0:
        out["warning"] = "No finite simulation statistics were supplied."
        return out

    n_lt = int(np.count_nonzero(sims < ps))
    n_gt = int(np.count_nonzero(sims > ps))
    n_eq = int(n - n_lt - n_gt)
    n_le = int(n_lt + n_eq)
    n_ge = int(n_gt + n_eq)

    p_lower = float((n_le + 1.0) / (n + 1.0))
    p_upper = float((n_ge + 1.0) / (n + 1.0))
    p_two_signed = float(min(1.0, 2.0 * min(p_lower, p_upper)))
    n_abs_ge = int(np.count_nonzero(np.abs(sims) >= abs(ps)))
    p_abs = float((n_abs_ge + 1.0) / (n + 1.0))
    p_direction = p_lower if ps < 0.0 else (p_upper if ps > 0.0 else 1.0)

    mean = float(np.mean(sims))
    median = float(np.median(sims))
    std = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    zscore = float((ps - mean) / std) if std > 0.0 and np.isfinite(std) else None
    q = np.percentile(sims, [1, 5, 16, 50, 84, 95, 99])

    if ps < 0.0:
        direction = "cold-tail side (negative S)"
    elif ps > 0.0:
        direction = "hot-tail side (positive S)"
    else:
        direction = "nearly symmetric (S = 0)"

    out.update({
        "planck_tail_direction": direction,
        "sim_mean_S": mean,
        "sim_median_S": median,
        "sim_std_S": std,
        "sim_min_S": float(np.min(sims)),
        "sim_max_S": float(np.max(sims)),
        "sim_percentile_1_S": float(q[0]),
        "sim_percentile_5_S": float(q[1]),
        "sim_percentile_16_S": float(q[2]),
        "sim_percentile_50_S": float(q[3]),
        "sim_percentile_84_S": float(q[4]),
        "sim_percentile_95_S": float(q[5]),
        "sim_percentile_99_S": float(q[6]),
        "planck_zscore_vs_sims": zscore,
        "planck_percentile_midrank_low_to_high": float(100.0 * (n_lt + 0.5 * n_eq) / n),
        "rank_ascending_1_is_most_cold_tail": int(n_lt + 1),
        "rank_descending_1_is_most_hot_tail": int(n_gt + 1),
        "n_sim_lt_planck": n_lt,
        "n_sim_le_planck": n_le,
        "n_sim_eq_planck": n_eq,
        "n_sim_gt_planck": n_gt,
        "n_sim_ge_planck": n_ge,
        "fraction_sims_le_planck": float(n_le / n),
        "fraction_sims_ge_planck": float(n_ge / n),
        "empirical_p_lower_tail_cold_plus_one": p_lower,
        "empirical_p_upper_tail_hot_plus_one": p_upper,
        "empirical_p_two_sided_signed_plus_one": p_two_signed,
        "empirical_p_abs_asymmetry_ge_abs_planck_plus_one": p_abs,
        "empirical_p_in_planck_tail_direction_plus_one": float(p_direction),
        "n_sims_abs_S_ge_abs_planck": n_abs_ge,
        "difference_planck_minus_sim_median_S": float(ps - median),
        "is_5pct_in_planck_tail_direction": bool(p_direction <= 0.05),
        "is_1pct_in_planck_tail_direction": bool(p_direction <= 0.01),
        "is_5pct_by_abs_asymmetry": bool(p_abs <= 0.05),
        "is_1pct_by_abs_asymmetry": bool(p_abs <= 0.01)
    })

    if std == 0.0:
        out["warning"] = "All finite simulation S values are identical; check map variation, mask propagation, and harmonic filtering."
    elif p_direction <= 0.01:
        out["interpretation"] = "Planck lies in the most extreme 1% of simulations in its signed tail-asymmetry direction."
    elif p_direction <= 0.05:
        out["interpretation"] = "Planck lies in the most extreme 5% of simulations in its signed tail-asymmetry direction."
    elif p_abs <= 0.05:
        out["interpretation"] = "Planck is not in a 5% one-sided signed tail, but its absolute tail-asymmetry magnitude is in the most extreme 5% of simulations."
    else:
        out["interpretation"] = "Planck is not in the most extreme 5% of simulations for this low-l tail-asymmetry statistic."

    return out
