Hypothesis

The observed Planck CMB temperature map exhibits a stronger dipole modulation of large-scale power (l ∈ [2,10]) than expected from lensed-ΛCDM simulations, as measured by the disc-local power dipole amplitude D.

Method

The SMICA temperature map is bandpass-filtered to l∈[2,10]. For each pixel at nside=256, local power is computed within a 15° disc via harmonic-space convolution (accounting for the mask). The monopole and dipole of the resulting local-power map are extracted, and the test statistic D = dipole amplitude / monopole is computed. D is compared against 1000 lensed-ΛCDM simulations processed identically.

Result Summary

Planck D = 0.377. Simulation mean D = 0.268 ± 0.111 (z-score ≈ 0.97σ). One-tailed upper p-value = 0.165; 16.5% of simulations exceed the data value. No significant excess is detected.

The measured dipole modulation of large-scale power (l∈[2,10]) is slightly elevated compared to the simulation mean but lies well within the expected scatter. The result is consistent with standard lensed-ΛCDM fluctuations and does not constitute evidence for hemispherical power asymmetry.

Agent Justification

The test description explicitly states "Large D indicates directional variation of large-scale power, consistent with hemispherical power asymmetry or dipole modulation." The scientific motivation is to detect an anomalous directional bias (an excess), not merely any deviation from the simulation mean. The code's p-value is calculated as the fraction of simulations with D ≥ the observed value (`sim_arr >= planck_stat`), confirming a one-tailed upper-tail test. There is no physical expectation or interest in a D value that is unusually small (lower-tail), making a two-tailed test inappropriate.

Comparison To Literature

The local variance estimator (LVE) for hemispherical power asymmetry has been widely applied to WMAP and Planck data (e.g., Aluri & Shafieloo 2017; Sanyal et al. 2026, arXiv:2602.10059). Previous studies report ~2–4% power asymmetry in polarization and modest signals in temperature, typically at marginal significance. The present implementation differs in using harmonic-space disc convolution rather than brute-force pixel averaging, and in restricting to l∈[2,10]. The null result here is broadly consistent with literature findings that temperature-based power asymmetry signals are weak in Planck data.

import numpy as np
import healpy as hp

test_description = ("Disc-local power dipole amplitude. The SMICA temperature map is bandpass-filtered to l ∈ [2,10]. "
    "For each pixel the local variance is computed within a 15 deg disc (unmasked pixels only). "
    "The monopole and dipole of the resulting local-power map are extracted via SH decomposition. "
    "The test statistic is D = sqrt(|a10|^2 + 2*|a11|^2) / |a00|, accounting for both m=±1 dipole terms. "
    "Large D indicates directional variation of large-scale power (hemispherical power asymmetry).")


def analyze_map(m):
    m = np.array(m, dtype=np.float64)
    nside = hp.npix2nside(len(m))
    mask = np.isfinite(m).astype(np.float64)

    lmin, lmax = 2, 10

    # Fill masked pixels with 0 for harmonic transforms
    m_fill = np.where(np.isfinite(m), m, 0.0)

    # Bandpass filter: keep only l in [2, 10] using almxfl for proper l-indexing
    alm = hp.map2alm(m_fill, lmax=lmax)
    fl = np.zeros(lmax + 1)
    fl[lmin:lmax + 1] = 1.0
    f_alm = hp.almxfl(alm, fl)
    f = hp.alm2map(f_alm, nside, lmax=lmax)

    # Zero out masked pixels in filtered map
    f = np.where(np.isfinite(m), f, 0.0)

    # Compute f^2
    f2 = f * f

    # === Disc kernel in harmonic space ===
    # For a disc of radius theta_disc, the normalized kernel coefficients are:
    #   kernel[0] = 1
    #   kernel[l] = W_l / W_0  for l >= 1
    # where W_l = int_{cos(theta)}^{1} P_l(x) dx
    # W_0 = 1 - cos(theta)
    # W_l = [P_{l-1}(cos_theta) - P_{l+1}(cos_theta)] / (2l+1)  for l >= 1

    lmax_kernel = 30
    theta_disc = np.radians(15.0)
    cos_theta = np.cos(theta_disc)

    # Compute P_l(cos_theta) for l = 0, ..., lmax_kernel+1 using standard recurrence
    P = np.zeros(lmax_kernel + 2)
    P[0] = 1.0
    P[1] = cos_theta
    for l in range(1, lmax_kernel + 1):
        P[l + 1] = ((2 * l + 1) * cos_theta * P[l] - l * P[l - 1]) / (l + 1)

    # Compute W_l
    kernel = np.zeros(lmax_kernel + 1)
    kernel[0] = 1.0 - cos_theta
    for l in range(1, lmax_kernel + 1):
        kernel[l] = (P[l - 1] - P[l + 1]) / (2 * l + 1)

    # Normalize so kernel[0] = 1 (disc-averaging operator)
    kernel /= kernel[0]

    # Convolve f^2 and mask with disc kernel via harmonic smoothing
    f2_smoothed = hp.smoothing(f2, beam_window=kernel, lmax=lmax_kernel)
    mask_smoothed = hp.smoothing(mask, beam_window=kernel, lmax=lmax_kernel)

    # Local power: kernel-weighted average of f^2 over unmasked pixels in disc
    P_map = np.where(mask_smoothed > 1e-10, f2_smoothed / mask_smoothed, 0.0)

    # Extract monopole and dipole of P(n)
    alm_P = hp.map2alm(P_map, lmax=1)
    # alm_P indices: [a_{00}, a_{10}, a_{1,1}]
    # For real maps, |a_{1,-1}|^2 = |a_{1,1}|^2
    # Total dipole amplitude: sqrt(|a10|^2 + |a11|^2 + |a1,-1|^2) = sqrt(|a10|^2 + 2*|a11|^2)

    a00 = alm_P[0]        # real (m=0)
    a10 = alm_P[1]        # real (m=0)
    a11 = alm_P[2]        # complex (m=1)

    dipole_amp = np.sqrt(np.abs(a10) ** 2 + 2.0 * np.abs(a11) ** 2)
    monopole = np.abs(a00)

    if monopole < 1e-30:
        return 0.0

    D = float(dipole_amp / monopole)
    return D


def summarize_results(planck_stat, sim_results):
    sim_arr = np.array(sim_results)
    finite_mask = np.isfinite(sim_arr)
    n_finite = finite_mask.sum()

    if n_finite > 0:
        p_value = float(np.sum(sim_arr[finite_mask] >= planck_stat) / n_finite)
    else:
        p_value = float('nan')

    result = {
        "test_description": test_description,
        "planck_stat": float(planck_stat),
        "p_value": p_value,
        "n_simulations": len(sim_results),
        "n_valid": int(n_finite),
        "sim_mean": float(np.nanmean(sim_arr)),
        "sim_std": float(np.nanstd(sim_arr)),
        "exceeds_95th_percentile": bool(p_value < 0.05),
    }

    result["plot_spec"] = {
        "kind": "histogram",
        "title": "Disc-Local Power Dipole Amplitude",
        "xlabel": "D = dipole_amplitude / monopole",
        "ylabel": "Number of simulations",
        "bins": 40,
        "caption": (f"Histogram of D values from {n_finite} lensed-LCDM simulations. "
                    f"Planck measured D = {planck_stat:.4f} (p = {p_value:.3f}). "
                    f"Large D indicates hemispherical power asymmetry / dipole modulation of large-scale power.")
    }

    return result
