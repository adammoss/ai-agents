Hypothesis

The fraction of CMB temperature power at multipoles ℓ=2 through 5 residing in ecliptic-symmetric modes (even m) is higher than the expected value of 0.5 for a random isotropic field.

Method

The fraction of total CMB temperature power at multipoles ℓ = 2–5 residing in ecliptic-symmetric (even-*m*) spherical harmonic modes is computed for the Planck map and 1000 lensed-ΛCDM simulations. For a random isotropic field the expected fraction is ~0.5; a significantly higher value would indicate anomalous concentration of low-ℓ power in modes symmetric about the ecliptic plane.

Result Summary

The Planck even-*m* power fraction is 0.388, notably *below* the simulation mean of 0.506 ± 0.140. The one-sided upper-tail p-value is 0.779 (z = −0.84σ), meaning 77.9% of simulations equal or exceed the Planck value. The observed statistic lies comfortably within the expected distribution.

The low-ℓ ecliptic-plane power fraction in Planck data is fully consistent with isotropic ΛCDM expectations and is actually marginally lower than the simulation median (0.512). There is no evidence of anomalous excess power in ecliptic-symmetric modes at these multipoles.

Agent Justification

The test description and code explicitly focus on detecting a significantly higher power fraction as anomalous, with the p-value calculated by counting simulations where the observed value is greater than or equal to the Planck statistic, indicating a clear directional preference for excess in ecliptic-symmetric modes.

Comparison To Literature

Ecliptic alignment of low-ℓ CMB modes has been reported in Planck Collaboration papers (e.g., Planck 2013 XVI; Planck 2018 VII) using various statistics, sometimes at modest significance. The current test, which isolates even-*m* power as a single scalar, finds no such excess. Differences in multipole ranges, masking, and statistic definitions make direct comparison with earlier claims non-trivial.

import numpy as np
import healpy as hp

test_description = ("Low-ℓ Ecliptic-Plane Power Fraction. Computes the fraction of total CMB "
    "temperature power at multipoles ℓ=2 through 5 residing in spherical harmonic modes "
    "with even m (symmetric about the ecliptic plane). For a random isotropic field the "
    "expected value is ~0.5; a significantly higher value indicates anomalous concentration "
    "of low-ℓ power in ecliptic-symmetric modes, as reported in Planck data.")


def analyze_map(m):
    m = np.asarray(m, dtype=np.float64)
    m_fill = np.where(np.isfinite(m), m, 0.0)

    lmax = 5
    alm = hp.map2alm(m_fill, lmax=lmax)

    even_power = 0.0
    total_power = 0.0

    for ell in range(2, 6):          # ℓ = 2, 3, 4, 5
        for me in range(0, ell + 1): # m = 0, 1, ..., ℓ
            idx = hp.Alm.getidx(lmax, ell, me)
            power = np.abs(alm[idx]) ** 2
            if me > 0:
                power *= 2.0         # account for the conjugate m < 0 mode
            total_power += power
            if me % 2 == 0:          # even m → ecliptic-symmetric
                even_power += power

    if total_power > 0.0:
        ratio = even_power / total_power
    else:
        ratio = 0.5

    return float(ratio)


def summarize_results(planck_stat, sim_results):
    sim_arr = np.array(sim_results, dtype=np.float64)
    finite_mask = np.isfinite(sim_arr)
    n_finite = int(finite_mask.sum())

    if n_finite > 0:
        valid = sim_arr[finite_mask]
        rank = int(np.sum(valid >= planck_stat))
        p_value = rank / n_finite
    else:
        rank = 0
        p_value = float('nan')

    sim_median = float(np.nanmedian(sim_arr))
    sim_mean = float(np.nanmean(sim_arr))
    sim_std = float(np.nanstd(sim_arr))

    interpretation = (
        "Anomalous: low-ℓ power concentrated in ecliptic-symmetric modes"
        if p_value < 0.05
        else "Low-ℓ ecliptic power fraction consistent with random expectations"
    )

    result = {
        "test_description": test_description,
        "planck_stat": float(planck_stat),
        "p_value": float(p_value),
        "rank": rank,
        "n_simulations": len(sim_results),
        "n_valid": n_finite,
        "sim_mean": sim_mean,
        "sim_std": sim_std,
        "sim_median": sim_median,
        "exceeds_95th_percentile": bool(p_value < 0.05),
        "interpretation": interpretation,
        "expected_random_fraction": float(0.5),
    }

    result["plot_spec"] = {
        "kind": "histogram",
        "title": "Low-ℓ Ecliptic-Plane Power Fraction",
        "xlabel": "Even-m power fraction (ℓ = 2–5)",
        "ylabel": "Number of simulations",
        "bins": 50,
        "caption": (
            f"Histogram of even-m power fraction from {n_finite} lensed-ΛCDM simulations. "
            f"Planck value = {planck_stat:.4f} (one-sided p = {p_value:.3f}, "
            f"rank {rank}/{n_finite}). "
            f"The expected value for isotropic random fields is ~0.5; "
            f"a value significantly above 0.5 indicates anomalous ecliptic symmetry."
        ),
    }

    return result
