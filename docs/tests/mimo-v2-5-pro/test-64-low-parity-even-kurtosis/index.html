Hypothesis

The CMB temperature map exhibits excess kurtosis in its parity-even low-ℓ components compared to Gaussian random field expectations.

Method

Computes the kurtosis of the parity-even field formed from CMB temperature multipoles ℓ=2 to 10 over the full sky.

Result Summary

The observed kurtosis is 2.557 (excess kurtosis = -0.443). Simulation mean ± σ = 2.873 ± 0.460. The Planck value is 0.69σ below the simulation mean, with a one-tailed p-value of 0.731 (fraction of sims ≥ observed).

The observed kurtosis is slightly lower than, but consistent with, the distribution expected from Gaussian realizations. The negative z-score indicates the Planck map is less kurtotic than average simulations, not more.

Agent Justification

The analysis code explicitly computes the p-value as the fraction of simulations with kurtosis greater than or equal to the observed value, indicating a directional interest in detecting unusually high kurtosis.

Comparison To Literature

Previous studies have examined parity asymmetry in power and multipole alignments (e.g., Aluri et al. 2017, Copi et al. 2004), but no prior work was found that specifically analyzes the kurtosis of the parity-even low-ℓ field.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Parity-Even Kurtosis: "
    "For the band-limited CMB temperature map containing multipoles ℓ=2 through 10, "
    "compute the parity-even field T_even(n) defined as the sum over all even-ℓ "
    "spherical harmonic components (ℓ = 2, 4, 6, 8, 10). "
    "Calculate the kurtosis (the fourth standardized moment) of the pixel values of "
    "T_even over the full unmasked sky. The statistic is this single scalar kurtosis value."
)


def analyze_map(m):
    """Return the kurtosis (4th standardized moment) of the parity-even low-ℓ CMB field."""

    # Identify unmasked pixels
    finite_mask = np.isfinite(m)

    # Replace NaN with 0 for healpy transforms (never pass NaN to healpy)
    m_clean = np.where(finite_mask, m, 0.0)

    nside = hp.npix2nside(len(m))
    lmax = 10

    # Compute spherical harmonic coefficients
    alm = hp.map2alm(m_clean, lmax=lmax)

    # Zero out all components except even ℓ in {2, 4, 6, 8, 10}
    for ell in range(lmax + 1):
        if ell < 2 or ell % 2 == 1:
            for em in range(ell + 1):
                idx = hp.Alm.getidx(lmax, ell, em)
                alm[idx] = 0.0

    # Reconstruct parity-even map
    T_even = hp.alm2map(alm, nside, lmax=lmax, verbose=False)

    # Extract pixel values on unmasked sky only
    values = T_even[finite_mask]

    # Compute kurtosis (fourth standardized moment): μ₄ / σ⁴
    n = len(values)
    mean = np.mean(values)
    diff = values - mean
    m2 = np.mean(diff ** 2)
    m4 = np.mean(diff ** 4)

    # Guard against degenerate case (negligible variance)
    if m2 < 1e-30:
        return 3.0  # Default to Gaussian expectation

    kurtosis = m4 / (m2 * m2)

    return float(kurtosis)


def summarize_results(planck_stat, sim_results):
    """Interpret the parity-even kurtosis relative to the simulation ensemble."""
    sims = np.array(sim_results, dtype=float)
    sims_finite = sims[np.isfinite(sims)]
    n_sims = len(sims_finite)

    if n_sims == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims_finite))
    std_s = float(np.std(sims_finite, ddof=1)) if n_sims > 1 else 0.0
    p_value = float(np.mean(sims_finite >= planck_stat)) if n_sims > 0 else np.nan
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else np.nan

    # For a Gaussian field the fourth standardized moment is exactly 3
    excess_kurtosis = planck_stat - 3.0

    return {
        "planck_stat": planck_stat,
        "excess_kurtosis": excess_kurtosis,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "z_score": z_score,
        "p_value": p_value,
        "n_sims": n_sims,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Parity-Even Kurtosis",
            "xlabel": "Kurtosis (4th standardized moment)",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck kurtosis = {planck_stat:.4f} (excess = {excess_kurtosis:.4f}).  "
                f"Sim mean ± σ = {mean_s:.4f} ± {std_s:.4f}.  "
                f"z = {z_score:.2f}, p = {p_value:.3f}  "
                f"(fraction of sims ≥ Planck)."
            ),
        },
    }
