Hypothesis

The normalized dipole amplitude of the low-ℓ CMB temperature gradient magnitude is consistent with the distribution from simulated isotropic Gaussian LCDM random fields.

Method

The low-ℓ CMB temperature map (ℓ=2–10) is extracted and the temperature gradient magnitude computed at each pixel via finite differences on the HEALPix grid, properly accounting for sin(θ) scaling. The dipole of the gradient magnitude map is then extracted and normalized by its monopole (median), yielding a dimensionless ratio that quantifies directional preference in large-scale temperature variation patterns.

Result Summary

The observed normalized dipole amplitude is 0.1414, compared to a simulation mean of 0.0665 ± 0.0443, placing it at the 94.1st percentile (~1.69σ above the mean). The two-tailed p-value is 0.118, with anomaly strength classified as negligible.

The Planck gradient magnitude dipole is mildly elevated relative to LCDM simulations but remains well within the expected distribution. No significant preferred direction is detected in the large-scale temperature gradient pattern. The ~1.7σ excess, while notable, does not reach statistical significance and is consistent with normal cosmic variance.

Agent Justification

The test is explicitly designed to check for any anomaly (either significantly large or significantly small) in the gradient magnitude dipole, which would indicate a break from statistical isotropy in the low-ℓ temperature field. The code computes both upper and lower tail p-values, and the interpretation handles both directions of deviation. This is a general exploratory test without a prior directional prediction, making a two-tailed approach appropriate.

Comparison To Literature

No prior studies were found examining the dipole of the CMB gradient magnitude field as a statistical isotropy probe. Related work on CMB dipole alignments (e.g., Luongo et al. 2022) and bipolar power spectrum analyses (Ghosh et al. 2007) address different statistics applied to large-scale CMB structure, but do not directly test this gradient-based observable.

import numpy as np
import healpy as hp

test_description = (
    "Compute the low-ell temperature map (multipoles ell=2 to ell=10) from the "
    "CMB map. At each pixel, calculate the magnitude of the temperature gradient "
    "using finite differences on the HEALPix grid, accounting for the sin(theta) "
    "scaling of the azimuthal component. Then compute the dipole (ell=1) of this "
    "gradient magnitude map. The statistic is the amplitude of this dipole "
    "normalized by the monopole (median gradient magnitude), i.e., "
    "sqrt(d1^2 + d2^2 + d3^2) / a0. For isotropic Gaussian random fields the "
    "gradient field should be statistically isotropic, yielding a dipole "
    "consistent with cosmic variance. A large normalized dipole amplitude "
    "indicates an anisotropic gradient pattern in the low-ell temperature field, "
    "which could signal a preferred direction of temperature variation (e.g., "
    "aligned with the CMB dipole or the ecliptic). Compare the observed value "
    "against the distribution from simulated LCDM maps to assess significance."
)


def analyze_map(m):
    """Return the dipole amplitude of the low-ell gradient magnitude map,
    normalized by the monopole (median gradient magnitude)."""
    mask = np.isfinite(m)
    n_valid = int(np.sum(mask))
    if n_valid < 500:
        return 0.0

    nside = hp.npix2nside(len(m))
    m_filled = np.where(mask, m, 0.0)

    # Band-limit to ell = 2-10
    lmin, lmax = 2, 10
    alm = hp.map2alm(m_filled, lmax=lmax, iter=0)
    # Zero out ell < lmin (monopole and dipole)
    for l in range(lmin):
        for i in range(2 * l + 1):
            idx = l * l + i
            alm[idx] = 0.0
    m_low = hp.alm2map(alm, nside, lmax=lmax, verbose=False)

    # Compute gradient magnitude using finite differences
    npix = hp.nside2npix(nside)
    res = hp.nside2resol(nside)  # radians per pixel
    theta, phi = hp.pix2ang(nside, np.arange(npix))
    stheta = np.sin(theta)

    # theta-derivative: (T_south - T_north) / (2 * res)
    # phi-derivative: (T_east - T_west) / (2 * res * sin(theta))
    # Gradient magnitude: sqrt((dT/dtheta)^2 + (1/sin(theta) * dT/dphi)^2)
    r = 1
    p_north = hp.get_interp_weights(nside, theta - res, phi, lonlat=False)[0][0].astype(int)
    p_south = hp.get_interp_weights(nside, theta + res, phi, lonlat=False)[0][0].astype(int)

    # Wrap phi with proper spherical geometry
    dphi = res / np.where(stheta > 1e-6, stheta, 1e-6)
    phi_east = phi + dphi
    phi_west = phi - dphi
    p_east = hp.get_interp_weights(nside, theta, phi_east, lonlat=False)[0][0].astype(int)
    p_west = hp.get_interp_weights(nside, theta, phi_west, lonlat=False)[0][0].astype(int)

    gt = (m_low[p_south] - m_low[p_north]) / (2.0 * res)
    gp = (m_low[p_east] - m_low[p_west]) / (2.0 * dphi)  # already includes 1/sin(theta)
    grad_mag = np.sqrt(gt**2 + gp**2)

    # Normalize by median to get a dimensionless anisotropy measure
    finite_gm = grad_mag[np.isfinite(grad_mag)]
    if len(finite_gm) == 0:
        return 0.0
    med_gm = float(np.median(finite_gm))
    if med_gm < 1e-20:
        return 0.0

    gm_norm = grad_mag / med_gm
    gm_norm = np.where(np.isfinite(gm_norm), gm_norm, 1.0)

    # Extract dipole and monopole of the gradient magnitude map
    alm_gm = hp.map2alm(gm_norm, lmax=1, iter=0)
    a0 = alm_gm[0]  # monopole coefficient
    a1_r = alm_gm[1]  # real dipole
    a1_i = alm_gm[2]  # imaginary dipole
    # The m=1 mode has two components (real and imaginary)
    # Total dipole amplitude: sqrt(a1_real^2 + a1_imag^2)
    dipole_amp = np.sqrt(a1_r**2 + a1_i**2)

    # Normalize: dipole amplitude / monopole (a0)
    if abs(a0) < 1e-20:
        return 0.0

    stat = float(dipole_amp / abs(a0))
    if not np.isfinite(stat):
        return 0.0
    return stat


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ell Gradient Magnitude Dipole Amplitude test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Gradient Magnitude Dipole Amplitude",
                "xlabel": "Dipole amplitude / monopole",
                "ylabel": "Count",
                "bins": 50,
                "caption": "No simulation data.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    p_value_upper = float(np.mean(sim >= planck_stat))
    p_value_lower = float(np.mean(sim <= planck_stat))
    p_value_two_sided = min(2.0 * min(p_value_upper, p_value_lower), 1.0)

    rank_above = int(np.sum(sim >= planck_stat))
    rank_below = int(np.sum(sim <= planck_stat))
    percentile = 100.0 * rank_below / n_sims
    deviation_sigma = (planck_stat - mean_sim) / std_sim if std_sim > 0 else 0.0

    if p_value_two_sided <= 0.01:
        strength = "strong"
    elif p_value_two_sided <= 0.05:
        strength = "moderate"
    else:
        strength = "negligible"

    interp = (
        f"Gradient magnitude dipole amplitude / monopole = {planck_stat:.6f}. "
        f"Sim mean = {mean_sim:.6f} ± {std_sim:.6f} "
        f"({deviation_sigma:+.2f}σ). Percentile = {percentile:.1f}%, "
        f"two-sided p = {p_value_two_sided:.4f}. Strength: {strength}. "
    )

    if planck_stat > mean_sim + 2 * std_sim:
        interp += (
            "The Planck map shows an anomalously large directional preference "
            "in the gradient magnitude of the low-ℓ temperature field compared "
            "to Gaussian LCDM expectations. This indicates that temperature "
            "variations at large angular scales are significantly stronger along "
            "one preferred direction than would be expected from a statistically "
            "isotropic field. Such an anisotropy could reflect alignment between "
            "large-scale temperature gradients and a preferred axis (e.g., the "
            "CMB dipole direction or the ecliptic pole)."
        )
    elif planck_stat < mean_sim - 2 * std_sim:
        interp += (
            "The Planck map shows an anomalously small directional preference "
            "in the gradient magnitude of the low-ℓ temperature field compared "
            "to Gaussian LCDM expectations. This suggests unusually uniform "
            "gradient magnitudes across the sky, potentially indicating an "
            "unexpectedly isotropic large-scale temperature structure."
        )
    else:
        interp += (
            "The gradient magnitude dipole amplitude is consistent with Gaussian "
            "LCDM expectations. No significant preferred direction detected in "
            "the large-scale temperature gradient pattern."
        )

    caption = (
        "Histogram of the normalized gradient magnitude dipole amplitude "
        "(dipole / monopole ratio) for the ℓ=2–10 band-limited CMB temperature "
        "map across LCDM simulations. Higher values indicate a stronger preferred "
        "direction of temperature variation at large angular scales. "
        "The red line marks the Planck observed value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value_one_sided_upper": p_value_upper,
        "p_value_one_sided_lower": p_value_lower,
        "p_value_two_sided": p_value_two_sided,
        "rank_above": rank_above,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": deviation_sigma,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Gradient Magnitude Dipole Amplitude",
            "xlabel": "Dipole amplitude / monopole",
            "ylabel": "Number of simulations",
            "bins": 50,
            "caption": caption,
        },
    }
