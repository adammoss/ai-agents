Hypothesis

The convex hull area ratio of low-ℓ hot spots in the Planck CMB temperature map is consistent with the distribution expected from isotropic Gaussian ΛCDM simulations, with no significant morphological asymmetry or clustering.

Method

The convex hull enclosing low-ℓ (ℓ ≤ 20) hot spots (positive temperature pixels exceeding the 1σ threshold) is computed in 3D Cartesian space on the unit sphere. The ratio of its solid angle to 4π provides a single morphological scalar sensitive to whether hot regions are unusually clustered or elongated relative to isotropic Gaussian realisations.

Result Summary

Planck's convex hull area ratio = 0.1450, compared to simulation mean = 0.1335 ± 0.0132. The value sits at the 80.5th percentile with a two-sided p-value of 0.39 (z ≈ 0.87σ). The anomaly strength is negligible.

The spatial extent of low-ℓ hot spots in the Planck map is fully consistent with random Gaussian realisations. There is no evidence for anomalous morphological clustering, elongation, or preferred directional compression of hot regions on the largest angular scales.

Agent Justification

The test is designed to detect anomalies in the morphology of hot regions, which could manifest as either an unusually small convex hull area (indicating elongation or clustering) or an unusually large area (indicating dispersion). The analysis code computes a two-sided empirical p-value and interprets deviations in both directions, indicating no prior directional preference. Therefore, a two-tailed test is appropriate for general exploration of potential anomalies.

Comparison To Literature

No prior studies applying convex hull morphology of low-ℓ CMB hot spots were identified in the search. Related morphological analyses of the CMB (e.g., genus statistics, Minkowski functionals) are well-established but do not employ this specific convex hull area ratio statistic.

import numpy as np
import healpy as hp
from scipy.spatial import ConvexHull

test_description = (
    "Low-ℓ Convex Hull Area Asymmetry: "
    "Compute the 1-sigma contour of the low-ℓ (ℓ ≤ 20) temperature map by thresholding. "
    "Calculate the area of the convex hull enclosing all positive-temperature pixels (hot spots) "
    "above the threshold. Compute the ratio of this area to the total area of the sphere. "
    "This single scalar measures how 'clustered' or 'elongated' the hot regions are morphologically, "
    "which can differ between Gaussian random fields and those with specific non-Gaussianities or anomalies."
)


def _solid_angle_triangle(A, B, C):
    """Compute the solid angle subtended by a triangle (A, B, C) at the origin.

    Uses the formula: tan(Ω/2) = (A · (B × C)) / (1 + A·B + A·C + B·C)
    and returns Ω = 2 * atan2(numerator, denominator).
    All vectors should be 3D numpy arrays.
    """
    cross = np.cross(B, C)
    num = np.dot(A, cross)
    den = 1.0 + np.dot(A, B) + np.dot(A, C) + np.dot(B, C)
    return 2.0 * np.arctan2(num, den)


def analyze_map(m):
    """Compute the convex hull area ratio of low-ℓ hot spots.

    Steps:
      1. Identify unmasked pixels (finite values).
      2. Fill masked (NaN) pixels with 0.0 for healpy compatibility.
      3. Compute alm coefficients up to ℓ_max = 20 and reconstruct the
         band-limited low-ℓ temperature map.
      4. Compute the 1-sigma threshold (std of the filtered map over unmasked pixels).
      5. Identify hot spot pixels: positive temperature AND above the 1-sigma threshold.
      6. Convert hot pixel centres to 3D Cartesian coordinates on the unit sphere.
      7. Compute the 3D convex hull of these points using scipy.spatial.ConvexHull.
      8. For each triangular face of the hull, compute its solid angle at the origin
         using the formula Ω = 2·atan2(A·(B×C), 1+A·B+A·C+B·C).
      9. Sum the absolute solid angles over all faces and divide by 4π.

    Returns
    -------
    float
        Ratio of convex hull area to total sphere area (dimensionless).
    """
    mask = np.isfinite(m)
    valid_idx = np.where(mask)[0]
    N = len(valid_idx)

    if N == 0:
        return 0.0

    nside = hp.npix2nside(len(m))

    # Fill masked pixels with 0 for healpy transforms
    m_filled = np.where(mask, m, 0.0)

    # Compute spherical harmonic coefficients up to ℓ = 20
    lmax = 20
    alm = hp.map2alm(m_filled, lmax=lmax)

    # Reconstruct band-limited map (ℓ ≤ 20)
    map_low = hp.alm2map(alm, nside, lmax=lmax, verbose=False)

    # Extract values at unmasked pixels
    vals = map_low[valid_idx]

    # 1-sigma threshold
    sigma = np.std(vals)

    if sigma == 0.0:
        return 0.0

    # Hot spot pixels: positive AND above 1-sigma
    hot_mask = (map_low > sigma) & mask
    hot_idx = np.where(hot_mask)[0]
    n_hot = len(hot_idx)

    # Fallback for very few hot pixels: approximate by pixel count
    if n_hot < 4:
        npix = hp.nside2npix(nside)
        return float(n_hot) / float(npix)

    # Convert hot pixel centres to 3D Cartesian coordinates on the unit sphere
    theta, phi = hp.pix2ang(nside, hot_idx)
    x = np.sin(theta) * np.cos(phi)
    y = np.sin(theta) * np.sin(phi)
    z = np.cos(theta)
    points = np.column_stack((x, y, z))

    # Compute the 3D convex hull
    try:
        hull = ConvexHull(points)
    except Exception:
        # Fallback if ConvexHull fails (e.g., degenerate configuration)
        npix = hp.nside2npix(nside)
        return float(n_hot) / float(npix)

    # Sum the solid angles of all triangular faces
    vertices = points[hull.simplices]  # shape: (n_faces, 3, 3)
    A = vertices[:, 0, :]
    B = vertices[:, 1, :]
    C = vertices[:, 2, :]

    # Vectorised: numerator = A · (B × C), denominator = 1 + A·B + A·C + B·C
    cross_BC = np.cross(B, C)  # (n_faces, 3)
    num = np.sum(A * cross_BC, axis=1)  # (n_faces,)
    den = (1.0
           + np.sum(A * B, axis=1)
           + np.sum(A * C, axis=1)
           + np.sum(B * C, axis=1))  # (n_faces,)

    omega = 2.0 * np.arctan2(num, den)  # (n_faces,)
    total_omega = np.sum(np.abs(omega))

    # Ratio to full sphere solid angle (4π)
    ratio = total_omega / (4.0 * np.pi)

    # Clamp to [0, 1] for safety
    ratio = float(np.clip(ratio, 0.0, 1.0))

    return ratio


def summarize_results(planck_stat, sim_results):
    """Summarise the Low-ℓ Convex Hull Area Asymmetry test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available for comparison.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Convex Hull Area Asymmetry",
                "xlabel": "Convex hull area / Sphere area",
                "ylabel": "Number of simulations",
                "bins": 40,
                "caption": "No simulation data available.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    # Two-sided empirical p-value
    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    # Direction interpretation
    if planck_stat < mean_sim - 2 * std_sim:
        direction = (
            "Planck's low-ℓ hot spot convex hull occupies a significantly smaller "
            "fraction of the sky than expected from isotropic Gaussian ΛCDM simulations. "
            "This indicates that the hot regions are unusually elongated, compressed, "
            "or morphologically distinct — potentially reflecting large-scale anisotropy "
            "or a preferred direction in the temperature field."
        )
    elif planck_stat > mean_sim + 2 * std_sim:
        direction = (
            "Planck's low-ℓ hot spot convex hull occupies a significantly larger "
            "fraction of the sky than expected. The hot regions are unusually spread "
            "out, which could indicate atypical large-scale temperature structure."
        )
    else:
        direction = (
            "The convex hull area of low-ℓ hot spots is consistent with isotropic "
            "Gaussian ΛCDM expectations; there is no anomalous morphological "
            "clustering or elongation of the hot regions."
        )

    interp = (
        f"Convex hull area ratio = {planck_stat:.6f}. "
        f"Simulation mean = {mean_sim:.6f} ± {std_sim:.6f}. "
        f"Percentile = {percentile:.1f}%, two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of the convex hull area ratio (convex hull enclosing low-ℓ hot spots "
        "divided by total sphere area) from isotropic ΛCDM simulations. Hot spots are "
        "defined as pixels with positive temperature exceeding the 1σ threshold of the "
        "ℓ ≤ 20 filtered map. For a statistically isotropic Gaussian field, the "
        "morphology of hot regions should vary randomly between realisations. "
        "Significantly low values indicate unusually elongated or clustered hot regions, "
        "while significantly high values indicate unusually dispersed ones. "
        "The red line marks the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Convex Hull Area Asymmetry",
            "xlabel": "Convex hull area / Sphere area",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
