Hypothesis

The observed cold spot depth statistic D (|T_min|/σ) at low multipoles (ℓ=2-20) is larger than expected from ΛCDM simulations, indicating an unusually deep cold spot.

Method

Measures the depth of the coldest feature in the CMB temperature map filtered to large angular scales (multipoles ℓ=2-20). The statistic is D = |T_min|/σ, where T_min is the minimum temperature in the filtered map and σ is its standard deviation, compared to 1000 ΛCDM simulations.

Result Summary

The observed Planck statistic D = 3.594 lies above the simulation mean (3.245±0.385) with a z-score of 0.97σ. The one-tailed upper p-value is 0.167, meaning ~16.7% of simulations show an equal or greater cold spot depth.

While the observed cold spot at low ℓ is deeper than the median simulation value (3.19), it is not statistically extreme. The result is consistent with the standard ΛCDM model at the ~1σ level.

Agent Justification

The test description explicitly states that a value significantly higher than the simulation median would indicate an unusually deep cold spot, indicating a directional expectation. Therefore, the anomaly test should focus on whether D is unusually large, making it a one-tailed test in the upper direction.

Comparison To Literature

Studies of the CMB's low-multipole anomalies, including the prominent non-Gaussian cold spot, typically analyze spatial isotropy or alignments rather than this specific depth-normalized statistic. Khan & Saha (2022) find no general anomaly in the clustering of hot/cold spots on the full sky, though partial-sky analyses show some signatures.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Cold Spot Depth. Applies a low-pass filter to the CMB temperature map "
    "keeping only multipoles ℓ = 2 through 20. Identifies the minimum temperature "
    "value T_min across the filtered map. Computes the standard deviation σ of the "
    "filtered map over the unmasked sky. The cold spot depth statistic is "
    "D = |T_min| / σ, measuring how extreme the coldest low-ℓ feature is relative "
    "to the overall large-scale temperature fluctuations. A value significantly "
    "higher than the simulation median would indicate an unusually deep cold spot."
)


def analyze_map(m):
    m = np.asarray(m, dtype=np.float64)
    mask = np.isfinite(m)
    m_clean = np.where(mask, m, 0.0)

    lmax = 20

    # Extract alm up to lmax=20
    alm = hp.map2alm(m_clean, lmax=lmax)

    # Zero out monopole (ℓ=0) and dipole (ℓ=1) so only ℓ=2..20 survive
    for ell in range(0, 2):
        for mm in range(ell + 1):
            idx = hp.Alm.getidx(lmax, ell, mm)
            alm[idx] = 0.0

    # Reconstruct filtered map (now contains only ℓ=2..20)
    nside = hp.npix2nside(len(m))
    m_filtered = hp.alm2map(alm, nside, lmax=lmax, verbose=False)

    # Evaluate only over unmasked pixels
    valid_pixels = m_filtered[mask]

    if len(valid_pixels) == 0:
        return 0.0

    T_min = np.min(valid_pixels)
    sigma = np.std(valid_pixels)

    if sigma == 0.0:
        return 0.0

    D = abs(T_min) / sigma
    return float(D)


def summarize_results(planck_stat, sim_results):
    sim_arr = np.array(sim_results, dtype=np.float64)
    finite_mask = np.isfinite(sim_arr)
    n_finite = int(finite_mask.sum())

    if n_finite > 0:
        valid = sim_arr[finite_mask]
        sim_median = float(np.median(valid))
        p_upper = float(np.sum(valid >= planck_stat) / n_finite)
        p_lower = float(np.sum(valid <= planck_stat) / n_finite)
        p_value = min(p_upper, p_lower)
        rank_upper = int(np.sum(valid >= planck_stat))
        rank_lower = int(np.sum(valid <= planck_stat))
    else:
        p_value = float("nan")
        rank_upper = rank_lower = 0
        sim_median = float("nan")

    sim_mean = float(np.nanmean(sim_arr))
    sim_std = float(np.nanstd(sim_arr))

    if n_finite > 0 and not np.isnan(p_value):
        if p_upper <= p_lower:
            direction, rank = "high", rank_upper
        else:
            direction, rank = "low", rank_lower
    else:
        direction, rank = "unknown", 0

    parts = [
        f"Planck cold spot depth D = |T_min|/σ at low ℓ (2–20) is {planck_stat:.4f}.",
        f"Two-sided p = {p_value:.3f} ({rank}/{n_finite} simulations "
        f"at least as extreme in the {direction} direction).",
        f"Simulation mean = {sim_mean:.4f} ± {sim_std:.4f}, "
        f"median = {sim_median:.4f}.",
    ]

    if not np.isnan(p_value) and p_value < 0.05:
        parts.append(
            "Anomalous: the cold spot depth at low ℓ is unusually large, "
            "suggesting the deepest large-scale temperature feature is "
            "more extreme than expected under ΛCDM."
        )
    else:
        parts.append(
            "The cold spot depth at low ℓ is consistent with ΛCDM simulation "
            "expectations — no evidence for an unusually deep cold spot."
        )

    result = {
        "test_description": test_description,
        "planck_stat": float(planck_stat),
        "p_value": float(p_value),
        "rank": rank,
        "n_simulations": len(sim_results),
        "n_valid": n_finite,
        "sim_mean": sim_mean,
        "sim_std": sim_std,
        "sim_median": sim_median,
        "exceeds_95th_percentile": bool(p_value < 0.05) if not np.isnan(p_value) else False,
        "anomaly_direction": direction,
        "interpretation": " ".join(parts),
    }

    result["plot_spec"] = {
        "kind": "histogram",
        "title": "Low-ℓ Cold Spot Depth",
        "xlabel": "Cold spot depth D = |T_min|/σ",
        "ylabel": "Number of simulations",
        "bins": 50,
        "caption": (
            f"Histogram of cold spot depths from {n_finite} ΛCDM simulations. "
            f"Planck value = {planck_stat:.4f} "
            f"(two-sided p = {p_value:.3f}, rank {rank}/{n_finite}, "
            f"{direction} tail). Simulation mean = {sim_mean:.4f} ± {sim_std:.4f}. "
            f"A large value indicates the coldest low-ℓ feature is unusually deep."
        ),
    }

    return result
