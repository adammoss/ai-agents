Hypothesis

The Pearson cross-correlation coefficient between the low-ℓ temperature map and its gradient magnitude is significantly different from zero.

Method

Compute the Pearson cross-correlation coefficient r between a band-limited (ℓ = 2–15) temperature map T and its gradient magnitude G = sqrt[(∂T/∂θ)² + (1/sin θ · ∂T/∂φ)²] over unmasked pixels. For a statistically isotropic Gaussian field, r is centered at zero by symmetry; significant nonzero values indicate spatially coherent non-Gaussianity where temperature extrema preferentially co-locate with regions of large or small gradient magnitude.

Result Summary

Planck yields r = 0.047, compared to simulation mean 0.003 ± 0.078. The z-score is 0.55σ, the two-tailed p-value is 0.584, and Planck sits at the 70.8th percentile. Anomaly strength is negligible.

The observed cross-correlation between low-ℓ temperature and gradient magnitude is fully consistent with isotropic Gaussian ΛCDM expectations. Temperature hot/cold spots show no anomalous preference for co-locating with steep or flat gradient regions, indicating no detectable spatial coherence anomaly in the large-scale CMB temperature field.

Agent Justification

The test is designed to detect any deviation from zero correlation, regardless of sign, as indicated by the two-sided p-value calculation in the summary function and the interpretation that checks for both significantly positive and negative anomalies. The description states that for a statistically isotropic Gaussian field, r is centered at zero by symmetry, and a significant nonzero value signals an anomaly, without specifying a directional preference. Therefore, a two-tailed test is appropriate.

Comparison To Literature

No prior studies of cross-correlation between CMB temperature and gradient magnitude were found in the literature. Related low-ℓ anomaly tests include multipole vector alignments (Copi et al. 2004) and various non-Gaussianity probes, but this specific spatial derivative statistic does not appear to have been previously investigated.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Temperature–Gradient-Magnitude Cross-Correlation Coefficient: "
    "Using the prepared low-ℓ temperature map T(θ,φ) at nside=256, retain "
    "multipoles ℓ = 2 through ℓ_max = 15. Compute the spherical-harmonic "
    "derivatives ∂T/∂θ and (1/sin θ) ∂T/∂φ via standard recurrence relations "
    "on the a_{ℓm} coefficients, then form the gradient magnitude map "
    "G(θ,φ) = sqrt[(∂T/∂θ)² + (1/sin θ · ∂T/∂φ)²] at each pixel. Restrict "
    "both T and G to unmasked pixels using the Planck common mask. Compute "
    "the Pearson cross-correlation coefficient r = Σ_i (T_i − T̄)(G_i − Ḡ) / "
    "[√Σ_i(T_i − T̄)² · √Σ_i(G_i − Ḡ)²], where the sum runs over unmasked "
    "pixels. The single scalar anomaly statistic is r. For a statistically "
    "isotropic Gaussian field r is centered at zero by symmetry; a significant "
    "nonzero value signals that temperature extrema preferentially co-locate "
    "with gradient extrema, probing non-Gaussianity or spatial coherence "
    "anomalies in the low-ℓ field."
)


def analyze_map(m):
    """Compute Pearson cross-correlation between low-ℓ temperature and gradient magnitude.

    Steps:
      1. Identify unmasked pixels (finite values).
      2. Fill masked (NaN) pixels with 0 for healpy compatibility.
      3. Compute alm coefficients up to ℓ_max = 15.
      4. Zero out coefficients for ℓ < 2 (band-limit to ℓ = 2..15).
      5. Use hp.alm2map_der1 to reconstruct the band-limited map and its
         angular derivatives ∂T/∂θ and ∂T/∂φ.
      6. Divide ∂T/∂φ by sin(θ) to get the physical phi-component of the gradient.
      7. Form gradient magnitude G = sqrt[(∂T/∂θ)² + (1/sin θ · ∂T/∂φ)²].
      8. Extract T and G at unmasked pixels, compute Pearson r.

    Returns
    -------
    float
        Pearson cross-correlation coefficient r between T and G.
    """
    nside = hp.npix2nside(len(m))
    mask = np.isfinite(m)
    valid_idx = np.where(mask)[0]
    N = len(valid_idx)

    if N == 0:
        return 0.0

    # Fill masked pixels with 0 for healpy transforms
    m_filled = np.where(mask, m, 0.0)

    # Compute alm up to lmax=15
    lmax = 15
    alm = hp.map2alm(m_filled, lmax=lmax)

    # Zero out coefficients for ℓ < 2 (retain only ℓ = 2 to 15)
    for l in range(0, 2):
        for m_val in range(0, l + 1):
            idx = hp.Alm.getidx(lmax, l, m_val)
            alm[idx] = 0.0

    # Reconstruct band-limited map and its angular derivatives
    # alm2map_der1 returns: (map, dT/dtheta, dT/dphi)
    m_band, dT_dtheta, dT_dphi_raw = hp.alm2map_der1(alm, nside, lmax=lmax)

    # Compute sin(theta) for each pixel to get the physical phi-gradient
    theta, phi = hp.pix2ang(nside, np.arange(hp.nside2npix(nside)))
    sin_theta = np.sin(theta)

    # Avoid division by zero at poles; set to 0 where sin_theta is tiny
    with np.errstate(divide='ignore', invalid='ignore'):
        dT_dphi = np.where(sin_theta > 1e-10, dT_dphi_raw / sin_theta, 0.0)

    # Form gradient magnitude G = sqrt[(dT/dtheta)^2 + ((1/sin_theta)*dT/dphi)^2]
    G = np.sqrt(dT_dtheta ** 2 + dT_dphi ** 2)

    # Extract values at unmasked pixels
    T_vals = m_band[valid_idx]
    G_vals = G[valid_idx]

    # Center the values
    T_bar = np.mean(T_vals)
    G_bar = np.mean(G_vals)
    T_centered = T_vals - T_bar
    G_centered = G_vals - G_bar

    # Compute Pearson correlation coefficient
    numerator = np.sum(T_centered * G_centered)
    denom_T = np.sqrt(np.sum(T_centered ** 2))
    denom_G = np.sqrt(np.sum(G_centered ** 2))

    if denom_T == 0.0 or denom_G == 0.0:
        return 0.0

    r = numerator / (denom_T * denom_G)

    return float(r)


def summarize_results(planck_stat, sim_results):
    """Summarise the Low-ℓ Temperature–Gradient-Magnitude Cross-Correlation test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results to compare.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ T–∇T Magnitude Cross-Correlation",
                "xlabel": "Pearson r(T, G)",
                "ylabel": "Number of simulations",
                "bins": 40,
                "caption": "No simulation data available.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    # Two-sided p-value
    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    direction = ""
    if abs(planck_stat) > abs(mean_sim) + 2 * std_sim:
        if planck_stat > 0:
            direction = (
                "Planck shows a significantly positive cross-correlation between "
                "low-ℓ temperature and gradient magnitude: temperature extrema "
                "(hot and cold spots) preferentially co-locate with sharp "
                "gradient regions. This could indicate spatially coherent "
                "non-Gaussian features, preferred angular scales for temperature "
                "variations, or alignment of low-ℓ modes beyond statistical "
                "isotropy expectations."
            )
        else:
            direction = (
                "Planck shows a significantly negative cross-correlation between "
                "low-ℓ temperature and gradient magnitude: temperature extrema "
                "preferentially occur where gradients are small (flat regions), "
                "suggesting that the largest-scale features have unusually smooth "
                "peaks/troughs with steeper gradients occurring away from extrema. "
                "This could indicate unusual mode structure or spatial coherence."
            )
    else:
        direction = (
            "The cross-correlation between temperature and gradient magnitude is "
            "consistent with isotropic Gaussian ΛCDM expectations; temperature "
            "extrema show no anomalous co-location with gradient extrema in the "
            "low-ℓ field."
        )

    interp = (
        f"Pearson r(T, G) = {planck_stat:.6f}. "
        f"Simulation mean = {mean_sim:.6f} ± {std_sim:.6f}. "
        f"Percentile = {percentile:.1f}%, two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of the Pearson cross-correlation coefficient r between the "
        "band-limited (ℓ = 2–15) temperature map T and its gradient magnitude "
        "G = sqrt[(∂T/∂θ)² + (1/sin θ · ∂T/∂φ)²] from isotropic ΛCDM "
        "simulations. For a statistically isotropic Gaussian field, r is "
        "centered at zero by symmetry: there is no preferred co-location of "
        "temperature extrema with gradient extrema. Significant nonzero r "
        "values indicate non-Gaussian spatial coherence in the low-ℓ field, "
        "where temperature peaks/troughs systematically co-occur with regions "
        "of high or low gradient magnitude. The red line marks the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "anomaly_strength": strength,

            "ylabel": "Number of simulations",

    }
