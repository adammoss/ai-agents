Hypothesis

The low-ℓ CMB temperature power is elongated along a preferred axis, leading to a significantly large elongation ratio λ₁/λ₃.

Method

The SMICA temperature map is smoothed to ℓ_max = 15 via spherical harmonic decomposition after masking. A 3×3 second-moment tensor M_{ij} = Σ_p T²(n_p) n_p^i n_p^j is computed over unmasked pixels. The statistic is the ratio of the largest to smallest eigenvalue (λ₁/λ₃) of M, which equals 1 under isotropy and exceeds 1 when large-angle temperature power is elongated along a preferred axis.

Result Summary

The Planck SMICA elongation ratio is λ₁/λ₃ = 1.416. From 1000 ΛCDM simulations the mean is 1.885 ± 0.330, with median 1.846. The Planck value sits 1.74σ below the simulation mean. The one-tailed upper p-value is 0.959 (i.e., 95.9% of simulations have a ratio at least as large as Planck's), indicating the Planck value is actually lower than typical simulations rather than higher.

The Planck low-ℓ elongation ratio is fully consistent with — and even slightly below — ΛCDM simulation expectations. There is no evidence that large-angle CMB temperature power is anomalously elongated along any preferred direction. The statistic is not elevated, so the hypothesis of directional anisotropy in the low-ℓ power is not supported.

Agent Justification

The statistic λ₁/λ₃ is inherently ≥ 1, with values near 1 expected under isotropy. The test description explicitly states that values significantly above 1 indicate directional anisotropy, and no anomaly is mentioned for values below 1. Thus, the hypothesis is specifically about an excess in the upper tail, making a one-tailed upper test appropriate.

Comparison To Literature

The specific statistic (eigenvalue ratio of the T²-weighted second-moment tensor at low ℓ) does not correspond closely to a well-established test in the literature. Related studies examine the quadrupole–octopole alignment or moment-based shape statistics, but this particular ratio formulation appears to be a novel construction. A search returned no directly comparable published analysis.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Anisotropy Elongation Ratio. Smooths the SMICA temperature map to "
    "ℓ_max = 15 using spherical harmonic decomposition (after masking). "
    "Computes the second-moment tensor M_{ij} = Σ_p T²(n_p) n_p^i n_p^j "
    "summed over unmasked pixels p, where n_p^i are Cartesian components of "
    "each pixel's unit vector. Diagonalizes M to obtain three eigenvalues "
    "λ₁ ≥ λ₂ ≥ λ₃. The statistic is the ratio λ₁/λ₃ (largest to smallest "
    "eigenvalue). Under isotropy this ratio is near unity; values "
    "significantly above 1 indicate that large-angle temperature power is "
    "elongated along a preferred axis, capturing a directional anisotropy in "
    "a single number."
)


def analyze_map(m):
    m = np.asarray(m, dtype=np.float64)
    mask = np.isfinite(m)

    # Isotropic limit for no valid pixels
    if not np.any(mask):
        return 1.0

    nside = hp.npix2nside(len(m))

    # Fill masked pixels with 0 for healpy SHT operations
    m_clean = np.where(mask, m, 0.0)

    # Step 1: Band-limit to ℓ_max = 15 via spherical harmonic transform
    lmax = 15
    alm = hp.map2alm(m_clean, lmax=lmax)
    m_low = hp.alm2map(alm, nside, lmax=lmax, verbose=False)

    # Step 2: Get pixel unit vectors for unmasked pixels only
    idx = np.where(mask)[0]
    theta, phi = hp.pix2ang(nside, idx)

    # Cartesian components of unit vectors
    nx = np.sin(theta) * np.cos(phi)
    ny = np.sin(theta) * np.sin(phi)
    nz = np.cos(theta)

    # T² values at unmasked pixels from the smoothed map
    T2 = m_low[idx] ** 2

    # Step 3: Compute the 3x3 second-moment tensor
    # M_{ij} = Σ_p T²(n_p) * n_p^i * n_p^j
    # n_vectors shape: (N_pix, 3)
    n_vecs = np.column_stack([nx, ny, nz])

    # Vectorized: M = (n_vecs.T * T2) @ n_vecs
    # This gives M[i,j] = Σ_p T2_p * n_p^i * n_p^j
    M = (n_vecs.T * T2) @ n_vecs

    # Step 4: Diagonalize to get eigenvalues
    eigenvalues = np.linalg.eigvalsh(M)
    eigenvalues = np.sort(eigenvalues)[::-1]  # descending: λ₁ ≥ λ₂ ≥ λ₃

    lam1 = eigenvalues[0]
    lam3 = eigenvalues[2]

    # Guard against degenerate cases
    if lam3 <= 0 or not np.isfinite(lam3):
        return 1.0

    ratio = lam1 / lam3

    # Return a single finite float
    if not np.isfinite(ratio):
        return 1.0

    return float(ratio)


def summarize_results(planck_stat, sim_results):
    sim_arr = np.array(sim_results, dtype=np.float64)
    finite_mask = np.isfinite(sim_arr)
    n_finite = int(finite_mask.sum())

    if n_finite > 0:
        valid = sim_arr[finite_mask]
        sim_median = float(np.median(valid))
        p_upper = float(np.sum(valid >= planck_stat) / n_finite)
        p_lower = float(np.sum(valid <= planck_stat) / n_finite)
        p_value = min(p_upper, p_lower)
        rank_upper = int(np.sum(valid >= planck_stat))
        rank_lower = int(np.sum(valid <= planck_stat))
    else:
        p_value = float("nan")
        rank_upper = rank_lower = 0
        sim_median = float("nan")

    sim_mean = float(np.nanmean(sim_arr))
    sim_std = float(np.nanstd(sim_arr))

    if n_finite > 0 and not np.isnan(p_value):
        if p_upper <= p_lower:
            direction, rank = "high", rank_upper
        else:
            direction, rank = "low", rank_lower
    else:
        direction, rank = "unknown", 0

    # Compute additional useful quantities
    p16 = float(np.percentile(valid, 16)) if n_finite > 0 else float("nan")
    p84 = float(np.percentile(valid, 84)) if n_finite > 0 else float("nan")
    p025 = float(np.percentile(valid, 2.5)) if n_finite > 0 else float("nan")
    p975 = float(np.percentile(valid, 97.5)) if n_finite > 0 else float("nan")

    parts = [
        f"Planck low-ℓ elongation ratio (λ₁/λ₃) is {planck_stat:.4f}.",
        f"Two-sided p = {p_value:.3f} ({rank}/{n_finite} simulations "
        f"at least as extreme in the {direction} direction).",
        f"Simulation mean = {sim_mean:.4f} ± {sim_std:.4f}, "
        f"median = {sim_median:.4f}.",
    ]

    if not np.isnan(p_value) and p_value < 0.05:
        parts.append(
            "Anomalous: the elongation ratio at low ℓ is unusually large, "
            "indicating that large-angle temperature power is elongated along "
            "a preferred axis — a signature of directional anisotropy in the "
            "spatial distribution of CMB temperature fluctuations at angular "
            "scales > 12°, suggestive of a preferred direction violating "
            "statistical isotropy."
        )
    else:
        parts.append(
            "The low-ℓ elongation ratio is consistent with ΛCDM simulation "
            "expectations — no evidence for anomalous directional elongation "
            "of large-scale temperature power."
        )

    result = {
        "test_description": test_description,
        "planck_stat": float(planck_stat),
        "p_value": float(p_value),
        "rank": rank,
        "n_simulations": len(sim_results),
        "n_valid": n_finite,
        "sim_mean": sim_mean,
        "sim_std": sim_std,
        "sim_median": sim_median,
        "sim_p16": p16,
        "sim_p84": p84,
        "sim_p025": p025,
        "sim_p975": p975,
        "exceeds_95th_percentile": bool(p_value < 0.05) if not np.isnan(p_value) else False,
        "anomaly_direction": direction,
        "interpretation": " ".join(parts),
    }

    result["plot_spec"] = {
        "kind": "histogram",
        "title": "Low-ℓ Anisotropy Elongation Ratio (λ₁/λ₃)",
        "xlabel": "Elongation ratio λ₁/λ₃",
        "ylabel": "Number of simulations",
        "bins": 50,
        "caption": (
            f"Histogram of elongation ratios (λ₁/λ₃ of the T²-weighted "
            f"second-moment tensor at ℓ ≤ 15) from {n_finite} ΛCDM simulations. "
            f"Planck value = {planck_stat:.4f} "
            f"(two-sided p = {p_value:.3f}, rank {rank}/{n_finite}, "
            f"{direction} tail). Simulation mean = {sim_mean:.4f} ± {sim_std:.4f}. "
            f"A ratio of 1 corresponds to isotropic power; values >> 1 indicate "
            f"temperature variance elongated along a preferred axis."
        ),
    }

    return result
