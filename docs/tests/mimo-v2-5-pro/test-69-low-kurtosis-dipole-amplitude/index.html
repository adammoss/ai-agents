Hypothesis

The dipole amplitude of the kurtosis over Nside=8 regions is larger than expected from isotropic ΛCDM simulations.

Method

Fit a dipole to the excess kurtosis of CMB temperature values within 768 Nside=8 regions, then take the fitted dipole amplitude as the test statistic.

Result Summary

The Planck dipole amplitude (0.0617) exceeds the simulation mean (0.0389 ± 0.0166), yielding z = 1.38 and an upper-tail p-value of 0.088.

The observed kurtosis dipole is larger than the ΛCDM simulation average, suggesting a possible directional preference in non-Gaussianity, but it does not reach the conventional 2σ (p < 0.05) anomaly threshold.

Agent Justification

The test measures the amplitude of a dipole fit to kurtosis values across the sky, where the dipole amplitude is non-negative. The anomaly interpretation explicitly focuses on whether the value is "significantly elevated," indicating a directional preference in non-Gaussianity. Since only large positive deviations from the expected null distribution (centered near zero) would constitute evidence for such an anomaly, an upper-tailed test is appropriate.

Comparison To Literature

Literature searches did not find direct prior studies of this specific kurtosis dipole statistic, making a direct comparison difficult. Related work on CMB dipoles and non-Gaussianity exists (e.g., Lineweaver 1996; Luongo et al. 2022; Yokoyama et al. 2011), but not this exact test.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Kurtosis Dipole Amplitude: For each Nside=8 region (768 pixels), compute "
    "the excess kurtosis of the underlying higher-resolution CMB temperature values. "
    "Fit a dipole model (monopole + dipole) to these kurtosis values via least squares "
    "and return the dipole amplitude as the anomaly statistic. This measures whether "
    "non-Gaussianity (heaviness of tails) has a preferred spatial direction."
)


def analyze_map(m):
    """Return the dipole amplitude of kurtosis values over Nside=8 regions."""
    nside_high = hp.npix2nside(len(m))
    nside_low = 8
    npix_low = hp.nside2npix(nside_low)  # 768

    # Handle trivial case: map is already at Nside=8 or lower
    if nside_high <= nside_low:
        return 0.0

    finite_mask = np.isfinite(m)
    if np.sum(finite_mask) < npix_low:
        return 0.0

    valid_pix = np.where(finite_mask)[0]
    valid_vals = m[valid_pix]

    # Map each higher-resolution pixel to its Nside=8 parent
    ratio = nside_high // nside_low
    parent_pix = valid_pix // (ratio * ratio)

    # Group valid pixels by parent region
    sort_idx = np.argsort(parent_pix)
    parent_sorted = parent_pix[sort_idx]
    vals_sorted = valid_vals[sort_idx]
    unique_parents, counts = np.unique(parent_sorted, return_counts=True)
    split_points = np.cumsum(counts)[:-1]
    grouped_vals = np.split(vals_sorted, split_points)

    # Compute excess kurtosis for each Nside=8 region
    kurtosis = np.full(npix_low, np.nan)
    min_pixels = 10  # minimum for a reliable estimate

    for i, (parent, vals) in enumerate(zip(unique_parents, grouped_vals)):
        n = len(vals)
        if n < min_pixels:
            continue
        mu = np.mean(vals)
        m2 = np.mean((vals - mu) ** 2)
        if m2 <= 0:
            continue
        m4 = np.mean((vals - mu) ** 4)
        kurtosis[parent] = m4 / (m2 ** 2) - 3.0

    # Collect finite kurtosis values
    finite_k = np.isfinite(kurtosis)
    n_valid = np.sum(finite_k)
    if n_valid < 4:
        return 0.0

    k_vals = kurtosis[finite_k]

    # Pixel centers for Nside=8 in Cartesian coordinates
    theta, phi = hp.pix2ang(nside_low, np.arange(npix_low))
    x = np.sin(theta) * np.cos(phi)
    y = np.sin(theta) * np.sin(phi)
    z = np.cos(theta)

    x_f = x[finite_k]
    y_f = y[finite_k]
    z_f = z[finite_k]

    # Fit dipole: kurtosis = a + d_x*x + d_y*y + d_z*z
    # Normalize coordinates to improve conditioning
    A = np.column_stack([np.ones(n_valid), x_f, y_f, z_f])
    coeffs, _, _, _ = np.linalg.lstsq(A, k_vals, rcond=None)

    d_x, d_y, d_z = coeffs[1], coeffs[2], coeffs[3]
    dipole_amplitude = np.sqrt(d_x**2 + d_y**2 + d_z**2)

    return float(dipole_amplitude)


def summarize_results(planck_stat, sim_results):
    """Interpret the kurtosis dipole amplitude."""
    sims = np.array(sim_results, dtype=float)
    sims_finite = sims[np.isfinite(sims)]
    n_sims = len(sims_finite)

    if n_sims == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims_finite))
    std_s = float(np.std(sims_finite, ddof=1)) if n_sims > 1 else 0.0
    p_upper = float(np.mean(sims_finite >= planck_stat))
    p_lower = float(np.mean(sims_finite <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else np.nan

    return {
        "planck_stat": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "z_score": z_score,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n_sims,
        "interpretation": (
            "This statistic measures whether the non-Gaussianity of the CMB temperature "
            "field (as characterized by excess kurtosis) has a preferred spatial direction. "
            "For each Nside=8 region (768 pixels covering the full sky), the excess kurtosis "
            "of temperature values from the higher-resolution map is computed. A dipole "
            "model (monopole + dipole) is then fit to these kurtosis values via least "
            "squares. The dipole amplitude quantifies the degree of directional preference "
            "in the tail-heaviness of the temperature distribution. In isotropic ΛCDM, the "
            "kurtosis dipole amplitude should be small; a significantly elevated value could "
            "indicate anisotropic non-Gaussianity, residual systematics, or foreground "
            "contamination with a preferred direction."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Kurtosis Dipole Amplitude",
            "xlabel": "Dipole Amplitude (kurtosis)",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck amplitude = {planck_stat:.4f}. "
                f"Sim mean ± σ = {mean_s:.4f} ± {std_s:.4f}. "
                f"z = {z_score:.2f}, p(two-sided) = {p_two:.3f}."
            ),
        },
    }
