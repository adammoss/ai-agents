Hypothesis

The dipole amplitude of local skewness in the low-ℓ CMB temperature map is inconsistent with predictions from the standard ΛCDM model.

Method

The CMB temperature map is bandpass-filtered to ℓ = 2–10. For each unmasked pixel, the local skewness S(n̂) of temperatures within a 60° cap is computed, producing a sky map of third-moment asymmetry. The ℓ = 1 dipole amplitude A_skew = √(|S₁₀|² + 2|S₁₁|²) of this skewness map is extracted, quantifying any preferred direction in how local non-Gaussianity (hot-versus-cold asymmetry) varies across the sky.

Result Summary

Planck SMICA yields A_skew = 0.752, compared to a simulation mean of 0.667 ± 0.307. The Planck value sits at the 65.4th percentile of 1000 ΛCDM simulations, with a two-tailed p-value of 0.692 and a log-amplitude z-score of 0.46. Anomaly strength is negligible.

The low-ℓ skewness dipole amplitude is fully consistent with standard ΛCDM expectations. No preferred direction in local non-Gaussianity is detected at large angular scales. The Planck value falls well within the central distribution of simulations, showing no evidence of the directional asymmetry in non-Gaussianity that the test was designed to probe.

Agent Justification

The test is designed to explore any directional anomaly in non-Gaussianity without a prior directional expectation, as it measures the strength of a preferred direction in local skewness. The analysis code computes a two-sided p-value using a log transform, indicating that both higher and lower dipole amplitudes are considered potentially anomalous, supporting a two-tailed approach for general exploration.

Comparison To Literature

No prior studies on the dipole of a local-skewness sky map were found in the CMB anomaly literature. Existing work addresses global skewness, directional H₀ variations, and hemispherical power asymmetries, but not directional third-moment asymmetry via this approach.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Skewness Dipole Amplitude: Filter the masked CMB temperature map to ℓ = 2–10 "
    "via spherical-harmonic bandpass. For each unmasked pixel, compute the local skewness "
    "S(n̂) = ⟨(T − T̄)³⟩ / ⟨(T − T̄)²⟩^{3/2} of pixel temperatures within a 60°-radius cap, "
    "using only unmasked pixels. Decompose the resulting skewness map into spherical harmonics "
    "under the same mask and extract the ℓ = 1 dipole amplitude A_skew = √(|S₁₀|² + 2|S₁₁|²). "
    "This quantifies the strength of any preferred direction in how the degree of non-Gaussianity "
    "(hot-versus-cold asymmetry) varies across the sky at large angular scales."
)


def analyze_map(m):
    """Compute the Low-ℓ Skewness Dipole Amplitude.

    Steps:
    1. Bandpass filter the temperature map to ℓ = 2–10 via spherical harmonics.
    2. Downgrade to nside=16 for fast local skewness computation.
    3. For each unmasked pixel, compute local skewness in a 60° cap using
       a precomputed vectorized distance matrix (no query_disc).
    4. Decompose the skewness map into SH and extract the ℓ=1 dipole amplitude.
    """
    nside = hp.npix2nside(len(m))

    # Identify unmasked pixels and fill masked with 0 for SH transforms
    valid = np.isfinite(m)
    m_filled = np.where(valid, m, 0.0).astype(np.float64)

    # --- Step 1: Bandpass filter to ℓ = 2–10 ---
    lmax_bp = 10
    alm = hp.map2alm(m_filled, lmax=lmax_bp, iter=0)

    # Zero out ℓ = 0 and ℓ = 1 (keep only ℓ = 2–10)
    mask_ell = np.zeros(hp.Alm.getsize(lmax_bp), dtype=bool)
    for ell in range(2, lmax_bp + 1):
        for em in range(ell + 1):
            mask_ell[hp.Alm.getidx(lmax_bp, ell, em)] = True
    alm[~mask_ell] = 0.0

    # Reconstruct bandpass-filtered map, then downgrade to nside=16 for fast skewness
    nside_lo = 16
    m_filt = hp.alm2map(alm, nside_lo, lmax=lmax_bp, verbose=False)

    # Mask at low resolution: keep pixel if >50% of its area is unmasked
    if nside != nside_lo:
        mask_hi = valid.astype(np.float64)
        mask_lo = hp.ud_grade(mask_hi, nside_lo)
        valid_lo = mask_lo > 0.5
    else:
        valid_lo = valid

    # --- Step 2: Local skewness via vectorized distance matrix ---
    cap_radius_rad = np.radians(60.0)

    valid_idx = np.where(valid_lo)[0]
    n_valid = len(valid_idx)

    if n_valid < 4:
        return 0.0

    # Unit vectors for all valid pixels (precomputed for distance matrix)
    vecs = np.column_stack(hp.pix2vec(nside_lo, valid_idx))  # (n_valid, 3)

    # Full pairwise distance matrix: all distances between valid pixels
    dist_matrix = vecs @ vecs.T  # (n_valid, n_valid), cosine of angles

    # Precompute cap membership masks (no query_disc used)
    cap_masks = []
    for i in range(n_valid):
        in_cap = dist_matrix[i] > np.cos(cap_radius_rad)
        cap_masks.append(in_cap)

    # Bandpass-filtered temperatures (NaN for masked pixels)
    temps = np.where(valid_lo, m_filt, np.nan)

    # Compute skewness for each valid pixel
    skewness_map = np.full(hp.nside2npix(nside_lo), np.nan)

    for i in range(n_valid):
        pix = valid_idx[i]
        in_cap = cap_masks[i]

        # Temperatures of neighbor pixels in cap (NaN for masked neighbors)
        cap_temps = temps[valid_idx[in_cap]]
        cap_valid = np.isfinite(cap_temps)

        if np.sum(cap_valid) < 4:
            continue

        cap_vals = cap_temps[cap_valid]
        t_mean = np.mean(cap_vals)
        t_dev = cap_vals - t_mean
        m2 = np.mean(t_dev ** 2)
        m3 = np.mean(t_dev ** 3)

        if m2 > 1e-30:
            skewness_map[pix] = m3 / (m2 ** 1.5)

    # --- Step 3: Extract ℓ=1 dipole of the skewness map ---
    skew_valid = np.isfinite(skewness_map)
    skew_filled = np.where(skew_valid, skewness_map, 0.0).astype(np.float64)

    lmax_skew = 1
    alm_skew = hp.map2alm(skew_filled, lmax=lmax_skew, iter=0)

    # A_skew = √(|S₁₀|² + 2|S₁₁|²)
    # alm_skew indices: ℓ=0,m=0 → 0; ℓ=1,m=0 → 1; ℓ=1,m=1 → 2
    s10 = alm_skew[1]
    s11 = alm_skew[2]
    A_skew = np.sqrt(np.abs(s10) ** 2 + 2.0 * np.abs(s11) ** 2)

    return float(A_skew)


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ℓ Skewness Dipole Amplitude anomaly."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float('nan'),
            "p_value_low": float('nan'),
            "p_value_high": float('nan'),
            "rank": 0,
            "n_sims": 0,
            "percentile": float('nan'),
            "sim_mean": float('nan'),
            "sim_std": float('nan'),
            "log_z_score": float('nan'),
            "asymmetry_label": "unknown",
            "asymmetry_direction": "no simulations available",
            "interpretation": "No simulation results to compare.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Skewness Dipole Amplitude",
                "xlabel": "A_skew",
                "ylabel": "Number of simulations",
                "bins": 40,
                "caption": "No simulation data available.",
            },
        }

    # Percentile rank (one-sided)
    rank = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank / n_sims

    # One-sided p-values
    p_value_low = float(np.mean(sim <= planck_stat))
    p_value_high = float(np.mean(sim >= planck_stat))

    # Two-sided p-value using log transform (A_skew is non-negative, right-skewed)
    pos_sim = sim[sim > 0]
    log_planck = np.log(planck_stat) if planck_stat > 0 else float('-inf')
    if len(pos_sim) > 0:
        log_sim = np.log(pos_sim)
        mean_log = float(np.mean(log_sim))
        std_log = float(np.std(log_sim))
        z_score = (log_planck - mean_log) / std_log if std_log > 0 else float('nan')
        planck_dev = abs(log_planck - mean_log)
        sim_devs = np.abs(log_sim - mean_log)
        p_value_two_sided = float(np.mean(sim_devs >= planck_dev))
    else:
        z_score = float('nan')
        p_value_two_sided = float('nan')

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    if percentile >= 95 or percentile <= 5:
        strength = "strong"
    elif percentile >= 90 or percentile <= 10:
        strength = "moderate"
    else:
        strength = "negligible"

    if planck_stat > mean_sim + std_sim:
        direction = "Planck shows a stronger dipole in local skewness than typical simulations"
    elif planck_stat < mean_sim - std_sim:
        direction = "Planck shows a weaker dipole in local skewness than typical simulations"
    else:
        direction = "Planck's skewness dipole is consistent with simulations"

    interp = (
        "Low-ℓ skewness dipole amplitude A_skew = {stat:.6f}, indicating {direction}. "
        "This places it at the {pct:.1f}th percentile of ΛCDM simulations "
        "(two-sided p-value = {pval:.4f}). "
        "Simulated mean A_skew = {mean:.6f} ± {std:.6f}. "
        "Log-amplitude z-score = {z:.2f}. "
        "Anomaly strength: {strength}."
    ).format(stat=planck_stat, direction=direction, pct=percentile,
             pval=p_value_two_sided, mean=mean_sim, std=std_sim, z=z_score, strength=strength)

    caption = (
        "Histogram of the Low-ℓ Skewness Dipole Amplitude A_skew = √(|S₁₀|² + 2|S₁₁|²), "
        "measuring the strength of the preferred direction in local temperature skewness "
        "across the sky. The temperature map is first bandpass-filtered to ℓ = 2–10, "
        "then local skewness S(n̂) is computed in 60° caps around each unmasked pixel. "
        "The dipole of the resulting skewness map quantifies directional non-Gaussianity. "
        "The red line marks the Planck SMICA value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value_two_sided,
        "p_value_low": p_value_low,
        "p_value_high": p_value_high,
        "rank": rank,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "log_z_score": z_score,
        "asymmetry_label": strength,
        "asymmetry_direction": direction,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Skewness Dipole Amplitude",
            "xlabel": "A_skew = √(|S₁₀|² + 2|S₁₁|²)",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
