Hypothesis

The multifractal spectrum width ΔD = D(−2) − D(2) of the low-ℓ CMB temperature map deviates significantly from the distribution predicted by ΛCDM simulations, indicating an anomaly in spatial heterogeneity.

Method

The width of the multifractal singularity spectrum ΔD = D(−2) − D(2) is computed for the Planck CMB temperature map band-limited to ℓ ≤ 30. The sky is partitioned into equal-area HEALPix nested patches at five scales (~2°–32° diameter). Rényi dimensions D(q) for q ∈ {−2, −1, 0, 1, 2} are estimated via linear regression of qth-order Rényi entropy against log(patch size). ΔD quantifies the range of scaling exponents: ≈0 for uniform Gaussian fields, larger for heterogeneous spatial structure.

Result Summary

Planck ΔD = 0.6856, compared to ΛCDM simulation mean 0.6528 ± 0.0284. The Planck value lies at +1.15σ (percentile 90.2%), yielding a two-tailed p-value of 0.196. Anomaly strength is negligible.

The multifractal spectrum width of the low-ℓ CMB temperature map is fully consistent with ΛCDM expectations. There is no indication of anomalous spatial heterogeneity or homogeneity at large angular scales compared to isotropic Gaussian realisations.

Agent Justification

The test is explicitly designed to detect deviations in both directions, as evidenced by the code calculating two-sided p-values and interpreting anomalies for both wider (larger ΔD) and narrower (smaller ΔD) spectra relative to simulations. The description of ΔD highlights that positive values indicate heterogeneity, but this is a descriptive measure rather than a directional prediction; the anomaly test is exploratory for any departure from the expected uniform Gaussian field, necessitating a two-tailed approach.

Comparison To Literature

Multifractal analysis has been applied to cosmological structures (e.g., galaxy clustering, cosmic web), but accessible literature does not appear to contain a direct study of Rényi-dimension-based singularity spectrum width applied specifically to the low-ℓ CMB temperature field. This limits contextualisation with prior work.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Multifractal Scaling Width: "
    "Compute the width of the multifractal singularity spectrum ΔD = D(−2) − D(2) "
    "for the low-ℓ (ℓ ≤ 30) CMB temperature map. Using HEALPix nested ordering, "
    "partition the sky into equal-area patches at five scales (angular diameters "
    "~2°–32°). For each scale, compute the total absolute temperature fluctuation "
    "within each patch. Rényi dimensions D(q) are estimated via linear regression "
    "of qth-order Rényi entropy against log(patch size). ΔD measures the range of "
    "scaling exponents: ΔD ≈ 0 for uniform Gaussian fields; larger positive values "
    "indicate heterogeneous spatial structure."
)


def analyze_map(m):
    mask = np.isfinite(m)
    n_valid = int(np.sum(mask))
    if n_valid < 100:
        return 0.0

    nside = hp.get_nside(m)
    npix = 12 * nside ** 2

    # Replace NaN with 0 for healpy compatibility
    clean = m.copy()
    clean[~mask] = 0.0

    # Band-limit to ℓ ≤ 30
    lmax = 30
    alm = hp.map2alm(clean, lmax=lmax)
    low_ell_map = hp.alm2map(alm, nside, lmax=lmax, verbose=False)

    # Remove monopole so we analyse fluctuations
    low_ell_map -= np.mean(low_ell_map)

    # Positive measure: absolute temperature fluctuation
    measure = np.abs(low_ell_map)

    # Five scales: side-length factors k = 2, 4, 8, 16, 32
    # In nested ordering, parent index at nside/k is ipix // k²
    # Patch angular diameter ∝ k  →  ε ∝ k  →  log(1/ε) = −log(k) + const
    scales = [2, 4, 8, 16, 32]
    q_values = [-2, -1, 0, 1, 2]

    H_q = {q: [] for q in q_values}
    x_vals = []  # −log(k) values for regression

    for k in scales:
        if nside < k:
            continue

        k_sq = k * k
        n_patches = 12 * (nside // k) ** 2

        # Map each base pixel to its parent patch via nested ordering
        parent = np.arange(npix) // k_sq

        # Aggregate measure into patches (efficient vectorised bincount)
        patch_mu = np.bincount(parent, weights=measure, minlength=n_patches)

        total_mu = patch_mu.sum()
        if total_mu <= 0:
            continue

        # Normalised probability per patch
        p = patch_mu / total_mu
        p = p[p > 0]
        if len(p) < 2:
            continue

        log_p = np.log(p)

        for q in q_values:
            if q == 0:
                # H₀ = log(number of non-empty patches)
                H = np.log(float(len(p)))
            elif q == 1:
                # Shannon entropy
                H = -np.sum(p * log_p)
            else:
                # H_q = [log(Σ p_i^q)] / (1−q), computed via logsumexp
                log_pq = q * log_p
                max_lpq = np.max(log_pq)
                log_sum_pq = max_lpq + np.log(np.sum(np.exp(log_pq - max_lpq)))
                H = log_sum_pq / (1.0 - q)

            if np.isfinite(H):
                H_q[q].append(H)

        x_vals.append(-np.log(float(k)))

    n_pts = len(x_vals)
    if n_pts < 2:
        return 0.0

    x = np.array(x_vals)

    # Linear regression: H_q = D(q) · log(1/ε) + const
    # With x = −log(k) ∝ log(1/ε), the slope is D(q)
    D_q = {}
    for q in q_values:
        y = np.array(H_q[q])
        if len(y) != n_pts:
            D_q[q] = 0.0
            continue
        coeffs = np.polyfit(x, y, 1)
        D_q[q] = coeffs[0]  # slope = D(q)

    delta_D = D_q[-2] - D_q[2]
    return float(delta_D) if np.isfinite(delta_D) else 0.0


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ℓ Multifractal Scaling Width test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Multifractal Scaling Width",
                "xlabel": "ΔD = D(−2) − D(2)",
                "ylabel": "Count",
                "bins": 40,
                "caption": "No simulation data.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims
    deviation_sigma = (planck_stat - mean_sim) / std_sim if std_sim > 0 else 0.0

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    if planck_stat > mean_sim + 2 * std_sim:
        direction = (
            "Planck's low-ℓ map shows a significantly wider multifractal "
            "spectrum than ΛCDM simulations, indicating more heterogeneous "
            "spatial structure across scales than expected from isotropic "
            "Gaussian fluctuations."
        )
    elif planck_stat < mean_sim - 2 * std_sim:
        direction = (
            "Planck's low-ℓ map shows a significantly narrower multifractal "
            "spectrum than ΛCDM simulations, indicating more uniform "
            "spatial structure than expected."
        )
    else:
        direction = (
            "The multifractal spectrum width is consistent with "
            "ΛCDM expectations."
        )

    interp = (
        f"Multifractal spectrum width ΔD = {planck_stat:.4f}. "
        f"Simulation mean = {mean_sim:.4f} ± {std_sim:.4f} "
        f"({deviation_sigma:+.2f}σ). Percentile = {percentile:.1f}%, "
        f"two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of the multifractal spectrum width ΔD = D(−2) − D(2) "
        "for the low-ℓ (ℓ ≤ 30) CMB temperature field. ΔD measures the "
        "range of scaling exponents present in the map: ΔD ≈ 0 for a "
        "uniform Gaussian field, while larger positive values indicate "
        "heterogeneous spatial structure across scales. Red line marks "
        "the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": deviation_sigma,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Multifractal Scaling Width",
            "xlabel": "ΔD = D(−2) − D(2)",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
