Hypothesis

The excess kurtosis of low-ℓ needlet coefficients for the observed CMB map deviates from the distribution expected under ΛCDM simulations.

Method

Excess kurtosis (kurtosis minus 3) is computed from the spatial distribution of needlet (spherical wavelet) coefficients at scale j=1, isolating large angular scales (ℓ = 2–20). This single scalar probes localized non-Gaussian tail behavior in the low-ℓ CMB temperature field, potentially capturing modality or outlier structure missed by standard harmonic analyses. The observed value is compared to the distribution from 1000 ΛCDM simulations.

Result Summary

The Planck excess kurtosis is 0.188, while simulations yield a mean of 0.005 ± 0.268. The observed value lies at +0.68σ from the simulation mean, corresponding to the 78.9th percentile. The two-tailed p-value is 0.422, well above any significance threshold. Anomaly strength is classified as negligible.

The tail behavior and modality of the spatial distribution of low-ℓ needlet coefficients is fully consistent with ΛCDM Gaussian expectations. There is no evidence for heavy tails (leptokurtic outliers such as anomalous hot/cold spots) or unusually light tails (platykurtic uniformity) at the largest angular scales.

Agent Justification

The test is designed to detect any anomaly in tail behavior (non-Gaussianity) at the largest angular scales, which could manifest as either heavier (leptokurtic) or lighter (platykurtic) tails. The code computes both one-sided p-values and the interpretation explicitly checks for deviations in both directions, with no clear prior prediction of a specific direction in the test description. Therefore, a two-tailed approach is appropriate for general exploration.

Comparison To Literature

Needlet-based analyses of CMB non-Gaussianity have been explored (e.g., Pietrobon et al. 2006; Marinucci et al. 2008), but typically involve bispectrum or skewness statistics rather than excess kurtosis of low-ℓ needlet coefficients specifically. No directly comparable published result was found in prior searches, though the general approach falls within the established needlet non-Gaussianity framework.

import numpy as np

test_description = (
    "Compute needlet (spherical wavelet) coefficients of the temperature map at a "
    "scale band centered on the low-ℓ regime (ℓ ≈ 2–20). Specifically, construct a "
    "needlet frame coefficient map by applying the needlet band-pass filter b_j(ℓ) "
    "for scale j=1 (where the support of b_1 covers 2 ≤ ℓ ≤ 20) to the spherical "
    "harmonic coefficients a_ℓm, yielding a spatial map of needlet coefficients "
    "β_j(γ) over pixels γ on the sphere. Then compute the excess kurtosis (kurtosis "
    "minus 3) of the distribution of these needlet coefficient values across all "
    "unmasked pixels. The resulting single scalar quantifies spatially localized "
    "non-Gaussianity at the largest angular scales, capturing modality or tail "
    "behavior that standard spherical harmonic analyses may miss. The statistic is "
    "compared against the distribution obtained from 1000 ΛCDM simulations processed "
    "identically, and its significance is quoted in units of the simulation standard "
    "deviation."
)


def analyze_map(m):
    """Return the excess kurtosis of low-ℓ needlet coefficients for the input map."""
    import numpy as np
    import healpy as hp

    nside = hp.npix2nside(len(m))

    # Identify valid (unmasked) pixels
    mask = np.isfinite(m)
    n_valid = int(mask.sum())
    if n_valid < 100:
        return 0.0

    # Replace NaN with 0.0 for healpy transforms (cannot pass NaN to anafast/map2alm)
    m_filled = np.where(mask, m, 0.0)

    # Compute spherical harmonic coefficients up to ℓ_max = 20
    ell_max = 20
    alm = hp.map2alm(m_filled, lmax=ell_max, iter=0)

    # Construct needlet band-pass filter b_1(ℓ) for scale j=1
    # Support covers ℓ = 2 to ℓ = 20
    # Using a simple rectangular window: b_1(ℓ) = 1 for 2 ≤ ℓ ≤ 20, 0 otherwise
    band_filter = np.zeros(ell_max + 1)
    band_filter[2:21] = 1.0  # ℓ = 2, 3, ..., 20

    # Apply band-pass filter to alms
    alm_filtered = hp.almxfl(alm, band_filter)

    # Transform back to get needlet coefficient map β_1(γ)
    beta_map = hp.alm2map(alm_filtered, nside, lmax=ell_max, verbose=False)

    # Extract needlet coefficients at valid (unmasked) pixels only
    beta_valid = beta_map[mask]

    # Need at least 4 values to compute kurtosis
    if len(beta_valid) < 4:
        return 0.0

    # Compute excess kurtosis (Fisher's definition: kurtosis - 3)
    # For a Gaussian field, excess kurtosis ≈ 0
    mean_val = np.mean(beta_valid)
    diffs = beta_valid - mean_val
    m2 = np.mean(diffs ** 2)
    m4 = np.mean(diffs ** 4)

    if m2 <= 0:
        return 0.0

    raw_kurtosis = m4 / (m2 ** 2)
    excess_kurtosis = raw_kurtosis - 3.0

    if not np.isfinite(excess_kurtosis):
        return 0.0

    return float(excess_kurtosis)


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ℓ Needlet Coefficient Kurtosis test."""
    import numpy as np

    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Needlet Coefficient Kurtosis",
                "xlabel": "Excess Kurtosis",
                "ylabel": "Count",
                "bins": 50,
                "caption": "No simulation data.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))
    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    rank_below = int(np.sum(sim <= planck_stat))
    percentile = 100.0 * rank_below / n_sims
    dev_sigma = (planck_stat - mean_sim) / std_sim if std_sim > 0 else 0.0

    strength = (
        "strong" if p_two <= 0.01
        else "moderate" if p_two <= 0.05
        else "negligible"
    )

    interp = (
        f"Low-ℓ needlet coefficient excess kurtosis = {planck_stat:.4f}. "
        f"Simulation mean = {mean_sim:.4f} ± {std_sim:.4f} ({dev_sigma:+.2f}σ). "
        f"Percentile = {percentile:.1f}%, two-sided p = {p_two:.4f}. "
        f"Strength: {strength}. "
    )

    if planck_stat > mean_sim + 2 * std_sim:
        interp += (
            "The low-ℓ needlet coefficients exhibit significantly heavier tails "
            "(leptokurtic) compared to ΛCDM simulations, suggesting localized "
            "non-Gaussian features at the largest angular scales. This could indicate "
            "anomalous hot or cold spots, or modality in the spatial distribution of "
            "large-scale CMB temperature fluctuations."
        )
    elif planck_stat < mean_sim - 2 * std_sim:
        interp += (
            "The low-ℓ needlet coefficients exhibit significantly lighter tails "
            "(platykurtic) compared to ΛCDM simulations, suggesting an unusually "
            "uniform spatial distribution at the largest angular scales. This could "
            "indicate suppression of extreme features or reduced variance in localized "
            "regions of the low-ℓ CMB temperature field."
        )
    else:
        interp += (
            "The kurtosis of low-ℓ needlet coefficients is consistent with ΛCDM "
            "expectations. The tail behavior and modality of the spatial distribution "
            "of large-scale temperature fluctuations shows no significant departure "
            "from Gaussian statistics."
        )

    return {
        "planck_stat": planck_stat,
        "p_value_one_sided_upper": p_upper,
        "p_value_one_sided_lower": p_lower,
        "p_value_two_sided": p_two,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": dev_sigma,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Needlet Coefficient Kurtosis",
            "xlabel": "Excess Kurtosis",
            "ylabel": "Number of simulations",
            "bins": 50,
            "caption": (
                "Histogram of the excess kurtosis (kurtosis − 3) of low-ℓ needlet "
                "coefficients (ℓ = 2–20) across ΛCDM simulations. Values near 0 "
                "indicate Gaussian-like tail behavior; positive values indicate heavy "
                "tails (leptokurtic); negative values indicate light tails (platykurtic). "
                "The red line marks the Planck observed value."
            ),
        },
    }
