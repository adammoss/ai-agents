Hypothesis

The peak-to-RMS temperature ratio for the CMB at ℓ ≤ 20 is consistent with the distribution predicted by isotropic Gaussian LCDM simulations.

Method

Band-limit the CMB temperature map to ℓ ≤ 20 via an alm cutoff. Compute the RMS temperature and the maximum absolute temperature over unmasked pixels. The statistic is max|T| / RMS, which for a Gaussian random field has a well-defined expectation (~3–4). Deviations would signal anomalous concentration of extreme temperature features at large angular scales.

Result Summary

Observed peak-to-RMS ratio = 3.52. LCDM simulation mean = 3.48 ± 0.36, yielding z = +0.11σ. Two-tailed p = 0.80 (percentile = 60%). Anomaly strength: negligible.

The peak-to-RMS ratio at low multipoles is fully consistent with isotropic Gaussian LCDM expectations. The most extreme temperature features at ℓ ≤ 20 are neither unusually large nor unusually small relative to the overall RMS fluctuation, indicating no anomalous clustering of hot or cold spots at the largest angular scales.

Agent Justification

The test is exploratory, checking for any significant deviation from the expected Gaussian random field behavior, as indicated by the two-sided p-value calculation in the code and the description's focus on detecting values "far from the LCDM-simulated distribution" without specifying a direction (e.g., only higher or lower).

Comparison To Literature

Low-ℓ CMB anomalies are extensively studied, including the quadrupole–octupole alignment (Tegmark et al., 2003; de Oliveira-Costa et al., 2004), the power deficit at ℓ ≲ 30 (Planck Collaboration XXIII, 2014), and the CMB Cold Spot (Cruz et al., 2005; Vielva et al., 2004). However, the specific peak-to-RMS ratio as a test statistic is not a standard probe in the anomaly literature; related work typically focuses on angular power spectrum anomalies, multipole alignments, or non-Gaussian signatures via Minkowski functionals or needlet analyses (e.g., Planck 2018 results VII).

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Peak-to-RMS Temperature Ratio: "
    "Band-limit the temperature map to ℓ ≤ 20 using an alm cutoff. "
    "Compute the RMS temperature over all unmasked pixels. Find the maximum "
    "absolute temperature value among the same unmasked pixels. The statistic "
    "is the ratio of the maximum absolute temperature to the RMS temperature. "
    "For a Gaussian random field this ratio has a well-defined expectation "
    "(typically 3–4 depending on resolution and mask), so a measured value far "
    "from the LCDM-simulated distribution would indicate anomalous concentration "
    "of extreme temperature features at large angular scales."
)


def analyze_map(m):
    """Compute the peak-to-RMS ratio of the low-ℓ (ℓ ≤ 20) temperature map.

    Steps:
      1. Identify unmasked (finite) pixels.
      2. Fill masked pixels with 0 for healpy compatibility.
      3. Band-limit to ℓ ≤ 20 via map2alm with lmax=20 → alm2map with lmax=20.
      4. Compute RMS temperature over unmasked pixels.
      5. Compute max absolute temperature over unmasked pixels.
      6. Return ratio max|T| / RMS.

    Returns
    -------
    float
        Peak-to-RMS ratio of the low-ℓ map.
    """
    nside = hp.get_nside(m)
    mask = np.isfinite(m)
    n_valid = int(np.sum(mask))

    if n_valid < 100:
        return 0.0

    # Fill masked pixels with 0 for healpy transforms
    m_filled = np.where(mask, m, 0.0)

    # Band-limit to ell <= 20 using consistent lmax for both transforms
    lmax = 20

    alm = hp.map2alm(m_filled, lmax=lmax)

    # Transform back to map with the same lmax
    m_low = hp.alm2map(alm, nside, lmax=lmax, verbose=False)

    # Extract unmasked pixel values
    vals = m_low[mask]

    # RMS temperature
    rms = np.sqrt(np.mean(vals ** 2))

    if rms == 0.0:
        return 0.0

    # Maximum absolute temperature
    max_abs = np.max(np.abs(vals))

    # Peak-to-RMS ratio
    ratio = float(max_abs / rms)

    return ratio


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ℓ Peak-to-RMS Temperature Ratio test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available for comparison.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Peak-to-RMS Temperature Ratio",
                "xlabel": "Peak / RMS ratio",
                "ylabel": "Number of simulations",
                "bins": 40,
                "caption": "No simulation data available.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    # Two-sided empirical p-value
    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims

    if std_sim > 0:
        deviation_sigma = (planck_stat - mean_sim) / std_sim
    else:
        deviation_sigma = 0.0

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    if planck_stat > mean_sim + 2 * std_sim:
        direction = (
            "Planck's peak-to-RMS ratio at low multipoles is unusually high "
            "compared to typical LCDM realizations. This suggests the largest "
            "temperature excursions at ℓ ≤ 20 are more extreme relative to the "
            "overall RMS than expected from a Gaussian random field, indicating "
            "possible anomalous concentration of power in localized hot or cold "
            "spots at large angular scales."
        )
    elif planck_stat < mean_sim - 2 * std_sim:
        direction = (
            "Planck's peak-to-RMS ratio at low multipoles is unusually low "
            "compared to typical LCDM realizations. This indicates the extreme "
            "temperature features at large angular scales are less pronounced "
            "than expected, suggesting a more uniform temperature field than "
            "predicted by the standard model."
        )
    else:
        direction = (
            "The peak-to-RMS ratio at low multipoles is consistent with "
            "isotropic Gaussian LCDM expectations. The most extreme temperature "
            "features at ℓ ≤ 20 are neither unusually large nor unusually small."
        )

    interp = (
        f"Peak-to-RMS ratio = {planck_stat:.4f}. "
        f"Simulation mean = {mean_sim:.4f} ± {std_sim:.4f} "
        f"({deviation_sigma:+.2f}σ from mean). "
        f"Percentile = {percentile:.1f}%, two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of the peak-to-RMS temperature ratio for the band-limited "
        "(ℓ ≤ 20) CMB temperature map, computed from isotropic LCDM simulations. "
        "The ratio measures how extreme the largest temperature excursion is "
        "relative to the overall RMS fluctuation. For a Gaussian random field "
        "this ratio typically falls between 3 and 4. "
        "The red line marks the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": deviation_sigma,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Peak-to-RMS Temperature Ratio",
            "xlabel": "Peak / RMS ratio",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
