Hypothesis

The observed CMB temperature map exhibits an anomalous low-ℓ angular momentum of its gradient field, indicating a statistically significant deviation in the expected Gaussian, isotropic pattern.

Method

The CMB temperature map is filtered to ℓ ≤ 10 and its surface gradient ∇T computed at each pixel. A vector field V = r × ∇T is integrated over the sphere to yield total angular momentum L. The anomaly statistic is |L|. For a real scalar field, |L| = 2√(2π)|Im(a₁,₁)|, so it depends solely on the ℓ=1 dipole mode and measures any net directional circulation in the low-ℓ gradient pattern.

Result Summary

Planck |L| = 5.905. Simulation mean ± std = 14.24 ± 10.95. The observed value lies below the simulation mean with z = −0.76 and an upper-tail p-value of 0.735. The anomaly ratio (Planck/sim mean) is 0.415.

The Planck angular momentum magnitude is entirely unremarkable—slelow the median expectation from isotropic Gaussian realizations but well within the central 68% of simulations. There is no evidence of an anomalous dipole alignment or directional preference in the low-ℓ temperature gradient field.

Agent Justification

The test statistic |L| measures the magnitude of a vector, which is intrinsically non-negative. The described anomaly is explicitly directional—an "unexpected directional preference" or "anomalous dipole alignment"—and the analysis code's interpretation and p-value focus are on whether Planck's value is anomalously *large* compared to simulations. The physics indicates that |L| directly amplifies the imaginary part of the ℓ=1 coefficient, making an unexpectedly high value a clear indicator of a preferred axis or non-Gaussianity. There is no physical motivation to consider an abnormally *low* value as the primary anomaly; thus, a one-tailed (upper tail) test is appropriate.

Comparison To Literature

The specific angular-momentum-of-gradient statistic does not appear in the searched literature. Related work examines low-ℓ multipole alignments using multipole vectors (Abramo et al. 2006, arXiv:astro-ph/0604361; Das & Souradeep 2012, arXiv:1210.2861), but those test inter-multipole correlations rather than a global circulation integral. Since |L| reduces to a dipole test, it connects loosely to existing dipole-anomaly studies but uses a distinct mathematical construction.

import numpy as np
import healpy as hp

test_description = (
    "For the temperature map filtered to multipoles l <= 10, compute the surface gradient "
    "dT at each pixel. Construct the vector field V = r x dT, where r is the unit radial "
    "vector. Integrate V over the full sphere to obtain the total angular momentum vector L. "
    "The anomaly statistic is the magnitude |L|. "
    "Key physics: for a scalar field on the sphere, the angular momentum integral picks out "
    "only the l=1 (dipole) mode of the gradient field. For real T, |L| = 2*sqrt(2*pi) * |Im(a_{1,1})|."
)


def analyze_map(m):
    """Return the magnitude of total angular momentum |L| of the gradient field.
    
    L = integral of r x grad(T) over the sphere.
    For a real scalar field, this depends only on the l=1, m=+/-1 spherical harmonic coefficients.
    """
    # --- 1. Identify valid (non-masked) pixels ---
    valid = np.isfinite(m)
    if not np.any(valid):
        return 0.0
    
    m_clean = np.where(valid, m, 0.0)
    nside = hp.npix2nside(len(m_clean))
    
    # --- 2. Filter to l <= 10 ---
    lmax = 10
    alm = hp.map2alm(m_clean, lmax=lmax)
    
    # --- 3. Extract a_{1,1} coefficient ---
    # In healpy's alm layout for lmax=10:
    # Index 0: (l=0, m=0)
    # Index 1: (l=1, m=0)
    # Index 2: (l=1, m=1)
    # ...
    ell_arr, m_arr = hp.Alm.getlm(lmax=lmax)
    
    # Find index for (l=1, m=1)
    idx_11 = np.where((ell_arr == 1) & (m_arr == 1))[0]
    if len(idx_11) == 0:
        return 0.0
    
    a11 = alm[idx_11[0]]
    
    # --- 4. Compute angular momentum magnitude ---
    # For a real scalar field:
    # L_z = 0 (from reality condition)
    # L_x = L_y = -2*sqrt(pi) * Im(a_{1,1})
    # |L| = sqrt(L_x^2 + L_y^2) = 2*sqrt(2*pi) * |Im(a_{1,1})|
    
    L_magnitude = 2.0 * np.sqrt(2.0 * np.pi) * np.abs(a11.imag)
    
    return float(L_magnitude)


def summarize_results(planck_stat, sim_results):
    """Interpret the Low-l Angular Momentum of Gradient Field statistic."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)
    
    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}
    
    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    median_s = float(np.median(sims))
    
    # P-values: upper tail (anomalously large |L|) and lower tail
    p_value_upper = float(np.mean(sims >= planck_stat))
    p_value_lower = float(np.mean(sims <= planck_stat))
    p_two = min(2.0 * min(p_value_upper, p_value_lower), 1.0)
    
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else 0.0
    anomaly_ratio = planck_stat / mean_s if mean_s > 0 else float("inf")
    
    return {
        "planck_stat": planck_stat,
        "planck_angular_momentum": planck_stat,
        "anomaly_ratio": anomaly_ratio,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "sim_median": median_s,
        "z_score": z_score,
        "p_value_upper": p_value_upper,
        "p_value_lower": p_value_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "interpretation": (
            f"Angular momentum |L| measures the net circulation of the low-l temperature "
            f"gradient field (l<=10). For a real scalar field, |L| = 2*sqrt(2*pi) * |Im(a_{{1,1}})| "
            f"depends only on the l=1 dipole mode of the temperature field. A large |L| indicates "
            f"an unexpected directional preference in the temperature gradient pattern. "
            f"Planck |L| = {planck_stat:.4f}. "
            f"Sim mean +/- std = {mean_s:.4f} +/- {std_s:.4f}. "
            f"Anomaly ratio (Planck/sim_mean) = {anomaly_ratio:.4f}. "
            f"z = {z_score:.2f}, two-sided p = {p_two:.4f}. "
            f"A significantly high value would suggest an anomalous dipole alignment in the "
            f"low-l temperature pattern."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-l Angular Momentum of Gradient Field",
            "xlabel": "Angular Momentum |L|",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck |L| = {planck_stat:.4f}. "
                f"Sim mean +/- std = {mean_s:.4f} +/- {std_s:.4f}. "
                f"Anomaly ratio = {anomaly_ratio:.4f}. "
                f"z = {z_score:.2f}, P(two-sided) = {p_two:.4f}."
            ),
        },
    }
