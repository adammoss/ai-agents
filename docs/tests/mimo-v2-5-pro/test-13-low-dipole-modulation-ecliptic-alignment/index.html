Hypothesis

The direction of the low-ℓ dipole modulation shows an anomalous alignment with the ecliptic plane (i.e., |cos(angle)| significantly less than 0.5) compared to ΛCDM simulations.

Method

Measures the orientation of the low-ℓ (ℓ=2–6) hemispherical power asymmetry dipole relative to the ecliptic plane. The CMB temperature map is bandpass-filtered to ℓ=2–6, squared to form a power proxy, and the ℓ=1 dipole direction of the squared map is extracted via spherical harmonic decomposition. The output is |cos(angle)| between this dipole direction and the ecliptic pole (l=96.3°, b=−29.8° Galactic). Values near 0 indicate the power dipole lies in the ecliptic plane; near 1 indicates alignment with the ecliptic pole. Under isotropy this is uniformly distributed with mean ≈0.5.

Result Summary

Planck observed |cos(angle)| = 0.215. Simulation mean = 0.509 ± 0.287 (median = 0.519). Two-tailed p-value = 0.39 (z ≈ −0.86), placing the observed value well within the ΛCDM simulation distribution. Approximately 39% of simulations produce values at least as extreme.

While the observed |cos| = 0.215 is below 0.5, suggesting some tendency toward ecliptic-plane alignment, the deviation is only ~0.86σ and far from significant. This is consistent with normal statistical fluctuation under ΛCDM.

Agent Justification

While the primary anomaly of interest in the literature is alignment with the ecliptic plane (low |cos|), the test statistic |cos| is symmetric under isotropy and the code explicitly computes a two-sided p-value, checking both extremes. The test description and summary function do not pre-specify a directional prediction, treating both low and high values as potentially anomalous. Therefore, a two-tailed test is appropriate for this exploratory analysis.

Comparison To Literature

Hemispherical power asymmetry at low multipoles is well-studied (e.g., Eriksen et al. 2004; Planck 2016 XXIII), typically focusing on dipole modulation amplitude rather than directional orientation relative to the ecliptic. No directly comparable statistic measuring this specific alignment was found in our search. The null result is broadly consistent with prior findings that ecliptic anomalies at low ℓ are subtle and inconsistent across analyses.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Dipole Modulation Ecliptic Alignment. Measures how the direction of "
    "hemispherical power asymmetry at multipoles ℓ = 2–6 aligns with the ecliptic "
    "plane. The map is bandpass-filtered to ℓ = 2–6, squared to form a power proxy, "
    "and its ℓ = 1 dipole direction is extracted via spherical harmonic decomposition. "
    "The returned value is |cos(α)| where α is the angle between this dipole and the "
    "ecliptic pole (l=96.3°, b=−29.8° Galactic). Under isotropy this is Uniform[0,1] "
    "with mean ≈0.5; values near 0 indicate the power dipole lies close to the ecliptic "
    "plane (anomalous alignment). This complements amplitude-based dipole statistics by "
    "targeting the orientation of the low-ℓ power modulation."
)


def analyze_map(m):
    m = np.asarray(m, dtype=np.float64)
    mask = np.isfinite(m)
    m_clean = np.where(mask, m, 0.0)
    nside = hp.npix2nside(len(m_clean))

    # Step 1: Bandpass filter to ℓ = 2–6
    lmax = 6
    alm = hp.map2alm(m_clean, lmax=lmax)
    # Zero out ℓ = 0, 1 (healpy stores m >= 0 only)
    for l in range(2):
        for mm in range(l + 1):
            alm[hp.Alm.getidx(lmax, l, mm)] = 0.0
    m_filt = hp.alm2map(alm, nside, lmax=lmax)

    # Step 2: Square to get power proxy.
    # A 90° disc convolution in harmonic space multiplies each a_lm by W_ℓ.
    # For ℓ=1, W_1 = 0.5, which scales the dipole amplitude but NOT its direction.
    # Since we only need the direction, we skip the convolution.
    m_sq = m_filt ** 2
    m_sq[~mask] = 0.0  # re-apply mask so masked pixels don't contribute

    # Step 3: Extract ℓ=1 dipole of the squared map
    alm_sq = hp.map2alm(m_sq, lmax=1)
    # healpy ordering for lmax=1: idx 0→(0,0), 1→(1,0), 2→(1,1)
    a10 = alm_sq[hp.Alm.getidx(1, 1, 0)]
    a11 = alm_sq[hp.Alm.getidx(1, 1, 1)]

    # Convert alm to Cartesian dipole vector
    # T(θ,φ) = dx sinθ cosφ + dy sinθ sinφ + dz cosθ
    #   a_{10} = dz · √(4π/3)
    #   a_{11} = √(2π/3) · (−dx + i·dy)
    s1 = np.sqrt(4.0 * np.pi / 3.0)
    s2 = np.sqrt(2.0 * np.pi / 3.0)
    dz = np.real(a10) / s1
    dx = -np.real(a11) / s2
    dy = np.imag(a11) / s2
    d = np.array([dx, dy, dz])
    amp = np.linalg.norm(d)

    if amp < 1e-30:
        return 0.5  # no dipole detected; return isotropic expectation

    d_hat = d / amp

    # Steps 4–5: Ecliptic pole in Galactic Cartesian coordinates
    l_ep, b_ep = np.radians(96.3), np.radians(-29.8)
    ep = np.array([
        np.cos(b_ep) * np.cos(l_ep),
        np.cos(b_ep) * np.sin(l_ep),
        np.sin(b_ep),
    ])

    # |cos(angle)| between dipole direction and ecliptic pole
    return float(np.abs(d_hat @ ep))


def summarize_results(planck_stat, sim_results):
    sim_arr = np.array(sim_results, dtype=np.float64)
    finite_mask = np.isfinite(sim_arr)
    n_finite = int(finite_mask.sum())

    if n_finite > 0:
        valid = sim_arr[finite_mask]
        sim_median = float(np.median(valid))
        p_upper = float(np.sum(valid >= planck_stat) / n_finite)
        p_lower = float(np.sum(valid <= planck_stat) / n_finite)
        p_value = min(p_upper, p_lower)
        rank_upper = int(np.sum(valid >= planck_stat))
        rank_lower = int(np.sum(valid <= planck_stat))
    else:
        p_value = float("nan")
        rank_upper = rank_lower = 0
        sim_median = float("nan")

    sim_mean = float(np.nanmean(sim_arr))
    sim_std = float(np.nanstd(sim_arr))

    if n_finite > 0 and not np.isnan(p_value):
        if p_upper <= p_lower:
            direction, rank = "high", rank_upper
        else:
            direction, rank = "low", rank_lower
    else:
        direction, rank = "unknown", 0

    # Build interpretation
    parts = [
        f"|cos(angle to ecliptic pole)| = {planck_stat:.4f}. "
        f"Low values indicate the low-ℓ power dipole lies near the ecliptic plane; "
        f"high values indicate it points toward the ecliptic pole.",
        f"Two-sided p = {p_value:.3f} ({rank}/{n_finite} simulations "
        f"at least as extreme in the {direction} direction).",
        f"Simulation mean = {sim_mean:.4f} ± {sim_std:.4f}, "
        f"median = {sim_median:.4f}.",
    ]

    if not np.isnan(p_value) and p_value < 0.05:
        if direction == "low":
            parts.append(
                "Anomalous: the low-ℓ power dipole direction is unusually close to "
                "the ecliptic plane compared to ΛCDM simulations, potentially "
                "indicating a systematic effect or non-standard cosmological signal."
            )
        else:
            parts.append(
                "Anomalous: the low-ℓ power dipole direction is unusually close to "
                "the ecliptic pole (anti-aligned with the ecliptic plane) relative "
                "to ΛCDM simulations."
            )
    else:
        parts.append(
            "The ecliptic alignment of the low-ℓ power dipole direction is "
            "consistent with ΛCDM simulation expectations."
        )

    result = {
        "test_description": test_description,
        "planck_stat": float(planck_stat),
        "p_value": float(p_value),
        "rank": rank,
        "n_simulations": len(sim_results),
        "n_valid": n_finite,
        "sim_mean": sim_mean,
        "sim_std": sim_std,
        "sim_median": sim_median,
        "exceeds_95th_percentile": bool(p_value < 0.05) if not np.isnan(p_value) else False,
        "anomaly_direction": direction,
        "interpretation": " ".join(parts),
        "ell_range": "2-6",
    }

    result["plot_spec"] = {
        "kind": "histogram",
        "title": "Low-ℓ Dipole–Ecliptic Alignment",
        "xlabel": "|cos(angle to ecliptic pole)|",
        "ylabel": "Number of simulations",
        "bins": 50,
        "caption": (
            f"Histogram of |cos(angle to ecliptic pole)| from {n_finite} "
            f"ΛCDM simulations. Planck value = {planck_stat:.4f} "
            f"(two-sided p = {p_value:.3f}, rank {rank}/{n_finite}, "
            f"{direction} tail). Simulation mean = {sim_mean:.4f} ± {sim_std:.4f}. "
            f"Low values → power dipole in the ecliptic plane (anomalous alignment); "
            f"high values → dipole toward the ecliptic pole."
        ),
    }

    return result
