Hypothesis

The fractal dimension D of the low-ℓ excursion-set boundary deviates from the distribution expected under Gaussian random fields, indicating potential non-Gaussian structure in the CMB temperature map.

Method

Box-counting fractal dimension D of the zero-threshold excursion-set boundary for the CMB temperature map band-limited to ℓ ≤ 30. Grids of side-length 1, 2, 4, 8, 16 pixels are laid over boundary pixels; log N(s) vs log(1/s) is fit to yield D, where D ≈ 1 indicates smooth contours (Gaussian expectation) and D → 2 indicates space-filling, non-Gaussian structure.

Result Summary

Planck D = 1.2307 (moderately smooth contours with some branching). Simulation mean ± std = 1.2254 ± 0.0044 over 1000 Gaussian realisations. z = 1.22, two-sided p = 0.232. The observed value ranks at the 88.4th percentile of simulations—slightly elevated but well within Gaussian expectations.

The Planck low-ℓ excursion-set boundary has a fractal dimension consistent with Gaussian random-field predictions. The modest positive offset (z ≈ 1.2) could reflect mild additional branching or roughness, but falls far short of statistical significance. There is no evidence for anomalous filamentary or patchy non-Gaussian geometry at large angular scales.

Agent Justification

The test is exploratory, designed to detect anomalies in either direction—D could be lower (smoother contours) or higher (more complex boundaries) than expected from Gaussian simulations. The code calculates both upper and lower p-values and combines them into a two-sided p-value, supporting a two-tailed approach. The description mentions that non-Gaussian features can push D toward 2, but it does not specify a strict directional bias, as Gaussian fields yield D ~ 1, making deviations in either direction noteworthy for anomaly detection.

Comparison To Literature

No prior studies were found applying box-counting fractal dimension of excursion-set boundaries to CMB temperature maps. Existing related work involves Minkowski functionals, genus statistics, lacunarity, and multifractal spectra of CMB fields, which summarise bulk topology or full multifractal spectra rather than the boundary's intrinsic scaling dimension. import numpy as np
import healpy as hp

test_description = (
    "Compute the fractal (box-counting) dimension of the excursion-set boundary "
    "at the median-zero threshold. Band-limit the map to l <= 30 and extract "
    "the excursion set S = {pixels where T > 0 after median subtraction}. "
    "Identify all boundary pixels (pixels in S with at least one valid neighbour "
    "not in S, or vice-versa). Perform box-counting on the boundary pixel set: "
    "lay grids of side-length s = 1, 2, 4, 8, 16 pixels using HEALPix nested "
    "hierarchy, counting the number of boxes N(s) containing at least one boundary "
    "pixel. Fit log N(s) versus log(1/s) via least-squares; slope D is the fractal "
    "dimension. D ~ 1 for smooth contours (Gaussian); D -> 2 for space-filling "
    "non-Gaussian structures."
)


def analyze_map(m):
    """Return fractal dimension D of the low-l excursion-set boundary."""
    valid = np.isfinite(m)
    n_valid = int(np.sum(valid))

    if n_valid < 100:
        return 1.0

    m_clean = np.where(valid, m, 0.0)
    nside = hp.npix2nside(len(m_clean))
    lmax = 30

    # Band-limit to l <= 30
    alm = hp.map2alm(m_clean, lmax=lmax)
    m_low = hp.alm2map(alm, nside=nside, lmax=lmax, verbose=False)

    # Median-zero threshold: excursion set S = {valid pixels where T > median}
    m_low_valid = m_low[valid]
    median_val = np.median(m_low_valid)
    excursion = (m_low > median_val) & valid

    # --- Boundary detection ---
    npix = len(m)
    all_pix = np.arange(npix)

    # Get all 8 neighbors in RING ordering
    neighbors = hp.get_all_neighbours(nside, all_pix)  # shape (8, npix)

    # Mask: which neighbor entries are valid (not -1) and point to unmasked pixels
    valid_nb = (neighbors >= 0) & valid[neighbors]  # (8, npix)

    # Excursion status of each neighbor (False if invalid)
    nb_exc = np.zeros((8, npix), dtype=bool)
    nb_exc[valid_nb] = excursion[neighbors[valid_nb]]

    # Boundary: pixel in S has valid neighbor outside S, or vice versa
    has_nb_outside = np.any(valid_nb & ~nb_exc, axis=0)
    has_nb_inside = np.any(valid_nb & nb_exc, axis=0)
    boundary = ((excursion & has_nb_outside) | (~excursion & has_nb_inside)) & valid

    boundary_pix = np.where(boundary)[0]
    n_boundary = len(boundary_pix)

    if n_boundary < 10:
        return 1.0

    # --- Box-counting using HEALPix nested hierarchy ---
    boundary_pix_nest = hp.ring2nest(nside, boundary_pix)

    scales = [1, 2, 4, 8, 16]
    xs = []  # log(1/s)
    ys = []  # log N(s)

    for s in scales:
        if s == 1:
            N_s = n_boundary
        else:
            if nside // s < 1:
                continue
            parent = boundary_pix_nest // (s * s)
            N_s = len(np.unique(parent))

        if N_s > 0:
            xs.append(np.log(1.0 / s))
            ys.append(np.log(N_s))

    if len(xs) < 2:
        return 1.0

    x = np.array(xs)
    y = np.array(ys)

    # Least-squares: log N = D * log(1/s) + c
    D = np.polyfit(x, y, 1)[0]

    if not np.isfinite(D):
        return 1.0

    return float(np.clip(D, 0.0, 3.0))


def summarize_results(planck_stat, sim_results):
    """Interpret the Low-l Excursion-Set Boundary Fractal Index."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)

    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    median_s = float(np.median(sims))
    rank = int(np.sum(sims <= planck_stat))
    p_upper = float(np.mean(sims >= planck_stat))
    p_lower = float(np.mean(sims <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else 0.0

    if planck_stat < 1.15:
        geom = "very smooth contours (nearly circular)"
    elif planck_stat < 1.35:
        geom = "moderately smooth contours with some branching"
    elif planck_stat < 1.6:
        geom = "moderately complex, filamentary or patchy boundary"
    else:
        geom = "highly complex, near-space-filling boundary structure"

    return {
        "planck_stat": planck_stat,
        "fractal_dimension": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "sim_median": median_s,
        "z_score": z_score,
        "rank": rank,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "boundary_geometry": geom,
        "interpretation": (
            f"The box-counting fractal dimension of the low-l (l<=30) excursion-set "
            f"boundary is D = {planck_stat:.4f}. For a Gaussian random field, D ~ 1 "
            f"(smooth contours); values approaching 2 indicate space-filling, non-Gaussian "
            f"structure. Planck D = {planck_stat:.4f} ({geom}). "
            f"Simulation mean +/- std = {mean_s:.4f} +/- {std_s:.4f}. "
            f"z = {z_score:.2f}, two-sided p = {p_two:.4f}."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Excursion-Set Boundary Fractal Dimension",
            "xlabel": "Fractal dimension D",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck D = {planck_stat:.4f}. "
                f"Sim mean +/- std = {mean_s:.4f} +/- {std_s:.4f}. "
                f"z = {z_score:.2f}, P(two-sided) = {p_two:.4f}."
            ),
        },
    }
