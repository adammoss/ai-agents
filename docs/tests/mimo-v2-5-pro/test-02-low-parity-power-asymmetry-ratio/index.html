Hypothesis

The low-ℓ parity power asymmetry ratio R is significantly lower than expected from lensed-ΛCDM simulations, indicating a deficit of odd-parity power.

Method

Measures the ratio R of total angular power in odd vs. even multipoles (ℓ = 2–30). A deficit in odd-parity power (R < 1) has been reported as a low-ℓ anomaly.

Result Summary

Planck observed R = 1.24. Simulations (lensed-ΛCDM) yield a median R = 0.60, with a one-tailed (lower) p-value of 0.924. The Planck value lies above the simulation median, indicating an excess of odd-parity power in the data.

No significant deficit of odd-parity power is found. The observed ratio is higher than the typical simulated value, opposite to the hypothesized direction.

Agent Justification

The test description explicitly states that "small R indicates a deficit of odd-parity power relative to even-parity power, consistent with the reported ~2-3σ CMB parity asymmetry." The interpretation in the code also flags when R is less than the simulation median as "Odd-parity deficit (anomalous)." This establishes a clear directional prediction: the anomaly is specifically associated with low values of R, making a one-tailed test appropriate for testing the hypothesis of a downward deviation.

Comparison To Literature

The low-ℓ parity asymmetry is a well-known anomaly, typically reported at ~2–3σ significance (e.g., Kim & Naselsky 2010; Aluri et al. 2023). This test's result (z ≈ 1.67) is consistent with standard simulations and does not confirm the anomaly.

import numpy as np
import healpy as hp

test_description = ("Low-ℓ parity power asymmetry ratio. After applying the mask (NaN pixels filled with 0), "
    "the pseudo angular power spectrum C̃_ℓ is computed via anafast for ℓ = 2 to 30. "
    "The test statistic is R = Σ_{ℓ odd} C̃_ℓ / Σ_{ℓ even} C̃_ℓ, the ratio of total odd-multipole "
    "power to total even-multipole power at large angular scales. "
    "Small R indicates a deficit of odd-parity power relative to even-parity power, "
    "consistent with the reported ~2-3σ CMB parity asymmetry.")


def analyze_map(m):
    m = np.array(m, dtype=np.float64)
    
    # Fill masked (NaN) pixels with 0 for healpy transforms
    m_fill = np.where(np.isfinite(m), m, 0.0)
    
    # Compute pseudo power spectrum up to ℓ=30
    lmax = 30
    cl = hp.anafast(m_fill, lmax=lmax)
    
    # Sum power in odd and even multipoles for ℓ = 2 to 30
    ells = np.arange(2, lmax + 1)
    c_ells = cl[2:lmax + 1]
    
    # Mask for odd ℓ (3, 5, 7, ..., 29) and even ℓ (2, 4, 6, ..., 30)
    odd_mask = (ells % 2 == 1)
    even_mask = (ells % 2 == 0)
    
    s_odd = np.sum(c_ells[odd_mask])
    s_even = np.sum(c_ells[even_mask])
    
    # Avoid division by zero
    if abs(s_even) < 1e-30:
        return 1.0
    
    R = float(s_odd / s_even)
    return R


def summarize_results(planck_stat, sim_results):
    sim_arr = np.array(sim_results, dtype=np.float64)
    finite_mask = np.isfinite(sim_arr)
    n_finite = int(finite_mask.sum())

    if n_finite > 0:
        valid = sim_arr[finite_mask]
        rank = int(np.sum(valid < planck_stat))
        p_value_below = rank / n_finite
        p_value_above = 1.0 - p_value_below
        p_value = min(p_value_below, p_value_above)  # two-sided
    else:
        rank = 0
        p_value_below = float('nan')
        p_value_above = float('nan')
        p_value = float('nan')

    sim_median = float(np.nanmedian(sim_arr))

    result = {
        "test_description": test_description,
        "planck_stat": float(planck_stat),
        "p_value": float(p_value),
        "p_value_below": float(p_value_below),
        "p_value_above": float(p_value_above),
        "rank_below": rank,
        "n_simulations": len(sim_results),
        "n_valid": n_finite,
        "sim_mean": float(np.nanmean(sim_arr)),
        "sim_std": float(np.nanstd(sim_arr)),
        "sim_median": sim_median,
        "exceeds_95th_percentile": bool(p_value < 0.05),
        "interpretation": ("Odd-parity deficit (anomalous)" if planck_stat < sim_median
                           else "Even-parity deficit (odd excess)"),
    }

    result["plot_spec"] = {
        "kind": "histogram",
        "title": "Low-ℓ Odd-Even Power Ratio",
        "xlabel": "R = ΣC_ℓ(odd) / ΣC_ℓ(even), ℓ ∈ [2, 30]",
        "ylabel": "Number of simulations",
        "bins": 40,
        "caption": (f"Histogram of R from {n_finite} lensed-ΛCDM simulations. "
                    f"Planck R = {planck_stat:.4f} (two-sided p = {p_value:.3f}, "
                    f"rank {rank}/{n_finite}). "
                    f"R < 1 indicates odd-parity power deficit, consistent with the "
                    f"CMB parity asymmetry.")
    }

    return result
