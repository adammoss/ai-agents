Hypothesis

The Pearson cross-correlation coefficient between |∇T| and ∇²T for the low-ℓ CMB temperature map is zero for a statistically isotropic Gaussian random field.

Method

Band-limits the CMB temperature map to ℓ = 2–10, computes the gradient magnitude |∇T| and Laplacian ∇²T maps, then measures their Pearson cross-correlation coefficient r. For a statistically isotropic Gaussian random field, r = 0 exactly (by the T → −T symmetry: |∇T| is invariant while ∇²T flips sign). A significant non-zero r flags a gradient–curvature coupling anomaly invisible to power-spectrum or T–|∇T|/T–∇²T tests.

Result Summary

Planck r = 0.0601. Simulations (1000 realisations) yield mean ± σ = −0.0006 ± 0.1005. z = 0.60, p(two-sided) = 0.584.

The observed correlation is well within the 68% simulation spread. No evidence for a gradient–curvature coupling anomaly at low multipoles. Regions of steep slope show no systematic preference for convex versus concave curvature beyond what random Gaussian fluctuations predict.

Agent Justification

The test is designed to detect any significant deviation from zero (positive or negative) because the anomaly could manifest as either a positive or negative gradient-curvature coupling. The symmetry T → -T forces the expected value to zero, and we are looking for a violation of this symmetry without a preferred direction. The code computes a two-sided p-value, and the interpretation does not specify a direction, indicating a two-tailed test.

Comparison To Literature

No directly comparable statistic found in CMB anomaly studies. Related low-ℓ investigations (e.g., Gurzadyan et al. 2003, focusing on ellipticity and geodesic mixing) employ different observables. Standard analyses examine T–|∇T| or T–∇²T correlations, not the |∇T|–∇²T cross-term.

import numpy as np
import healpy as hp

test_description = (
    "Band-limits the temperature map to ℓ = 2–10, then computes the Pearson "
    "cross-correlation coefficient between the gradient magnitude |∇T| and the "
    "Laplacian ∇²T over unmasked pixels. For any statistically isotropic Gaussian "
    "random field the expected value is exactly zero (by the T → −T symmetry: "
    "|∇T| is unchanged while ∇²T flips sign). A significant non-zero value flags "
    "a gradient–curvature coupling anomaly—invisible to T–|∇T|, T–∇²T, or pure "
    "power-spectrum tests."
)


def analyze_map(m):
    """Return the Pearson cross-correlation r = corr(|∇T|, ∇²T) for ℓ = 2–10."""
    nside = hp.npix2nside(len(m))

    # --- 1. Clean masked pixels for healpy compatibility ---
    mask_valid = np.isfinite(m)
    m_clean = np.where(mask_valid, m, 0.0)

    # --- 2. Compute a_ℓm and zero out ℓ = 0, 1 ---
    lmax = 10
    alm = hp.map2alm(m_clean, lmax=lmax)

    # Zero out ℓ = 0 and ℓ = 1 modes
    for ell in range(2):
        for m_idx in range(ell + 1):
            alm[hp.Alm.getidx(lmax, ell, m_idx)] = 0.0

    # --- 3. Gradient components via spherical-harmonic derivatives ---
    # alm2map_der1 may return 2 or 3 arrays depending on healpy version;
    # we take only the first two: (d_theta, d_phi)
    der1_out = hp.alm2map_der1(alm, nside, lmax=lmax)
    grad_theta = der1_out[0]
    grad_phi = der1_out[1]

    # --- 4. Gradient magnitude |∇T| in pixel space ---
    grad_mag = np.sqrt(grad_theta**2 + grad_phi**2)

    # --- 5. Laplacian ∇²T via -ℓ(ℓ+1) multiplication in harmonic space ---
    alm_lap = alm.copy()
    ell_arr = np.arange(lmax + 1, dtype=float)
    factors = -ell_arr * (ell_arr + 1)
    
    # Apply factors efficiently using Alm indexing
    for ell in range(lmax + 1):
        idx_start = hp.Alm.getidx(lmax, ell, 0)
        idx_end = hp.Alm.getidx(lmax, ell, ell) + 1
        alm_lap[idx_start:idx_end] *= factors[ell]

    laplacian = hp.alm2map(alm_lap, nside, lmax=lmax)

    # --- 6. Restrict to unmasked pixels ---
    valid = mask_valid
    g = grad_mag[valid]
    lap = laplacian[valid]

    if len(g) < 10:
        return 0.0

    # --- 7. Pearson cross-correlation coefficient ---
    g_c = g - np.mean(g)
    lap_c = lap - np.mean(lap)

    std_g = np.sqrt(np.mean(g_c**2))
    std_lap = np.sqrt(np.mean(lap_c**2))

    if std_g < 1e-30 or std_lap < 1e-30:
        return 0.0

    r = np.mean(g_c * lap_c) / (std_g * std_lap)

    return float(np.clip(r, -1.0, 1.0))


def summarize_results(planck_stat, sim_results):
    """Interpret the gradient-magnitude–Laplacian cross-correlation coefficient."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)

    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    p_upper = float(np.mean(sims >= planck_stat))
    p_lower = float(np.mean(sims <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else np.nan

    return {
        "planck_stat": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "z_score": z_score,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "interpretation": (
            "This statistic measures the cross-correlation between the gradient "
            "magnitude |∇T| and the Laplacian ∇²T of the low-ℓ (ℓ = 2–10) CMB "
            "temperature field. For a statistically isotropic Gaussian random field, "
            "this correlation is exactly zero by the T → −T symmetry: |∇T| is "
            "invariant under field sign flip while ∇²T changes sign. A significant "
            "non-zero r indicates a gradient–curvature coupling anomaly—regions with "
            "large slope systematically prefer either convex (∇²T < 0) or concave "
            "(∇²T > 0) curvature—an effect invisible to power-spectrum, T–|∇T|, or "
            "T–∇²T diagnostics."
        ),
        "plot_spec": {
            "kind": "histogram",

            "xlabel": "r = corr(|∇T|, ∇²T)",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck r = {planck_stat:.4f}. "
                f"Sim mean ± σ = {mean_s:.4f} ± {std_s:.4f}. "
                f"z = {z_score:.2f}, p(two-sided) = {p_two:.3f}."
            ),
        },
    }
