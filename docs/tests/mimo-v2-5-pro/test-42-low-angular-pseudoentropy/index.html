Hypothesis

The low-ℓ angular pseudoentropy of the observed CMB map deviates from the distribution predicted by Gaussian isotropic LCDM simulations.

Method

Compute the angular pseudoentropy S_angular = -∫ ρ(θ,φ) ln ρ(θ,φ) dΩ of the low-ℓ (ℓ ≤ 10) CMB temperature field, where ρ is the normalised absolute temperature distribution over the sphere. This rotationally invariant scalar quantifies spatial uniformity of low-ℓ power; deviations from the characteristic Gaussian value signal non-Gaussianity or anisotropy. Compared against 1000 Gaussian isotropic LCDM simulations at nside=256.

Result Summary

Observed S = 2.231 nats vs. simulation mean 2.212 ± 0.024 nats (z = +0.76σ). Percentile = 77.6%, two-tailed p = 0.448. Anomaly strength: negligible. The maximum possible entropy (uniform sphere) is 2.531 nats.

The low-ℓ angular pseudoentropy lies well within the distribution expected from Gaussian isotropic simulations. No evidence for anomalous non-uniformity or concentration of low-ℓ temperature power on the sphere. This is consistent with the absence of statistically significant large-scale anisotropy in the temperature field.

Agent Justification

The test is exploratory without a prior directional prediction; the analysis code calculates two-sided p-values, and the interpretation considers both higher and lower entropy as potential anomalies.

Comparison To Literature

The concept of pseudoentropy for CMB analysis was introduced and applied to Planck 2015 data by Pinkwart, Schupp & Schwarz (2019), who linked it to multipole vectors for detecting statistical anisotropy. The present test focuses specifically on low multipoles using the normalised absolute temperature field, rather than the full multipole structure studied previously. No direct prior application of this exact statistic to ℓ ≤ 10 was identified, though the underlying entropy framework is established.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Angular Pseudoentropy: "
    "For the low-ℓ (ℓ ≤ 10) component of the CMB temperature map, "
    "normalize the |temperature| field to create a probability density "
    "ρ(θ,φ) that integrates to unity over the full sphere. Compute "
    "the angular pseudoentropy S_angular = -∫ ρ(θ,φ) ln ρ(θ,φ) dΩ. "
    "This scalar quantifies the uniformity of the low-ℓ temperature "
    "distribution; Gaussian random skies have a characteristic entropy "
    "value, so deviations signal non-Gaussianity or anisotropy. The "
    "statistic is rotationally invariant and computationally cheap."
)


def analyze_map(m):
    """Low-ℓ angular pseudoentropy.

    Steps:
    1. Replace NaN pixels with 0 for healpy compatibility.
    2. Decompose into spherical harmonics up to ℓ=10 and reconstruct
       a smooth low-ℓ map (fills masked regions naturally via the
       band-limited basis).
    3. Form ρ(θ,φ) = |T_low(θ,φ)| / [ΔΩ · Σ|T_low|] so that
       ∫ ρ dΩ = 1 over the full sphere.
    4. Compute S = -Σ w_i ln w_i + ln(ΔΩ) where w_i = |T_i|/Σ|T_j|,
       which is equivalent to -∫ ρ ln ρ dΩ on the pixel grid.
    """
    mask = np.isfinite(m)
    n_valid = int(np.sum(mask))
    if n_valid < 100:
        return 0.0

    nside = hp.get_nside(m)
    npix = 12 * nside ** 2

    # Replace NaN with 0 for healpy spherical-harmonic operations
    clean = m.copy()
    clean[~mask] = 0.0

    # Filter to ℓ ≤ 10
    lmax = 10
    alm = hp.map2alm(clean, lmax=lmax)
    filtered = hp.alm2map(alm, nside, lmax=lmax, verbose=False)

    # Absolute value gives a non-negative field suitable as a density
    abs_map = np.abs(filtered)

    # Normalised weights that sum to 1
    total = np.sum(abs_map)
    if total <= 0:
        return 0.0

    w = abs_map / total

    # Shannon entropy of the discrete weight distribution (in nats)
    w_pos = w[w > 0]
    H = -np.sum(w_pos * np.log(w_pos))

    # Convert to continuous angular pseudoentropy:
    #   ρ_i = w_i / ΔΩ,  ΔΩ = 4π / N_pix
    #   S = -Σ w_i ln(w_i / ΔΩ) = -Σ w_i ln w_i + ln ΔΩ
    delta_omega = 4.0 * np.pi / npix
    S = H + np.log(delta_omega)

    return float(S) if np.isfinite(S) else 0.0


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ℓ Angular Pseudoentropy test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    S_max = np.log(4.0 * np.pi)  # uniform distribution on the sphere

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Angular Pseudoentropy",
                "xlabel": "S_angular (nats)",
                "ylabel": "Count",
                "bins": 40,
                "caption": "No simulation data.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims
    deviation_sigma = (planck_stat - mean_sim) / std_sim if std_sim > 0 else 0.0

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    if planck_stat > mean_sim + 2 * std_sim:
        direction = (
            f"Planck has significantly higher angular pseudoentropy than "
            f"LCDM simulations (closer to the theoretical maximum "
            f"S_max = {S_max:.4f} nats), indicating a more uniform spatial "
            f"distribution of low-ℓ power across the sky than expected "
            f"from Gaussian isotropy."
        )
    elif planck_stat < mean_sim - 2 * std_sim:
        direction = (
            "Planck has significantly lower angular pseudoentropy than "
            "LCDM simulations, suggesting more concentrated or anisotropic "
            "distribution of low-ℓ temperature fluctuations than expected "
            "from Gaussian isotropy."
        )
    else:
        direction = (
            "The angular pseudoentropy is consistent with Gaussian LCDM "
            "expectations."
        )

    interp = (
        f"Angular pseudoentropy S = {planck_stat:.4f} nats. "
        f"Maximum possible (uniform) = {S_max:.4f} nats. "
        f"Simulation mean = {mean_sim:.4f} ± {std_sim:.4f} "
        f"({deviation_sigma:+.2f}σ). Percentile = {percentile:.1f}%, "
        f"two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of the angular pseudoentropy S = -∫ ρ ln ρ dΩ of the "
        "low-ℓ (ℓ ≤ 10) CMB temperature field, where ρ is the normalised "
        "|T| distribution over the sphere. Higher entropy indicates more "
        "uniform spatial distribution of low-ℓ power; lower entropy "
        "indicates more concentrated or anisotropic structure. "
        "Red line marks the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": deviation_sigma,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Angular Pseudoentropy",
            "xlabel": "S_angular (nats)",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
