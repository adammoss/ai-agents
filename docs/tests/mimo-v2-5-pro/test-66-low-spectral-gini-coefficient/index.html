Hypothesis

The Gini coefficient of the low-ℓ (ℓ = 2–10) power spectrum for the Planck CMB temperature map is statistically different from the distribution of the Gini coefficient predicted by ΛCDM simulations.

Method

The Gini coefficient is computed over C_ℓ values at ℓ = 2–10 from a masked Planck temperature map. The nine power spectrum values are sorted and the Gini formula is applied to quantify distributional inequality: G → 0 for uniform power spread, G → 1 for extreme concentration in a few multipoles. This statistic is insensitive to overall calibration but probes whether the known quadrupole deficit and octopole dominance produce anomalous power concentration compared to ΛCDM predictions.

Result Summary

Planck yields G = 0.3511, while 1000 ΛCDM simulations give mean ± σ = 0.5198 ± 0.0890. The z-score is −1.89, indicating Planck's power distribution is more uniform than typical simulations. The two-tailed p-value is 0.05, meaning approximately 5% of simulations exhibit equal or greater deviation from the mean.

Planck's low-ℓ power is distributed more evenly across multipoles than ΛCDM simulations predict. While the known quadrupole deficit and octopole relative strength are present, the overall inequality among ℓ = 2–10 is lower than expected, suggesting either a mild uniformity anomaly or a statistical fluctuation near the detection boundary.

Agent Justification

The test is designed to investigate whether the distribution of power among the low multipoles is anomalous compared to simulations, but the description and code do not explicitly predict a specific direction of deviation (e.g., more concentrated or more uniform). The analysis code computes both upper and lower p-values, and the summary interpretation states it is probing "whether the known quadrupole deficit and octopole dominance manifest as anomalous power concentration," which is a general check for inequality, not a directional prediction. Therefore, a two-tailed test is appropriate to detect deviations in either direction.

Comparison To Literature

No prior application of the Gini coefficient to low-ℓ CMB power concentration was identified in searches. Related low-ℓ anomaly studies typically examine mean power deficit, quadrupole–octopole alignment, or hemispherical asymmetry rather than distributional inequality across multipoles.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Spectral Gini Coefficient: "
    "Compute the angular power spectrum C_ℓ for multipoles ℓ = 2 through 10 (inclusive) "
    "from the masked, band-limited temperature map. Sort these nine C_ℓ values into ascending "
    "order and compute the Gini coefficient as a measure of power concentration across multipoles. "
    "G = (2 Σ i·C_(i)) / (9 Σ C_(i)) − (9+1)/9, where i runs from 1 to 9 over sorted values. "
    "High G indicates power concentrated in a few dominant multipoles (e.g., octopole dominance); "
    "low G indicates uniformly spread power. This probes whether the known quadrupole deficit "
    "and octopole dominance manifest as anomalous power concentration compared to ΛCDM simulations."
)


def analyze_map(m):
    """Return Gini coefficient of low-ℓ (ℓ=2..10) power spectrum."""
    lmax = 10  # We need ℓ up to 10

    finite_mask = np.isfinite(m)
    if not np.any(finite_mask):
        return 0.0

    # Replace NaN with 0 for healpy transforms
    m_clean = np.where(finite_mask, m, 0.0)

    # Compute angular power spectrum up to lmax
    cl_full = hp.anafast(m_clean, lmax=lmax)

    # Extract ℓ = 2 through 10 (indices 2..10 in the cl array)
    cl_low = cl_full[2:11]  # 9 values: ℓ = 2, 3, ..., 10

    # Sort in ascending order
    cl_sorted = np.sort(cl_low)

    n = len(cl_sorted)  # should be 9
    total_power = np.sum(cl_sorted)

    if total_power < 1e-30:
        return 0.0

    # Gini coefficient:
    # G = (2 * sum(i * C_(i))) / (n * sum(C_(i))) - (n + 1) / n
    # where i runs from 1 to n over sorted values
    indices = np.arange(1, n + 1, dtype=float)
    gini = (2.0 * np.dot(indices, cl_sorted)) / (n * total_power) - (n + 1.0) / n

    # Clamp to [0, 1] for robustness (Gini can be negative for very unusual distributions)
    gini = float(np.clip(gini, 0.0, 1.0))

    return gini


def summarize_results(planck_stat, sim_results):
    """Interpret low-ℓ spectral Gini coefficient relative to simulations."""
    sims = np.array(sim_results, dtype=float)
    sims_finite = sims[np.isfinite(sims)]
    n_sims = len(sims_finite)

    if n_sims == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims_finite))
    std_s = float(np.std(sims_finite, ddof=1)) if n_sims > 1 else 0.0
    p_value_upper = float(np.mean(sims_finite >= planck_stat))
    p_value_lower = float(np.mean(sims_finite <= planck_stat))
    p_value_two_sided = 2.0 * min(p_value_upper, p_value_lower)
    p_value_two_sided = min(p_value_two_sided, 1.0)
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else np.nan

    # Determine which tail is anomalous
    if planck_stat > mean_s:
        tail_note = "Planck has higher power concentration (more unequal distribution) than simulations."
    else:
        tail_note = "Planck has lower power concentration (more uniform distribution) than simulations."

    return {
        "planck_stat": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "z_score": z_score,
        "p_value_upper": p_value_upper,
        "p_value_lower": p_value_lower,
        "p_value_two_sided": p_value_two_sided,
        "n_sims": n_sims,
        "tail_note": tail_note,
        "interpretation": (
            "The Gini coefficient measures inequality in power allocation among low-ℓ multipoles. "
            "Values near 0 indicate uniform power across ℓ=2..10; values near 1 indicate power "
            "concentrated in one or a few multipoles. In standard ΛCDM, the quadrupole (ℓ=2) is "
            "known to be low and the octopole (ℓ=3) relatively high, which produces moderate "
            "inequality. An anomalously high Gini would indicate extreme power concentration; "
            "an anomalously low G would suggest more uniformity than expected."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Spectral Gini Coefficient",
            "xlabel": "Gini coefficient (ℓ = 2–10)",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck G = {planck_stat:.4f}. "
                f"Sim mean ± σ = {mean_s:.4f} ± {std_s:.4f}. "
                f"z = {z_score:.2f}, p(two-sided) = {p_value_two_sided:.3f}. "
                f"Higher G indicates power concentrated in fewer multipoles."
            ),
        },
    }
