Hypothesis

The Pearson cross-correlation coefficient between low-ℓ (2≤ℓ≤10) and high-ℓ (11≤ℓ≤32) band-limited temperature maps is significantly different from zero, indicating anomalous cross-scale coherence inconsistent with a statistically isotropic Gaussian field.

Method

The Pearson correlation coefficient is computed between two band-limited CMB temperature maps: a low-ℓ band (2≤ℓ≤10) and a high-ℓ band (11≤ℓ≤32). For a statistically isotropic Gaussian field, these multipole bands should be independent, so the expected correlation is zero. The test isolates pixel-space cross-correlation between large and intermediate angular scales within the low-multipole regime.

Result Summary

Planck yields Pearson r = 0.0692, compared to simulations with mean 0.0386 ± 0.0083. The observed value sits at the 99.9th percentile of the simulation distribution (z ≈ 3.7σ), with a two-tailed p-value of 0.002. The anomaly strength is classified as strong.

Planck exhibits a significantly positive cross-correlation between the ℓ=2–10 and ℓ=11–32 band-limited temperature maps. Large-scale and intermediate-scale temperature features are unusually aligned, indicating coherent spatial structure across angular scales that should be independent under statistical isotropy. This cross-scale coherence may be related to known low-ℓ anomalies, such as the quadrupole–octupole alignment or the hemispherical asymmetry, suggesting a common origin in a possible departure from isotropy or Gaussianity.

Agent Justification

The test description and code implementation explicitly compute a two-sided empirical p-value to assess deviations in both positive and negative directions from the expected zero correlation under isotropic Gaussian ΛCDM simulations. The interpretation in the code checks for both significant positive and negative anomalies, and there is no prior directional bias specified in the hypothesis; thus, a two-tailed test is appropriate for general exploration.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Band-Limited Cross-Correlation Coefficient: "
    "Compute the spherical harmonic coefficients a_ℓm of the temperature map "
    "for 2 ≤ ℓ ≤ 10 (low band) and 11 ≤ ℓ ≤ 32 (high band). Construct two "
    "filtered maps by zeroing all a_ℓm outside each band and performing an "
    "inverse spherical harmonic transform. The statistic is the Pearson "
    "correlation coefficient between the pixel values of these two filtered "
    "maps, computed over the unmasked sky region. This measures the unexpected "
    "spatial correlation between large and intermediate angular scales within "
    "the low-ℓ regime."
)


def analyze_map(m):
    """Compute Pearson cross-correlation between low-ℓ and high-ℓ band-limited maps.

    Steps:
      1. Identify unmasked pixels (finite values).
      2. Fill masked (NaN) pixels with 0.0 for healpy compatibility.
      3. Compute alm coefficients up to ℓ_max = 32.
      4. Create two band-limited alm arrays:
         - Low band: 2 ≤ ℓ ≤ 10
         - High band: 11 ≤ ℓ ≤ 32
      5. Transform each band-limited alm back to pixel space.
      6. Extract values at unmasked pixels and compute Pearson r.

    Returns
    -------
    float
        Pearson cross-correlation coefficient between low-ℓ and high-ℓ filtered maps.
    """
    mask = np.isfinite(m)
    valid_idx = np.where(mask)[0]
    N = len(valid_idx)

    if N == 0:
        return 0.0

    # Fill masked pixels with 0 for healpy transforms
    m_filled = np.where(mask, m, 0.0)

    # Only need ℓ up to 32
    lmax = 32

    # Compute spherical harmonic coefficients
    alm = hp.map2alm(m_filled, lmax=lmax)

    # Get ℓ values for each alm index
    l_arr, _ = hp.Alm.getlm(lmax)

    # Low band: 2 ≤ ℓ ≤ 10
    alm_low = np.copy(alm)
    alm_low[(l_arr < 2) | (l_arr > 10)] = 0.0

    # High band: 11 ≤ ℓ ≤ 32
    alm_high = np.copy(alm)
    alm_high[(l_arr < 11) | (l_arr > 32)] = 0.0

    # Transform back to pixel space
    nside = hp.npix2nside(len(m))
    map_low = hp.alm2map(alm_low, nside, lmax=lmax)
    map_high = hp.alm2map(alm_high, nside, lmax=lmax)

    # Extract values at unmasked pixels
    vals_low = map_low[valid_idx]
    vals_high = map_high[valid_idx]

    # Center values
    mean_low = np.mean(vals_low)
    mean_high = np.mean(vals_high)
    low_c = vals_low - mean_low
    high_c = vals_high - mean_high

    # Pearson correlation coefficient
    num = np.sum(low_c * high_c)
    den_low = np.sqrt(np.sum(low_c * low_c))
    den_high = np.sqrt(np.sum(high_c * high_c))

    if den_low == 0.0 or den_high == 0.0:
        return 0.0

    r = num / (den_low * den_high)
    return float(r)


def summarize_results(planck_stat, sim_results):
    """Summarise the Low-ℓ Band-Limited Cross-Correlation test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available for comparison.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Band-Limited Cross-Correlation",
                "xlabel": "Pearson r(T_{2≤ℓ≤10}, T_{11≤ℓ≤32})",
                "ylabel": "Number of simulations",
                "bins": 40,
                "caption": "No simulation data available.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    # Two-sided empirical p-value
    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    if abs(planck_stat) > abs(mean_sim) + 2 * std_sim:
        if planck_stat > 0:
            direction = (
                "Planck shows a significantly positive cross-correlation between "
                "the low-ℓ (2≤ℓ≤10) and high-ℓ (11≤ℓ≤32) band-limited temperature "
                "maps. This indicates that large-scale and intermediate-scale "
                "temperature features are unusually aligned, suggesting coherent "
                "spatial structure across angular scales that should be independent "
                "in a statistically isotropic Gaussian field."
            )
        else:
            direction = (
                "Planck shows a significantly negative cross-correlation between "
                "the low-ℓ (2≤ℓ≤10) and high-ℓ (11≤ℓ≤32) band-limited temperature "
                "maps. This indicates that large-scale and intermediate-scale "
                "temperature features are unusually anti-correlated, suggesting "
                "anomalous phase relationships between angular scales."
            )
    else:
        direction = (
            "The cross-correlation between the low-ℓ and high-ℓ band-limited "
            "temperature maps is consistent with isotropic Gaussian ΛCDM "
            "expectations; there is no anomalous alignment between large and "
            "intermediate angular scales."
        )

    interp = (
        f"Pearson r(T_{{2≤ℓ≤10}}, T_{{11≤ℓ≤32}}) = {planck_stat:.6f}. "
        f"Simulation mean = {mean_sim:.6f} ± {std_sim:.6f}. "
        f"Percentile = {percentile:.1f}%, two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of the Pearson cross-correlation coefficient between the "
        "low-ℓ band (2≤ℓ≤10) and high-ℓ band (11≤ℓ≤32) filtered temperature "
        "maps from isotropic ΛCDM simulations. For a statistically isotropic "
        "Gaussian field, different ℓ bands are independent, so this correlation "
        "should be centered at zero. Significant nonzero values indicate "
        "anomalous cross-scale coherence in the temperature field, potentially "
        "related to the low-ℓ anomaly or other large-angle CMB anomalies. "
        "The red line marks the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "anomaly_strength": strength,
        "interpretation": interp,
