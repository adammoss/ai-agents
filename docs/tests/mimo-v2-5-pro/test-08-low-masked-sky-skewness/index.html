Hypothesis

The null hypothesis is that the CMB temperature map at large scales (ℓ=2-20) exhibits skewness consistent with zero, as expected from a Gaussian random field under ΛCDM simulations. The alternative hypothesis is that the skewness deviates from zero, indicating non-Gaussianity.

Method

Applies the Planck foreground mask and bandpass-filters the temperature map to retain only multipoles ℓ = 2–20. Computes the skewness (third central moment normalised by σ³) of pixel values in unmasked regions, providing a single scalar probe of large-scale non-Gaussianity.

Result Summary

Planck skewness S = −0.209; simulation mean = 0.001, σ_sim = 0.167; z-score = −1.26; two-tailed p-value = 0.206. The observed value lies comfortably within the distribution of 1000 lensed-ΛCDM simulations.

The negative skewness indicates a slight tendency toward cold-spot excess, but the deviation is well within Gaussian expectations. No evidence for unexpected large-scale non-Gaussianity is found in the ℓ = 2–20 range after applying the Planck mask.

Agent Justification

The test description emphasizes significant deviations from zero skewness in either direction, and the analysis code computes a two-sided p-value to assess if the Planck value is extreme relative to simulations without directional bias. No specific directional prediction is provided, making this an exploratory test for non-Gaussian anomalies at large scales, necessitating a two-tailed approach.

Comparison To Literature

Previous studies have probed low-ℓ anomalies using multipole vectors, planarity, and alignment diagnostics (e.g., Copi et al. 2004), but a direct comparison to a masked-sky pixel skewness statistic at these multipoles could not be found in the literature. Searches for "CMB low multipole skewness" returned work on primordial f_NL and ellipticity rather than this specific observable. The result is broadly consistent with the general finding that most low-ℓ non-Gaussianity tests show no strong departure from ΛCDM.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Masked-Sky Skewness. For each map, the Planck mask is applied (masked "
    "pixels are np.nan) and the temperature anisotropies are filtered to retain only "
    "multipoles ℓ = 2 to 20 via spherical harmonic decomposition. The skewness of "
    "the pixel values in the unmasked region is then computed as the third central "
    "moment divided by the cube of the standard deviation. This single scalar "
    "quantifies large-scale non-Gaussianity in the CMB temperature map. Significant "
    "deviations from zero (the Gaussian expectation) would indicate unexpected "
    "non-Gaussian features at the largest angular scales."
)


def analyze_map(m):
    m = np.asarray(m, dtype=np.float64)

    # Identify unmasked pixels (those that are finite)
    valid_mask = np.isfinite(m)

    # Replace NaN with 0.0 for healpy spherical-harmonic transforms
    m_clean = np.where(valid_mask, m, 0.0)

    # Compute spherical harmonic coefficients up to lmax=20
    lmax = 20
    alm = hp.map2alm(m_clean, lmax=lmax)

    # Zero out monopole (l=0, index 0) and dipole (l=1, indices 1-2)
    # so only l=2..20 survive
    hp.remove_dipole(m_clean, copy=False)  # this handles l=0 and l=1
    # Actually, let's do it properly via alm manipulation
    # Re-compute cleanly: zero l=0 (1 coeff) and l=1 (2 coeffs) = first 3 indices
    alm[:3] = 0.0

    # Reconstruct the filtered map containing only l=2..20
    nside = hp.npix2nside(len(m))
    m_filtered = hp.alm2map(alm, nside=nside, lmax=lmax, verbose=False)

    # Extract pixel values only in the unmasked region
    values = m_filtered[valid_mask]

    # Compute skewness: third central moment / sigma^3
    n = len(values)
    if n < 3:
        return 0.0

    mu = np.mean(values)
    centered = values - mu
    sigma = np.std(values, ddof=0)

    if sigma == 0.0:
        return 0.0

    skewness = np.mean(centered ** 3) / (sigma ** 3)

    return float(skewness)


def summarize_results(planck_stat, sim_results):
    sim_arr = np.array(sim_results, dtype=np.float64)
    finite_mask = np.isfinite(sim_arr)
    n_finite = int(finite_mask.sum())

    if n_finite > 0:
        valid = sim_arr[finite_mask]
        # Two-sided p-value: how extreme is Planck relative to simulations?
        rank = int(np.sum(np.abs(valid) >= np.abs(planck_stat)))
        p_value = rank / n_finite
    else:
        rank = 0
        p_value = float('nan')

    sim_median = float(np.nanmedian(sim_arr))
    sim_mean = float(np.nanmean(sim_arr))
    sim_std = float(np.nanstd(sim_arr))

    # Determine direction of skewness
    if planck_stat > 0:
        skew_dir = "positive (tail toward hot spots)"
    else:
        skew_dir = "negative (tail toward cold spots)"

    interpretation = (
        f"Anomalous: low-ℓ skewness S = {planck_stat:.4f} ({skew_dir}), "
        f"two-sided p = {p_value:.3f} — inconsistent with Gaussian ΛCDM"
        if p_value < 0.05
        else f"Low-ℓ skewness S = {planck_stat:.4f} ({skew_dir}) is consistent "
             f"with Gaussian ΛCDM expectations (two-sided p = {p_value:.3f})"
    )

    result = {
        "test_description": test_description,
        "planck_stat": float(planck_stat),
        "p_value": float(p_value),
        "rank": rank,
        "n_simulations": len(sim_results),
        "n_valid": n_finite,
        "sim_mean": sim_mean,
        "sim_std": sim_std,
        "sim_median": sim_median,
        "exceeds_95th_percentile": bool(p_value < 0.05),
        "interpretation": interpretation,
        "skewness_direction": skew_dir,
        "expected_gaussian_skewness": "≈0 for a Gaussian random field",
    }

    result["plot_spec"] = {
        "kind": "histogram",
        "title": "Low-ℓ Masked-Sky Skewness (ℓ = 2–20)",
        "xlabel": "Skewness S = ⟨(T−μ)³⟩ / σ³",
        "ylabel": "Number of simulations",
        "bins": 50,
        "caption": (
            f"Histogram of low-ℓ skewness S from {n_finite} lensed-ΛCDM simulations "
            f"(ℓ = 2–20, Planck mask applied). Planck value = {planck_stat:.4f} "
            f"(two-sided p = {p_value:.3f}, rank {rank}/{n_finite}). "
            f"Skewness measures large-scale non-Gaussianity; a value of 0 is expected "
            f"for a statistically isotropic Gaussian CMB sky."
        ),
    }

    return result
