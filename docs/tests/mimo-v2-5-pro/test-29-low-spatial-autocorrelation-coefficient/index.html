Hypothesis

The Moran's I spatial autocorrelation coefficient of the Planck CMB temperature map deviates significantly from the distribution expected under isotropic Gaussian ΛCDM simulations.

Method

The Planck CMB temperature map is smoothed to ~10° FWHM to retain only large-scale structure. Moran's I spatial autocorrelation statistic is computed on the HEALPix grid using 8-neighbor adjacency, measuring whether neighboring pixels tend to have similar (positive I) or dissimilar (negative I) temperatures compared to isotropic Gaussian ΛCDM expectations.

Result Summary

Moran's I = 1.000753, compared to simulation mean = 1.001712 ± 0.000332. The Planck value falls at the 0.6th percentile of simulations, yielding a two-tailed p-value of 0.012 (z ≈ −2.89σ). Anomaly strength classified as strong.

Planck exhibits significantly negative spatial autocorrelation at large angular scales. The smoothed temperature map displays an unusually checkerboard-like pattern, with adjacent pixels tending to have opposite-sign temperatures more frequently than expected under ΛCDM. This suggests anti-correlated spatial structure or anomalous interference among low-ℓ spherical harmonic modes, consistent with previously noted large-angle anomalies.

Agent Justification

The test is designed to detect anomalies in both directions—positive autocorrelation indicating spatial clustering and negative autocorrelation indicating checkerboard-like structure—without a prior directional preference. The code explicitly computes a two-sided p-value and interprets deviations in either tail, supporting a non-directional hypothesis.

Comparison To Literature

No published studies were found applying Moran's I spatial autocorrelation specifically to CMB temperature maps. However, the finding is broadly consistent with known low-ℓ anomalies—including the quadrupole–octupole alignment, hemispherical power asymmetry, and cold spot—collectively suggesting departures from statistical isotropy at large angular scales. The checkerboard-like anti-correlation pattern detected here may relate to anomalous phase correlations among low-ℓ multipoles reported in other anomaly studies.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Spatial Autocorrelation Coefficient: Smooth the temperature map to an "
    "effective FWHM of ~10° to retain only large-scale (low-ℓ) structure. Using a "
    "nearest-neighbor adjacency weight matrix on the HEALPix grid (8 neighbors per "
    "unmasked pixel), compute Moran's I spatial autocorrelation statistic "
    "I = (N/W) × [Σᵢ Σⱼ wᵢⱼ (Tᵢ - T̄)(Tⱼ - T̄)] / [Σᵢ (Tᵢ - T̄)²], where N is "
    "the number of unmasked pixels, T̄ is the masked-sky mean temperature, wᵢⱼ = 1 "
    "if pixels i and j are spatial neighbors and both unmasked (0 otherwise), and "
    "W = Σᵢ Σⱼ wᵢⱼ is the total neighbor-pair weight. Values near zero are expected "
    "for Gaussian ΛCDM; significant positive values indicate anomalous spatial "
    "clustering of hot/cold regions; significant negative values indicate unusual "
    "checkerboard-like spatial structure."
)


def analyze_map(m):
    """Compute Moran's I spatial autocorrelation on a 10°-smoothed low-ℓ map.

    Steps:
      1. Identify unmasked pixels (finite values).
      2. Fill masked (NaN) pixels with 0 for healpy compatibility.
      3. Smooth the map with a Gaussian beam of FWHM ≈ 10° to retain only
         large-scale structure.
      4. Extract smoothed values at unmasked pixels, compute deviation from
         the masked-sky mean.
      5. Build a full-sky lookup array of centered values (0 for masked pixels).
      6. For each unmasked pixel, find its 8 HEALPix neighbors (4 side + 4
         corner). Use the lookup array to sum centered values of unmasked
         neighbors and compute Moran's I numerator and total weight.
      7. Return Moran's I = (N/W) × (Σᵢ Σⱼ wᵢⱼ cᵢ cⱼ) / (Σᵢ cᵢ²).

    Returns
    -------
    float
        Moran's I spatial autocorrelation coefficient.
    """
    nside = hp.npix2nside(len(m))
    npix = hp.nside2npix(nside)
    mask = np.isfinite(m)
    valid_idx = np.where(mask)[0]
    N = len(valid_idx)

    if N == 0:
        return 0.0

    # Fill masked pixels with 0 for healpy transforms
    m_filled = np.where(mask, m, 0.0)

    # Smooth to ~10° FWHM to isolate low-ℓ structure
    fwhm_rad = 10.0 * np.pi / 180.0
    m_smooth = hp.smoothing(m_filled, fwhm=fwhm_rad, verbose=False)

    # Extract smoothed values at unmasked pixels and center
    vals = m_smooth[valid_idx]
    T_bar = np.mean(vals)
    centered = vals - T_bar

    # Denominator: sum of squared deviations
    denom = np.sum(centered ** 2)
    if denom == 0.0:
        return 0.0

    # Build full-sky lookup of centered values: 0 for masked pixels
    centered_full = np.zeros(npix, dtype=np.float64)
    centered_full[valid_idx] = centered

    # Validity lookup: True if pixel is unmasked
    is_valid = np.zeros(npix, dtype=bool)
    is_valid[valid_idx] = True

    # Get all 8 neighbors for valid pixels (shape: 8 × npix)
    # hp.get_all_neighbours returns -1 for missing neighbors
    all_neighbors = hp.get_all_neighbours(nside, valid_idx)  # shape (8, N)

    # Check which neighbors are valid (not -1 and unmasked)
    has_neighbor = all_neighbors >= 0
    # Safe index: replace -1 with 0 so we can use it for array lookup
    safe_idx = np.where(has_neighbor, all_neighbors, 0)
    neighbor_valid = np.zeros_like(all_neighbors, dtype=bool)
    neighbor_valid[has_neighbor] = is_valid[safe_idx[has_neighbor]]

    # Get centered values of neighbors (0 for invalid ones via safe_idx)
    neighbor_centered = centered_full[safe_idx]  # shape (8, N)
    # Zero out invalid neighbors
    neighbor_centered = np.where(neighbor_valid, neighbor_centered, 0.0)

    # Moran's I numerator: Σᵢ Σⱼ wᵢⱼ cᵢ cⱼ
    # For each pixel i, contribution = cᵢ × Σⱼ(valid neighbor) cⱼ
    sum_neighbor = np.sum(neighbor_centered, axis=0)  # shape (N,)
    numerator = np.sum(centered * sum_neighbor)

    # Total weight W = total number of valid neighbor pairs
    W = float(np.sum(neighbor_valid))

    if W == 0.0:
        return 0.0

    # Moran's I
    morans_I = (N / W) * (numerator / denom)

    return float(morans_I)


def summarize_results(planck_stat, sim_results):
    """Summarise the Low-ℓ Spatial Autocorrelation Coefficient test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results to compare.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Spatial Autocorrelation (Moran's I)",
                "xlabel": "Moran's I",
                "ylabel": "Number of simulations",
                "bins": 40,
                "caption": "No simulation data available.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    # Two-sided p-value
    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    direction = ""
    if planck_stat > mean_sim + 2 * std_sim:
        direction = (
            "Planck shows significantly positive spatial autocorrelation: "
            "hot and cold regions on the large-scale (10°-smoothed) map are "
            "more clustered together than expected from isotropic Gaussian "
            "ΛCDM simulations. This could indicate anomalous alignment of "
            "low-ℓ modes, preferred directions, or spatially coherent "
            "temperature structure beyond statistical isotropy."
        )
    elif planck_stat < mean_sim - 2 * std_sim:
        direction = (
            "Planck shows significantly negative spatial autocorrelation: "
            "the large-scale temperature map has an unusually checkerboard-like "
            "structure with adjacent pixels tending to have opposite signs "
            "more than expected, suggesting anti-correlated spatial patterns "
            "or unusual mode interference."
        )
    else:
        direction = (
            "The spatial autocorrelation is consistent with isotropic Gaussian "
            "ΛCDM expectations; the large-scale temperature field shows no "
            "anomalous clustering or anti-clustering of hot/cold regions."
        )

    interp = (
        f"Moran's I = {planck_stat:.6f}. "
        f"Simulation mean = {mean_sim:.6f} ± {std_sim:.6f}. "
        f"Percentile = {percentile:.1f}%, two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of Moran's I spatial autocorrelation statistic from "
        "isotropic ΛCDM simulations. The temperature map is smoothed to "
        "~10° FWHM to isolate large-scale (low-ℓ) structure. Moran's I "
        "measures whether neighboring pixels on the HEALPix grid (8 "
        "neighbors per pixel) tend to have similar (positive I) or "
        "dissimilar (negative I) temperature values. Under Gaussian "
        "statistical isotropy, I should be near zero; significantly "
        "positive values indicate spatial clustering of hot/cold regions, "
        "while significantly negative values indicate checkerboard-like "
        "structure. The red line marks the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Spatial Autocorrelation (Moran's I)",
            "xlabel": "Moran's I (10°-smoothed map, 8-neighbor adjacency)",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
