Hypothesis

The octopole-hexadecupole tensor alignment statistic S for the Planck CMB data is inconsistent with the distribution expected from the standard ΛCDM model.

Method

For each CMB multipole restricted to ℓ=3 and ℓ=4, a real symmetric trace-free rank-2 tensor T_ij is constructed from the quadrupole moment of the squared multipole pattern, capturing each mode's preferred plane. The scalar S = Tr(T^(3)T^(4))/(||T^(3)||_F ||T^(4)||_F) measures the cosine of the Frobenius angle between the two tensors, yielding a rotationally invariant alignment measure in [−1,1].

Result Summary

Planck SMICA gives S = 0.467 (moderate coplanar alignment), placing it at the 8.8th percentile of 1000 ΛCDM simulations (two-tailed p = 0.176, |S| p = 0.916). The z-score is −1.35σ, with simulated mean S = 0.793 ± 0.250. The observed alignment is weaker than typical simulations but not significantly so.

The Planck octopole-hexadecupole tensor alignment is unremarkable. The modest two-tailed p-value (0.176) and z-score (−1.35σ) are well within ΛCDM expectations. The lower-than-average observed S may reflect modest statistical fluctuation rather than a physical effect.

Agent Justification

The test is designed to detect anomalies by comparing the observed alignment against simulations, with no specified directional preference in the test description or code. The alignment can be either positive (coplanar) or negative (anti-coplanar), and the code computes a two-sided p-value (p_two_sided) to assess significance for deviations in both directions, making a two-tailed test appropriate.

Comparison To Literature

Abramo et al. (2006) [arXiv:astro-ph/0604336] reported possible anomalies in octopole-hexadecupole alignment using multipole vector methods, and Marcos-Caballero & Martínez-González (2019) [arXiv:1903.03223] found that scale-dependent dipolar modulation reduces quadrupole-octopole alignment significance. The present tensor-based Frobenius approach measures alignment through a different mathematical object than multipole vector plane correlations, yielding a distinct statistic.

import numpy as np
import healpy as hp

test_description = ("Octopole-Hexadecupole Tensor Alignment: For each CMB multipole restricted "
    "to ell=3 and ell=4, constructs a real symmetric trace-free rank-2 tensor T_ij that captures "
    "the preferred plane of anisotropy via the quadrupole of the squared multipole pattern. "
    "The scalar S = T^(3)_ij T^(4)_ij / (||T^(3)_F ||T^(4)||_F) is the normalized Frobenius inner "
    "product (cosine of angle between tensors), yielding a rotationally invariant alignment measure "
    "in [-1, 1]. Values near ±1 indicate strong planar alignment between the octopole and "
    "hexadecupole, while values near 0 indicate no preferred alignment. This complements the "
    "quadrupole-octopole alignment test already in the suite.")


def analyze_map(m):
    """Compute the octopole-hexadecupole tensor alignment statistic.

    For multipole ell, we form the squared field pattern f_ell^2 and compute its
    trace-free quadrupole moment tensor T_ij = int f_ell^2 * (n_i n_j - delta_ij/3) dOmega.
    This 3x3 symmetric trace-free tensor captures the preferred plane of anisotropy.
    The alignment S = Tr(T3 @ T4) / (||T3||_F * ||T4||_F) is the cosine of the
    Frobenius-angle between the two tensors.
    """
    m_filled = np.where(np.isfinite(m), m, 0.0)
    nside = hp.npix2nside(len(m))
    lmax = 4
    npix = hp.nside2npix(nside)

    # Spherical harmonic decomposition up to ell=4
    alm_full = hp.map2alm(m_filled, lmax=lmax)

    # Precompute pixel unit vectors (vectorized over all pixels)
    theta, phi = hp.pix2ang(nside, np.arange(npix))
    sin_t = np.sin(theta)
    n_x = sin_t * np.cos(phi)
    n_y = sin_t * np.sin(phi)
    n_z = np.cos(theta)
    n_vecs = [n_x, n_y, n_z]
    dOmega = 4.0 * np.pi / npix

    def tracefree_tensor(ell):
        """Trace-free rank-2 tensor from quadrupole of f_ell^2."""
        # Isolate coefficients for this multipole
        alm_ell = np.zeros_like(alm_full)
        i0 = hp.Alm.getidx(lmax, ell, 0)
        i1 = hp.Alm.getidx(lmax, ell, ell) + 1
        alm_ell[i0:i1] = alm_full[i0:i1]

        # Synthesize multipole map, square it to highlight ridges
        map_ell = hp.alm2map(alm_ell, nside, lmax=lmax, verbose=False)
        w = (map_ell ** 2) * dOmega  # pixel-weighted squared field

        # T_ij = sum_pix w * n_i * n_j  (symmetric 3x3)
        T = np.empty((3, 3))
        for i in range(3):
            for j in range(i, 3):
                val = np.dot(w, n_vecs[i] * n_vecs[j])
                T[i, j] = val
                if j != i:
                    T[j, i] = val

        # Make trace-free: T -= (trace/3) * I
        tr = (T[0, 0] + T[1, 1] + T[2, 2]) / 3.0
        T[0, 0] -= tr
        T[1, 1] -= tr
        T[2, 2] -= tr
        return T

    T3 = tracefree_tensor(3)
    T4 = tracefree_tensor(4)

    # Frobenius inner product and norms
    inner = np.sum(T3 * T4)
    norm3 = np.sqrt(np.sum(T3 * T3))
    norm4 = np.sqrt(np.sum(T4 * T4))

    if norm3 < 1e-30 or norm4 < 1e-30:
        return 0.0

    alignment = float(np.clip(inner / (norm3 * norm4), -1.0, 1.0))
    return alignment


def summarize_results(planck_stat, sim_results):
    """Summarize the octopole-hexadecupole alignment anomaly."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]

    # Percentile rank
    rank = int(np.sum(sim < planck_stat))
    n_sims = len(sim)
    percentile = 100.0 * rank / n_sims if n_sims > 0 else float('nan')

    # P-values
    p_lower = float(np.mean(sim <= planck_stat)) if n_sims > 0 else float('nan')
    p_upper = float(np.mean(sim >= planck_stat)) if n_sims > 0 else float('nan')
    p_two_sided = min(2.0 * min(p_lower, p_upper), 1.0)

    # Absolute alignment significance (alignment can be + or -)
    abs_planck = abs(planck_stat)
    abs_sim = np.abs(sim)
    p_abs = float(np.mean(abs_sim >= abs_planck)) if n_sims > 0 else float('nan')

    # Descriptive statistics
    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))
    z_score = (planck_stat - mean_sim) / std_sim if std_sim > 0 else float('nan')
    mean_abs = float(np.mean(abs_sim))
    std_abs = float(np.std(abs_sim))

    # Interpretation labels
    if abs_planck > 0.8:
        strength = "very strong"
    elif abs_planck > 0.6:
        strength = "strong"
    elif abs_planck > 0.4:
        strength = "moderate"
    elif abs_planck > 0.2:
        strength = "weak"
    else:
        strength = "negligible"

    direction = "coplanar (same preferred plane)" if planck_stat > 0 else "anti-coplanar (orthogonal preferred planes)"

    interp = (
        "Octopole-hexadecupole tensor alignment S = {stat:.4f}, indicating {strength} {direction} alignment. "
        "This places it at the {pct:.1f}th percentile of LCDM simulations "
        "(p_two-sided = {p2s:.3f}, p_|S| = {pabs:.3f}). "
        "Simulated mean S = {mean:.4f} +/- {std:.4f}, mean |S| = {ma:.4f} +/- {sa:.4f}."
    ).format(stat=planck_stat, strength=strength, direction=direction,
             pct=percentile, p2s=p_two_sided, pabs=p_abs,
             mean=mean_sim, std=std_sim, ma=mean_abs, sa=std_abs)

    caption = (
        "Histogram of the octopole-hexadecupole tensor alignment statistic S = "
        "Tr(T^(3) @ T^(4)) / (||T^(3)||_F ||T^(4)||_F), measuring the degree of planar alignment "
        "between the ell=3 and ell=4 multipoles. Each tensor T_ij is the trace-free quadrupole "
        "moment of the squared multipole pattern f_ell^2. Values near ±1 indicate strong "
        "alignment of preferred planes, values near 0 indicate no preferred alignment. "
        "The red line marks the Planck SMICA value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_two_sided,
        "p_value_abs": p_abs,
        "rank": rank,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "sim_mean_abs": mean_abs,
        "sim_std_abs": std_abs,
        "z_score": z_score,
        "alignment_label": strength,
        "direction_label": direction,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Octopole–Hexadecupole Tensor Alignment",
            "xlabel": "S = Tr(T$^{(3)}$ T$^{(4)}$) / (||T$^{(3)}$||$_F$ ||T$^{(4)}$||$_F$)",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
