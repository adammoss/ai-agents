Hypothesis

The low-ℓ (2-5) power-modulation dipole direction in the CMB temperature map is preferentially aligned with the galactic north pole.

Method

Decomposes the CMB temperature map (ℓ = 2–5) into a squared-field map T², extracts its dipole to find the power-modulation direction n̂, and measures its alignment with the galactic north pole via the signed dot product n̂ · n̂_gal.

Result Summary

Planck SMICA value = -0.584 (z = -0.92). Two-sided p = 0.524; one-sided (upper) p = 0.738. The result lies at the 26.2nd percentile of simulations, indicating a south-pointing modulation axis.

The observed low-ℓ power modulation points strongly toward the galactic south pole, contrary to the hypothesis. However, this alignment is well within the range expected from random LCDM fluctuations.

Agent Justification

The test statistic is defined as the signed dot product between the modulation direction and the galactic north pole, where a value of +1 indicates perfect alignment with galactic north, -1 indicates alignment with galactic south, and 0 indicates no preferred galactic direction. The description and interpretation code explicitly focus on testing for alignment in a specific direction (toward galactic north), with the alternative hypothesis being an excess of positive values (alignment toward north) rather than any general deviation from zero. Therefore, the test is directional and one-tailed, focusing on the upper tail of the distribution.

Comparison To Literature

The specific statistic (low-ℓ T² dipole galactic alignment) is not widely studied. Standard dipole modulation analyses (e.g., Planck 2013, 2016) focus on full-sky or high-ℓ modulations and directional alignments with ecliptic or other axes, not this exact low-ℓ squared-field method. No directly comparable studies were identified.

import numpy as np
import healpy as hp

test_description = ("Low-ell Dipole Modulation Galactic Alignment: Decomposes the masked map "
    "into spherical harmonics, isolates ell=2-5, constructs the squared-field map T^2 from "
    "those multipoles, extracts the dipole of T^2 to find the power-modulation direction, "
    "and measures its alignment with the galactic north pole via the signed dot product "
    "n_hat . n_hat_gal. Values near +1 indicate modulation peak toward galactic north, "
    "near -1 toward galactic south, and near 0 indicate no preferred galactic alignment.")


def analyze_map(m):
    nside = hp.npix2nside(len(m))
    
    # Step 1: Fill masked pixels with 0.0 for healpy compatibility
    m_filled = np.where(np.isfinite(m), m, 0.0)
    
    # Step 2: Decompose into spherical harmonics up to ell=5
    lmax = 5
    alm = hp.map2alm(m_filled, lmax=lmax)
    
    # Step 3: Isolate low-ell range ell=2 to ell=5
    # Zero out ell=0 and ell=1 components
    alm_low = alm.copy()
    # In HEALPix triangular scheme: index = m*(2*lmax+1-m)//2 + l
    # For ell=0: index 0; for ell=1: indices 1,2 (m=0,1)
    # Set ell=0 and ell=1 to zero
    alm_low[0] = 0.0  # ell=0, m=0
    alm_low[1] = 0.0  # ell=1, m=0
    alm_low[2] = 0.0  # ell=1, m=1
    
    # Step 4: Construct the low-ell map
    m_low = hp.alm2map(alm_low, nside=nside, lmax=lmax)
    
    # Step 5: Construct squared-field map T^2 restricted to low-ell multipoles
    m_sq = m_low * m_low
    
    # Step 6: Compute spherical harmonic transform of T^2 and extract dipole (ell=1)
    alm_sq = hp.map2alm(m_sq, lmax=1)
    
    # Step 7: Extract dipole components
    # In HEALPix triangular indexing with lmax=1:
    # index 0: (l=0, m=0)
    # index 1: (l=1, m=0)
    # index 2: (l=1, m=1)
    a10 = alm_sq[1]   # (l=1, m=0) - should be real for real field
    a11 = alm_sq[2]   # (l=1, m=1) - complex
    
    # For a real field: a_{1,-1} = -a_{11}^*
    a1_neg1 = -np.conj(a11)
    
    # Step 8: Convert dipole to Cartesian direction vector
    # The spherical harmonics Y_{1m} relate to Cartesian as:
    # Y_{1,0} ~ z (cos theta)
    # Y_{1,1} and Y_{1,-1} ~ x, y (sin theta * cos/sin phi)
    # 
    # Using the standard real-valued mapping:
    # x-component: (a_{1,-1} - a_{1,1}) / sqrt(2)  [from Y_{1,-1} - Y_{1,1}]
    # y-component: i*(a_{1,1} + a_{1,-1}) / sqrt(2) [from i*(Y_{1,1} + Y_{1,-1})]
    # z-component: a_{1,0}
    #
    # Since a_{1,-1} = -a_{11}^*, let a11 = a + ib:
    # a_{1,-1} = -a + ib
    # x: ((-a+ib) - (a+ib)) / sqrt(2) = -2a/sqrt(2) = -sqrt(2)*a = -sqrt(2)*Re(a11)
    # y: i*((a+ib) + (-a+ib)) / sqrt(2) = i*(2ib)/sqrt(2) = -sqrt(2)*b = -sqrt(2)*Im(a11)
    # z: a10 (real)
    
    nx = -np.sqrt(2.0) * a11.real
    ny = -np.sqrt(2.0) * a11.imag
    nz = float(a10.real)
    
    # Step 9: Normalize to get unit vector n_hat
    norm = np.sqrt(nx*nx + ny*ny + nz*nz)
    if norm < 1e-30:
        return 0.0  # No dipole modulation detected
    
    nx /= norm
    ny /= norm
    nz /= norm
    
    # Step 10: Galactic north pole in Cartesian coordinates
    # n_hat_gal = (0, 0, 1)
    # The signed dot product is simply nz (the z-component)
    dot_product = nz  # n_hat . n_hat_gal
    
    # Clip to [-1, 1] for numerical safety
    dot_product = float(np.clip(dot_product, -1.0, 1.0))
    
    if not np.isfinite(dot_product):
        return 0.0
    
    return dot_product


def summarize_results(planck_stat, sim_results):
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    
    p_value_lower = float(np.mean(sim <= planck_stat)) if len(sim) > 0 else np.nan
    p_value_upper = float(np.mean(sim >= planck_stat)) if len(sim) > 0 else np.nan
    # Two-sided p-value
    p_value = 2.0 * min(p_value_lower, p_value_upper)
    p_value = min(p_value, 1.0)
    
    rank = int(np.sum(sim < planck_stat)) if len(sim) > 0 else 0
    percentile = 100.0 * rank / len(sim) if len(sim) > 0 else np.nan
    mean_sim = float(np.mean(sim)) if len(sim) > 0 else np.nan
    std_sim = float(np.std(sim)) if len(sim) > 0 else np.nan
    
    abs_planck = abs(planck_stat)
    abs_sim = np.abs(sim)
    p_value_abs = float(np.mean(abs_sim >= abs_planck)) if len(sim) > 0 else np.nan
    
    z_score = (planck_stat - mean_sim) / std_sim if std_sim > 0 else np.nan
    
    # Interpret alignment direction
    if planck_stat > 0.5:
        direction_label = "strong galactic north alignment"
    elif planck_stat > 0.2:
        direction_label = "moderate galactic north alignment"
    elif planck_stat < -0.5:
        direction_label = "strong galactic south alignment"
    elif planck_stat < -0.2:
        direction_label = "moderate galactic south alignment"
    elif abs(planck_stat) < 0.1:
        direction_label = "no preferred galactic direction"
    else:
        direction_label = "weak galactic alignment"
    
    # Check if the magnitude is unusual
    mean_abs = float(np.mean(abs_sim)) if len(sim) > 0 else np.nan
    std_abs = float(np.std(abs_sim)) if len(sim) > 0 else np.nan
    
    interp = (
        "Planck dipole modulation alignment with galactic north: n_hat . n_hat_gal = {:.4f}, "
        "indicating {}. This places it at the {:.1f}th percentile of simulations "
        "(p_two-sided = {:.3f}, p_|alignment| = {:.3f}). "
        "Simulated mean alignment = {:.4f} +/- {:.4f}."
    ).format(planck_stat, direction_label, percentile, p_value, p_value_abs, 
             mean_sim, std_sim)
    
    caption = (
        "Histogram of the signed dot product n_hat . n_hat_gal between the low-ell (2-5) "
        "power-modulation dipole direction and the galactic north pole. Values near +1 "
        "indicate modulation toward galactic north, near -1 toward galactic south. "
        "The red line marks the Planck SMICA value."
    )
    
    result = {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_abs": p_value_abs,
        "rank": rank,
        "n_sims": len(sim),
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "sim_mean_abs": mean_abs,
        "sim_std_abs": std_abs,
        "z_score": z_score,
        "direction_label": direction_label,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ell Dipole Modulation Galactic Alignment",
            "xlabel": "n_hat . n_hat_gal (signed alignment with galactic north)",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
    return result
