Hypothesis

The two-point correlation coherence length θ_half of the low-ℓ CMB temperature map from Planck data deviates from the predictions of the isotropic Gaussian ΛCDM model as estimated from simulations.

Method

The temperature map is band-limited to ℓ_max = 30, and the two-point correlation function C(θ) is computed in 1° bins from 0° to 180°, normalized by C(0). The statistic θ_half is the smallest angle where C(θ_half) ≤ 0.5, characterising how quickly spatial correlations decorrelate on large angular scales.

Result Summary

Planck yields θ_half = 7°, compared to a simulation mean of 8.5° ± 1.3°, corresponding to −1.11σ from the mean. The two-tailed p-value is 0.36 (percentile ≈ 0.0%). Anomaly strength is negligible.

Planck's coherence length is slightly shorter than the simulation average but well within the expected scatter. The two-point correlation function decorrelates at a rate fully consistent with isotropic Gaussian ΛCDM realisations. There is no evidence for unusually coherent large-scale patterns or unusually patchy low-ℓ structure.

Agent Justification

The test is exploratory, designed to detect anomalies in large-scale temperature correlations in either direction—unusually coherent patterns (large θ_half) or unusually patchy structures (small θ_half). The analysis code computes a flat_ direction

Comparison To Literature

The large-angle two-point correlation function has been extensively studied, beginning with Hinshaw et al. (1996) who noted a lack of correlations above ~60°, and later investigated in detail by Copi et al. (2007, 2009, 2010) and Slosar & Seljak (2004). Zhang (2012) [arXiv:1202.3469] examined directional dependence of the angular correlation function at low ℓ. While those works typically focused on the correlation function amplitude at large angles or its integrated absence, the present test uses a coherence-length metric (C(θ)/C(0) = 0.5 crossing angle), which is a different but closely related characterisation of the same underlying physics. The null result here is broadly consistent with analyses that find the low-ℓ two-point function anomaly depends on how the statistic is defined (e.g., Slosar & Seljak 2004 argued the significance is sensitive to sky cut choices).

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Two-Point Correlation Coherence Length: "
    "Band-limit the temperature map to ℓ_max = 30. Compute the two-point correlation "
    "function C(θ) for angular separations θ from 0° to 180° in 1° bins. Normalize "
    "C(θ) by C(0). The statistic is the smallest angle θ_half (in degrees) such that "
    "C(θ_half) ≤ 0.5. If no such angle exists, set θ_half = 180°. This coherence "
    "length characterizes how quickly spatial temperature correlations decay with "
    "angular separation, probing the typical angular scale of large-scale temperature "
    "structure. Anomalously large θ_half suggests unusually coherent large-scale patterns, "
    "while anomalously small θ_half indicates unusually patchy or incoherent structure."
)


def analyze_map(m):
    """Compute the coherence length θ_half of the band-limited two-point correlation function.

    Steps:
      1. Identify unmasked pixels; bail out if too few.
      2. Fill masked pixels with 0 for healpy compatibility.
      3. Compute angular power spectrum C_ℓ for ℓ = 0..30.
      4. Reconstruct normalized correlation function C(θ)/C(0) via Legendre recurrence.
      5. Find smallest θ (in degrees) where C(θ)/C(0) ≤ 0.5.

    Returns
    -------
    float
        Coherence length θ_half in degrees (1.0 to 180.0).
    """
    mask = np.isfinite(m)
    n_valid = np.sum(mask)

    if n_valid < 100:
        return 180.0

    # Fill masked pixels with 0 for healpy transforms
    m_filled = np.where(mask, m, 0.0)

    # Compute angular power spectrum up to ℓ_max = 30
    lmax = 30
    cl = hp.anafast(m_filled, lmax=lmax)

    # Coefficients: a_ℓ = (2ℓ+1)/(4π) C_ℓ
    ell = np.arange(lmax + 1, dtype=float)
    leg_coeffs = (2.0 * ell + 1.0) / (4.0 * np.pi) * cl

    # Evaluate at θ = 0°, 1°, ..., 180°
    thetas_deg = np.arange(181)
    thetas_rad = np.deg2rad(thetas_deg)
    x = np.cos(thetas_rad)  # cos(θ)

    # Use Legendre recurrence to compute C(θ) = Σ a_ℓ P_ℓ(cos θ)
    # P_0(x) = 1, P_1(x) = x
    # (ℓ+1) P_{ℓ+1}(x) = (2ℓ+1) x P_ℓ(x) - ℓ P_{ℓ-1}(x)
    c_theta = np.zeros_like(x)
    P_prev = np.ones_like(x)   # P_0
    c_theta += leg_coeffs[0] * P_prev

    if lmax >= 1:
        P_curr = x  # P_1
        c_theta += leg_coeffs[1] * P_curr
        P_prev2 = P_prev  # P_0
        P_prev = P_curr   # P_1

        for l in range(2, lmax + 1):
            P_next = ((2 * l - 1) * x * P_prev - (l - 1) * P_prev2) / l
            c_theta += leg_coeffs[l] * P_next
            P_prev2 = P_prev
            P_prev = P_next

    # Normalize by C(0) — c_theta[0] corresponds to θ=0°
    c0 = c_theta[0]
    if c0 <= 0.0:
        return 180.0

    c_norm = c_theta / c0

    # Find smallest angle θ ≥ 1° where C(θ)/C(0) ≤ 0.5
    indices = np.where(c_norm[1:] <= 0.5)[0]
    if len(indices) > 0:
        return float(indices[0] + 1)

    # No crossing found within 180°
    return 180.0


def summarize_results(planck_stat, sim_results):
    """Summarise the Low-ℓ Two-Point Correlation Coherence Length test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available for comparison.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Two-Point Correlation Coherence Length",
                "xlabel": "Coherence length θ_half (degrees)",
                "ylabel": "Number of simulations",
                "bins": 40,
                "caption": "No simulation data available.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    # Two-sided empirical p-value
    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    if std_sim > 0:
        deviation_sigma = (planck_stat - mean_sim) / std_sim
    else:
        deviation_sigma = 0.0

    if planck_stat > mean_sim + 2 * std_sim:
        direction = (
            "Planck's two-point correlation function maintains coherence to unusually "
            "large angular scales, suggesting the large-scale temperature pattern is "
            "more spatially coherent than typical ΛCDM realizations. This could reflect "
            "anomalous large-scale alignments or suppressed small-scale structure in the "
            "low-ℓ temperature field."
        )
    elif planck_stat < mean_sim - 2 * std_sim:
        direction = (
            "Planck's two-point correlation function decorrelates unusually quickly, "
            "suggesting the large-scale temperature pattern is more patchy or incoherent "
            "than typical ΛCDM realizations. This could indicate suppressed large-scale "
            "power or unusual spatial structure at low multipoles."
        )
    else:
        direction = (
            "The coherence length is consistent with isotropic Gaussian ΛCDM "
            "expectations; the two-point correlation function decays at a typical rate "
            "for a statistically isotropic temperature field."
        )

    interp = (
        f"Coherence length θ_half = {planck_stat:.0f}°. "
        f"Simulation mean = {mean_sim:.1f}° ± {std_sim:.1f}° "
        f"({deviation_sigma:+.2f}σ from mean). "
        f"Percentile = {percentile:.1f}%, two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of the coherence length θ_half — the smallest angular separation "
        "at which the normalized two-point correlation function C(θ)/C(0) drops to 0.5 "
        "or below — computed from the ℓ_max = 30 band-limited temperature map of "
        "isotropic ΛCDM simulations. Larger θ_half indicates greater spatial coherence "
        "of the large-scale temperature pattern; smaller θ_half indicates more rapid "
        "decorrelation. The red line marks the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": deviation_sigma,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Two-Point Correlation Coherence Length",
            "xlabel": "Coherence length θ_half (degrees)",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
