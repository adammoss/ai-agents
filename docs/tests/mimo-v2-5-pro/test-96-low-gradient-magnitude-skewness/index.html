Hypothesis

The skewness of the gradient magnitude of the low-ℓ CMB temperature map differs from the expected distribution under a Gaussian random field model (i.e., the null hypothesis is that the skewness matches simulations, and the alternative is that it does not).

Method

Compute the temperature gradient magnitude map from the SMICA CMB map filtered to ℓ ≤ 30 using spherical harmonic derivatives on a HEALPix grid (nside=256). The gradient magnitude |∇T| = sqrt((dT/dθ)² + ((1/sinθ)dT/dφ)²) is calculated at each pixel. The sample skewness of the gradient magnitude distribution across unmasked pixels serves as the test statistic, probing asymmetry in the intensity of spatial temperature fluctuations to distinguish Gaussian from non-Gaussian features.

Result Summary

Planck gradient magnitude skewness = 0.6501. Simulation mean ± std = 0.6443 ± 0.0697. The observed value falls at the 57th percentile of simulations, yielding z = 0.08 and a two-tailed p-value of 0.862. This is entirely consistent with Gaussian random field expectations.

Since |∇T| ≥ 0, the gradient magnitude distribution is inherently positively skewed (Rayleigh-like) even for Gaussian fields. The test correctly compares against simulations rather than assuming zero skewness. The Planck data show no discernible departure from the expected distribution—there is no evidence for excess or deficit in sharp gradient features relative to ΛCDM predictions.

Agent Justification

The test is designed to detect any departure from Gaussianity in the gradient magnitude distribution, whether positive or negative. The code and description explicitly consider both "positive excess skew" and "negative excess skew" as potential anomalies, and the summary statistics compute two-sided p-values. There is no clear directional expectation; the test is exploratory, aiming to identify non-Gaussian features in either direction. Therefore, a two-tailed test is appropriate.

Comparison To Literature

Standard CMB non-Gaussianity tests typically employ Minkowski functionals, genus statistics, or temperature-space skewness/kurtosis. Specific tests of gradient magnitude skewness at low multipoles do not appear prominently in published CMB anomaly surveys. This statistic probes a different aspect (gradient field asymmetry) than conventional approaches, though the underlying physics overlaps with existing studies of CMB topology.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Gradient Magnitude Skewness. "
    "Compute the temperature gradient magnitude map from the low-ℓ (ℓ ≤ 30) "
    "filtered SMICA map using finite differences on the HEALPix grid (nside=256). "
    "The gradient magnitude is calculated at each pixel as the Euclidean norm of "
    "the spatial gradient vector (dT/dθ, (1/sinθ)dT/dφ). Then, compute the "
    "skewness of the distribution of these gradient magnitude values across all "
    "unmasked pixels. The final scalar is the sample skewness (third moment divided "
    "by the cube of the standard deviation) of the gradient magnitude distribution. "
    "This statistic probes the asymmetry in the intensity of temperature fluctuations, "
    "potentially distinguishing between Gaussian and non-Gaussian features in the "
    "large-scale structure of the CMB."
)


def analyze_map(m):
    """Return the skewness of the gradient magnitude of the low-ℓ filtered map.

    For a band-limited function (ℓ ≤ 30), alm2map_der1 computes exact spherical
    harmonic derivatives—mathematically equivalent to finite differences but
    vectorized and numerically stable, especially near the poles.
    """
    nside = hp.npix2nside(len(m))
    lmax = 30
    valid = np.isfinite(m)

    if np.sum(valid) < 10:
        return 0.0

    # Replace masked pixels with 0 for harmonic transforms
    m_filled = np.where(valid, m, 0.0)

    # Decompose into spherical harmonics and compute first derivatives
    nside_int = int(nside)
    alm = hp.map2alm(m_filled, lmax=lmax)
    # derivs = [T, dT/dθ, (1/sinθ) dT/dφ]
    derivs = hp.alm2map_der1(alm, nside_int, lmax=lmax)

    dT_dtheta = derivs[1]       # ∂T/∂θ
    dT_dphi_sin = derivs[2]     # (1/sinθ) ∂T/∂φ

    # Gradient magnitude: |∇T| = sqrt( (∂T/∂θ)² + ((1/sinθ) ∂T/∂φ)² )
    grad_mag = np.sqrt(dT_dtheta ** 2 + dT_dphi_sin ** 2)

    # Restrict to unmasked pixels
    valid_grad = grad_mag[valid]

    if len(valid_grad) < 10:
        return 0.0

    mean_g = np.mean(valid_grad)
    var_g = np.var(valid_grad)

    if var_g < 1e-30:
        return 0.0

    skew = float(np.mean((valid_grad - mean_g) ** 3) / (var_g ** 1.5))

    if not np.isfinite(skew):
        return 0.0
    return skew


def summarize_results(planck_stat, sim_results):
    """Interpret the Low-ℓ Gradient Magnitude Skewness."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)
    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    median_s = float(np.median(sims))
    rank = int(np.sum(sims <= planck_stat))
    p_upper = float(np.mean(sims >= planck_stat))
    p_lower = float(np.mean(sims <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else 0.0
    pct_rank = rank / n * 100 if n > 0 else 50.0

    if pct_rank < 2.5 or pct_rank > 97.5:
        sig_desc = "highly anomalous (<2.5% or >97.5% of sims)"
    elif pct_rank < 5 or pct_rank > 95:
        sig_desc = "marginally anomalous (<5% or >95% of sims)"
    elif pct_rank < 25 or pct_rank > 75:
        sig_desc = "noteworthy (outside IQR of sims)"
    else:
        sig_desc = "typical (within IQR of sims)"

    if planck_stat > mean_s + std_s:
        grad_desc = (
            "positive excess skew: the gradient magnitude distribution has a heavier "
            "right tail than simulations predict, suggesting localized sharp features "
            "(hot/cold spot boundaries) are more prominent in the Planck data than "
            "expected from a Gaussian random field"
        )
    elif planck_stat < mean_s - std_s:
        grad_desc = (
            "negative excess skew: the gradient magnitude distribution is more "
            "compact/symmetric than simulations predict, suggesting the low-ℓ "
            "temperature field has fewer or less prominent sharp features than "
            "expected from a Gaussian random field"
        )
    else:
        grad_desc = (
            "gradient magnitude skewness is consistent with Gaussian random field "
            "expectations (symmetric gradient intensity distribution)"
        )

    # Gradient magnitude is inherently ≥ 0, so even for a Gaussian field the
    # skewness will be positive (Rayleigh-like tail). The test compares against
    # simulations rather than a fixed null value.

    return {
        "planck_stat": planck_stat,
        "gradient_mag_skewness": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "sim_median": median_s,
        "z_score": z_score,
        "rank": rank,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "percentile_rank": pct_rank,
        "gradient_asymmetry": grad_desc,
        "significance_description": sig_desc,
        "interpretation": (
            f"Gradient magnitude skewness = {planck_stat:.4f}. "
            f"Since |∇T| ≥ 0, the skewness is inherently positive even for Gaussian fields "
            f"(Rayleigh-like distribution). The test compares against simulations to detect "
            f"departures from Gaussianity. "
            f"{grad_desc.capitalize()} "
            f"Sim mean ± std = {mean_s:.4f} ± {std_s:.4f}. "
            f"{sig_desc.capitalize()}. z = {z_score:.2f}, two-sided p = {p_two:.4f}."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Gradient Magnitude Skewness",
            "xlabel": "Skewness of |∇T| (dimensionless)",
            "ylabel": "Count",
