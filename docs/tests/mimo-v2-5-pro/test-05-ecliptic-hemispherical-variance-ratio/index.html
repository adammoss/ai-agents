Hypothesis

The CMB temperature map exhibits a higher variance in the northern ecliptic hemisphere compared to the southern ecliptic hemisphere (i.e., the north/south variance ratio is greater than 1).

Method

The variance of the masked Planck CMB temperature map is computed separately for the northern ecliptic hemisphere (latitude > 0°) and the southern ecliptic hemisphere (latitude < 0°), after rotating pixel coordinates from Galactic to Ecliptic. The statistic is the ratio of northern-to-southern variance. Under isotropy, the expected ratio is ~1.0.

Result Summary

The observed Planck variance ratio is 1.0019, compared to a simulation mean of 1.0047 and standard deviation of 0.0321. The z-score is only 0.065σ from the simulation mean. The one-sided upper-tail p-value is 0.526, indicating the Planck value is well within the distribution of isotropic ΛCDM simulations.

The ecliptic hemispherical variance ratio is entirely consistent with isotropic expectations derived from 1000 lensed-ΛCDM simulations. No evidence for a north–south ecliptic power asymmetry is found using this particular real-space variance statistic.

Agent Justification

The analysis code explicitly computes the p-value as the fraction of simulations with a variance ratio greater than or equal to the observed Planck value (`rank = int(np.sum(valid >= planck_stat))`), which is an upper-tail test. This aligns with the known ecliptic hemispherical power asymmetry in Planck data, where the northern ecliptic hemisphere shows more power than the southern. The test statistic is directional (ratio > 1), so a one-tailed test is appropriate.

Comparison To Literature

The ecliptic hemispherical power asymmetry is a well-documented anomaly, studied using various statistics in Planck Collaboration XXIII (2013) and subsequent work (e.g., arXiv:1608.02234). Some analyses report anomalously low variance in the northern ecliptic hemisphere, contrary to the tested hypothesis of higher northern variance. This null result with a simple variance ratio is broadly consistent with studies finding the asymmetry to be statistic-dependent and not robustly captured by pixel-space variance alone.

import numpy as np
import healpy as hp

test_description = ("Ecliptic Hemispherical Variance Ratio. Computes the ratio of CMB "
    "temperature variance in the northern ecliptic hemisphere (ecliptic latitude > 0°) "
    "to the southern ecliptic hemisphere (ecliptic latitude < 0°). For a random isotropic "
    "field the expected ratio is ~1; significant deviation from unity indicates an "
    "ecliptic hemispherical power asymmetry, as reported in Planck data.")

def analyze_map(m):
    m = np.asarray(m, dtype=np.float64)
    nside = hp.npix2nside(len(m))
    npix = hp.nside2npix(nside)
    pix = np.arange(npix)

    # Get pixel unit vectors in Galactic coordinates (default for healpy)
    vec = hp.pix2vec(nside, pix)

    # Rotate from Galactic to Ecliptic coordinates
    rot = hp.Rotator(coord=['G', 'E'])
    vec_ecl = rot(vec)

    # Convert ecliptic unit vectors to (colatitude, longitude)
    theta_ecl, phi_ecl = hp.vec2ang(vec_ecl)

    # Ecliptic latitude = 90° - colatitude
    ecl_lat = 90.0 - np.degrees(theta_ecl)

    # Northern ecliptic hemisphere: ecliptic latitude > 0°
    north_mask = ecl_lat > 0.0
    # Southern ecliptic hemisphere: ecliptic latitude < 0°
    south_mask = ecl_lat < 0.0

    # Combine with valid (finite / unmasked) pixel mask
    valid = np.isfinite(m)
    north_vals = m[valid & north_mask]
    south_vals = m[valid & south_mask]

    # Compute variance in each hemisphere
    north_var = np.var(north_vals) if len(north_vals) > 1 else 0.0
    south_var = np.var(south_vals) if len(south_vals) > 1 else 0.0

    # Compute the ratio; fall back to 1.0 (isotropic expectation)
    # if the denominator is zero.
    if south_var > 0.0:
        ratio = north_var / south_var
    else:
        ratio = 1.0

    return float(ratio)


def summarize_results(planck_stat, sim_results):
    sim_arr = np.array(sim_results, dtype=np.float64)
    finite_mask = np.isfinite(sim_arr)
    n_finite = int(finite_mask.sum())

    if n_finite > 0:
        valid = sim_arr[finite_mask]
        rank = int(np.sum(valid >= planck_stat))
        p_value = rank / n_finite
    else:
        rank = 0
        p_value = float('nan')

    sim_median = float(np.nanmedian(sim_arr))
    sim_mean = float(np.nanmean(sim_arr))
    sim_std = float(np.nanstd(sim_arr))

    interpretation = (
        "Anomalous: ecliptic hemispherical variance asymmetry detected"
        if p_value < 0.05
        else "Ecliptic hemispherical variance ratio consistent with isotropic expectations"
    )

    result = {
        "test_description": test_description,
        "planck_stat": float(planck_stat),
        "p_value": float(p_value),
        "rank": rank,
        "n_simulations": len(sim_results),
        "n_valid": n_finite,
        "sim_mean": sim_mean,
        "sim_std": sim_std,
        "sim_median": sim_median,
        "exceeds_95th_percentile": bool(p_value < 0.05),
        "interpretation": interpretation,
        "expected_random_ratio": float(1.0),
    }

    result["plot_spec"] = {
        "kind": "histogram",
        "title": "Ecliptic Hemispherical Variance Ratio",
        "xlabel": "North / South ecliptic hemisphere variance ratio",
        "ylabel": "Number of simulations",
        "bins": 50,
        "caption": (
            f"Histogram of ecliptic hemispherical variance ratio from {n_finite} "
            f"lensed-ΛCDM simulations. Planck value = {planck_stat:.4f} "
            f"(one-sided p = {p_value:.3f}, rank {rank}/{n_finite}). "
            f"The expected ratio for isotropic random fields is ~1.0; "
            f"a value significantly different from unity indicates an ecliptic "
            f"hemispherical power asymmetry."
        ),
    }

    return result
