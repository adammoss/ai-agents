Hypothesis

The Shannon entropy of the 8-bit LBP code distribution in the low-ℓ (ℓ=2–32) CMB temperature map deviates from the mean expected under standard ΛCDM simulations, indicating anomalous local texture diversity.

Method

The masked Planck temperature map is band-pass filtered to multipoles 2 ≤ ℓ ≤ 32 and downgraded to nside=32. For each unmasked pixel, an 8-bit Local Binary Pattern (LBP) code is computed from the 8 HEALPix neighbors. A normalized histogram of 256 possible codes is constructed, and its Shannon entropy H = −Σ pᵢ log₂(pᵢ) serves as the anomaly statistic, quantifying diversity and uniformity of local texture patterns in the low-ℓ CMB field.

Result Summary

Planck LBP entropy = 5.796 bits (72.4% of the theoretical maximum 8.0). Simulation mean = 5.745 ± 0.028 bits. The Planck value falls at the 95.8th percentile, yielding z ≈ 1.82σ and a two-tailed p-value of 0.084. Anomaly strength is classified as moderate.

The low-ℓ Planck map shows moderately elevated LBP texture diversity compared to ΛCDM simulations, suggesting slightly richer local pattern configurations than expected. However, the deviation does not reach conventional statistical significance (2σ / p < 0.05). This could hint at subtle large-scale spatial organization (e.g., hemispherical asymmetry or localized features) but is consistent with Gaussian fluctuations at marginal levels.

Agent Justification

The test description and implementation do not specify a directional expectation (e.g., predicting only higher or only lower entropy). The anomaly statistic measures "diversity and uniformity," which could deviate in either direction—excess entropy (unusually diverse textures) or deficit entropy (unusually homogeneous textures). The summarize_results function explicitly computes both upper and lower p-values and combines them into a two-sided p-value, confirming the two-tailed nature. Since no prior physical or observational evidence points specifically to one direction, a two-tailed test is appropriate for general anomaly detection.

Comparison To Literature

No prior studies applying Local Binary Pattern texture analysis or Shannon entropy of LBP code distributions to CMB anomaly testing were identified. LBP is a well-established tool in computer vision and texture analysis but its use in CMB statistical isotropy studies appears unexplored in the literature.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Local Texture Diversity Entropy: "
    "Filter the masked temperature map to retain only multipoles 2 ≤ ℓ ≤ 32. "
    "On this band-limited map, compute the Local Binary Pattern (LBP) code for "
    "each unmasked pixel using a radius of 1.5 pixels (3×3 neighborhood). "
    "Construct a normalized histogram of the 8-bit LBP codes (256 bins) across "
    "all unmasked pixels. The anomaly statistic is the Shannon entropy of this "
    "histogram, H = -Σ p_i log2(p_i), where p_i is the probability of the "
    "i-th LBP code. This single scalar measures the diversity and uniformity "
    "of local texture patterns in the low-ℓ CMB map."
)


def analyze_map(m):
    """Low-ℓ Local Texture Diversity Entropy.

    Band-pass filter the CMB temperature map to ℓ=2–32, downgrade to nside=32
    (band-limit means all spatial info is captured), compute 8-bit LBP codes
    for each unmasked pixel using the 8 HEALPix neighbors, then return the
    Shannon entropy of the LBP code histogram.
    """
    nside = hp.get_nside(m)
    mask = np.isfinite(m)
    n_valid = int(np.sum(mask))
    if n_valid < 100:
        return 0.0

    # Step 1: Band-pass filter to ℓ = 2–32 at original nside
    clean = np.where(mask, m, 0.0)
    lmax = 32
    alm = hp.map2alm(clean, lmax=lmax)
    # Zero monopole (ℓ=0) and dipole (ℓ=1)
    alm[0] = 0.0
    alm[1] = 0.0
    alm[2] = 0.0
    filtered = hp.alm2map(alm, nside, lmax=lmax, verbose=False)

    # Step 2: Downgrade to nside=32 for LBP computation
    # Band-limit ℓ≤32 means nside=32 (Nyquist ℓ≈96) captures all info
    tns = 32
    tnpix = 12 * tns ** 2

    filt_filled = np.where(mask, filtered, 0.0)
    filt_ds = hp.ud_grade(filt_filled, tns)
    wt_ds = hp.ud_grade(mask.astype(np.float64), tns)

    # Only keep coarse pixels where majority of fine pixels were valid
    ok = wt_ds > 0.5
    n_ok = int(np.sum(ok))
    if n_ok < 50:
        return 0.0

    corrected = np.full(tnpix, np.nan)
    corrected[ok] = filt_ds[ok] / wt_ds[ok]

    # Step 3: Compute LBP codes using HEALPix 8-neighbor topology
    all_neigh = hp.get_all_neighbours(tns, np.arange(tnpix))

    vpix = np.where(ok)[0]
    temps = corrected[vpix]

    # Map global pixel index → local index for valid pixels
    p2i = np.full(tnpix, -1, dtype=np.int32)
    p2i[vpix] = np.arange(n_ok, dtype=np.int32)

    # Build LBP codes vectorized over all valid pixels
    lbp_codes = np.zeros(n_ok, dtype=np.int32)
    for bit in range(8):
        neigh_global = all_neigh[bit, vpix]  # neighbor pixel indices
        # Check if neighbor exists and is valid
        valid = (neigh_global >= 0) & ok[neigh_global]
        # Default: comparison is False (0) for invalid neighbors
        # Compare neighbor value >= center value
        # Use -inf for invalid neighbors so they never satisfy >= condition
        neigh_vals = np.where(valid, corrected[neigh_global], -np.inf)
        bit_set = neigh_vals >= temps
        lbp_codes[bit_set] += (1 << bit)

    # Step 4: Build histogram of LBP codes (256 bins for 8-bit)
    code_counts = np.bincount(lbp_codes, minlength=256).astype(np.float64)
    total = code_counts.sum()
    if total == 0:
        return 0.0

    probs = code_counts / total

    # Step 5: Compute Shannon entropy H = -Σ p_i log2(p_i)
    # Only consider bins with non-zero probability
    nz = probs > 0
    entropy = -np.sum(probs[nz] * np.log2(probs[nz]))

    return float(entropy) if np.isfinite(entropy) else 0.0


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ℓ Local Texture Diversity Entropy test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ LBP Texture Entropy",
                "xlabel": "Shannon Entropy (bits)",
                "ylabel": "Count",
                "bins": 40,
                "caption": "No simulation data.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))
    deviation_sigma = (planck_stat - mean_sim) / std_sim if std_sim > 0 else 0.0

    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    max_entropy = 8.0  # max possible for 256-bin distribution
    uniformity_pct = 100.0 * planck_stat / max_entropy if max_entropy > 0 else 0.0

    if planck_stat > mean_sim + 2 * std_sim:
        direction = (
            "Planck shows significantly higher LBP texture entropy than LCDM "
            "simulations, indicating unusually diverse local texture patterns "
            "in the low-ℓ field — possibly reflecting non-standard spatial "
            "organization such as hemispherical asymmetry, localized cold/hot "
            "spots, or anisotropic features that create a richer variety of "
            "local intensity configurations."
        )
    elif planck_stat < mean_sim - 2 * std_sim:
        direction = (
            "Planck shows significantly lower LBP texture entropy than LCDM "
            "simulations, indicating unusually homogeneous local texture "
            "patterns — the low-ℓ field may have fewer distinct local "
            "intensity configurations than expected, suggesting large-scale "
            "coherence or reduced small-scale structure."
        )
    else:
        direction = (
            "The LBP texture entropy is consistent with Gaussian LCDM "
            "expectations, indicating standard diversity of local texture "
            "patterns in the low-ℓ CMB field."
        )

    interp = (
        f"LBP texture entropy = {planck_stat:.4f} bits "
        f"(max possible = {max_entropy:.1f}, uniformity = {uniformity_pct:.1f}%). "
        f"Simulation mean = {mean_sim:.4f} ± {std_sim:.4f} "
        f"({deviation_sigma:+.2f}σ). Percentile = {percentile:.1f}%, "
        f"two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of the Shannon entropy of the 8-bit LBP code distribution "
        "from the low-ℓ (ℓ=2–32) CMB temperature field. Higher entropy "
        "indicates more diverse local texture patterns (codes are more "
        "uniformly distributed), while lower entropy indicates a few "
        "dominant texture patterns. The maximum possible entropy is 8 bits "
        "(all 256 codes equally likely). Red line marks the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": deviation_sigma,
        "max_entropy": max_entropy,
        "uniformity_pct": uniformity_pct,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ LBP Texture Diversity Entropy",
            "xlabel": "Shannon Entropy of LBP Code Histogram (bits)",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
