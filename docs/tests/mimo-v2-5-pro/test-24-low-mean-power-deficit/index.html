Hypothesis

The CMB temperature map exhibits a significant deficit in power at large scales (ℓ = 2 to 10) compared to ΛCDM predictions, as indicated by a negative weighted mean normalized deviation D.

Method

Computes the (2ℓ+1)-weighted mean normalized deviation D between the observed pseudo-Cℓ power spectrum and the Planck 2018 best-fit ΛCDM theoretical spectrum across multipoles ℓ = 2–10. A significantly negative D would indicate large-scale power suppression relative to ΛCDM expectations.

Result Summary

The Planck SMICA map yields D = −0.7737, compared to a simulation mean of −0.7238 ± 0.0419 (1σ). The observed value falls at the 10.4th percentile of the ΛCDM simulation distribution, corresponding to z = −1.26σ. The one-tailed (lower) p-value is 0.104, and the two-sided p-value is 0.208. Anomaly strength is classified as negligible.

While the Planck map shows a mildly negative D consistent with a slight power deficit at low multipoles, the deviation is well within the expected ΛCDM fluctuation range. At ~1.3σ below the simulation mean, this does not reach statistical significance. The result is consistent with isotropic ΛCDM expectations.

Agent Justification

The test is explicitly designed to detect a "power deficit" anomaly, and the description highlights that a significantly negative D indicates large-scale power suppression. This directional expectation (deficit, not excess) makes a one-tailed lower test appropriate for testing the hypothesis of suppressed large-scale power.

Comparison To Literature

The low-ℓ power deficit is one of the longest-standing CMB anomalies, first noted with COBE and extensively studied with WMAP and Planck (e.g., Planck Collaboration 2020, Paper VII; Copi et al. 2010). Previous works have reported deficits in the quadrupole and octupole, as well correlations like the quadrupole–octupole alignment and the "axis of evil." The present test's measured D is consistent with prior reports showing mild but not highly significant suppression, and is broadly consistent with the ~10% tension level reported in comprehensive Planck anomaly analyses.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Mean Power Deficit: Compute the pseudo-Cℓ power spectrum from "
    "the processed temperature map for multipoles ℓ = 2 to 10. Compare to "
    "a reference ΛCDM theoretical spectrum (Planck 2018 best-fit). For each "
    "ℓ, compute the normalized deviation dℓ = (Cℓ − Cℓ^ref) / Cℓ^ref, then "
    "compute the weighted mean D = Σ(2ℓ+1)·dℓ / Σ(2ℓ+1) over ℓ = 2 to 10. "
    "A significantly negative D indicates large-scale power suppression "
    "relative to ΛCDM expectations."
)


def analyze_map(m):
    """Compute the weighted mean normalized power deficit D at ℓ=2–10.

    Compares the pseudo-Cℓ of the (masked) map to the Planck 2018
    best-fit ΛCDM TT power spectrum and returns the (2ℓ+1)-weighted
    mean of normalized deviations as a single scalar.
    """

    # Replace masked (NaN) pixels with 0.0 for healpy compatibility
    m_filled = np.where(np.isfinite(m), m, 0.0)

    # Compute pseudo-Cℓ power spectrum up to ℓ=10
    lmax = 10
    cl = hp.anafast(m_filled, lmax=lmax)

    # Reference ΛCDM D_ℓ values from Planck 2018 best-fit (TT, μK²)
    # D_ℓ = ℓ(ℓ+1)C_ℓ/(2π)
    # Source: Planck 2018 best-fit base_plikHM_TTTEEE_lowl_lowE_lensing
    dl_ref = np.array([
        0.0,     # ℓ=0 (unused)
        0.0,     # ℓ=1 (unused, monopole/dipole removed)
        1760.0,  # ℓ=2
        2350.0,  # ℓ=3
        2570.0,  # ℓ=4
        2510.0,  # ℓ=5
        2630.0,  # ℓ=6
        2740.0,  # ℓ=7
        2690.0,  # ℓ=8
        2630.0,  # ℓ=9
        2600.0,  # ℓ=10
    ])

    # Convert D_ℓ to C_ℓ: C_ℓ = D_ℓ * 2π / (ℓ(ℓ+1))
    ell = np.arange(lmax + 1, dtype=float)
    ell[0] = 1.0  # avoid division by zero; cl_ref[0] set explicitly below
    cl_ref = dl_ref * 2.0 * np.pi / (ell * (ell + 1))
    cl_ref[0] = 0.0
    cl_ref[1] = 0.0

    # Compute (2ℓ+1)-weighted mean of normalized deviations for ℓ = 2 to 10
    ell_idx = np.arange(2, lmax + 1)
    weights = 2.0 * ell_idx + 1.0
    d_ell = (cl[ell_idx] - cl_ref[ell_idx]) / cl_ref[ell_idx]

    D = float(np.sum(weights * d_ell) / np.sum(weights))
    return D


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ℓ Mean Power Deficit test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results to compare.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Mean Power Deficit",
                "xlabel": "Weighted mean normalized deviation D",
                "ylabel": "Number of simulations",
                "bins": 40,
                "caption": "No simulation data available.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    interp = (
        f"Weighted mean normalized deviation D = {planck_stat:.4f}. "
        f"Simulation mean = {mean_sim:.4f} \u00b1 {std_sim:.4f}. "
        f"Percentile = {percentile:.1f}%, two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. "
    )

    if planck_stat < mean_sim - 2 * std_sim:
        interp += (
            "Planck shows a significantly negative D, indicating a deficit "
            "of large-scale power (\u2113=2\u201310) compared to \u039bCDM "
            "expectations. This is consistent with the well-known low-\u2113 "
            "CMB power deficit anomaly."
        )
    elif planck_stat > mean_sim + 2 * std_sim:
        interp += (
            "Planck shows a significantly positive D, indicating excess "
            "large-scale power compared to \u039bCDM expectations."
        )
    else:
        interp += (
            "The statistic is consistent with isotropic \u039bCDM expectations, "
            "showing no significant low-\u2113 power deficit."
        )

    caption = (
        "Histogram of the weighted mean normalized power deviation D at "
        "\u2113=2\u201310 from \u039bCDM simulations. D compares the "
        "pseudo-C\u2113 spectrum to the Planck 2018 best-fit \u039bCDM "
        "theoretical spectrum with (2\u2113+1) weighting. Under \u039bCDM, "
        "D fluctuates around a value set by masking and cosmic variance; "
        "a significantly more negative Planck value indicates large-scale "
        "power suppression. The red line marks the Planck SMICA value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-\u2113 Mean Power Deficit",
            "xlabel": "Weighted mean normalized deviation D",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
