Hypothesis

The coefficient of variation of the low-ℓ gradient magnitude in the Planck CMB map is consistent with the distribution predicted by the standard ΛCDM model (no anomaly in gradient variability).

Method

The gradient magnitude of the CMB temperature field is computed via spherical harmonic derivatives up to ℓ_max=10. For each unmasked pixel, the gradient magnitude |∇T| = √((dT/dθ)² + (1/sinθ · dT/dφ)²) is calculated. The coefficient of variation (CoV = std/mean of |∇T|) is then extracted as a single scalar probing the relative spatial variability of low-ℓ temperature gradients, sensitive to anisotropic or non-Gaussian structures.

Result Summary

Planck CoV = 0.515; simulation mean ± std = 0.528 ± 0.028. The Planck value corresponds to z = −0.44 and a two-sided p-value of 0.702, placing it at the 35th percentile of 1000 ΛCDM realizations—well within the interquartile range.

The gradient variability in Planck data is entirely typical of ΛCDM expectations. No evidence for unusually high variability (which would suggest anisotropic/non-Gaussian structure) or unusually low variability (suggesting an overly uniform gradient field) is found. The CoV lies comfortably within one standard deviation of the simulation mean.

Agent Justification

The test is designed to detect deviations from the ΛCDM expectation in either direction—either unusually high variability (which could indicate anisotropic or non-Gaussian structures) or unusually low variability (which could indicate an overly uniform gradient field). The analysis code explicitly computes both p-values and a two-sided p-value, and the interpretation logic in `summarize_results` checks for both higher and lower variability relative to simulations. Since there is no strong prior prediction for a specific direction of deviation, a two-tailed test is appropriate.

Comparison To Literature

No prior studies using gradient magnitude coefficient of variation as a CMB anomaly statistic were identified in available literature. Related low-ℓ anomaly work—such as multipole vector analyses (Copi et al. 2004) and bipolar power spectrum tests (Ghosh et al. 2007)—addresses statistical isotropy through entirely different statistics. This gradient-based approach probes complementary spatial structure but has no direct published precedent.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Gradient Magnitude Coefficient of Variation. "
    "Compute the gradient magnitude map of the temperature anisotropy field "
    "using spherical harmonic derivatives up to ℓ_max=10. For each pixel, "
    "calculate the magnitude of the gradient vector sqrt((dT/dθ)² + (1/sinθ dT/dφ)²). "
    "Then compute the coefficient of variation: the standard deviation of gradient "
    "magnitudes divided by the mean of gradient magnitudes across the masked sky. "
    "This single scalar measures the relative variability of temperature gradients, "
    "which can distinguish between Gaussian isotropic fluctuations and anisotropic "
    "or non-Gaussian structures."
)


def analyze_map(m):
    """Return the coefficient of variation of the gradient magnitude at low ℓ."""
    nside = hp.npix2nside(len(m))
    lmax = 10

    # Replace masked pixels with 0.0 for spherical harmonic transforms
    valid = np.isfinite(m)
    m_filled = np.where(valid, m, 0.0)

    # Forward transform to get a_{ℓm}
    alm = hp.map2alm(m_filled, lmax=lmax)

    # Compute gradient maps using healpy's alm2map_der1
    # Returns (T_map, dT_dtheta, dT_dphi_over_sintheta)
    _, dT_dtheta, dT_dphi_sintheta = hp.alm2map_der1(alm, nside, lmax=lmax)

    # Compute gradient magnitude: sqrt((dT/dθ)² + (1/sinθ dT/dφ)²)
    grad_mag = np.sqrt(dT_dtheta**2 + dT_dphi_sintheta**2)

    # Extract valid (unmasked) pixel values
    grad_valid = grad_mag[valid]

    n = len(grad_valid)
    if n < 2:
        return 0.0

    # Compute coefficient of variation
    mean_grad = np.mean(grad_valid)
    if mean_grad <= 0 or not np.isfinite(mean_grad):
        return 0.0

    std_grad = np.std(grad_valid, ddof=1)
    if not np.isfinite(std_grad):
        return 0.0

    cv = std_grad / mean_grad

    if not np.isfinite(cv):
        return 0.0

    return float(cv)


def summarize_results(planck_stat, sim_results):
    """Interpret the Low-ℓ Gradient Magnitude Coefficient of Variation."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)
    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    median_s = float(np.median(sims))
    rank = int(np.sum(sims <= planck_stat))
    p_upper = float(np.mean(sims >= planck_stat))
    p_lower = float(np.mean(sims <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else 0.0
    pct_rank = rank / n * 100 if n > 0 else 50.0

    if pct_rank < 2.5 or pct_rank > 97.5:
        sig_desc = "highly anomalous (<2.5% or >97.5% of sims)"
    elif pct_rank < 5 or pct_rank > 95:
        sig_desc = "marginally anomalous (<5% or >95% of sims)"
    elif pct_rank < 25 or pct_rank > 75:
        sig_desc = "noteworthy (outside IQR of sims)"
    else:
        sig_desc = "typical (within IQR of sims)"

    if planck_stat > mean_s + std_s:
        cv_desc = "higher gradient variability than typical LCDM (possible anisotropic structures)"
    elif planck_stat < mean_s - std_s:
        cv_desc = "lower gradient variability than typical LCDM (unusually uniform gradient field)"
    else:
        cv_desc = "gradient variability consistent with LCDM expectations"

    return {
        "planck_stat": planck_stat,
        "coefficient_of_variation": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "sim_median": median_s,
        "z_score": z_score,
        "rank": rank,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "percentile_rank": pct_rank,
        "cv_description": cv_desc,
        "significance_description": sig_desc,
        "interpretation": (
            f"Gradient magnitude CoV = {planck_stat:.6f}. {cv_desc}. "
            f"Sim mean ± std = {mean_s:.6f} ± {std_s:.6f}. "
            f"{sig_desc}. z = {z_score:.2f}, two-sided p = {p_two:.4f}."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Gradient Magnitude Coefficient of Variation",
            "xlabel": "Coefficient of Variation (std/mean of |∇T|)",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck CoV = {planck_stat:.6f}. "
                f"Sim mean ± std = {mean_s:.6f} ± {std_s:.6f}. "
                f"z = {z_score:.2f}, P(two-sided) = {p_two:.4f}."
            ),
        },
