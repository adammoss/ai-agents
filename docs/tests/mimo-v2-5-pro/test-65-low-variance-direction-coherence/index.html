Hypothesis

The low-ℓ variance direction coherence (mean resultant length of dipole axes across multipole scales) is greater than expected from isotropic Gaussian fluctuations.

Method

For each multipole ℓ=2 through 20, a local variance map is computed using a 30° Gaussian beam-weighted aperture, and a dipole is fitted to extract the preferred axis of maximal variance. The 19 resulting dipole directions are combined into a mean resultant vector; its magnitude (normalized by bin count) measures how consistently variance directions align across multipole scales. Values near 1 indicate coherent alignment; values near 0 indicate random orientations (expected ~0.23 for 19 random directions).

Result Summary

Planck statistic = 0.3195. Simulation mean ± σ = 0.3243 ± 0.1363. Z-score = −0.03. One-tailed upper p-value = 0.487 (49% of simulations exceed the Planck value). The observed coherence is essentially at the simulation mean, well within the 1σ expectation from isotropic Gaussian skies.

The low-ℓ variance dipole directions across multipole bins show no evidence of preferential alignment in the Planck data. The result is entirely consistent with random orientation as expected under isotropy, with no deviation from the null distribution.

Agent Justification

The test statistic is specifically designed to detect consistent alignment of variance directions, with a high value (near 1) indicating alignment and being anomalous. The code's interpretation states a high value is anomalous, and the expected random value (~0.23) serves as the null expectation. There is no indication that low values are anomalous, so only upper-tail departures are of interest.

Comparison To Literature

Low-ℓ alignment anomalies (e.g., quadrupole–octupole alignment, the "axis of evil") have been extensively studied in CMB literature. However, this specific statistic—testing coherence of variance-map dipole directions rather than the multipole moments themselves—did not appear in the search results. It addresses a related but distinct observable, and no directly comparable prior analysis was identified.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Variance-Direction Coherence: "
    "For each multipole ℓ=2 through 20, compute a local variance map using a "
    "30° Gaussian beam-weighted aperture, fit a dipole to extract the preferred "
    "axis of maximal variance, and collect the 19 dipole directions. "
    "The statistic is the magnitude of the mean resultant vector of these 19 "
    "unit vectors, measuring how consistently the variance direction aligns "
    "across multipole scales."
)


def analyze_map(m):
    """Return mean resultant length of per-ℓ variance-map dipole directions."""
    lmax = 20
    nside_target = 32  # sufficient for lmax=20

    finite_mask = np.isfinite(m)
    if not np.any(finite_mask):
        return 0.0

    m_clean = np.where(finite_mask, m, 0.0)
    nside_orig = hp.npix2nside(len(m))
    nside_work = min(nside_orig, nside_target)

    # Downgrade map and mask to working resolution for speed
    if nside_orig > nside_work:
        m_work = hp.ud_grade(m_clean, nside_work)
        mask_low = hp.ud_grade(finite_mask.astype(float), nside_work)
        valid = mask_low > 0.5
    else:
        m_work = m_clean
        valid = finite_mask.copy()

    # Spherical harmonic coefficients at working resolution
    alm = hp.map2alm(m_work, lmax=lmax)

    valid_idx = np.where(valid)[0]
    n_valid = len(valid_idx)
    if n_valid < 4:
        return 0.0

    # 30° FWHM Gaussian beam window function
    sigma_beam = np.radians(30.0) / (2.0 * np.sqrt(2.0 * np.log(2.0)))
    ell_arr = np.arange(lmax + 1, dtype=float)
    beam = np.exp(-ell_arr * (ell_arr + 1.0) * sigma_beam ** 2 / 2.0)

    # Dipole design matrix [monopole, x, y, z] at valid pixels
    theta, phi = hp.pix2ang(nside_work, valid_idx)
    A = np.column_stack([
        np.ones(n_valid),
        np.sin(theta) * np.cos(phi),
        np.sin(theta) * np.sin(phi),
        np.cos(theta),
    ])
    pinv_A = np.linalg.pinv(A)  # (4, n_valid)

    directions = []

    for ell in range(2, 21):
        # Extract single-ℓ alm
        alm_ell = np.zeros_like(alm)
        idx = hp.Alm.getidx(lmax, ell, np.arange(ell + 1))
        alm_ell[idx] = alm[idx]

        # Reconstruct single-ℓ map
        T = hp.alm2map(alm_ell, nside_work, lmax=lmax, verbose=False)

        # Smoothed T: for single-ℓ, just scale by beam window
        T_smooth = beam[ell] * T

        # Smoothed T²: transform T² to alm, apply beam, transform back
        T_sq = T ** 2
        alm_sq = hp.map2alm(T_sq, lmax=lmax)
        T_sq_smooth = hp.alm2map(
            hp.almxfl(alm_sq, beam), nside_work, lmax=lmax, verbose=False
        )

        # Local variance map
        var_map = T_sq_smooth - T_smooth ** 2

        # Fit dipole via least-squares at valid pixels
        coeffs = pinv_A @ var_map[valid_idx]
        dipole = coeffs[1:4]
        norm = np.linalg.norm(dipole)

        if norm > 1e-30:
            directions.append(dipole / norm)
        else:
            directions.append(np.array([1.0, 0.0, 0.0]))

    # Mean resultant vector length = magnitude of average unit vector
    directions = np.array(directions)  # (19, 3)
    mean_vec = np.mean(directions, axis=0)
    return float(np.linalg.norm(mean_vec))


def summarize_results(planck_stat, sim_results):
    """Interpret variance-direction coherence relative to simulation ensemble."""
    sims = np.array(sim_results, dtype=float)
    sims_finite = sims[np.isfinite(sims)]
    n_sims = len(sims_finite)

    if n_sims == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims_finite))
    std_s = float(np.std(sims_finite, ddof=1)) if n_sims > 1 else 0.0
    p_value = float(np.mean(sims_finite >= planck_stat))
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else np.nan

    return {
        "planck_stat": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "z_score": z_score,
        "p_value": p_value,
        "n_sims": n_sims,
        "interpretation": (
            "A high value indicates that the direction of maximal low-ℓ power "
            "variance is consistently aligned across multipole scales, which "
            "would be anomalous for isotropic Gaussian fluctuations."
        ),
        "expected_random": "~0.23 for 19 random directions (1/sqrt(19))",
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Variance-Direction Coherence",
            "xlabel": "Mean resultant length of dipole axes",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck stat = {planck_stat:.4f}. "
                f"Sim mean ± σ = {mean_s:.4f} ± {std_s:.4f}. "
                f"z = {z_score:.2f}, p = {p_value:.3f} "
                f"(fraction of sims ≥ Planck). "
                f"Values near 1 indicate aligned variance directions; "
                f"values near 0 indicate random orientations."
            ),
        },
    }
