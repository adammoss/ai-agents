Hypothesis

The observed excess kurtosis κ of the low-ℓ (ℓ ≤ 5) masked-sky temperature distribution from the Planck SMICA map is inconsistent with the distribution of κ from ΛCDM simulations, indicating an anomaly in large-scale non-Gaussianity.

Method

The low-ℓ temperature map (ℓ ≤ 5) is reconstructed from SMICA spherical harmonic coefficients, the Planck common mask is applied, and the excess kurtosis κ of the standardized unmasked pixel temperatures is computed. This statistic probes the heaviness of tails in the large-scale temperature distribution on the masked sky. The observed κ is compared to the distribution from 1000 ΛCDM simulations processed identically.

Result Summary

The observed excess kurtosis is κ = −0.483. The ΛCDM simulation mean is −0.182 ± 0.448, giving a z-score of 0.63σ and a two-tailed p-value of 0.528. The Planck value lies comfortably within the simulation envelope, showing no significant departure from Gaussian ΛCDM expectations.

The low-ℓ masked-sky temperature distribution has marginally lighter tails than the median simulation but is fully consistent with the scatter expected from ΛCDM. There is no evidence for anomalous large-scale non-Gaussianity in the tail behaviour of the temperature field.

Agent Justification

The test is designed to capture both heavier tails (positive κ) and lighter tails (negative κ) as indicated in the test description, which mentions "positive κ indicates heavier tails" and "negative κ indicates lighter tails." The analysis code computes a two-sided p-value by taking the minimum of the upper and lower ranks, showing no directional preference. Since the test is exploratory and aims to detect any deviation from ΛCDM expectations without a strong prior on the direction of kurtosis change, a two-tailed test is appropriate.

Comparison To Literature

Previous low-ℓ anomaly studies have focused on quadrupole–octupole alignment, hemispherical power asymmetry, and the cold spot, rather than on masked-sky kurtosis specifically. Searches for direct prior studies of this exact statistic returned no relevant results, though excess kurtosis is a standard non-Gaussianity probe used in other CMB contexts (e.g., local f_NL analyses). This test therefore extends the low-ℓ anomaly battery to a new observable.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Masked-Sky Kurtosis. Reconstructs the low-ℓ temperature map (ℓ ≤ 5) "
    "from the spherical harmonic coefficients of the SMICA map, applies the Planck "
    "common mask, and computes the excess kurtosis of the standardized unmasked "
    "pixel temperatures: κ = (1/N) Σ t_i⁴ − 3, where t_i = (T_i − μ)/σ. This "
    "captures the non-Gaussian heaviness of the low-ℓ temperature distribution "
    "tails on the masked sky. Positive κ indicates heavier tails than Gaussian "
    "(potential localized anomalies), while negative κ indicates lighter tails "
    "(suppressed large-scale structure). Evaluated by comparing the Planck value "
    "against 1000 FFP ΛCDM simulations processed identically."
)


def analyze_map(m):
    m = np.asarray(m, dtype=np.float64)

    # Determine mask: unmasked pixels are finite
    mask = np.isfinite(m)

    # Replace NaN with 0.0 for healpy spherical harmonic transforms
    m_clean = np.where(mask, m, 0.0)

    # Extract spherical harmonic coefficients up to ℓ = 5
    lmax_reconstruct = 5
    alm = hp.map2alm(m_clean, lmax=lmax_reconstruct)

    # Reconstruct the low-ℓ map from these coefficients
    m_low_ell = hp.alm2map(alm, nside=hp.npix2nside(len(m)))

    # Apply the mask: only consider unmasked pixels
    m_masked = m_low_ell[mask]

    N = len(m_masked)
    if N < 10:
        return 0.0

    # Compute mean and std of unmasked pixels
    mu = np.mean(m_masked)
    sigma = np.std(m_masked)

    if sigma < 1e-15 or not np.isfinite(sigma):
        return 0.0

    # Standardized temperatures: t_i = (T_i - mu) / sigma
    t = (m_masked - mu) / sigma

    # Excess kurtosis: kappa = (1/N) * sum(t_i^4) - 3
    kappa = float(np.mean(t ** 4) - 3.0)

    if not np.isfinite(kappa):
        return 0.0

    return kappa


def summarize_results(planck_stat, sim_results):
    sim_arr = np.array(sim_results, dtype=np.float64)
    finite_mask = np.isfinite(sim_arr)
    n_finite = int(finite_mask.sum())

    if n_finite > 0:
        valid = sim_arr[finite_mask]
        sim_median = float(np.median(valid))
        p_upper = float(np.sum(valid >= planck_stat) / n_finite)
        p_lower = float(np.sum(valid <= planck_stat) / n_finite)
        p_value = min(p_upper, p_lower)
        rank_upper = int(np.sum(valid >= planck_stat))
        rank_lower = int(np.sum(valid <= planck_stat))
    else:
        p_value = float('nan')
        rank_upper = 0
        rank_lower = 0
        sim_median = float('nan')

    sim_mean = float(np.nanmean(sim_arr))
    sim_std = float(np.nanstd(sim_arr))

    if n_finite > 0 and not np.isnan(p_value):
        if p_upper <= p_lower:
            direction = "high"
            rank = rank_upper
        else:
            direction = "low"
            rank = rank_lower
    else:
        direction = "unknown"
        rank = 0

    interpretation_parts = []
    interpretation_parts.append(
        f"Low-ℓ masked-sky excess kurtosis κ = {planck_stat:.4f}."
    )
    interpretation_parts.append(
        f"Two-sided p = {p_value:.3f} ({rank}/{n_finite} simulations "
        f"at least as extreme in the {direction} direction)."
    )
    interpretation_parts.append(
        f"Simulation mean = {sim_mean:.4f}, std = {sim_std:.4f}, "
        f"median = {sim_median:.4f}."
    )

    if not np.isnan(p_value) and p_value < 0.05:
        if direction == "high":
            interpretation_parts.append(
                "Anomalous: the kurtosis is unusually HIGH, indicating heavier tails "
                "than expected from Gaussian ΛCDM — the low-ℓ temperature distribution "
                "on the masked sky has more extreme outliers (potential localized cold "
                "or hot spots at large angular scales)."
            )
        else:
            interpretation_parts.append(
                "Anomalous: the kurtosis is unusually LOW (more negative), indicating "
                "lighter/thinner tails than expected — the low-ℓ temperature distribution "
                "is unusually platykurtic on the masked sky, suggesting suppressed "
                "large-scale fluctuations or an overly uniform pattern."
            )
    else:
        interpretation_parts.append(
            "The excess kurtosis is consistent with ΛCDM simulation expectations."
        )

    result = {
        "test_description": test_description,
        "planck_stat": float(planck_stat),
        "p_value": float(p_value),
        "rank": rank,
        "n_simulations": len(sim_results),
        "n_valid": n_finite,
        "sim_mean": sim_mean,
        "sim_std": sim_std,
        "sim_median": sim_median,
        "exceeds_95th_percentile": bool(p_value < 0.05) if not np.isnan(p_value) else False,
        "anomaly_direction": direction,
        "interpretation": " ".join(interpretation_parts),
        "ell_max": 5,
    }

    result["plot_spec"] = {
        "kind": "histogram",
        "title": "Low-ℓ Masked-Sky Excess Kurtosis",
        "xlabel": "Excess kurtosis κ",
        "ylabel": "Number of simulations",
        "bins": 50,
        "caption": (
            f"Histogram of the excess kurtosis κ of the standardized low-ℓ (ℓ ≤ 5) "
            f"temperature distribution on the masked sky from {n_finite} FFP ΛCDM "
            f"simulations. Planck value = {planck_stat:.4f} "
            f"(two-sided p = {p_value:.3f}, rank {rank}/{n_finite}, "
            f"{direction} direction). Simulation mean = {sim_mean:.4f} ± {sim_std:.4f}. "
            f"Positive κ signals heavier tails (localized anomalies); "
            f"negative κ signals lighter tails (suppressed structure)."
        ),
    }

    return result
