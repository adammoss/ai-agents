Hypothesis

The slope of Δ_ℓ versus (-1)^ℓ is different from zero.

Method

For the Planck-masked SMICA map, ecliptic north and south hemispheres are defined. Pseudo-C_ℓ are computed for ℓ = 2–32 in each hemisphere. The fractional power difference Δ_ℓ = (C_ℓ^N − C_ℓ^S)/(C_ℓ^N + C_ℓ^S) is calculated at each ℓ, then regressed against (−1)^ℓ. The resulting slope tests whether the ecliptic hemispherical power asymmetry correlates with multipole parity (even vs. odd ℓ).

Result Summary

Planck slope = 0.0111. Simulations yield mean ± σ = 0.0010 ± 0.0111. The z-score is 0.99, corresponding to a two-tailed p-value of 0.32. Planck shows even-ℓ modes with relatively more power in ecliptic north than odd-ℓ modes, compared to simulations.

The observed slope is well within the distribution expected from isotropic ΛCDM simulations. At roughly 1σ from the simulated mean, the result shows no evidence for a parity-dependent hemispherical asymmetry. The small positive slope (even-ℓ slightly favored in ecliptic north) is fully consistent with statistical fluctuations.

Agent Justification

The test statistic measures the correlation between hemispherical power asymmetry and multipole parity. Under the null hypothesis of isotropy, the slope should be zero. The alternative hypothesis is that the slope is non-zero, without a specified direction. The code computes a two-sided p-value, indicating that both positive and negative deviations from zero are considered interesting. There is no prior expectation in the test description that the asymmetry should favor even or odd ℓ modes in a particular hemisphere, so a two-tailed test is appropriate.

Comparison To Literature

Parity asymmetry in the CMB low-ℓ spectrum has been studied extensively (Naselsky et al. 2012; Zhao et al. 2017, arXiv:1702.04541; Chang et al. 2018), typically as a global even-ℓ/odd-ℓ power imbalance. Hemispherical power asymmetry is also well-documented (McDonald 2014, arXiv:1405.2076). However, these two phenomena have generally been examined independently. The specific statistic combining hemispherical decomposition with a parity-dependent regression was not identified in literature searches.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Parity-Dependent Hemispherical Power Asymmetry Slope: "
    "For ecliptic north and south hemispheres, compute the pseudo angular power spectra "
    "C_ℓ for 2 ≤ ℓ ≤ 32 from the Planck-masked SMICA map. At each ℓ, form the fractional "
    "power difference Δ_ℓ = (C_ℓ^N - C_ℓ^S) / (C_ℓ^N + C_ℓ^S). The statistic is the "
    "linear-regression slope of Δ_ℓ against the parity sign (-1)^ℓ, measuring whether the "
    "ecliptic hemispherical asymmetry correlates with multipole parity. Positive slope: "
    "even-ℓ power relatively stronger in ecliptic north; negative: odd-ℓ relatively stronger."
)


def analyze_map(m):
    """Return the slope of ecliptic hemispherical Δ_ℓ vs (-1)^ℓ for ℓ = 2–32."""
    lmin, lmax = 2, 32
    finite_mask = np.isfinite(m)
    if not np.any(finite_mask):
        return 0.0

    nside = hp.npix2nside(len(m))
    npix = hp.nside2npix(nside)

    # Replace NaN with 0 for healpy transforms
    m_clean = np.where(finite_mask, m, 0.0)

    # Convert pixel positions to ecliptic coordinates
    # coord='ge' : Galactic → Ecliptic rotation
    rot = hp.Rotator(coord='ge')
    theta_ecl, phi_ecl = rot(hp.pix2ang(nside, np.arange(npix)))

    # Ecliptic north: colatitude θ_ecl ≤ π/2 (latitude β ≥ 0)
    ecl_north_mask = (theta_ecl <= np.pi / 2.0) & finite_mask
    ecl_south_mask = (theta_ecl >  np.pi / 2.0) & finite_mask

    m_north = np.where(ecl_north_mask, m_clean, 0.0)
    m_south = np.where(ecl_south_mask, m_clean, 0.0)

    # Pseudo-C_ℓ for each hemisphere
    cl_north = hp.anafast(m_north, lmax=lmax)[lmin:lmax + 1]
    cl_south = hp.anafast(m_south, lmax=lmax)[lmin:lmax + 1]

    # Fractional power difference per ℓ
    denom = cl_north + cl_south
    power_floor = np.max(denom) * 1e-30 if np.max(denom) > 0 else 1e-30
    delta_ell = np.where(denom > power_floor,
                         (cl_north - cl_south) / np.maximum(denom, power_floor),
                         0.0)

    # Slope of Δ_ℓ regressed on (-1)^ℓ
    ell = np.arange(lmin, lmax + 1)
    parity = (-1.0) ** ell            # +1 for even ℓ, -1 for odd ℓ
    slope = float(np.polyfit(parity, delta_ell, 1)[0])

    return slope


def summarize_results(planck_stat, sim_results):
    """Interpret the parity-dependent hemispherical asymmetry slope."""
    sims = np.array(sim_results, dtype=float)
    sims_finite = sims[np.isfinite(sims)]
    n_sims = len(sims_finite)

    if n_sims == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims_finite))
    std_s = float(np.std(sims_finite, ddof=1)) if n_sims > 1 else 0.0
    p_upper = float(np.mean(sims_finite >= planck_stat))
    p_lower = float(np.mean(sims_finite <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else np.nan

    if planck_stat > 0:
        tail_note = ("Planck shows even-ℓ modes with relatively more power in ecliptic north "
                     "than odd-ℓ modes, compared to simulations.")
    else:
        tail_note = ("Planck shows odd-ℓ modes with relatively more power in ecliptic north "
                     "than even-ℓ modes, compared to simulations.")

    return {
        "planck_stat": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "z_score": z_score,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n_sims,
        "tail_note": tail_note,
        "interpretation": (
            "This statistic measures whether the ecliptic hemispherical power asymmetry at "
            "low multipoles (ℓ = 2–32) correlates with parity (-1)^ℓ. For each multipole, "
            "the fractional power difference between the ecliptic north and south hemispheres "
            "is computed as Δ_ℓ = (C_ℓ^N − C_ℓ^S) / (C_ℓ^N + C_ℓ^S). A linear regression "
            "of Δ_ℓ against (-1)^ℓ yields the reported slope. A positive slope means even-ℓ "
            "modes contribute relatively more power to the ecliptic north, while a negative "
            "slope means odd-ℓ modes dominate the north. In isotropic ΛCDM the slope should "
            "be consistent with zero; a significant non-zero value could indicate parity-"
            "dependent large-scale anomalies or foreground contamination."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Parity-Dependent Hemispherical Asymmetry Slope",

                f"Positive: even-ℓ power relatively stronger in ecliptic north."
