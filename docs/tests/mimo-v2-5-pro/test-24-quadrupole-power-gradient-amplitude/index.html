Hypothesis

The quadrupole power gradient amplitude in the Planck CMB map is consistent with the distribution predicted by ΛCDM simulations.

Method

This statistic computes the RMS amplitude of the directional gradient of the quadrupole (ℓ=2) temperature field across the sphere, normalized by the mean absolute quadrupole temperature. It captures the overall spatial coherence and directional structure of the quadrupole, probing whether it exhibits anomalously strong or weak modulation patterns compared to ΛCDM expectations.

Result Summary

The Planck SMICA map yields a gradient amplitude of 2.889, compared to a simulation mean of 2.901 ± 0.063 (1σ). The z-score is −0.18σ, with a two-sided p-value of 0.782 and Planck falling at the 39.1st percentile of the simulated distribution. Anomaly strength is classified as negligible.

Planck's quadrupole gradient amplitude is completely consistent with ΛCDM predictions, lying well within 1σ of the simulation mean. This indicates that the quadrupole's spatial structure—its directional modulation pattern—is unremarkable and shows no evidence of anomalous alignment or unusual coherence at this level of characterization.

Agent Justification

The test is designed to detect any anomalous deviation in the gradient amplitude, whether higher or lower than expected, as indicated by the two-sided p-value calculation in the code and the interpretation that considers both directions of deviation (e.g., unusually strong or weak spatial structure). There is no clear directional preference specified in the test description or code.

Comparison To Literature

Previous studies have focused on quadrupole power deficit, quadrupole-octopole alignment (e.g., Copi et al. 2004; Weeks 2004), and multipole vector statistics. No prior work specifically examines the RMS gradient amplitude of the quadrupole temperature field as an anomaly probe. The negative result here is consistent with the broader picture: while some directional statistics of low-ℓ modes show mild anomalies, this particular spatial-coherence measure does not.

import numpy as np
import healpy as hp

test_description = (
    "Quadrupole Power Gradient Amplitude: For each map, compute the quadrupole "
    "(ℓ=2) spherical harmonic coefficients a_{2m}, then evaluate the quadrupole "
    "temperature field on the nside=256 map. Compute the directional gradient of "
    "this quadrupole field across the sphere using a finite-difference scheme on "
    "the pixel grid. The statistic is the root-mean-square (RMS) amplitude of "
    "this gradient vector field across all pixels, normalized by the mean "
    "absolute temperature of the quadrupole component. This single scalar "
    "captures the overall strength and spatial coherence of any dipole-like "
    "modulation pattern imprinted specifically on the quadrupole moment, which "
    "is a low-ℓ structure that could be anomalously aligned or prominent."
)


def analyze_map(m):
    """Compute RMS gradient amplitude of the quadrupole temperature field,
    normalized by mean absolute quadrupole temperature.

    Steps:
    1. Replace masked (NaN) pixels with 0 for healpy compatibility.
    2. Extract spherical harmonic coefficients a_{lm} up to l=2.
    3. Isolate the quadrupole (l=2) coefficients and reconstruct the
       quadrupole-only temperature map on the full sphere.
    4. Compute the gradient of the quadrupole field using central finite
       differences with healpy bilinear interpolation (get_interp_val).
    5. Return RMS(|∇T₂|) / mean(|T₂|).
    """
    nside = hp.npix2nside(len(m))
    npix = len(m)

    # Step 1: Replace NaN with 0 for healpy transforms
    valid = np.isfinite(m)
    m_filled = np.where(valid, m, 0.0)

    # Step 2: Extract a_{lm} up to l=2
    alm = hp.map2alm(m_filled, lmax=2)

    # Step 3: Keep only l=2 (healpy alm ordering: idx = l*(l+1)/2 + m)
    #   idx 0: (0,0), idx 1: (1,0), idx 2: (1,1),
    #   idx 3: (2,0), idx 4: (2,1), idx 5: (2,2)
    alm_q = np.zeros_like(alm)
    alm_q[3:] = alm[3:]

    # Reconstruct quadrupole-only temperature map
    quad = hp.alm2map(alm_q, nside=nside, lmax=2)

    # Step 4: Compute gradient via central finite differences
    theta, phi = hp.pix2ang(nside, np.arange(npix))
    delta = 0.01  # finite-difference step in radians (~0.57°)

    # Theta positions for central difference (clipped to stay on sphere)
    theta_p = np.clip(theta - delta, 1e-10, np.pi - 1e-10)
    theta_m = np.clip(theta + delta, 1e-10, np.pi - 1e-10)

    # Evaluate quadrupole at all offset positions in a single vectorized call
    all_theta = np.concatenate([theta_p, theta_m, theta, theta])
    all_phi = np.concatenate([phi, phi, phi + delta, phi - delta])
    all_vals = hp.get_interp_val(quad, all_theta, all_phi)
    T_tp, T_tm, T_pp, T_pm = np.split(all_vals, 4)

    # Partial derivatives
    dT_dtheta = (T_tp - T_tm) / (theta_p - theta_m)
    dT_dphi = (T_pp - T_pm) / (2.0 * delta)

    # Physical gradient components on the sphere:
    #   grad_theta = dT/dθ
    #   grad_phi   = (1/sinθ) * dT/dφ
    sin_theta = np.sin(theta)
    safe_sin = np.maximum(sin_theta, 1e-10)
    grad_theta = dT_dtheta
    grad_phi = dT_dphi / safe_sin

    # Step 5: RMS gradient magnitude normalized by mean |T₂|
    grad_mag_sq = grad_theta ** 2 + grad_phi ** 2
    rms_grad = np.sqrt(np.mean(grad_mag_sq))
    mean_abs_quad = np.mean(np.abs(quad))

    if mean_abs_quad < 1e-30:
        return 0.0

    return float(rms_grad / mean_abs_quad)


def summarize_results(planck_stat, sim_results):
    """Summarize the Quadrupole Power Gradient Amplitude anomaly test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results to compare.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Quadrupole Power Gradient Amplitude",
                "xlabel": "RMS gradient / mean |T₂|",
                "ylabel": "Number of simulations",
                "bins": 40,
                "caption": "No simulation data available.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    # Two-sided p-value
    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    # Rank and percentile
    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims

    # Anomaly classification
    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    # Contextual direction
    if planck_stat > mean_sim + std_sim:
        direction = (
            f"Planck's gradient amplitude ({planck_stat:.4f}) exceeds the "
            f"simulation mean by more than 1σ, indicating the quadrupole has "
            f"an unusually strong directional structure or modulation pattern."
        )
    elif planck_stat < mean_sim - std_sim:
        direction = (
            f"Planck's gradient amplitude ({planck_stat:.4f}) is more than 1σ "
            f"below the simulation mean, suggesting an unusually uniform "
            f"quadrupole with less spatial structure than typical realizations."
        )
    else:
        direction = (
            f"Planck's gradient amplitude is within 1σ of the simulation mean, "
            f"consistent with isotropic ΛCDM expectations."
        )

    interp = (
        f"Quadrupole gradient amplitude = {planck_stat:.4f}. "
        f"Simulation mean = {mean_sim:.4f} ± {std_sim:.4f}. "
        f"Percentile = {percentile:.1f}%, two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of the RMS gradient amplitude of the quadrupole (ℓ=2) "
        "temperature field, normalized by the mean absolute quadrupole "
        "temperature, across simulated CMB maps. This statistic probes the "
        "spatial coherence and directional structure of the quadrupole: a "
        "higher value indicates stronger angular variation (e.g., power "
        "concentrated in higher-m modes), while a lower value indicates a "
        "more uniform pattern. Anomalously high or low values could signal "
        "alignment or non-standard physics at large angular scales. "
        "The red line marks the Planck SMICA value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Quadrupole Power Gradient Amplitude",
            "xlabel": "RMS gradient / mean |T₂|",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
