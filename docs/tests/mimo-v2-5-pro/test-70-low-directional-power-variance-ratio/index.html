Hypothesis

The directional power variance ratio for multipoles ℓ=2–10 relative to ℓ=2 alone does not match the expected distribution in standard ΛCDM cosmology, indicating an anomaly in the low-ℓ power structure.

Method

Computes the variance of directional power (squared temperature) across the full sky for multipoles ℓ=2–10, then divides by the corresponding variance from the ℓ=2 quadrupole alone. The resulting ratio quantifies how much spatial structure the higher low-ℓ modes add to the anisotropy pattern of T² relative to the quadrupole baseline.

Result Summary

Planck ratio = 354.23. Simulation mean ± σ = 333.78 ± 5916.25 (median = 38.74), indicating a highly skewed distribution. z-score = 0.003, two-tailed p-value = 0.098. The Planck value lies extremely close to the simulation mean and well within the bulk of the distribution.

The directional power variance ratio for Planck is fully consistent with ΛCDM simulations. The very small z-score confirms negligible deviation from expectations. The distribution is heavily right-skewed (mean ≫ median), so standard symmetric thresholds are less informative, but the Planck value does not approach either tail at the 2σ level.

Agent Justification

The test is explicitly designed to quantify whether the power distribution is "more or less structured" than the quadrupole alone, with the code's interpretation noting that "a significantly elevated or suppressed value could indicate anomalous alignment." There is no directional preference specified in the description or code; instead, both higher and lower ratios are considered potentially anomalous, making it appropriate for a two-tailed test to detect deviations in either direction from the ΛCDM expectation.

Comparison To Literature

The directional power variance ratio of squared temperature across low multipoles does not correspond to any well-known CMB anomaly statistic found in standard literature. Previous searches returned no relevant studies quantifying this specific ratio. The test appears to probe spatial structure in low-ℓ power concentrations but lacks established precedent for direct comparison.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Directional Power Variance Ratio: Restrict the CMB map to multipoles "
    "ℓ = 2–10 and compute the directional power (squared temperature) at each pixel. "
    "Calculate the variance of this power across the full sky. Repeat for a map "
    "containing only the ℓ = 2 quadrupole component. The statistic is the ratio of "
    "the former variance to the latter, quantifying whether the low-ℓ power distribution "
    "is more or less structured than what the quadrupole alone produces."
)


def analyze_map(m):
    """Return the ratio of directional-power variance (ℓ=2–10) to (ℓ=2 only)."""
    nside = hp.npix2nside(len(m))

    # Replace NaN (masked) pixels with 0.0 for healpy transforms
    mask_invalid = ~np.isfinite(m)
    m_clean = m.copy()
    m_clean[mask_invalid] = 0.0

    # Compute spherical harmonic coefficients up to ℓ=10
    lmax = 10
    alm = hp.map2alm(m_clean, lmax=lmax)

    # Build alm for ℓ=2–10: zero out ℓ < 2
    alm_low = alm.copy()
    for ell in range(2):  # ℓ=0 and ℓ=1
        for em in range(ell + 1):
            idx = hp.Alm.getidx(lmax, ell, em)
            alm_low[idx] = 0.0

    # Build alm for ℓ=2 only: zero out everything except ℓ=2
    alm_quad = np.zeros_like(alm)
    for em in range(3):  # ℓ=2 has m = 0, 1, 2
        idx = hp.Alm.getidx(lmax, 2, em)
        alm_quad[idx] = alm[idx]

    # Reconstruct band-limited maps
    map_low = hp.alm2map(alm_low, nside, lmax=lmax, verbose=False)
    map_quad = hp.alm2map(alm_quad, nside, lmax=lmax, verbose=False)

    # Directional power = squared amplitude at each pixel
    power_low = map_low ** 2
    power_quad = map_quad ** 2

    # Variance of directional power across the sky
    var_low = np.var(power_low)
    var_quad = np.var(power_quad)

    # Guard against degenerate case
    if var_quad == 0.0:
        return 1.0

    ratio = float(var_low / var_quad)
    return ratio


def summarize_results(planck_stat, sim_results):
    """Interpret the directional power variance ratio."""
    sims = np.array(sim_results, dtype=float)
    sims_finite = sims[np.isfinite(sims)]
    n_sims = len(sims_finite)

    if n_sims == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims_finite))
    std_s = float(np.std(sims_finite, ddof=1)) if n_sims > 1 else 0.0
    p_upper = float(np.mean(sims_finite >= planck_stat))
    p_lower = float(np.mean(sims_finite <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else np.nan

    return {
        "planck_stat": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "z_score": z_score,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n_sims,
        "interpretation": (
            "This statistic compares how spatially structured the directional power "
            "(squared temperature) is across the full sky for the low-ℓ band (ℓ=2–10) "
            "relative to the quadrupole alone (ℓ=2). The band ℓ=2–10 contains 45 "
            "spherical harmonic modes whose interference pattern determines the "
            "anisotropy of T²(p). The ratio of variances quantifies whether adding "
            "multipoles ℓ=3–10 increases or redistributes the spatial contrast of "
            "hot/cold power concentrations relative to the quadrupole pattern. In "
            "standard ΛCDM, this ratio has a well-defined expectation; a significantly "
            "elevated or suppressed value could indicate anomalous alignment, power "
            "asymmetry, or unexpected mode coupling at the largest angular scales."
        ),
        "plot_spec": {
            "kind": "histogram",

            "xlabel": "Variance Ratio (ℓ=2–10) / (ℓ=2)",
            "ylabel": "Count",
            "bins": 30,

                f"Planck ratio = {planck_stat:.4f}. "
                f"Sim mean ± σ = {mean_s:.4f} ± {std_s:.4f}. "
                f"z = {z_score:.2f}, p(two-sided) = {p_two:.3f}."
            ),
        },
