Hypothesis

The observed low-ℓ (ℓ ≤ 10) CMB temperature distribution from the Planck SMICA map exhibits positive excess kurtosis (κ > 0) compared to the Gaussian expectation, indicating heavier-than-expected tails due to the presence of outliers such as the CMB Cold Spot or other non-Gaussian features.

Method

Band-limit the Planck SMICA map to ℓ ≤ 10 via spherical harmonic transform, apply Galactic mask, and compute sample excess kurtosis κ of unmasked pixel temperatures. Compare observed κ against 1000 masked ΛCDM simulations. Positive κ indicates heavier-than-Gaussian tails (more extreme outliers such as the Cold Spot).

Result Summary

Observed κ = −0.001, essentially at the Gaussian expectation of 0. Simulation mean ± std = −0.025 ± 0.380. z-score = 0.06, upper-tail p-value = 0.408. The Planck value ranks at the 59th percentile of simulations—well within the central distribution.

The low-ℓ temperature distribution is mesokurtic (near-Gaussian), with no evidence for enhanced tails from outliers like the Cold Spot at this multipoles resolution. The statistic is completely typical relative to ΛCDM realizations. No signature of the hypothesized heavy-tailed behavior is detected.

Agent Justification

The test description explicitly states that it is "sensitive to isolated extreme-temperature features such as the Cold Spot." The Cold Spot is a well-known, significant anomaly in the CMB that would manifest as a large cold pixel, pulling the temperature distribution toward heavier tails (positive excess kurtosis). Therefore, there is a clear directional hypothesis that the observed map will show an excess of such outliers, making an upper-tailed test appropriate.

Comparison To Literature

The Planck Collaboration (2020, Planck 2018 results VII: Isotropy and Statistics of the CMB; arXiv:1906.02552) performed extensive non-Gaussianity tests at low multipoles using Minkowski functionals, bispectrum analyses, and temperature distribution moments. Their findings generally showed consistency with Gaussianity at large angular scales, though certain statistics (e.g., local non-Gaussianity from the Cold Spot region) showed mild tension. The present excess kurtosis result is consistent with these broader findings of no significant departure from Gaussianity in the low-ℓ temperature distribution.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Temperature Distribution Excess Kurtosis. "
    "Band-limit the input Planck SMICA map to ℓ ≤ 10 via spherical harmonic transform "
    "(alm up to lmax=10) and inverse-transform back to a Healpix map at nside=256. "
    "Apply the Galactic mask so only unmasked pixels are retained. From the surviving "
    "pixel temperatures, compute the sample excess kurtosis: "
    "κ = (1/N) Σᵢ [(Tᵢ − T̄)/σ]⁴ − 3. "
    "Positive κ indicates heavier-than-Gaussian tails (more extreme hot/cold pixels "
    "than expected), while negative κ indicates lighter-than-Gaussian tails. "
    "This statistic is sensitive to isolated extreme-temperature features such as "
    "the Cold Spot, broad non-Gaussian excursions, or any systematic enhancement/"
    "suppression of temperature outliers relative to a Gaussian expectation."
)


def analyze_map(m):
    """Return excess kurtosis κ of the low-ℓ (ℓ≤10) temperature distribution."""
    nside = hp.npix2nside(len(m))
    lmax = 10

    valid = np.isfinite(m)
    n_valid = int(np.sum(valid))

    if n_valid < 100:
        return 0.0

    # Band-limit to ℓ ≤ 10 via spherical harmonic transform
    # Replace NaN with 0 for healpy compatibility
    m_filled = np.where(valid, m, 0.0)
    alm = hp.map2alm(m_filled, lmax=lmax)
    m_low = hp.alm2map(alm, nside, lmax=lmax, verbose=False)

    # Extract temperatures of unmasked pixels only
    temps = m_low[valid]
    N = len(temps)

    if N < 10:
        return 0.0

    # Sample mean and standard deviation
    T_bar = np.mean(temps)
    sigma = np.std(temps, ddof=0)

    if sigma == 0.0:
        return 0.0

    # Sample excess kurtosis: κ = (1/N) Σ [(Tᵢ - T̄)/σ]⁴ - 3
    z = (temps - T_bar) / sigma
    kappa = float(np.mean(z ** 4) - 3.0)

    return kappa if np.isfinite(kappa) else 0.0


def summarize_results(planck_stat, sim_results):
    """Interpret the Low-ℓ Temperature Distribution Excess Kurtosis."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)
    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    median_s = float(np.median(sims))
    rank = int(np.sum(sims <= planck_stat))
    p_upper = float(np.mean(sims >= planck_stat))
    p_lower = float(np.mean(sims <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else 0.0
    pct_rank = rank / n * 100 if n > 0 else 50.0

    if pct_rank < 2.5 or pct_rank > 97.5:
        sig_desc = "highly anomalous (<2.5% or >97.5% of sims)"
    elif pct_rank < 5 or pct_rank > 95:
        sig_desc = "marginally anomalous (<5% or >95% of sims)"
    elif pct_rank < 25 or pct_rank > 75:
        sig_desc = "noteworthy (outside IQR of sims)"
    else:
        sig_desc = "typical (within IQR of sims)"

    if planck_stat > 0.1:
        kurt_desc = (
            "heavy-tailed (leptokurtic), indicating more extreme temperature "
            "outliers than expected from a Gaussian field—possibly linked to features "
            "like the CMB Cold Spot or other non-Gaussian hot/cold excursions"
        )
    elif planck_stat < -0.1:
        kurt_desc = (
            "light-tailed (platykurtic), indicating fewer extreme temperature "
            "outliers than expected from a Gaussian field, suggesting suppressed "
            "or smoothed temperature fluctuations at large angular scales"
        )
    else:
        kurt_desc = (
            "near-Gaussian (mesokurtic), consistent with the expected temperature "
            "distribution for a Gaussian random CMB field at low multipoles"
        )

    return {
        "planck_stat": planck_stat,
        "excess_kurtosis_kappa": planck_stat,
        "gaussian_expectation": 0.0,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "sim_median": median_s,
        "z_score": z_score,
        "rank": rank,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "percentile_rank": pct_rank,
        "tail_behavior": kurt_desc,
        "significance_description": sig_desc,
        "interpretation": (
            f"Low-ℓ excess kurtosis κ = {planck_stat:.4f} (Gaussian expectation = 0). "
            f"Distribution is {kurt_desc}. "
            f"Sim mean ± std = {mean_s:.4f} ± {std_s:.4f}. "
            f"{sig_desc.capitalize()}. z = {z_score:.2f}, two-sided p = {p_two:.4f}."
        ),
        "plot_spec": {
            "kind": "histogram",

            "xlabel": r"$\kappa = \langle [(T - \bar{T})/\sigma]^4 \rangle - 3$",
            "ylabel": "Count",
