Hypothesis

The asphericity ratio A of the squared temperature pattern in the Planck low-ℓ (ℓ ≤ 30) map is consistent with the distribution predicted by isotropic, Gaussian ΛCDM simulations.

Method

A weighted inertia tensor I_{ij} is computed using T(p)² as weights over pixels in the low-ℓ (ℓ ≤ 30) masked CMB map. After diagonalizing the 3×3 tensor to obtain eigenvalues λ₁ ≥ λ₂ ≥ λ₃ (summing to zero), the asphericity ratio A = (λ₁ − λ₃)/λ₁ is computed to quantify the elongation of the squared temperature pattern. The Planck SMICA map value is compared to distributions from 1000 ΛCDM simulations.

Result Summary

Planck asphericity ratio A = 2.386 compared to simulation mean = 1.889 ± 0.250, giving a deviation of +1.99σ (percentile = 94.5%). Two-sided p-value = 0.11.

The low-ℓ squared temperature pattern shows marginally higher asphericity than typical ΛCDM realizations, but the deviation does not reach the 2σ threshold. The result falls within expectations for a Gaussian, isotropic sky. The specific physical interpretation—whether prolate or oblate elongation—depends on the eigenvalue structure, but statistically no significant departure from ΛCDM is detected.

Agent Justification

The test is exploratory, checking for any significant deviation from ΛCDM expectations in the asphericity of the squared temperature pattern—either more elongated/flattened than expected or more spherical. The implementation computes both upper and lower p-values and combines them into a two-sided p-value, confirming the two-tailed nature. No prior directional hypothesis is specified in the test description or code; it's a general check for anomalies.

Comparison To Literature

Standard low-ℓ CMB anomaly studies (e.g., Planck Collaboration VII, 2020) examine quadrupole–octupole alignment, parity asymmetry, and hemispherical power asymmetry rather than T²-weighted asphericity statistics. No directly comparable studies using this specific T²-weighted inertia tensor approach were identified in the literature.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ T² Asphericity Eigenvalue Ratio: "
    "Compute the weighted inertia tensor I_{ij} = Σ_p T(p)² · "
    "(3 n_i(p) n_j(p) − δ_{ij}) ΔΩ over pixels with temperature T(p) "
    "in the low-ℓ (ℓ ≤ 30) CMB map. Diagonalize I to obtain eigenvalues "
    "λ₁ ≥ λ₂ ≥ λ₃ (with λ₁+λ₂+λ₃ = 0). The statistic is the asphericity "
    "ratio A = (λ₁ − λ₃) / λ₁, which quantifies the elongation of the "
    "squared temperature pattern. Compare observed A from the Planck SMICA "
    "map to the distribution from 1000 simulated ΛCDM maps."
)


def analyze_map(m):
    """Low-ℓ T² Asphericity Eigenvalue Ratio.

    Steps:
    1. Replace NaN pixels with 0 for healpy compatibility.
    2. Decompose into spherical harmonics up to ℓ=30 and reconstruct
       a smooth low-ℓ map (fills masked regions naturally via the
       band-limited basis).
    3. Compute T² for each pixel.
    4. Build the weighted inertia tensor I_{ij} = Σ T²(3 n_i n_j − δ_{ij})
       using vectorised matrix multiplication over all pixels.
    5. Diagonalize the 3×3 real symmetric matrix to get eigenvalues.
    6. Return A = (λ₁ − λ₃) / λ₁.
    """
    mask = np.isfinite(m)
    n_valid = int(np.sum(mask))
    if n_valid < 100:
        return 0.0

    nside = hp.get_nside(m)
    npix = 12 * nside ** 2

    # Replace NaN with 0 for healpy spherical-harmonic operations
    clean = m.copy()
    clean[~mask] = 0.0

    # Filter to ℓ ≤ 30
    lmax = 30
    alm = hp.map2alm(clean, lmax=lmax)
    low_ell_map = hp.alm2map(alm, nside, lmax=lmax, verbose=False)

    # Compute T² weighting
    T2 = low_ell_map ** 2

    # Get pixel directions in Cartesian coordinates
    theta, phi = hp.pix2ang(nside, np.arange(npix))
    sin_theta = np.sin(theta)
    nx = sin_theta * np.cos(phi)
    ny = sin_theta * np.sin(phi)
    nz = np.cos(theta)

    # Stack direction vectors: shape (3, npix)
    n_vec = np.array([nx, ny, nz])

    # Weighted inertia tensor (drop ΔΩ factor since it cancels in the ratio):
    #   I_{ij} = 3 Σ_p T²_p n_i n_j  −  δ_{ij} Σ_p T²_p
    total_T2 = np.sum(T2)

    # Efficient vectorised computation: (n_vec * T²) @ n_vec.T
    # Entry (i,j) = Σ_p T²_p n_i(p) n_j(p)
    weighted_n = n_vec * T2[np.newaxis, :]
    I = 3.0 * (weighted_n @ n_vec.T) - total_T2 * np.eye(3)

    # Diagonalize (I is real symmetric, use eigvalsh)
    eigenvalues = np.linalg.eigvalsh(I)
    eigenvalues = np.sort(eigenvalues)[::-1]  # descending: λ₁ ≥ λ₂ ≥ λ₃
    lam1, lam2, lam3 = eigenvalues

    # Asphericity ratio A = (λ₁ − λ₃) / λ₁
    # λ₁ ≥ 0, λ₃ ≤ 0, and λ₁ + λ₂ + λ₃ = 0
    if abs(lam1) < 1e-30:
        return 0.0

    A = (lam1 - lam3) / lam1

    return float(A) if np.isfinite(A) else 0.0


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ℓ T² Asphericity Eigenvalue Ratio test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ T² Asphericity Eigenvalue Ratio",
                "xlabel": "A = (λ₁ − λ₃) / λ₁",
                "ylabel": "Count",
                "bins": 40,
                "caption": "No simulation data.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims
    deviation_sigma = (planck_stat - mean_sim) / std_sim if std_sim > 0 else 0.0

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    # A = 1: perfectly prolate (λ₂ = λ₃ = −λ₁/2 → A = 3/2 actually)
    # A = 3/2: axisymmetric prolate
    # A = 3: perfectly oblate (λ₁ = λ₂, λ₃ = −2λ₁)
    if planck_stat > mean_sim + 2 * std_sim:
        direction = (
            "Planck's low-ℓ T² pattern is significantly more aspheric than "
            "ΛCDM simulations, indicating the squared temperature field is "
            "more elongated or flattened than expected from isotropic "
            "Gaussian fluctuations."
        )
    elif planck_stat < mean_sim - 2 * std_sim:
        direction = (
            "Planck's low-ℓ T² pattern is significantly more spherical than "
            "ΛCDM simulations, indicating less elongation/flattening of the "
            "squared temperature field than expected."
        )
    else:
        direction = (
            "The asphericity of the low-ℓ T² pattern is consistent with "
            "ΛCDM expectations."
        )

    interp = (
        f"Asphericity ratio A = {planck_stat:.4f}. "
        f"Simulation mean = {mean_sim:.4f} ± {std_sim:.4f} "
        f"({deviation_sigma:+.2f}σ). Percentile = {percentile:.1f}%, "
        f"two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of the asphericity ratio A = (λ₁ − λ₃)/λ₁ of the "
        "T²-weighted inertia tensor for the low-ℓ (ℓ ≤ 30) CMB temperature "
        "field. Larger A indicates the squared temperature pattern is more "
        "elongated or flattened (oblate); A ≈ 3/2 for a prolate distribution "
        "and A → 3 for a strongly oblate one. Red line marks the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": deviation_sigma,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ T² Asphericity Eigenvalue Ratio",
            "xlabel": "A = (λ₁ − λ₃) / λ₁",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
