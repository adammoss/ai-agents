Hypothesis

The skewness of low-ℓ temperature fluctuations in the northern Galactic hemisphere is zero, consistent with the isotropic and Gaussian predictions of the ΛCDM model.

Method

Skewness of pixel temperatures in the northern Galactic hemisphere (b > 0) computed on a masked, low-ℓ (ℓ = 2–20) CMB temperature map, probing asymmetric tails in the large-scale fluctuation distribution.

Result Summary

Planck SMICA skewness = 0.0073. Simulation mean = 0.0001 ± 0.0225. The observed value sits at the 63.8th percentile of 1000 ΛCDM simulations (z = 0.35σ, two-tailed p = 0.724). Anomaly strength classified as negligible.

The slight positive skewness indicates a mild excess of warm pixels in the northern Galactic hemisphere, but the value is entirely consistent with isotropic, Gaussian ΛCDM expectations. No directional asymmetry in the third moment of the temperature distribution is detected at low multipoles.

Agent Justification

The test is designed as a general exploration for asymmetry without a prior directional preference. The test description states that under isotropic ΛCDM, the skewness "should be symmetric between hemispheres and centered near zero," and the code's `summarize_results` function explicitly computes both left and right one-sided p-values and a two-sided p-value. The interpretation considers both significant negative skewness (excess cold) and positive skewness (excess warm) as potential anomalies. Since the test is motivated by a search for any unexpected deviation from symmetry, rather than a strong, pre-specified prediction of a particular sign, a two-tailed test is appropriate.

Comparison To Literature

Hemispherical asymmetry studies using the local variance estimator (LVE) find ~2σ–3σ power asymmetries at low multipoles (e.g., Sanyal et al. 2026, arXiv:2502.10768; McDonald 2014). However, those studies probe the *second* moment (variance/power), not the *third* moment (skewness). I found no prior published work specifically measuring hemispherical skewness of low-ℓ CMB temperature maps. The null result here is consistent with the absence of a detected third-moment anomaly, complementing but not directly contradicting the power-asymmetry findings.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Galactic North Skewness: On the masked, low-ℓ (ℓ = 2–20) temperature "
    "map, compute the skewness (the third standardized moment) of the pixel "
    "temperature values restricted to the northern Galactic hemisphere (Galactic "
    "latitude b > 0). This statistic probes whether the large-scale temperature "
    "fluctuations in the Galactic north exhibit asymmetric tails in their "
    "distribution — a property that could signal anomalous alignment, preferred "
    "directions, or non-Gaussian features at low multipoles. Under isotropic "
    "ΛCDM the skewness should be symmetric between hemispheres and centered "
    "near zero, so any significant deviation warrants investigation."
)


def analyze_map(m):
    """Compute skewness of low-ℓ temperatures in the northern Galactic hemisphere.

    Steps:
    1. Identify unmasked pixels with finite values.
    2. Determine which pixels lie in the northern Galactic hemisphere (b > 0),
       equivalent to colatitude θ < π/2.
    3. Extract the temperature values at those pixels.
    4. Compute and return the skewness (third standardized moment).
    """
    nside = hp.npix2nside(len(m))
    npix = len(m)

    # Step 1: Valid (unmasked) pixels
    valid = np.isfinite(m)

    # Step 2: Northern Galactic hemisphere — b > 0 ↔ θ < π/2
    # hp.pix2ang returns (theta, phi) in Healpix convention
    theta, _ = hp.pix2ang(nside, np.arange(npix))
    north = theta < (np.pi / 2.0)

    # Combined mask: valid AND northern
    mask = valid & north

    # Step 3: Extract temperatures in the selected region
    temps = m[mask]

    # Safety: need at least 3 values for skewness
    if len(temps) < 3:
        return 0.0

    # Step 4: Compute skewness = E[(X - mu)^3] / sigma^3
    mu = np.mean(temps)
    diffs = temps - mu
    n = len(temps)
    # Population skewness (third standardized moment)
    m3 = np.mean(diffs ** 3)
    m2 = np.mean(diffs ** 2)
    sigma = np.sqrt(m2)
    if sigma < 1e-30:
        return 0.0
    skewness = m3 / (sigma ** 3)

    return float(skewness)


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ℓ Galactic North Skewness anomaly.

    Skewness near zero is expected. Significant negative skewness means the
    northern hemisphere has an excess of cold pixels (heavy left tail), while
    significant positive skewness means an excess of hot pixels. We test
    both tails.
    """
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float('nan'),
            "rank": 0,
            "n_sims": 0,
            "percentile": float('nan'),
            "sim_mean": float('nan'),
            "sim_std": float('nan'),
            "interpretation": "No simulation results to compare.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Galactic North Skewness",
                "xlabel": "Skewness",
                "ylabel": "Number of simulations",
                "bins": 40,
                "caption": "No simulation data available.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    # Two-sided p-value: fraction of sims at least as extreme as Planck
    p_value = float(np.mean(np.abs(sim) >= np.abs(planck_stat)))

    # One-sided p-values
    p_left = float(np.mean(sim <= planck_stat))
    p_right = float(np.mean(sim >= planck_stat))

    # Rank (among simulations)
    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims

    # Anomaly classification
    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    if planck_stat < 0:
        tail_desc = (
            f"Planck shows negative skewness ({planck_stat:.4f}), indicating "
            f"an excess of cold pixels in the northern Galactic hemisphere. "
        )
    else:
        tail_desc = (
            f"Planck shows positive skewness ({planck_stat:.4f}), indicating "
            f"an excess of warm pixels in the northern Galactic hemisphere. "
        )

    if p_value <= 0.05:
        direction = (
            f"This is at the {percentile:.1f}th percentile of simulations "
            f"(two-sided p = {p_value:.4f}), which is anomalously extreme."
        )
    elif p_value <= 0.10:
        direction = (
            f"This is at the {percentile:.1f}th percentile of simulations "
            f"(two-sided p = {p_value:.4f}), showing a moderate tension."
        )
    else:
        direction = (
            f"This is at the {percentile:.1f}th percentile of simulations "
            f"(two-sided p = {p_value:.4f}), consistent with ΛCDM expectations."
        )

    interp = (
        f"Low-ℓ (ℓ = 2–20) Galactic North skewness = {planck_stat:.4f}. "
        f"{tail_desc}{direction} "
        f"Simulation mean = {mean_sim:.4f} ± {std_sim:.4f}. "
        f"Anomaly strength: {strength}."
    )

    caption = (
        "Histogram of the skewness of pixel temperatures in the northern "
        "Galactic hemisphere (b > 0) for the low-ℓ (ℓ = 2–20) masked map. "
        "Under isotropic Gaussian ΛCDM, the skewness is centered near zero "
        "with scatter determined by cosmic variance. A significant negative "
        "value indicates an excess of cold fluctuations in the north; a "
        "significant positive value indicates an excess of hot fluctuations. "
        "The red line marks the Planck SMICA value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_left": p_left,
        "p_value_right": p_right,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Galactic North Skewness",
            "xlabel": "Skewness",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
