Hypothesis

The low-ℓ (ℓ=2–10) temperature-temperature correlation coherence length in the Planck SMICA map is larger than the average coherence length predicted by standard ΛCDM simulations.

Method

Compute the temperature-temperature two-point correlation function C(θ) restricted to multipoles ℓ=2–10 from the Planck SMICA map. Define the coherence length L_c as the smallest angle where C(θ) first drops to 1/e of its maximum value. The anomaly statistic is the ratio L_c(Planck) / mean(L_c(1000 ΛCDM simulations)), testing whether the real sky's low-ℓ pattern is more spatially coherent than typical random realizations.

Result Summary

Planck L_c = 0.2919 rad (16.7°), while the simulation mean ± σ = 0.3406 ± 0.0395 rad (19.5° ± 2.3°). The anomaly ratio is 0.857, meaning the Planck coherence length is actually *smaller* than the simulation average. z = −1.23, one-tailed upper p-value = 0.917, two-sided p = 0.166.

The hypothesis predicted an anomalously *large* coherence length in the real sky. Instead, the Planck map shows a coherence length ~14% shorter than the ΛCDM mean, sitting at roughly the 8th percentile of simulations toward smaller scales. This deviation is modest (1.2σ below the mean) and falls well within the range expected from normal statistical fluctuation. There is no evidence that the low-ℓ correlation pattern is unusually coherent.

Agent Justification

The test description and code explicitly define the anomaly as a ratio greater than 1, indicating "an anomalously large coherence length." The interpretation within the code states: "A ratio > 1 means the low-ℓ pattern in the real sky is more spatially coherent than typical ΛCDM realizations." This establishes a clear, directional prediction that the real sky's coherence length is greater than the simulated average. Therefore, a one-tailed test examining the upper tail (ratio > 1) is appropriate.

Comparison To Literature

No prior studies were found that specifically test a "coherence length" statistic derived from the band-limited (ℓ=2–10) two-point correlation function. Related work exists on large-angle C(θ) anomalies and low-ℓ alignment/planarity, but this particular statistic does not appear in the searched literature.

import numpy as np
import healpy as hp

test_description = (
    "For the masked Planck SMICA map, compute the two-point temperature correlation function "
    "C(θ) restricted to multipoles ℓ=2–10. Define the coherence length L_c as the smallest "
    "angle θ at which C(θ) first drops to 1/e of its maximum value (which occurs at θ=0). "
    "The per-map statistic is L_c in radians. The final anomaly statistic is the ratio "
    "L_c(Planck) / mean(L_c(simulations)), measuring how the spatial coherence scale of the "
    "low-ℓ pattern in the real sky compares to the typical scale expected in random ΛCDM "
    "realizations. A ratio > 1 indicates an anomalously large coherence length."
)


def analyze_map(m):
    """Return coherence length L_c (radians) for the low-ℓ correlation function.

    Computes C(θ) = Σ_{ℓ=2}^{10} (2ℓ+1)/(4π) C_ℓ P_ℓ(cos θ) on a fine grid,
    then finds the smallest angle where C(θ) first drops to C(0)/e.
    """
    # Replace NaN masked pixels with 0 for healpy compatibility
    valid = np.isfinite(m)
    m_clean = np.where(valid, m, 0.0)

    # Compute pseudo-C_ℓ up to ℓ=10
    cl = hp.anafast(m_clean, lmax=10)

    # Fine angular grid from 0 to π
    n_theta = 2000
    theta = np.linspace(0.0, np.pi, n_theta)
    cos_theta = np.cos(theta)

    # Compute C(θ) = Σ_{ℓ=2}^{10} (2ℓ+1)/(4π) * C_ℓ * P_ℓ(cos θ)
    c_theta = np.zeros(n_theta)
    for ell in range(2, 11):
        coeffs = np.zeros(ell + 1)
        coeffs[ell] = 1
        P_ell = np.polynomial.legendre.legval(cos_theta, coeffs)
        c_theta += (2 * ell + 1) / (4 * np.pi) * cl[ell] * P_ell

    # C(0) is the global maximum since all C_ℓ >= 0 and P_ℓ(1) = 1
    c_max = c_theta[0]

    if c_max <= 0:
        return 1.0  # fallback for maps with no power at low ℓ

    # Find L_c: smallest θ where C(θ) first drops to c_max / e
    threshold = c_max / np.e

    below = np.where(c_theta < threshold)[0]

    if len(below) == 0:
        # Correlation never drops below threshold within [0, π]
        return float(np.pi)

    idx = below[0]
    if idx > 0:
        # Linear interpolation for a more accurate crossing point
        frac = (threshold - c_theta[idx - 1]) / (c_theta[idx] - c_theta[idx - 1])
        L_c = theta[idx - 1] + frac * (theta[idx] - theta[idx - 1])
    else:
        L_c = theta[0]

    return float(L_c)


def summarize_results(planck_stat, sim_results):
    """Interpret the Low-ℓ Band-Limited Correlation Coherence Length."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)

    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    median_s = float(np.median(sims))

    # The anomaly ratio: Planck coherence / mean sim coherence
    ratio = planck_stat / mean_s if mean_s > 0 else float('inf')

    # p-values
    p_value_upper = float(np.mean(sims >= planck_stat))  # tail toward large L_c
    p_value_lower = float(np.mean(sims <= planck_stat))  # tail toward small L_c
    p_two = min(2.0 * min(p_value_upper, p_value_lower), 1.0)

    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else 0.0

    return {
        "planck_stat": planck_stat,
        "planck_coherence_length_rad": planck_stat,
        "planck_coherence_length_deg": float(np.degrees(planck_stat)),
        "anomaly_ratio": ratio,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "sim_mean_deg": float(np.degrees(mean_s)),
        "sim_median": median_s,
        "z_score": z_score,
        "p_value_upper": p_value_upper,
        "p_value_lower": p_value_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "interpretation": (
            f"The coherence length L_c is the smallest angle at which the low-ℓ (2–10) "
            f"two-point correlation function C(θ) first drops to 1/e of its maximum value. "
            f"Planck L_c = {planck_stat:.4f} rad ({np.degrees(planck_stat):.1f}°). "
            f"Sim mean ± σ = {mean_s:.4f} ± {std_s:.4f} rad "
            f"({np.degrees(mean_s):.1f}° ± {np.degrees(std_s):.1f}°). "
            f"Anomaly ratio (Planck/sim_mean) = {ratio:.4f}. "
            f"A ratio > 1 means the low-ℓ pattern in the real sky is more spatially "
            f"coherent than typical ΛCDM realizations. "
            f"z = {z_score:.2f}, two-sided p = {p_two:.4f}."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Band-Limited Correlation Coherence Length",
            "xlabel": "Coherence length L_c (radians)",
            "ylabel": "Count",
            "bins": 30,

                f"Planck L_c = {planck_stat:.4f} rad ({np.degrees(planck_stat):.1f}°). "

            ),
