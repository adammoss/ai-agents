Hypothesis

The Planck CMB temperature map exhibits an unusually large magnitude of the dipole vector formed by the hottest and coldest spots in the low-ℓ smoothed map, indicating a net directional alignment (rather than antipodality) compared to simulations from the standard cosmological model.

Method

Smooth the Planck CMB temperature map to ℓ=10, identify the single hottest and coldest pixels, and form unit vectors from their sky positions. The statistic |D| = |n_hot + n_cold| ranges from 0 (perfectly antipodal extremes) to 2 (co-located extremes), testing whether the two most extreme low-ℓ temperature features show an unexpected directional alignment rather than the expected antipodal arrangement.

Result Summary

Planck |D| = 1.1626, compared to simulation mean ± σ = 1.4255 ± 0.3964. The z-score is −0.66 with an upper-tail p-value of 0.769. The Planck value lies below the simulation median (1.5327), indicating the extremes are slightly more antipodal than typical LCDM realizations.

There is no evidence of anomalous alignment among the hottest and coldest low-ℓ features. If anything, the Planck extremes are marginally more antipodal than the simulation average, entirely consistent with statistical fluctuations in the standard model. The high p-value (0.769) rules out any upper-tail anomaly.

Agent Justification

The test statistic |D| ranges from 0 (perfect antipodality) to 2 (perfect alignment). The test description explicitly states it measures "the net directional preference" and is "testing for an unexpected dipole in their spatial arrangement." The summary interpretation highlights that values significantly above the simulation mean indicate the extremes are "preferentially aligned rather than antipodal, suggesting a dipolar directional preference." This defines a clear directional hypothesis favoring high |D|, justifying a one-tailed upper test.

Comparison To Literature

No prior studies were found testing this specific statistic—constructing a dipole vector from the hottest and coldest pixel positions in a low-ℓ smoothed CMB map. Related literature examines CMB dipole alignments with large-scale structure (e.g., Luongo et al. 2022 on H₀ hemispherical variations along the CMB dipole direction), but none use this particular directional-extreme diagnostic.

import numpy as np
import healpy as hp

test_description = (
    "Smooth the temperature map to a maximum multipole of ℓ=10. "
    "Identify the single hottest and single coldest pixel in the smoothed map. "
    "Treat their sky positions as unit vectors n_hot and n_cold. "
    "Compute the dipole vector D = n_hot + n_cold. "
    "The statistic is the magnitude |D|. This measures the net directional preference "
    "of the two most extreme temperature features at large angular scales, testing for "
    "an unexpected dipole in their spatial arrangement."
)


def analyze_map(m):
    """Return |D|: magnitude of dipole vector from hot and cold extreme directions at low-ℓ."""
    nside = hp.npix2nside(len(m))
    lmax = 10

    # Track which pixels are unmasked
    valid = np.isfinite(m)

    # Replace NaNs with 0 for healpy compatibility
    m_clean = np.where(valid, m, 0.0)

    # Decompose into spherical harmonics up to lmax=10 and reconstruct
    alm = hp.map2alm(m_clean, lmax=lmax, iter=10)
    m_smooth = hp.alm2map(alm, nside=nside, lmax=lmax)

    # Only consider unmasked pixels for finding extremes
    valid_indices = np.where(valid)[0]
    if len(valid_indices) == 0:
        return 0.0

    m_valid = m_smooth[valid_indices]

    # Find hottest and coldest pixels among unmasked set (vectorized)
    idx_hot_local = np.argmax(m_valid)
    idx_cold_local = np.argmin(m_valid)

    pix_hot = valid_indices[idx_hot_local]
    pix_cold = valid_indices[idx_cold_local]

    # Get unit vectors for these pixels
    n_hot = np.array(hp.pix2vec(nside, pix_hot))
    n_cold = np.array(hp.pix2vec(nside, pix_cold))

    # Compute dipole vector D = n_hot + n_cold, return magnitude |D|
    D = n_hot + n_cold
    return float(np.sqrt(np.sum(D ** 2)))


def summarize_results(planck_stat, sim_results):
    """Interpret the Low-ℓ Extreme-Direction Dipole Magnitude statistic."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)

    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    median_s = float(np.median(sims))

    # p-value upper: fraction of sims with |D| >= Planck (anomalously aligned extremes)
    p_upper = float(np.mean(sims >= planck_stat))
    # p-value lower: fraction of sims with |D| <= Planck (anomalously antipodal extremes)
    p_lower = float(np.mean(sims <= planck_stat))
    # Two-sided p-value
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)

    # z-score relative to simulation ensemble
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else 0.0

    return {
        "planck_stat": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "sim_median": median_s,
        "z_score": z_score,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "interpretation": (
            f"|D| = magnitude of dipole vector from hot and cold extreme directions at low-ℓ. "
            f"|D|=0 means hottest and coldest spots are perfectly antipodal; "
            f"|D|=2 means they are co-located (physically unlikely). "
            f"Values significantly above the simulation mean indicate the two most extreme "
            f"temperature features are preferentially aligned rather than antipodal, "
            f"suggesting a dipolar directional preference in the arrangement of low-ℓ extremes. "
            f"Planck z-score = {z_score:.2f} relative to simulations. "
            f"P(|D| ≥ Planck) = {p_upper:.4f}."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Extreme-Direction Dipole Magnitude",
            "xlabel": "|D| (Dipole magnitude from extreme directions)",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck |D| = {planck_stat:.4f}. "
                f"Sim mean ± σ = {mean_s:.4f} ± {std_s:.4f}. "
                f"z = {z_score:.2f}, P(≥ Planck) = {p_upper:.4f}."
            ),
        },
    }
