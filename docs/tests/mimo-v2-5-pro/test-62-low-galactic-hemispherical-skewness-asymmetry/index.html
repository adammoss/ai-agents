Hypothesis

The CMB temperature map shows a significant hemispherical skewness asymmetry at low multipoles (ℓ ≤ 30), with the absolute difference in skewness between northern and southern galactic hemispheres exceeding expectations from isotropic simulations.

Method

Band-limit the Planck temperature map to ℓ ≤ 30. Compute the pixel-value skewness separately in the northern galactic hemisphere (b > 0°) and the southern hemisphere (b < 0°). The statistic is the absolute difference |Skew_N − Skew_S|.

Result Summary

Planck asymmetry = 0.3962. From 1000 ΛCDM simulations: mean ± σ = 0.2019 ± 0.1559. z = 1.25, one-tailed upper p = 0.104.

The Planck map shows modestly elevated hemispherical skewness asymmetry at low multipoles compared to isotropic simulations, but the excess is not statistically significant. At z ≈ 1.25, the signal remains well below the 2σ threshold (p = 0.104 > 0.05). No evidence for a preferred direction or breakdown of isotropy in the skewness distribution on large angular scales.

Agent Justification

The test statistic is the absolute difference in skewness, which is always non-negative. The p-value is computed as the fraction of simulations with a statistic greater than or equal to the observed Planck value, focusing on detecting unusually large asymmetries. Since the test is designed to identify anomalies where the magnitude of asymmetry is significant, without specifying a direction for which hemisphere is more skewed, a one-tailed upper test is appropriate.

Comparison To Literature

The well-established hemispherical power asymmetry at low ℓ (e.g., Lew 2008; Eriksen et al. 2004) is typically quantified through variance or power ratios in ecliptic or galactic coordinates. McDonald (2014) and Assadullahi et al. (2014) discuss theoretical origins of the power asymmetry. Skewness-based hemispherical asymmetry statistics at low multipoles in galactic coordinates have not been identified in prior literature from the searches conducted, though the general phenomenon of large-scale CMB asymmetries is well studied.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Galactic Hemispherical Skewness Asymmetry: "
    "Band-limit the temperature map to ℓ ≤ 30. Compute the skewness of "
    "pixel temperatures in the northern (b > 0°) and southern (b < 0°) "
    "galactic hemispheres separately. The statistic is the absolute "
    "difference between these two skewness values."
)


def analyze_map(m):
    """Return |skewness(N) - skewness(S)| for the ℓ ≤ 30 band-limited map."""

    def skewness(values):
        """Compute sample skewness (bias-corrected / Fisher) for a 1-D array."""
        n = len(values)
        if n < 3:
            return 0.0
        mean = np.mean(values)
        diff = values - mean
        m2 = np.mean(diff ** 2)
        if m2 <= 0.0:
            return 0.0
        m3 = np.mean(diff ** 3)
        skew = (n * (n - 1)) ** 0.5 / (n - 2) * m3 / (m2 ** 1.5)
        return skew

    nside = hp.npix2nside(len(m))

    # Replace NaN with 0 for healpy spherical-harmonic operations
    masked = ~np.isfinite(m)
    m_clean = m.copy()
    m_clean[masked] = 0.0

    # Forward transform -> filter to l <= 30 -> inverse transform
    lmax = 30
    alm = hp.map2alm(m_clean, lmax=lmax)
    m_limited = hp.alm2map(alm, nside, lmax=lmax)

    # Build galactic-latitude mask (vectorised, no per-pixel Python loop)
    npix = hp.nside2npix(nside)
    theta, phi = hp.pix2ang(nside, np.arange(npix))  # co-latitude, longitude
    gal_lat = np.pi / 2.0 - theta  # latitude in radians
    north = gal_lat > 0.0
    south = gal_lat < 0.0

    # Combine with the original mask (keep only originally-valid pixels)
    valid = ~masked
    north_vals = m_limited[valid & north]
    south_vals = m_limited[valid & south]

    skew_n = skewness(north_vals)
    skew_s = skewness(south_vals)

    asymmetry = abs(skew_n - skew_s)
    return float(asymmetry)


def summarize_results(planck_stat, sim_results):
    """Interpret the asymmetry relative to the simulation ensemble."""
    sims = np.array(sim_results, dtype=float)
    sims_finite = sims[np.isfinite(sims)]
    n_sims = len(sims_finite)

    if n_sims == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims_finite))
    std_s = float(np.std(sims_finite, ddof=1)) if n_sims > 1 else 0.0
    p_value = float(np.mean(sims_finite >= planck_stat)) if n_sims > 0 else np.nan

    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else np.nan

    return {
        "planck_stat": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "z_score": z_score,
        "p_value": p_value,
        "n_sims": n_sims,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Galactic Hemispherical Skewness Asymmetry",
            "xlabel": r"|Skew$_N$ − Skew$_S$|",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck asymmetry = {planck_stat:.4f}.  "
                f"Simulation mean ± σ = {mean_s:.4f} ± {std_s:.4f}.  "
                f"z = {z_score:.2f},  p = {p_value:.3f}  "
                f"(fraction of simulations ≥ Planck value)."
            ),
        },
    }
