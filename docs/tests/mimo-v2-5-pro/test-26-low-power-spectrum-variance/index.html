Hypothesis

The observed variance of C_ℓ for ℓ=2 to 10 is inconsistent with the distribution predicted by ΛCDM simulations.

Method

The angular power spectrum C_ℓ for multipoles ℓ = 2–10 is computed, and the sample variance of these nine values is taken. This scalar quantifies how unevenly power is distributed across the lowest multipoles. Under ΛCDM each C_ℓ is χ²(2ℓ+1)-distributed, so the expected variance is set by the fiducial spectrum. Both unusually high and low variance can signal anomalies such as suppressed low-ℓ power or oscillatory features in the Sachs–Wolfe plateau.

Result Summary

Var(C_ℓ) = 5,300 μK⁴, well below the simulation mean of 90,377 ± 149,257 μK⁴. The observed value sits at the 1.7th percentile of 1,000 ΛCDM simulations, yielding a two-tailed p-value of 0.034 (z ≈ −0.57; the wide spread reflects the skewed distribution of the variance estimator). Anomaly strength is moderate.

The Planck low-ℓ power spectrum variance is notably low relative to ΛCDM expectations. The very low observed value suggests power among ℓ = 2–10 is more uniformly suppressed than typical simulations predict, consistent with the well-documented low quadrupole and overall low-ℓ power deficit. However, the two-tailed p = 0.034 falls just above the conventional 2σ threshold (p = 0.046 corresponds to 2σ for a Gaussian; here the non-Gaussianity of the test statistic complicates interpretation).

Agent Justification

The test is designed for general exploration of anomalies in low-ℓ power dispersion, checking for significant deviations in both high and low variance directions. The code explicitly calculates two-sided p-values and interprets both high and low variance cases as potential anomalies, indicating no clear directional preference.

Comparison To Literature

The low-ℓ power deficit, particularly the low quadrupole, is a long-standing anomaly noted since COBE and confirmed by WMAP and Planck (e.g., arXiv:2509.03719; Eriksen et al. 2004). However, testing the *variance* of C_ℓ across ℓ = 2–10 as a single scalar statistic is not a standard approach in the literature; most studies examine individual multipoles or ratios. No directly comparable published analysis was found.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Power Spectrum Variance: Compute the angular power spectrum C_ℓ "
    "for multipoles ℓ=2 to 10, then calculate the variance of these nine "
    "C_ℓ values. This scalar measures the dispersion of power among low "
    "multipoles. Under ΛCDM, each C_ℓ is a χ²-distributed random variable "
    "with 2ℓ+1 degrees of freedom, so the expected variance depends on the "
    "underlying power spectrum. An unusually high or low variance may "
    "indicate anomalies in how power is distributed across low multipoles, "
    "such as the low quadrupole or oscillatory patterns in the Sachs-Wolfe "
    "plateau."
)


def analyze_map(m):
    """Compute the variance of C_ℓ for ℓ=2..10 from the angular power spectrum.

    Steps:
      1. Replace masked (NaN) pixels with 0.0 for healpy compatibility.
      2. Compute the pseudo power spectrum via hp.anafast up to lmax=10.
      3. Extract C_ℓ for ℓ = 2, 3, ..., 10 (9 values).
      4. Return the variance of these 9 values.

    Returns
    -------
    float
        The sample variance of C_ℓ for ℓ in [2, 10].
    """
    m_filled = np.where(np.isfinite(m), m, 0.0)

    lmax = 10
    cl = hp.anafast(m_filled, lmax=lmax)

    # cl[ℓ] = C_ℓ; we want ℓ = 2 through 10 inclusive
    cl_low = cl[2:lmax + 1]  # 9 values

    var_cl = np.var(cl_low)

    return float(var_cl)


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ℓ Power Spectrum Variance test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results to compare.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Power Spectrum Variance",
                "xlabel": "Var(C_ℓ), ℓ = 2–10 [μK⁴]",
                "ylabel": "Number of simulations",
                "bins": 40,
                "caption": "No simulation data available.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    direction = ""
    if planck_stat > mean_sim + 2 * std_sim:
        direction = (
            "Planck shows significantly high low-ℓ power spectrum variance, "
            "indicating an unusual distribution of power among the lowest "
            "multipoles. This could reflect features such as excess power in "
            "specific ℓ bands or oscillatory behavior in the Sachs–Wolfe plateau."
        )
    elif planck_stat < mean_sim - 2 * std_sim:
        direction = (
            "Planck shows significantly low low-ℓ power spectrum variance, "
            "indicating an unusually uniform distribution of power across "
            "multipoles ℓ=2–10. This is consistent with the well-known low "
            "quadrupole anomaly if power is suppressed across the board."
        )
    else:
        direction = (
            "The variance of low-ℓ power is consistent with isotropic ΛCDM "
            "expectations, showing no anomalous dispersion of power among "
            "the lowest multipoles."
        )

    interp = (
        f"Var(C_ℓ) = {planck_stat:.4f} μK⁴. "
        f"Simulation mean = {mean_sim:.4f} ± {std_sim:.4f} μK⁴. "
        f"Percentile = {percentile:.1f}%, two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of the variance of C_ℓ for multipoles ℓ=2–10 from "
        "isotropic ΛCDM simulations. The variance measures how unevenly "
        "power is distributed among the lowest multipoles. Under ΛCDM, each "
        "C_ℓ is χ²(2ℓ+1)-distributed with known mean and variance, so the "
        "expected variance of the nine C_ℓ values is determined by the "
        "fiducial power spectrum. Significantly high variance may indicate "
        "localized excess or deficit of power; low variance suggests "
        "suppressed fluctuations. The red line marks the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Power Spectrum Variance",
            "xlabel": "Var(C_ℓ), ℓ = 2–10 [μK⁴]",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
