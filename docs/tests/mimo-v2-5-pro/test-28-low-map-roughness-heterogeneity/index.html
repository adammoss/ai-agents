Hypothesis

The CV of per-patch Laplacian variance for the Planck map is significantly greater than expected from isotropic ΛCDM simulations, indicating spatially anisotropic temperature structure at low multipoles.

Method

The spherical Laplacian of the band-limited SMICA temperature map (ℓ = 2–30) is computed via the eigenvalue −ℓ(ℓ+1), and the sky is divided into ~60 equal-area azimuthal patches (10 iso-latitude rings with φ-sectors scaled by sin θ). The coefficient of variation (CV) of per-patch Laplacian variances quantifies how unevenly spatial "roughness" is distributed. An elevated CV would signal anisotropic temperature texture.

Result Summary

Planck CV = 0.4121. ΛCDM simulation mean = 0.4379 ± 0.0394 (1000 sims). The Planck value lies at the 25.3rd percentile (z ≈ −0.67σ). One-tailed upper p-value = 0.747; two-tailed p ≈ 0.506. Anomaly strength is negligible.

The Laplacian roughness distribution across the Planck sky is entirely consistent with isotropic Gaussian realizations. The observed CV is actually *slightly below* the simulation mean, opposite to the hypothesized excess. No evidence of spatially anisotropic temperature texture at low multipoles is found.

Agent Justification

The test description explicitly states that an "elevated CV" would indicate anisotropic temperature structure, and the code includes directional interpretation for an elevated CV as a potential anomaly. The motivation focuses on detecting "roughness heterogeneity" where some regions are systematically rougher or smoother—this is naturally a deviation toward higher CV, not lower. The alternative hypothesis is directional: we expect greater heterogeneity, not less.

Comparison To Literature

No prior study using per-patch Laplacian variance heterogeneity was found in the CMB literature. Related analyses include hemisphere-based power asymmetry tests (e.g., Eriksen et al. 2004; Planck 2020 XI) and local variance maps, but these operate on the temperature field directly rather than its Laplacian texture. The null result is consistent with the broader finding that low-ℓ spatial anomalies are localized to specific signatures (quadrupole–octopole alignment, cold spot) rather than global texture heterogeneity.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Map Roughness Heterogeneity: Compute the spherical Laplacian of the "
    "low-ℓ temperature map (multipoles ℓ=2–30), then measure how unevenly this "
    "roughness is distributed across the sky. The sky is divided into ~100 "
    "equal-area azimuthal patches (10 iso-latitude rings with equal Δcos θ, each "
    "subdivided into φ-sector counts scaled by sin θ to approximately equalise "
    "solid angle, with a minimum of 3 sectors per ring). For each patch j, the "
    "population variance σ²_j of the Laplacian values is computed from unmasked "
    "pixels. The scalar output is the coefficient of variation "
    "CV = std({σ²_j}) / mean({σ²_j}), measuring spatial heterogeneity in "
    "temperature roughness. In a statistically isotropic Gaussian field the "
    "roughness should be nearly uniform (CV near zero after accounting for patch "
    "geometry and pixel-count noise); a significantly elevated CV compared to "
    "ΛCDM simulations would indicate spatially anisotropic temperature structure."
)


def analyze_map(m):
    """Compute the coefficient of variation of per-patch Laplacian variance.

    Steps:
      1. Replace masked (NaN) pixels with 0 for healpy compatibility.
      2. Compute alm up to lmax=30, zero out ℓ < 2, apply the spherical
         Laplacian eigenvalue -ℓ(ℓ+1), and reconstruct the Laplacian field.
      3. Divide the sky into ~60 equal-area patches: 10 iso-latitude rings
         (equal Δcosθ = 0.2) with φ-sector counts proportional to sin(θ_mid),
         minimum 3 sectors per ring.
      4. Use vectorised bincount to compute per-patch mean, count, and
         variance of Laplacian values at unmasked pixels.
      5. Return CV = std(patch_vars) / mean(patch_vars).

    Returns
    -------
    float
        Coefficient of variation of per-patch Laplacian variances.
    """
    nside = hp.npix2nside(len(m))
    mask = np.isfinite(m)
    m_filled = np.where(mask, m, 0.0)
    npix = len(m)

    # --- Band-limit to ℓ = 2–30 and compute Laplacian ---
    lmax = 30
    alm = hp.map2alm(m_filled, lmax=lmax)

    # Zero out ℓ < 2
    for ell in range(2):
        for em in range(ell + 1):
            alm[hp.Alm.getidx(lmax, ell, em)] = 0.0

    # Apply spherical Laplacian eigenvalue -ℓ(ℓ+1) using almxfl
    fl = np.zeros(lmax + 1)
    for ell in range(2, lmax + 1):
        fl[ell] = -ell * (ell + 1.0)
    alm = hp.almxfl(alm, fl)

    # Reconstruct Laplacian map
    laplacian = hp.alm2map(alm, nside, verbose=False)

    # --- Define equal-area patches ---
    n_theta = 10
    cos_edges = np.linspace(-1.0, 1.0, n_theta + 1)
    theta_edges = np.arccos(np.clip(cos_edges, -1.0, 1.0))
    theta_mid = 0.5 * (theta_edges[:-1] + theta_edges[1:])

    # Sectors per ring: proportional to sin(θ), min 3 to avoid tiny polar patches
    n_phi = np.maximum(3, np.round(10.0 * np.sin(theta_mid)).astype(int))

    # --- Vectorised patch assignment for valid pixels ---
    valid_pixels = np.where(mask)[0]
    if len(valid_pixels) == 0:
        return 0.0

    theta_v, phi_v = hp.pix2ang(nside, valid_pixels)

    # Assign latitude ring: digitize cos(θ) into bins
    cos_theta_v = np.cos(theta_v)
    ring_idx = np.digitize(cos_theta_v, cos_edges[1:-1])  # values 0..n_theta-1
    ring_idx = np.clip(ring_idx, 0, n_theta - 1)

    # Assign azimuthal sector within each ring
    phi_v_mod = phi_v % (2.0 * np.pi)  # wrap to [0, 2π)
    n_phi_per_pix = n_phi[ring_idx]     # sectors in the ring of each pixel
    sector_idx = np.floor(phi_v_mod / (2.0 * np.pi / n_phi_per_pix)).astype(int)
    sector_idx = np.clip(sector_idx, 0, n_phi_per_pix - 1)

    # Flatten patch index
    patch_id = ring_idx * 100 + sector_idx

    # --- Vectorised per-patch statistics via bincount ---
    lap_valid = laplacian[valid_pixels]

    counts = np.bincount(patch_id)
    sum_lap = np.bincount(patch_id, weights=lap_valid)
    sum_lap2 = np.bincount(patch_id, weights=lap_valid ** 2)

    # Per-patch mean and variance (population)
    mean_lap = np.zeros_like(sum_lap)
    valid_counts = counts > 0
    mean_lap[valid_counts] = sum_lap[valid_counts] / counts[valid_counts]
    var_lap = np.zeros_like(sum_lap)
    var_lap[valid_counts] = sum_lap2[valid_counts] / counts[valid_counts] - mean_lap[valid_counts] ** 2
    var_lap = np.maximum(var_lap, 0.0)  # numerical safety

    # Keep patches with ≥ 2 valid pixels and finite variance
    valid_patch = (counts >= 2) & np.isfinite(var_lap)
    patch_vars = var_lap[valid_patch]

    if len(patch_vars) == 0:
        return 0.0

    mean_pv = np.mean(patch_vars)
    if mean_pv <= 0.0:
        return 0.0

    cv = np.std(patch_vars) / mean_pv
    return float(cv)


def summarize_results(planck_stat, sim_results):
    """Summarise the Low-ℓ Map Roughness Heterogeneity test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results to compare.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Map Roughness Heterogeneity",
                "xlabel": "CV of per-patch Laplacian variance",
                "ylabel": "Number of simulations",
                "bins": 40,
                "caption": "No simulation data available.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    # Two-sided p-value
    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    direction = ""
    if planck_stat > mean_sim + 2 * std_sim:
        direction = (
            "Planck shows significantly elevated roughness heterogeneity: "
            "some sky regions appear systematically rougher or smoother than "
            "others in the ℓ=2–30 Laplacian field. This could indicate "
            "spatially anisotropic temperature structure, a localised "
            "departure from isotropy, or texture variation beyond ΛCDM "
            "expectations."
        )
    elif planck_stat < mean_sim - 2 * std_sim:
        direction = (
            "Planck shows unusually low roughness heterogeneity: the "
            "Laplacian field is more uniformly textured across the sky than "
            "expected from isotropic simulations, suggesting unusually "
            "homogeneous large-scale temperature structure."
        )
    else:
        direction = (
            "The roughness heterogeneity is consistent with isotropic ΛCDM "
            "expectations; the Laplacian field is distributed across the sky "
            "without significant anisotropic texture variation."
        )

    interp = (
        f"CV = {planck_stat:.6f}. "
        f"Simulation mean = {mean_sim:.6f} ± {std_sim:.6f}. "
        f"Percentile = {percentile:.1f}%, two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of the coefficient of variation (CV) of per-patch "
        "Laplacian variance from isotropic ΛCDM simulations. The Laplacian "
        "of the ℓ=2–30 band-limited temperature map measures local spatial "
        "roughness. The sky is divided into ~60 equal-area patches (10 "
        "iso-latitude rings with equal Δcos θ, each with φ-sector counts "
        "proportional to sin θ and minimum 3 sectors per ring). For each "
        "patch, the population variance of Laplacian values at unmasked "
        "pixels is computed; the CV of these variances quantifies how "
        "unevenly roughness is distributed. Under statistical isotropy, "
        "CV should be small and determined by sampling noise; a "
        "significantly elevated CV in Planck would indicate spatially "
        "anisotropic temperature texture. The red line marks the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Map Roughness Heterogeneity",
            "xlabel": "CV of per-patch Laplacian variance (ℓ = 2–30)",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
