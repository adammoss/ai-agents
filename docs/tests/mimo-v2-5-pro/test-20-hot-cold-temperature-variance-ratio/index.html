Hypothesis

The hot-cold temperature variance ratio R = σ²_hot / σ²_cold is not equal to 1 under statistical isotropy and Gaussianity.

Method

Compute the variance of the CMB temperature field separately for pixels with T > 0 ("hot") and T < 0 ("cold"), then take the ratio R = σ²_hot / σ²_cold. Under statistical isotropy and Gaussianity, R ≈ 1. A low-resolution SMICA map (nside=256) with the standard low-ℓ mask is used. This scalar statistic probes morphological asymmetry between positive and negative temperature excursions, sensitive to features like the Cold Spot and hemispherical power differences.

Result Summary

The Planck SMICA value is R = 0.9954, corresponding to a log-ratio z-score of −0.15. The two-tailed p-value is 0.894, placing the observation at the 44.7th percentile of 1000 ΛCDM simulations (simulated mean R = 1.0009 ± 0.0351). The asymmetry is classified as negligible.

The hot and cold temperature variances are approximately equal in the Planck data, fully consistent with the ΛCDM expectation of statistical isotropy and Gaussianity. There is no evidence for morphological asymmetry between positive and negative temperature excursions. The result is unremarkable and well within the bulk of the simulation distribution.

Agent Justification

The test is designed to detect any morphological asymmetry between positive and negative temperature excursions, regardless of direction. The description and code explicitly consider both R < 1 (indicating cold pixels have higher variance) and R > 1 (indicating hot pixels have higher variance). There is no strong prior expectation for a specific direction, making a two-tailed test appropriate for exploring deviations from the null expectation of R ≈ 1.

Comparison To Literature

While hemispherical power asymmetry and temperature parity have been extensively studied (e.g., Planck 2018 VII), this specific ratio of variances computed over pixel subsets split by sign of temperature is not a standard statistic in the literature. Related analyses exist but use different definitions and decomposition methods. No directly comparable published result was identified in searches.

import numpy as np

test_description = (
    "Hot-Cold Temperature Variance Ratio: Compute the variance of the CMB temperature field "
    "separately in the positive and negative hemispheres of the temperature distribution, then "
    "take their ratio. Specifically: (1) Identify unmasked pixels with T > 0 ('hot' pixels) and "
    "T < 0 ('cold' pixels). (2) Compute σ²_hot = mean(T²) over hot pixels and σ²_cold = mean(T²) "
    "over cold pixels. (3) The statistic is R = σ²_hot / σ²_cold. Under statistical isotropy and "
    "Gaussianity, hot and cold fluctuations should have equal variance, yielding R ≈ 1. Deviations "
    "indicate morphological asymmetry between positive and negative temperature excursions. This is "
    "sensitive to features like the Cold Spot and hemispherical power differences, while remaining "
    "a simple single scalar that is cheap to compute and distinct from existing alignment, parity, "
    "and hemispherical power ratio tests."
)


def analyze_map(m):
    """Compute the hot-cold temperature variance ratio.

    1. Select unmasked (finite) pixels.
    2. Split into hot (T > 0) and cold (T < 0) subsets.
    3. Compute mean(T²) over each subset.
    4. Return R = σ²_hot / σ²_cold.

    For a statistically isotropic Gaussian field, R ≈ 1.
    R < 1 means cold pixels have larger variance (e.g., Cold Spot influence).
    R > 1 means hot pixels have larger variance.
    """
    # Select only unmasked pixels
    m_valid = m[np.isfinite(m)]

    # Separate hot and cold pixels
    hot = m_valid[m_valid > 0]
    cold = m_valid[m_valid < 0]

    # Guard against degenerate cases (shouldn't happen for CMB maps)
    if len(hot) == 0 or len(cold) == 0:
        return 1.0

    # Variance as mean of squared temperatures for each subset
    var_hot = np.mean(hot ** 2)
    var_cold = np.mean(cold ** 2)

    # Protect against zero variance (extremely unlikely)
    if var_cold == 0.0:
        return float('inf')

    R = var_hot / var_cold

    return float(R)


def summarize_results(planck_stat, sim_results):
    """Summarize the hot-cold temperature variance ratio anomaly."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    # For a two-sided test: we care about deviation from 1 in either direction
    # Compute log(R) for symmetric treatment, since R is a ratio
    log_planck = np.log(planck_stat) if planck_stat > 0 else 0.0
    log_sim = np.log(sim[sim > 0]) if np.any(sim > 0) else np.array([0.0])

    mean_log = float(np.mean(log_sim))
    std_log = float(np.std(log_sim))
    z_score = (log_planck - mean_log) / std_log if std_log > 0 else float('nan')

    # Two-sided p-value: fraction of simulations more extreme than Planck
    # "More extreme" = farther from 1 (i.e., |log(R)| larger)
    planck_dev = abs(log_planck)
    sim_devs = np.abs(log_sim - mean_log)
    p_value_two_sided = float(np.mean(sim_devs >= planck_dev)) if n_sims > 0 else float('nan')

    # One-sided p-values for each direction
    p_value_low = float(np.mean(sim <= planck_stat)) if n_sims > 0 else float('nan')
    p_value_high = float(np.mean(sim >= planck_stat)) if n_sims > 0 else float('nan')

    # Percentile rank
    rank = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank / n_sims if n_sims > 0 else float('nan')

    # Descriptive statistics
    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    # Interpretation
    if planck_stat < 0.85:
        direction = "cold pixels have substantially higher variance than hot pixels"
        strength = "strong"
    elif planck_stat < 0.95:
        direction = "cold pixels have moderately higher variance than hot pixels"
        strength = "moderate"
    elif planck_stat > 1.15:
        direction = "hot pixels have substantially higher variance than cold pixels"
        strength = "strong"
    elif planck_stat > 1.05:
        direction = "hot pixels have moderately higher variance than cold pixels"
        strength = "moderate"
    else:
        direction = "hot and cold variances are approximately equal"
        strength = "negligible"

    interp = (
        "Hot-cold temperature variance ratio R = {stat:.4f}, indicating {direction}. "
        "This places it at the {pct:.1f}th percentile of LCDM simulations "
        "(two-sided p-value = {pval:.4f}). "
        "Simulated mean R = {mean:.4f} ± {std:.4f}. "
        "Log-ratio z-score = {z:.2f}."
    ).format(stat=planck_stat, direction=direction, pct=percentile,
             pval=p_value_two_sided, mean=mean_sim, std=std_sim, z=z_score)

    caption = (
        "Histogram of the hot-cold temperature variance ratio R = σ²_hot / σ²_cold, "
        "measuring the morphological asymmetry between positive and negative temperature "
        "excursions in the CMB. σ²_hot = mean(T²) over pixels with T > 0 and "
        "σ²_cold = mean(T²) over pixels with T < 0. Under statistical isotropy and "
        "Gaussianity, R ≈ 1. Values below 1 indicate cold features (like the Cold Spot) "
        "dominate the variance; values above 1 indicate hot features dominate. "
        "The red line marks the Planck SMICA value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value_two_sided,
        "p_value_low": p_value_low,
        "p_value_high": p_value_high,
        "rank": rank,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "log_z_score": z_score,
        "asymmetry_label": strength,
        "asymmetry_direction": direction,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Hot–Cold Temperature Variance Ratio",
            "xlabel": "R = σ²_hot / σ²_cold",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
