Hypothesis

The low-ℓ (ℓ ≤ 32) CMB temperature map exhibits a mean Hessian anisotropy index that is significantly different from the standard ΛCDM expectation, indicating an anomalous degree of local curvature anisotropy.

Method

The CMB temperature map is smoothed to ℓ_max = 32. At each unmasked pixel, the covariant Hessian matrix of the temperature field is computed via spherical harmonic transforms and diagonalised to obtain two eigenvalues (principal curvatures). A per-pixel anisotropy index A = | |λ₁| − |λ₂| | / (|λ₁| + |λ₂| + ε) quantifies directional bias in local curvature. The test statistic is the sky-averaged A, measuring mean local curvature anisotropy in the low-ℓ map.

Result Summary

Planck mean anisotropy index = 0.4930; simulation mean ± std = 0.4921 ± 0.0042. The Planck value lies at the 59.4th percentile of ΛCDM simulations (rank 594/1000). z-score = 0.22, two-tailed p-value = 0.812.

The Planck low-ℓ map exhibits mean Hessian anisotropy fully consistent with ΛCDM expectations. The statistic falls comfortably within the interquartile range of simulations, indicating no preference for either enhanced or suppressed directional bias in local curvature structure.

Agent Justification

The test is designed to measure the average degree of local curvature anisotropy by comparing the magnitudes of the two principal curvature directions (Hessian eigenvalues). There is no strong theoretical prior predicting the CMB should be specifically more or less anisotropic than ΛCDM; the anomaly could manifest as either more directionally biased curvature (A closer to 1) or more isotropic curvature (A closer to 0). The provided `summarize_results` function explicitly calculates both upper and lower p-values and identifies anomalies on either side of the distribution (e.g., "higher mean... anisotropy" or "lower mean... anisotropy"), confirming a two-tailed approach is intended for general exploration of this statistic.

Comparison To Literature

Previous online searches returned no prior studies employing a Hessian eigenvalue-based anisotropy index on CMB temperature maps. Related work on CMB statistical isotropy (e.g., Ghosh et al. 2007, bipolar power spectrum methods) addresses different observables and techniques. No directly comparable statistic was identified.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Mean Hessian Anisotropy Index. "
    "Smooth the temperature map to ℓ_max=32. At each unmasked pixel, compute the "
    "covariant Hessian matrix ∇_i ∇_j T via spherical harmonic transforms. "
    "Diagonalize the Hessian to obtain eigenvalues λ₁ and λ₂. For each pixel, "
    "compute the anisotropy index A = | |λ₁| - |λ₂| | / (|λ₁| + |λ₂| + ε), "
    "where ε = 10⁻³ × global mean of (|λ₁| + |λ₂|). The scalar statistic is "
    "the sky-averaged A over all unmasked pixels. This measures the average degree "
    "of local curvature anisotropy in the low-ℓ map."
)


def analyze_map(m):
    """Return the mean Hessian anisotropy index at low ℓ."""
    nside = hp.npix2nside(len(m))
    lmax = 32

    # Replace masked pixels with 0.0 for spherical harmonic transforms
    valid = np.isfinite(m)
    m_filled = np.where(valid, m, 0.0)

    # Get a_ℓm coefficients (band-limited to ℓ_max=32)
    alm = hp.map2alm(m_filled, lmax=lmax)

    # Get ℓ for each alm index (vectorised)
    l_arr, _ = hp.Alm.getlm(lmax)
    ell = l_arr.astype(np.float64)

    # --- Laplacian: ∇²T has harmonic factor −ℓ(ℓ+1) ---
    lap_alm = alm * (-ell * (ell + 1.0))

    # --- Trace-free Hessian via spin-2 field ð²T ---
    # c_ℓ = sqrt(ℓ(ℓ+1)(ℓ+2)(ℓ-1)), nonzero only for ℓ ≥ 2
    c_ell = np.zeros_like(ell)
    mask2 = l_arr >= 2
    ls = ell[mask2]
    c_ell[mask2] = np.sqrt(ls * (ls + 1.0) * (ls + 2.0) * (ls - 1.0))
    e_alm = alm * c_ell
    b_alm = np.zeros_like(alm)

    # Single alm2map call with pol=True:
    #   input  = (T_alm, E_alm, B_alm)
    #   output = (T_map, Q_map, U_map)
    # We use lap_alm as T_alm to get the Laplacian map directly.
    lap_map, Q_map, U_map = hp.alm2map(
        [lap_alm, e_alm, b_alm], nside, lmax=lmax, pol=True
    )

    # --- Eigenvalues of the 2×2 Hessian in orthonormal frame ---
    # H = [[L/2 + Q/2,  U/2],
    #      [U/2,        L/2 - Q/2]]
    # eigenvalues = L/2 ± sqrt(Q² + U²)/2
    half_L = lap_map / 2.0
    half_R = np.hypot(Q_map, U_map) / 2.0

    lambda1 = half_L + half_R
    lambda2 = half_L - half_R

    # --- Anisotropy index A at each pixel ---
    abs_l1 = np.abs(lambda1)
    abs_l2 = np.abs(lambda2)
    sum_abs = abs_l1 + abs_l2

    # Regularisation constant: ε = 10⁻³ × mean(|λ₁| + |λ₂|) over valid pixels
    mean_sum = np.mean(sum_abs[valid])
    if mean_sum < 1e-30:
        return 0.0
    epsilon = 1e-3 * mean_sum

    A = np.abs(abs_l1 - abs_l2) / (sum_abs + epsilon)

    # Sky-average over unmasked pixels
    result = float(np.mean(A[valid]))

    if not np.isfinite(result):
        return 0.0
    return result


def summarize_results(planck_stat, sim_results):
    """Interpret the Low-ℓ Mean Hessian Anisotropy Index."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)
    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    median_s = float(np.median(sims))
    rank = int(np.sum(sims <= planck_stat))
    p_upper = float(np.mean(sims >= planck_stat))
    p_lower = float(np.mean(sims <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else 0.0
    pct_rank = rank / n * 100 if n > 0 else 50.0

    if pct_rank < 2.5 or pct_rank > 97.5:
        sig_desc = "highly anomalous (<2.5% or >97.5% of sims)"
    elif pct_rank < 5 or pct_rank > 95:
        sig_desc = "marginally anomalous (<5% or >95% of sims)"
    elif pct_rank < 25 or pct_rank > 75:
        sig_desc = "noteworthy (outside IQR of sims)"
    else:
        sig_desc = "typical (within IQR of sims)"

    if planck_stat > mean_s + std_s:
        aniso_desc = (
            "higher mean Hessian anisotropy than typical LCDM (curvature is more "
            "directionally biased / elongated)"
        )
    elif planck_stat < mean_s - std_s:
        aniso_desc = (
            "lower mean Hessian anisotropy than typical LCDM (curvature is more "
            "isotropic than expected)"
        )
    else:
        aniso_desc = "Hessian anisotropy consistent with LCDM expectations"

    return {
        "planck_stat": planck_stat,
        "hessian_anisotropy_index": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "sim_median": median_s,
        "z_score": z_score,
        "rank": rank,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "percentile_rank": pct_rank,
        "anisotropy_description": aniso_desc,
        "significance_description": sig_desc,
        "interpretation": (
            f"Mean Hessian anisotropy index = {planck_stat:.6f}. {aniso_desc}. "
            f"Sim mean ± std = {mean_s:.6f} ± {std_s:.6f}. "
            f"{sig_desc}. z = {z_score:.2f}, two-sided p = {p_two:.4f}."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Mean Hessian Anisotropy Index",
            "xlabel": "Mean Anisotropy Index A",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck anisotropy index = {planck_stat:.6f}. "
                f"Sim mean ± std = {mean_s:.6f} ± {std_s:.6f}. "
                f"z = {z_score:.2f}, P(two-sided) = {p_two:.4f}."
            ),
        },
    }
