Hypothesis

There is an asymmetry between the number of hot and cold spots in the low-ℓ CMB temperature map (R ≠ 0), which may indicate a departure from isotropy or Gaussianity in the large-scale CMB.

Method

The number of local maxima (hot spots) and local minima (cold spots) are identified in the low-ℓ (ℓ ≤ 32) Planck temperature map using an 8-neighbor stencil on the HEALPix grid. The asymmetry ratio R = (N_hot − N_cold) / (N_hot + N_cold) quantifies the imbalance between hot and cold extrema. The statistic is compared against 1000 isotropic ΛCDM simulations.

Result Summary

Planck yields R = −0.0065. The simulation mean is −0.0011 ± 0.0316, placing the observed value at −0.17σ from the mean (43.8th percentile). The two-tailed p-value is 0.88.

The hot and cold spot counts in the large-scale CMB are well-balanced, fully consistent with isotropic Gaussian ΛCDM expectations. The slight excess of cold spots over hot spots is negligible and well within the typical simulation scatter. No evidence for a preferred direction or handedness in the low-ℓ temperature pattern is found.

Agent Justification

The test statistic R measures the imbalance between hot and cold spots, and can be positive or negative. The code computes a two-sided p-value by considering both tails, and the interpretation handles deviations in either direction without a prior directional expectation. Since the anomaly could manifest as an excess of either hot or cold spots, a two-tailed test is appropriate for exploratory analysis.

Comparison To Literature

No directly comparable studies using this specific formal statistic were found. Related work includes CMB peak count statistics and studies of the CMB Cold Spot anomaly, but the formal asymmetry ratio between hot and cold local extrema at low multipoles does not appear to have been previously investigated as a dedicated anomaly test.

import numpy as np
import healpy as hp

test_description = (
    "Hot-Cold Spot Count Asymmetry Ratio: "
    "Compute the number of local maxima (hot spots) and local minima (cold spots) "
    "in the low-ℓ (ℓ ≤ 32) temperature map. A spot is a pixel whose temperature is "
    "higher (or lower) than all its immediate neighbors using an 8-neighbor stencil "
    "on the HEALPix grid. The statistic is R = (N_hot - N_cold) / (N_hot + N_cold), "
    "measuring the asymmetry between hot and cold extrema in the large-scale CMB field. "
    "An anomalous R indicates an imbalance in the number of hot versus cold peaks, "
    "which could reflect a preferred direction or handedness in the large-scale temperature pattern."
)


def analyze_map(m):
    """Compute the hot-cold spot count asymmetry ratio R for the low-ℓ temperature map.

    Steps:
      1. Identify unmasked (finite) pixels; bail out if too few.
      2. Fill masked pixels with 0 for healpy compatibility.
      3. Band-limit the map to ℓ ≤ 32 via spherical harmonic transform.
      4. For each unmasked pixel, find all valid (unmasked) HEALPix neighbors via
         hp.get_all_neighbours (8-neighbor stencil).
      5. Identify local maxima (hot spots) and local minima (cold spots): a pixel
         must be strictly greater (or less) than ALL of its valid neighbors.
      6. Compute R = (N_hot - N_cold) / (N_hot + N_cold).

    Returns
    -------
    float
        Asymmetry ratio R in [-1, 1]. Returns 0.0 if no valid spots are found.
    """
    mask = np.isfinite(m)
    n_valid = np.sum(mask)

    if n_valid < 100:
        return 0.0

    # Fill masked pixels with 0 for healpy transforms
    m_filled = np.where(mask, m, 0.0)

    # Band-limit to ℓ ≤ 32 via spherical harmonics
    lmax = 32
    alm = hp.map2alm(m_filled, lmax=lmax)
    nside = hp.get_nside(m_filled)
    m_low = hp.alm2map(alm, nside, lmax=lmax, verbose=False)

    npix = hp.nside2npix(nside)

    # Get up to 8 neighbors for every pixel (shape: 8 × npix)
    # Missing neighbors are marked with -1
    all_neighbors = hp.get_all_neighbours(nside, np.arange(npix))

    # Mask indicating which neighbor entries are valid (>= 0)
    valid_nb = all_neighbors >= 0

    # Look up neighbor temperature values; use -inf for missing so they never
    # block a pixel from being a hot spot, and +inf so they never block a cold spot
    # (but we also track which pixels have at least one valid neighbor)
    has_any_valid = np.any(valid_nb, axis=0)

    # Neighbor values: where invalid, use +inf for hot-spot test (never blocks)
    # and -inf for cold-spot test (never blocks)
    nb_vals_hot = np.where(valid_nb, m_low[np.clip(all_neighbors, 0, npix - 1)], np.inf)
    nb_vals_cold = np.where(valid_nb, m_low[np.clip(all_neighbors, 0, npix - 1)], -np.inf)

    # Vectorized comparison: pixel must be strictly above ALL valid neighbors (hot spot)
    is_hot = np.all(m_low > nb_vals_hot, axis=0) & mask & has_any_valid

    # Pixel must be strictly below ALL valid neighbors (cold spot)
    is_cold = np.all(m_low < nb_vals_cold, axis=0) & mask & has_any_valid

    N_hot = int(np.sum(is_hot))
    N_cold = int(np.sum(is_cold))
    total = N_hot + N_cold

    if total == 0:
        return 0.0

    return float((N_hot - N_cold) / total)


def summarize_results(planck_stat, sim_results):
    """Summarise the Hot-Cold Spot Count Asymmetry Ratio test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available for comparison.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Hot-Cold Spot Count Asymmetry Ratio",
                "xlabel": "Asymmetry ratio R",
                "ylabel": "Number of simulations",
                "bins": 40,
                "caption": "No simulation data available.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    # Two-sided empirical p-value
    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims

    if std_sim > 0:
        deviation_sigma = (planck_stat - mean_sim) / std_sim
    else:
        deviation_sigma = 0.0

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    if planck_stat > mean_sim + 2 * std_sim:
        direction = (
            "Planck's large-scale temperature field has significantly more hot spots "
            "than cold spots compared to typical ΛCDM realizations. This asymmetry "
            "could indicate a preferred direction or anomalous power distribution "
            "in the low-ℓ CMB."
        )
    elif planck_stat < mean_sim - 2 * std_sim:
        direction = (
            "Planck's large-scale temperature field has significantly more cold spots "
            "than hot spots compared to typical ΛCDM realizations. This asymmetry "
            "could indicate a preferred direction or anomalous power distribution "
            "in the low-ℓ CMB."
        )
    else:
        direction = (
            "The hot-cold spot count is consistent with isotropic Gaussian ΛCDM "
            "expectations; the number of local maxima and minima in the low-ℓ "
            "temperature map are balanced as expected."
        )

    interp = (
        f"Asymmetry ratio R = {planck_stat:+.4f}. "
        f"Simulation mean = {mean_sim:+.4f} ± {std_sim:.4f} "
        f"({deviation_sigma:+.2f}σ from mean). "
        f"Percentile = {percentile:.1f}%, two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of the hot-cold spot count asymmetry ratio R = (N_hot - N_cold) / "
        "(N_hot + N_cold) computed from the ℓ ≤ 32 band-limited temperature map of "
        "isotropic ΛCDM simulations. Local maxima and minima are identified using "
        "an 8-neighbor stencil on the HEALPix grid. R near 0 indicates balanced "
        "hot and cold extrema; R near ±1 indicates all extrema are of one type. "
        "The red line marks the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": deviation_sigma,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Hot-Cold Spot Count Asymmetry Ratio",
            "xlabel": "Asymmetry ratio R",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
