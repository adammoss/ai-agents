Hypothesis

The skewness of the Gaussian curvature K of the low-ℓ (ℓ ≤ 20) CMB temperature field is significantly different from zero, indicating a departure from the symmetric distribution expected for a statistically isotropic Gaussian random field.

Method

Compute the Gaussian curvature K = T_θθ·T_φφ − (T_θφ)² of the low-ℓ (ℓ ≤ 20) SMICA temperature surface at each pixel (nside=256) via finite-difference Hessian approximations, then measure the spatial skewness γ_K of K across unmasked pixels. For a Gaussian random field, the distribution of K is symmetric (γ_K ≈ 0); significant skewness indicates preferential concentration of elliptic (peak/valley) versus saddle-point regions, probing geometric non-Gaussianity in contour topology.

Result Summary

Planck γ_K = −60.18. Simulation mean = −98.02 ± 74.19 (z = +0.51σ). Percentile = 86.3%, two-tailed p = 0.274. Anomaly strength: negligible.

The Gaussian curvature skewness of the low-ℓ temperature field is fully consistent with ΛCDM Gaussian expectations. The balance between elliptic and saddle-point regions shows no significant anomaly in the geometric structure of temperature contours. The result lies well within the simulation envelope.

Agent Justification

The test is designed to detect any significant deviation (positive or negative) from the null expectation of zero skewness, as stated in the description: "For a statistically isotropic Gaussian random field, the distribution of K is symmetric and γ_K ≈ 0; a significant departure indicates preferential concentration of either convex/concave structure or saddle-like structure." There is no prior directional hypothesis (e.g., expecting an excess of peaks over saddle points); it is a general geometric non-Gaussianity probe. The code also computes a two-sided p-value, confirming the exploratory, two-tailed nature.

Comparison To Literature

Barreiro et al. (2000, astro-ph/0004218) studied the Gaussian curvature of CMB excursion sets as a non-Gaussianity estimator and found it effective for detecting perturbations with skewness/kurtosis. However, that work examined the topology of excursion set surfaces at specific thresholds, whereas this test measures the skewness of the pixel-by-pixel Gaussian curvature of the temperature field itself—a related but distinct statistic. No prior studies using this specific skewness-of-K approach were identified in the literature.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Gaussian Curvature Skewness: Compute the Hessian matrix of the "
    "temperature field T(θ,φ) at each pixel on the low-ℓ SMICA map (ℓ ≤ 20) "
    "on nside=256, using finite-difference approximations for the second spatial "
    "derivatives (T_θθ, T_φφ, T_θφ). At each pixel, calculate the Gaussian "
    "curvature K = T_θθ · T_φφ − (T_θφ)², which measures the intrinsic curvature "
    "of the temperature surface as a 2D manifold embedded in 3D space. K > 0 "
    "indicates elliptic points (concave or convex regions), while K < 0 indicates "
    "saddle points. Then compute the skewness of K across all unmasked pixels: "
    "γ_K = ⟨(K − ⟨K⟩)³⟩ / σ_K³. For a statistically isotropic Gaussian random "
    "field, γ_K ≈ 0; a significant departure indicates preferential concentration "
    "of either convex/concave structure or saddle-like structure."
)


def analyze_map(m):
    """Compute skewness of Gaussian curvature K of the low-ℓ temperature surface."""
    mask = np.isfinite(m)
    n_valid = int(np.sum(mask))
    if n_valid < 500:
        return 0.0

    nside = hp.npix2nside(len(m))
    npix = len(m)
    lmax = 20

    # Replace NaN with 0 for healpy operations
    clean = np.where(mask, m, 0.0)

    # Reconstruct band-limited map (ℓ ≤ 20)
    alm = hp.map2alm(clean, lmax=lmax, iter=0)
    m_low = hp.alm2map(alm, nside, lmax=lmax, verbose=False)

    # Get pixel coordinates (colatitude θ, longitude φ)
    theta, phi = hp.pix2ang(nside, np.arange(npix))

    # Get all 8 neighbors for every pixel
    # hp.get_all_neighbours returns (8, npix) when given an array of pixels
    neighbors = hp.get_all_neighbours(nside, np.arange(npix))  # shape: (8, npix)

    # Extract neighbor indices for cardinal and diagonal directions
    # HEALPix neighbor ordering: 0=SW, 1=S, 2=SE, 3=W, 4=E, 5=NW, 6=N, 7=NE
    n_idx = neighbors[6]   # North
    s_idx = neighbors[1]   # South
    e_idx = neighbors[4]   # East
    w_idx = neighbors[3]   # West
    ne_idx = neighbors[7]  # North-East
    nw_idx = neighbors[5]  # North-West
    se_idx = neighbors[2]  # South-East
    sw_idx = neighbors[0]  # South-West

    # Create safe indices (clip to valid range for array access)
    n_safe = np.clip(n_idx, 0, npix - 1)
    s_safe = np.clip(s_idx, 0, npix - 1)
    e_safe = np.clip(e_idx, 0, npix - 1)
    w_safe = np.clip(w_idx, 0, npix - 1)
    ne_safe = np.clip(ne_idx, 0, npix - 1)
    nw_safe = np.clip(nw_idx, 0, npix - 1)
    se_safe = np.clip(se_idx, 0, npix - 1)
    sw_safe = np.clip(sw_idx, 0, npix - 1)

    # Create masks for valid neighbors (exists AND unmasked)
    valid_n = (n_idx >= 0) & mask[n_safe]
    valid_s = (s_idx >= 0) & mask[s_safe]
    valid_e = (e_idx >= 0) & mask[e_safe]
    valid_w = (w_idx >= 0) & mask[w_safe]
    valid_ne = (ne_idx >= 0) & mask[ne_safe]
    valid_nw = (nw_idx >= 0) & mask[nw_safe]
    valid_se = (se_idx >= 0) & mask[se_safe]
    valid_sw = (sw_idx >= 0) & mask[sw_safe]

    # Pixels where all 4 cardinal neighbors are valid
    valid_hess = mask & valid_n & valid_s & valid_e & valid_w

    # Pixels where all 8 neighbors are valid (for diagonal derivatives)
    valid_diag = valid_hess & valid_ne & valid_nw & valid_se & valid_sw

    # Get neighbor temperature values
    T_n = m_low[n_safe]
    T_s = m_low[s_safe]
    T_e = m_low[e_safe]
    T_w = m_low[w_safe]
    T_ne = m_low[ne_safe]
    T_nw = m_low[nw_safe]
    T_se = m_low[se_safe]
    T_sw = m_low[sw_safe]
    T_c = m_low

    # Get neighbor coordinates
    theta_n = theta[n_safe]; phi_n = phi[n_safe]
    theta_s = theta[s_safe]; phi_s = phi[s_safe]
    theta_e = theta[e_safe]; phi_e = phi[e_safe]
    theta_w = theta[w_safe]; phi_w = phi[w_safe]
    theta_ne = theta[ne_safe]; phi_ne = phi[ne_safe]
    theta_nw = theta[nw_safe]; phi_nw = phi[nw_safe]
    theta_se = theta[se_safe]; phi_se = phi[se_safe]
    theta_sw = theta[sw_safe]; phi_sw = phi[sw_safe]

    # Compute step sizes for finite differences
    # For colatitude (θ): north has larger θ
    dtheta_n = theta_n - theta
    dtheta_s = theta - theta_s
    # Average step size for central differences
    dtheta = 0.5 * (dtheta_n + dtheta_s)
    dtheta = np.where(dtheta > 1e-10, dtheta, 1e-10)

    # For longitude (φ): handle wrap-around
    dphi_e = phi_e - phi
    dphi_e = (dphi_e + np.pi) % (2 * np.pi) - np.pi
    dphi_w = phi - phi_w
    dphi_w = (dphi_w + np.pi) % (2 * np.pi) - np.pi
    dphi = 0.5 * (np.abs(dphi_e) + np.abs(dphi_w))
    dphi = np.where(dphi > 1e-10, dphi, 1e-10)

    # Second derivatives using central finite differences
    # T_θθ = (T_n - 2*T_c + T_s) / dtheta²
    T_tt = (T_n - 2.0 * T_c + T_s) / (dtheta ** 2)

    # T_φφ = (T_e - 2*T_c + T_w) / dphi²
    T_pp = (T_e - 2.0 * T_c + T_w) / (dphi ** 2)

    # Mixed derivative T_θφ using diagonal neighbors
    # For diagonal step sizes, use actual coordinate differences
    dtheta_diag = 0.5 * ((theta_ne - theta_sw) + (theta_nw - theta_se))
    dphi_diag_ne = phi_ne - phi_nw
    dphi_diag_ne = (dphi_diag_ne + np.pi) % (2 * np.pi) - np.pi
    dphi_diag_se = phi_se - phi_sw
    dphi_diag_se = (dphi_diag_se + np.pi) % (2 * np.pi) - np.pi
    dphi_diag = 0.5 * (np.abs(dphi_diag_ne) + np.abs(dphi_diag_se))
    dphi_diag = np.where(dphi_diag > 1e-10, dphi_diag, 1e-10)
    dtheta_diag = np.where(np.abs(dtheta_diag) > 1e-10, dtheta_diag, dtheta)

    # Mixed derivative using 4 diagonal neighbors: (NE - NW - SE + SW) / (4 * dθ * dφ)
    T_tp_diag = (T_ne - T_nw - T_se + T_sw) / (4.0 * dtheta_diag * dphi_diag)

    # Fallback: use cardinal neighbors when diagonals unavailable
    # T_θφ ≈ (T_n - T_s)(T_e - T_w) / (4 dθ dφ)
    T_tp_cardinal = ((T_n - T_s) * (T_e - T_w)) / (4.0 * dtheta * dphi)

    # Use diagonal version where available, cardinal otherwise
    T_tp = np.where(valid_diag, T_tp_diag, T_tp_cardinal)

    # Gaussian curvature: K = T_θθ * T_φφ - (T_θφ)²
    K = T_tt * T_pp - T_tp ** 2

    # Only use pixels where Hessian is valid (all 4 cardinal neighbors valid)
    K_valid = K[valid_hess]

    if len(K_valid) < 100:
        return 0.0

    # Compute skewness: γ_K = ⟨(K - ⟨K⟩)³⟩ / σ_K³
    K_mean = np.mean(K_valid)
    K_centered = K_valid - K_mean
    K_std = np.std(K_valid)

    if K_std < 1e-30:
        return 0.0

    skewness = float(np.mean(K_centered ** 3) / (K_std ** 3))

    if not np.isfinite(skewness):
        return 0.0

    return skewness


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ℓ Gaussian Curvature Skewness test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Gaussian Curvature Skewness",
                "xlabel": "γ_K (skewness of Gaussian curvature)",
                "ylabel": "Count",
                "bins": 40,
                "caption": "No simulation data.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    # Two-sided p-value
    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims
    deviation_sigma = (planck_stat - mean_sim) / std_sim if std_sim > 0 else 0.0

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    if planck_stat > mean_sim + 2 * std_sim:
        direction = (
            "Planck shows a significant positive skewness in the Gaussian curvature "
            "of the low-ℓ temperature field, beyond what ΛCDM simulations predict. "
            "This indicates an excess of elliptic (convex/concave) points relative "
            "to saddle points in the temperature surface, suggesting preferential "
            "concentration of peaks or troughs — a geometric non-Gaussianity "
            "signature that may reflect intrinsic anomalies or systematic effects."
        )
    elif planck_stat < mean_sim - 2 * std_sim:
        direction = (
            "Planck shows a significant negative skewness in the Gaussian curvature "
            "of the low-ℓ temperature field, beyond what ΛCDM simulations predict. "
            "This indicates an excess of saddle points relative to elliptic "
            "(convex/concave) points, suggesting the temperature surface has "
            "unusually complex contour topology — a geometric non-Gaussianity "
            "probe that may indicate departures from Gaussian random field statistics."
        )
    else:
        direction = (
            "The Gaussian curvature skewness of the low-ℓ temperature field is "
            "consistent with ΛCDM Gaussian expectations. The balance between "
            "elliptic (convex/concave) and saddle-point regions shows no significant "
            "anomaly in the geometric structure of temperature contours."
        )

    interp = (
        f"γ_K = {planck_stat:.4f}. "
        f"Sim mean = {mean_sim:.4f} ± {std_sim:.4f} "
        f"({deviation_sigma:+.2f}σ). Percentile = {percentile:.1f}%, "
        f"two-sided p = {p_value:.4f}. "
        f"Strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of the Gaussian curvature skewness γ_K for the low-ℓ "
        "(ℓ ≤ 20) CMB temperature field. K = T_θθ·T_φφ − (T_θφ)² is computed "
        "from finite-difference Hessian approximations at each pixel. K > 0 "
        "corresponds to elliptic points (peaks/valleys); K < 0 to saddle points. "
        "The skewness γ_K = ⟨(K−⟨K⟩)³⟩/σ_K³ measures the asymmetry between "
        "these regions. For a Gaussian random field, γ_K ≈ 0. Red line marks "
        "the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": deviation_sigma,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Gaussian Curvature Skewness",
            "xlabel": "γ_K (skewness of Gaussian curvature)",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
