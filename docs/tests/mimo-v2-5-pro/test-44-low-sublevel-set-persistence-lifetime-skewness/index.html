Hypothesis

The Planck CMB temperature map has a persistence lifetime skewness statistic that differs from the mean of the LCDM simulations.

Method

Band-pass filter the Planck CMB temperature map to retain multipoles 2 ≤ ℓ ≤ 30, then apply a sublevel-set filtration on HEALPix pixel centres, tracking connected components as the temperature threshold rises. Each component's persistence lifetime (death − birth temperature) quantifies its topological significance. The skewness of positive lifetimes captures whether the low-ℓ field is dominated by many short-lived features (negative skewness) or a few very persistent large-scale structures (positive skewness).

Result Summary

Planck persistence lifetime skewness = 1.631. Simulation mean = 1.585 ± 0.264. The Planck value deviates by only +0.17σ from the LCDM simulation mean, yielding a two-tailed p-value of 0.788. Anomaly strength is negligible.

The persistence lifetime skewness of the Planck low-ℓ map is fully consistent with Gaussian LCDM expectations. There is no evidence that large-scale topological features (e.g., the cold spot, hemispherical asymmetry) produce an unusual excess of persistent connected components relative to the simulation ensemble.

Agent Justification

The test is designed to explore any significant deviation in the skewness statistic, which can be either positive (indicating dominance of a few persistent large-scale features) or negative (indicating dominance of many short-lived features). The analysis code explicitly checks for both directions by computing a two-sided p-value and interpreting anomaly strength for both high and low values. There is no strong prior directional prediction stated; the test is general exploration for an anomaly. Therefore, a two-tailed test is appropriate.

Comparison To Literature

Prior online searches for persistent homology applied to CMB anomaly detection returned no directly relevant studies. The general statistical theory of persistent homology has been explored (e.g., Bubenik & Kim 2007), but application of persistence lifetime skewness to CMB maps as an anomaly test does not appear in the literature to my knowledge.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Sublevel-Set Persistence Lifetime Skewness: "
    "Band-pass filter the CMB temperature map to retain multipoles 2 ≤ ℓ ≤ 30. "
    "Build a sublevel-set filtration on HEALPix pixel centres, tracking connected "
    "components as the temperature threshold rises from minimum to maximum. "
    "Each component's persistence lifetime (death − birth temperature) quantifies "
    "its topological significance. The skewness of positive lifetimes captures "
    "whether short-lived small-scale features dominate (negative skewness) or "
    "a few very persistent large-scale structures dominate (positive skewness). "
    "This statistic probes the spatial organization of temperature extrema and "
    "is sensitive to features like the CMB cold spot and hemispherical asymmetry."
)


def analyze_map(m):
    """Low-ℓ Sublevel-Set Persistence Lifetime Skewness.

    Efficient implementation: band-pass filter at full nside, then downgrade
    to nside=32 for fast topological computation (band-limit ℓ≤30 means
    nside=32 captures all spatial information). Uses MST + array-based
    union-find with path halving for 0-dim persistent homology.
    """
    from scipy.sparse import csr_matrix
    from scipy.sparse.csgraph import minimum_spanning_tree

    nside = hp.get_nside(m)
    npix = 12 * nside ** 2

    mask = np.isfinite(m)
    n_valid = int(np.sum(mask))
    if n_valid < 100:
        return 0.0

    # Step 1: Band-pass filter to ℓ = 2–30 at original nside
    clean = np.where(mask, m, 0.0)
    lmax = 30
    alm = hp.map2alm(clean, lmax=lmax)
    # Zero monopole (ℓ=0) and dipole (ℓ=1)
    alm[0] = 0.0
    alm[1] = 0.0
    alm[2] = 0.0
    filtered = hp.alm2map(alm, nside, lmax=lmax, verbose=False)

    # Step 2: Downgrade to nside=32 for topology computation
    # The band-limit ℓ≤30 means nside=32 (Nyquist ℓ≈96) captures all info
    tns = 32
    tnpix = 12 * tns ** 2

    filt_filled = np.where(mask, filtered, 0.0)
    filt_ds = hp.ud_grade(filt_filled, tns)
    wt_ds = hp.ud_grade(mask.astype(np.float64), tns)

    # Mask coarse pixels where too many fine pixels were masked
    ok = wt_ds > 0.5

    # Correct for partial masking (divide by weight to recover true mean)
    corrected = np.full(tnpix, np.nan)
    corrected[ok] = filt_ds[ok] / wt_ds[ok]

    # Step 3: Build edge list from HEALPix adjacency at target nside
    vpix = np.where(ok)[0]
    nv = len(vpix)
    if nv < 50:
        return 0.0

    temps = corrected[vpix]

    p2i = np.full(tnpix, -1, dtype=np.int32)
    p2i[vpix] = np.arange(nv, dtype=np.int32)

    all_neigh = hp.get_all_neighbours(tns, np.arange(tnpix))

    es_parts = []
    ed_parts = []
    for d in range(8):
        nd = all_neigh[d, vpix]
        vm = (nd >= 0) & ok[nd]
        s = np.where(vm)[0]
        d_ = p2i[nd[s]]
        k = s < d_
        es_parts.append(s[k])
        ed_parts.append(d_[k])

    es = np.concatenate(es_parts)
    ed = np.concatenate(ed_parts)

    if len(es) == 0:
        return 0.0

    # Edge weights: max of endpoint temperatures (sublevel-set filtration value)
    wts = np.maximum(temps[es], temps[ed])

    # Step 4: Compute MST
    graph = csr_matrix((wts, (es, ed)), shape=(nv, nv), dtype=np.float64)
    graph = graph + graph.T
    mst = minimum_spanning_tree(graph)
    mc = mst.tocoo()

    if len(mc.data) == 0:
        return 0.0

    # Sort MST edges by weight (filtration order)
    order = np.argsort(mc.data)
    ms = mc.row[order].astype(np.intp)
    md = mc.col[order].astype(np.intp)
    mw = mc.data[order]

    # Step 5: Array-based Union-Find with path halving
    parent = np.arange(nv, dtype=np.intp)
    rank = np.zeros(nv, dtype=np.int32)
    birth = temps.copy()

    lt_buf = np.empty(len(mw), dtype=np.float64)
    n_lt = 0

    for k in range(len(mw)):
        u = int(ms[k])
        v = int(md[k])

        # Find root of u with path halving
        while parent[u] != u:
            parent[u] = parent[parent[u]]
            u = int(parent[u])
        # Find root of v with path halving
        while parent[v] != v:
            parent[v] = parent[parent[v]]
            v = int(parent[v])

        if u == v:
            continue

        # Union: younger component (higher birth temp) dies
        bu = birth[u]
        bv = birth[v]

        if bu > bv:
            parent[u] = v
            lt = mw[k] - bu
        elif bv > bu:
            parent[v] = u
            lt = mw[k] - bv
        else:
            # Equal birth: pick one as survivor
            parent[v] = u
            lt = mw[k] - bv
            if rank[u] == rank[v]:
                rank[u] += 1

        if lt > 0:
            lt_buf[n_lt] = lt
            n_lt += 1

    # Step 6: Compute sample skewness of positive lifetimes
    if n_lt < 3:
        return 0.0

    pos = lt_buf[:n_lt]
    mu = np.mean(pos)
    sigma = np.std(pos)
    if sigma <= 0:
        return 0.0

    skew = float(np.mean(((pos - mu) / sigma) ** 3))
    return skew if np.isfinite(skew) else 0.0


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ℓ Sublevel-Set Persistence Lifetime Skewness test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Persistence Lifetime Skewness",
                "xlabel": "Skewness",
                "ylabel": "Count",
                "bins": 40,
                "caption": "No simulation data.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))
    deviation_sigma = (planck_stat - mean_sim) / std_sim if std_sim > 0 else 0.0

    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    if planck_stat > mean_sim + 2 * std_sim:
        direction = (
            "Planck shows significantly more positively skewed persistence "
            "lifetimes than LCDM simulations, indicating that a few very "
            "persistent large-scale structures dominate the low-ℓ field — "
            "consistent with features like the CMB cold spot or "
            "hemispherical asymmetry."
        )
    elif planck_stat < mean_sim - 2 * std_sim:
        direction = (
            "Planck shows significantly less positive (or more negative) "
            "persistence lifetime skewness than LCDM simulations, suggesting "
            "that short-lived small-scale topological features are "
            "unusually dominant."
        )
    else:
        direction = (
            "The persistence lifetime skewness is consistent with Gaussian "
            "LCDM expectations."
        )

    interp = (
        f"Persistence lifetime skewness = {planck_stat:.4f}. "
        f"Simulation mean = {mean_sim:.4f} ± {std_sim:.4f} "
        f"({deviation_sigma:+.2f}σ). Percentile = {percentile:.1f}%, "
        f"two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of the skewness of persistence lifetimes from a "
        "sublevel-set filtration of the low-ℓ (ℓ=2–30) CMB temperature "
        "field. Positive skewness indicates a few very persistent "
        "large-scale topological features dominate (long right tail), "
        "while negative skewness indicates many short-lived features "
        "bunched near zero. Red line marks the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": deviation_sigma,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Persistence Lifetime Skewness",
            "xlabel": "Skewness of persistence lifetimes",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
