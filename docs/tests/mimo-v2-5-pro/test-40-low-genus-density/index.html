Hypothesis

The genus density (n_hot - n_cold)/(4π) of the CMB temperature map at large angular scales (ℓ ≤ 32) deviates from the expected distribution under the standard ΛCDM Gaussian isotropic model.

Method

Computes the topological asymmetry between hot and cold connected regions in the CMB temperature map at a single zero-temperature threshold, defined as genus density G/(4π) = (n_hot − n_cold)/(4π). The map is band-limited to ℓ ≤ 32. This compact scalar summary is sensitive to non-Gaussian features distinct from peak-count or moment-based tests.

Result Summary

The Planck genus density is 0.637, while 1000 ΛCDM simulations yield a mean of 0.010 ± 0.770. The Planck value falls at the 78.1st percentile (z = +0.81σ), yielding a two-sided p-value of 0.438. Anomaly strength: negligible.

The asymmetry between hot and cold connected regions in the low-ℓ CMB is fully consistent with Gaussian isotropic ΛCDM expectations. There is no indication of topological excess in either hot or cold connected components at large angular scales.

Agent Justification

The test statistic is fundamentally symmetric—it can be positive (more hot regions) or negative (more cold regions)—and the anomaly could plausibly manifest in either direction (e.g., an excess of hot or cold spots). The analysis code explicitly computes a two-sided p‑value (taking twice the minimum of the upper and lower tail probabilities), and there is no stated prior expectation for a specific direction in the description. Thus, the test is exploratory, checking for any deviation without directional preference.

Comparison To Literature

The genus statistic is well-established in CMB topology studies (e.g., Gott, Melott & Dickinson 1986; Park et al. 2001; Ducos et al. 2020), typically computed at multiple threshold levels to reconstruct the full genus curve across the temperature distribution. However, no published studies were found specifically applying a single zero-threshold genus density to low-ℓ CMB maps. The search returned no directly relevant CMB genus results, suggesting this particular formulation has limited prior coverage.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Genus Density: "
    "Compute the genus number of the CMB temperature map at a zero-temperature "
    "threshold, defined as G = N_hot − N_cold, where N_hot and N_cold are the "
    "numbers of connected hot (T > 0) and cold (T < 0) regions respectively. "
    "Divide by the total sky area in steradians to obtain genus density. "
    "This topological statistic measures the asymmetry between hot and cold "
    "connected components, providing a compact summary sensitive to "
    "non-Gaussian features in the large-scale temperature distribution."
)


def analyze_map(m):
    """Compute genus density: (n_hot - n_cold) / (4π) at zero threshold."""
    from scipy.sparse import csr_matrix
    from scipy.sparse.csgraph import connected_components
    
    nside = hp.get_nside(m)
    npix = hp.nside2npix(nside)

    mask = np.isfinite(m)
    n_valid = int(np.sum(mask))
    if n_valid < 100:
        return 0.0

    vals = m.copy()
    vals[~mask] = 0.0

    # Band-limit to ell <= 32 for low-ℓ analysis
    alm = hp.map2alm(vals, lmax=32)
    vals = hp.alm2map(alm, nside, lmax=32, verbose=False)
    vals[~mask] = 0.0

    # Classify pixels
    is_hot = mask & (vals > 0)
    is_cold = mask & (vals < 0)

    if not np.any(is_hot) and not np.any(is_cold):
        return 0.0

    # Vectorized adjacency construction using scipy sparse
    # Get neighbors: shape (8, npix)
    neighbors = hp.get_all_neighbours(nside, np.arange(npix))

    rows_list = []
    cols_list = []
    for i in range(8):
        nbrs = neighbors[i]
        # Valid: both pixel and neighbor are unmasked, and neighbor index is valid
        valid = mask & (nbrs >= 0) & mask[nbrs]
        # Same sign: both hot or both cold
        same = valid & ((is_hot & is_hot[nbrs]) | (is_cold & is_cold[nbrs]))
        idx = np.where(same)[0]
        rows_list.append(idx)
        cols_list.append(nbrs[idx])

    if len(rows_list) > 0:
        rows = np.concatenate(rows_list)
        cols = np.concatenate(cols_list)
    else:
        rows = np.array([], dtype=int)
        cols = np.array([], dtype=int)

    # Build sparse adjacency matrix (symmetric, so one direction suffices for connectivity)
    data = np.ones(len(rows), dtype=np.int8)
    graph = csr_matrix((data, (rows, cols)), shape=(npix, npix))

    # Find connected components
    n_components, labels = connected_components(graph, directed=False)

    # Count hot and cold components
    hot_labels = labels[is_hot]
    cold_labels = labels[is_cold]

    n_hot_components = len(np.unique(hot_labels)) if len(hot_labels) > 0 else 0
    n_cold_components = len(np.unique(cold_labels)) if len(cold_labels) > 0 else 0

    genus = n_hot_components - n_cold_components
    sky_area = 4.0 * np.pi
    genus_density = float(genus) / sky_area

    return genus_density if np.isfinite(genus_density) else 0.0


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ℓ Genus Density test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Genus Density",
                "xlabel": "Genus density G / (4π)",
                "ylabel": "Count",
                "bins": 40,
                "caption": "No simulation data.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims
    deviation_sigma = (planck_stat - mean_sim) / std_sim if std_sim > 0 else 0.0

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    if planck_stat > mean_sim + 2 * std_sim:
        direction = (
            "Planck has significantly more hot connected regions than cold "
            "connected regions compared to LCDM simulations, suggesting a "
            "topological asymmetry inconsistent with Gaussian isotropy at "
            "large scales."
        )
    elif planck_stat < mean_sim - 2 * std_sim:
        direction = (
            "Planck has significantly fewer hot connected regions relative "
            "to cold connected regions compared to LCDM simulations, "
            "indicating an unusual cold-spot topology at large scales."
        )
    else:
        direction = (
            "The asymmetry between hot and cold connected regions is "
            "consistent with Gaussian LCDM expectations."
        )

    interp = (
        f"Genus density (n_hot - n_cold) / (4π) = {planck_stat:.6f}. "
        f"Simulation mean = {mean_sim:.6f} ± {std_sim:.6f} "
        f"({deviation_sigma:+.2f}σ). Percentile = {percentile:.1f}%, "
        f"two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of genus density G / (4π), where G = n_hot − n_cold "
        "counts the difference between connected hot (T > 0) and cold "
        "(T < 0) regions in the zero-thresholded low-ℓ CMB map. "
        "Positive values indicate more hot regions; negative values "
        "indicate more cold regions. Red line marks the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": deviation_sigma,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Genus Density",
            "xlabel": "Genus density G / (4π)",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
