Hypothesis

The ratio R = mean(C_ℓ, ℓ=2–5) / mean(C_ℓ, ℓ=6–10) deviates from the distribution predicted by the standard ΛCDM cosmological model, indicating an unexpected tilt or curvature in the primordial power spectrum at very low multipoles.

Method

Computes the ratio R = mean(C_ℓ, ℓ=2–5) / mean(C_ℓ, ℓ=6–10) from the angular power spectrum, testing for unexpected spectral tilt or curvature in the primordial power spectrum at the lowest multipoles compared to ΛCDM FFP simulations (1000 realisations).

Result Summary

Planck R = 3.613, simulation mean = 5.953 ± 2.519 (median = 5.483). Two-sided p = 0.29 (z-score ≈ 1.06σ). The observed ratio is lower than the simulation mean, indicating slightly less relative power in ℓ = 2–5 compared to ℓ = 6–10, but well within expected statistical fluctuations.

The Planck low-ℓ spectral tilt ratio is consistent with ΛCDM expectations. The modest deficit in the lower band relative to simulations (~1σ below the mean) is unremarkable and does not constitute evidence for an anomalous spectral index or curvature at very low multipoles. No compelling deviation from the standard model is detected.

Agent Justification

The test description and analysis code are explicitly designed to detect any unexpected deviation in the spectral ratio R, whether it is anomalously high (excess low-ℓ power) or anomalously low (low-ℓ power deficit). The code calculates a two-sided p-value by considering both the upper and lower tails of the simulation distribution and classifies the anomaly direction after the fact. This is a general exploratory test for spectral anomalies, not a targeted test for a specific predicted directional change, so a two-tailed approach is appropriate.

Comparison To Literature

Previous work has extensively studied low-ℓ anomalies including the quadrupole–octupole alignment (Copi et al. 2004) and the overall low-ℓ power deficit. The specific band-power ratio with this binning does not appear to be a standard statistic in the literature, making direct comparison difficult. The result is broadly consistent with prior findings that low-ℓ power is somewhat suppressed but not significantly anomalous.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Spectral Tilt Ratio. Computes the angular power spectrum C_ℓ for "
    "multipoles ℓ = 2 through 20 using healpy anafast. The statistic is the ratio "
    "R = mean(C_ℓ, ℓ=2–5) / mean(C_ℓ, ℓ=6–10). In standard ΛCDM with a nearly "
    "scale-invariant primordial spectrum, C_ℓ at these low multipoles follows the "
    "Sachs–Wolfe plateau with modest structure from the first acoustic peak rising "
    "around ℓ ≈ 7–8. An anomalously low ratio R would indicate an unexpected dip "
    "or suppression at the very lowest multipoles (the 'low-ℓ power deficit'), "
    "while an anomalously high ratio could signal excess large-scale power or an "
    "unusual spectral tilt. Evaluated by comparing the Planck value against "
    "FFP ΛCDM simulations."
)


def analyze_map(m):
    m = np.asarray(m, dtype=np.float64)

    # Replace NaN (masked) pixels with 0.0 for healpy transforms
    m_clean = np.where(np.isfinite(m), m, 0.0)

    # Compute angular power spectrum up to ℓ = 10
    cl = hp.anafast(m_clean, lmax=10)

    # cl array indices: cl[0] = C_0, cl[1] = C_1, ..., cl[10] = C_10
    # We need ℓ = 2 through 5 and ℓ = 6 through 10
    lower_mean = np.mean(cl[2:6])   # C_2, C_3, C_4, C_5
    upper_mean = np.mean(cl[6:11])  # C_6, C_7, C_8, C_9, C_10

    # Guard against division by zero or tiny denominators
    if upper_mean <= 0.0 or not np.isfinite(upper_mean):
        return 1.0  # fallback neutral value

    ratio = lower_mean / upper_mean

    if not np.isfinite(ratio):
        return 1.0

    return float(ratio)


def summarize_results(planck_stat, sim_results):
    sim_arr = np.array(sim_results, dtype=np.float64)
    finite_mask = np.isfinite(sim_arr)
    n_finite = int(finite_mask.sum())

    if n_finite > 0:
        valid = sim_arr[finite_mask]
        # Two-sided p-value: fraction of simulations as extreme or more extreme
        sim_median = float(np.median(valid))
        # Compute one-sided p-values in both directions
        p_upper = float(np.sum(valid >= planck_stat) / n_finite)
        p_lower = float(np.sum(valid <= planck_stat) / n_finite)
        p_value = min(p_upper, p_lower)
        rank_upper = int(np.sum(valid >= planck_stat))
        rank_lower = int(np.sum(valid <= planck_stat))
    else:
        p_value = float('nan')
        rank_upper = 0
        rank_lower = 0
        sim_median = float('nan')

    sim_mean = float(np.nanmean(sim_arr))
    sim_std = float(np.nanstd(sim_arr))

    # Classify anomaly direction
    if n_finite > 0 and not np.isnan(p_value):
        if p_upper <= p_lower:
            direction = "high"
            rank = rank_upper
        else:
            direction = "low"
            rank = rank_lower
    else:
        direction = "unknown"
        rank = 0

    interpretation_parts = []
    interpretation_parts.append(
        f"Low-ℓ spectral tilt ratio R = mean(C_ℓ, ℓ=2–5) / mean(C_ℓ, ℓ=6–10) "
        f"= {planck_stat:.4f}."
    )
    interpretation_parts.append(
        f"Two-sided p = {p_value:.3f} ({rank}/{n_finite} simulations "
        f"at least as extreme in the {direction} direction)."
    )
    interpretation_parts.append(
        f"Simulation mean = {sim_mean:.4f}, std = {sim_std:.4f}, "
        f"median = {sim_median:.4f}."
    )
    if not np.isnan(p_value) and p_value < 0.05:
        if direction == "low":
            interpretation_parts.append(
                "Anomalous: the ratio is unusually LOW, suggesting excess power in "
                "ℓ = 6–10 relative to ℓ = 2–5 — consistent with the known CMB "
                "low-ℓ power deficit or an unexpected spectral upturn at the "
                "lowest multipoles."
            )
        else:
            interpretation_parts.append(
                "Anomalous: the ratio is unusually HIGH, suggesting excess power in "
                "ℓ = 2–5 relative to ℓ = 6–10 — indicating an unusual spectral "
                "tilt or suppression of power just above the quadrupole."
            )
    else:
        interpretation_parts.append(
            "The ratio is consistent with ΛCDM simulation expectations."
        )

    result = {
        "test_description": test_description,
        "planck_stat": float(planck_stat),
        "p_value": float(p_value),
        "rank": rank,
        "n_simulations": len(sim_results),
        "n_valid": n_finite,
        "sim_mean": sim_mean,
        "sim_std": sim_std,
        "sim_median": sim_median,
        "exceeds_95th_percentile": bool(p_value < 0.05) if not np.isnan(p_value) else False,
        "anomaly_direction": direction,
        "interpretation": " ".join(interpretation_parts),
        "lower_band_range": "ℓ = 2–5",
        "upper_band_range": "ℓ = 6–10",
    }

    result["plot_spec"] = {
        "kind": "histogram",
        "title": "Low-ℓ Spectral Tilt Ratio",
        "xlabel": "R = mean(C_ℓ, ℓ=2–5) / mean(C_ℓ, ℓ=6–10)",
        "ylabel": "Number of simulations",
        "bins": 50,
        "caption": (
            f"Histogram of the low-ℓ spectral tilt ratio R = "
            f"mean(C_ℓ, ℓ=2–5) / mean(C_ℓ, ℓ=6–10) from {n_finite} "
            f"FFP ΛCDM simulations. "
            f"Planck value = {planck_stat:.4f} "
            f"(two-sided p = {p_value:.3f}, rank {rank}/{n_finite}, "
            f"{direction} direction). "
            f"Simulation mean = {sim_mean:.4f} ± {sim_std:.4f}. "
            f"A low ratio signals the low-ℓ power deficit; a high ratio "
            f"suggests excess large-scale power."
        ),
    }

    return result
