Hypothesis

The mean eccentricity of saddle points in the low-ℓ (ℓ≤30) CMB temperature map is consistent with the distribution from LCDM simulations.

Method

The SMICA temperature map is band-limited to ℓ ≤ 30 and saddle points are identified as pixels where the Hessian has negative determinant. At each saddle, the Hessian is diagonalised and an eccentricity e = 1 − |λ₂|/|λ₁| is computed (0 = circular, 1 = maximally elongated). The mean eccentricity over all saddles is compared against 1000 ΛCDM simulations to test whether real saddle points show anomalous elongation or compression.

Result Summary

Observed mean eccentricity = 0.5768. Simulation mean = 0.5696 ± 0.0044, yielding z = +1.62σ (94.2nd percentile, two-sided p = 0.116). No strong departure from the ΛCDM distribution is present.

The observed saddle-point eccentricity sits comfortably within the simulation envelope. At +1.62σ it is mildly elevated but well within the expected scatter, indicating that the local curvature geometry at saddle-type critical points in the low-ℓ CMB is consistent with Gaussian random-field expectations.

Agent Justification

The test assesses whether saddle points are anomalously elongated or compressed, indicating a two-tailed investigation. The code computes both upper and lower p-values and the interpretation considers deviations in both directions. There is no stated directional preference.

Comparison To Literature

No prior studies measuring the eccentricity of CMB saddle points could be identified in the literature. Saddle-point curvature geometry has not been previously explored in CMB anomaly analyses, unlike extrema eccentricity or genus statistics.

import numpy as np
import healpy as hp

test_description = (
    "Band-limit the SMICA temperature map to ell <= 30. Identify all saddle points "
    "as pixels where the Hessian of the filtered map has negative determinant "
    "(det H < 0), computed via finite differences on the HEALPix grid. At each "
    "saddle point, diagonalise the Hessian to obtain eigenvalues lambda1 and lambda2 "
    "(with |lambda1| >= |lambda2| > 0). Define the eccentricity of each saddle as "
    "e = 1 - |lambda2|/|lambda1|, which equals 0 for a perfectly circular saddle and "
    "approaches 1 for a highly elongated one. Average e over all detected saddle "
    "points to yield a single scalar. This statistic probes the local curvature "
    "geometry of the temperature field at its saddle-type critical points and is "
    "topologically distinct from genus density and from extrema eccentricity."
)


def analyze_map(m):
    """Return mean eccentricity of saddle points in the ell<=30 band-limited map."""
    mask = np.isfinite(m)
    n_valid = int(np.sum(mask))

    if n_valid < 500:
        return 0.0

    # Fill masked pixels with 0 for healpy compatibility
    m_filled = np.where(mask, m, 0.0)

    # Band-limit to ell <= 30
    lmax = 30
    alm = hp.map2alm(m_filled, lmax=lmax, iter=0)

    # Work at nside=32 (sufficient resolution for lmax=30, keeps things fast)
    nside = 32
    npix = hp.nside2npix(nside)  # 12288
    m_smooth = hp.alm2map(alm, nside, lmax=lmax, verbose=False)

    # Pixel centers
    theta, phi = hp.pix2ang(nside, np.arange(npix))
    sin_theta = np.sin(theta)

    # Get 8 neighbors for each pixel (shape: 8 x npix)
    neighbors = hp.get_all_neighbours(nside, np.arange(npix))
    valid_nbr = (neighbors >= 0)  # (8, npix)
    safe_nbr = np.where(valid_nbr, neighbors, 0)  # safe index for gathering

    # Gather neighbor data
    nbr_theta = theta[safe_nbr]  # (8, npix)
    nbr_phi = phi[safe_nbr]
    nbr_vals = m_smooth[safe_nbr]

    # Local tangent-plane displacements
    dtheta = nbr_theta - theta[None, :]  # (8, npix)
    dphi = nbr_phi - phi[None, :]
    # Handle azimuthal wraparound
    dphi = (dphi + np.pi) % (2 * np.pi) - np.pi

    # Tangent plane coordinates (npix, 8)
    dx = dtheta.T
    dy = (sin_theta[None, :] * dphi).T
    dv = (nbr_vals - m_smooth[None, :]).T  # residuals from center value

    # Validity weights (npix, 8): 1 for valid neighbors, 0 for missing
    w = valid_nbr.T.astype(np.float64)

    # Zero out invalid neighbor contributions (no NaN propagation)
    dx = dx * w
    dy = dy * w
    dv = dv * w

    # Build weighted feature matrix for quadratic fit:
    #   dv ≈ b·dx + c·dy + d·dx² + e·dx·dy + f·dy²
    # Stack features: (npix, 8, 5)
    features = np.stack([dx, dy, dx ** 2, dx * dy, dy ** 2], axis=-1)
    wf = features * w[:, :, None]   # weighted features
    wdv = dv * w                     # weighted residuals

    # Normal equations: (wfᵀ wf) c = wfᵀ wdv
    AtA = np.einsum('pji,pjk->pik', wf, wf)  # (npix, 5, 5)
    Atb = np.einsum('pji,pj->pi', wf, wdv)    # (npix, 5)

    # Tiny regularisation for numerical stability
    AtA += 1e-12 * np.eye(5)[None, :, :]

    # Solve via matrix inverse (avoids gufunc shape issues with np.linalg.solve)
    try:
        inv_AtA = np.linalg.inv(AtA)  # (npix, 5, 5)
    except np.linalg.LinAlgError:
        return 0.0

    coeffs = (inv_AtA @ Atb[:, :, None]).squeeze(-1)  # (npix, 5)

    # Hessian from quadratic coefficients: H = [[2d, e], [e, 2f]]
    H_xx = 2.0 * coeffs[:, 2]
    H_xy = coeffs[:, 3]
    H_yy = 2.0 * coeffs[:, 4]

    det_H = H_xx * H_yy - H_xy ** 2

    # Select saddle points: det(H) < 0, away from poles where tangent plane is ill-defined
    away_from_poles = (theta > 0.2) & (theta < np.pi - 0.2)
    is_saddle = away_from_poles & (det_H < 0)

    saddle_idx = np.where(is_saddle)[0]
    n_saddles = len(saddle_idx)

    if n_saddles < 5:
        return 0.0

    # Eigenvalues at saddle points
    Hxx_s = H_xx[saddle_idx]
    Hxy_s = H_xy[saddle_idx]
    Hyy_s = H_yy[saddle_idx]

    trace = Hxx_s + Hyy_s
    det_s = Hxx_s * Hyy_s - Hxy_s ** 2
    disc = np.maximum(trace ** 2 - 4.0 * det_s, 0.0)
    sqrt_disc = np.sqrt(disc)

    lam1 = (trace + sqrt_disc) / 2.0
    lam2 = (trace - sqrt_disc) / 2.0

    abs1 = np.abs(lam1)
    abs2 = np.abs(lam2)
    max_abs = np.maximum(abs1, abs2)
    min_abs = np.minimum(abs1, abs2)

    # Eccentricity: e = 1 - |λ₂|/|λ₁|
    valid_eig = max_abs > 0
    eccentricity = np.where(valid_eig, 1.0 - min_abs / max_abs, 0.0)
    eccentricity = eccentricity[np.isfinite(eccentricity)]

    if len(eccentricity) < 5:
        return 0.0

    return float(np.mean(eccentricity))


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ell Saddle-Point Mean Eccentricity test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Saddle-Point Mean Eccentricity",
                "xlabel": "Mean eccentricity e = 1 − |λ₂|/|λ₁|",
                "ylabel": "Count",
                "bins": 50,
                "caption": "No simulation data.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    p_value_upper = float(np.mean(sim >= planck_stat))
    p_value_lower = float(np.mean(sim <= planck_stat))
    p_value_two_sided = min(2.0 * min(p_value_upper, p_value_lower), 1.0)

    rank_above = int(np.sum(sim >= planck_stat))
    rank_below = int(np.sum(sim <= planck_stat))
    percentile = 100.0 * rank_below / n_sims
    deviation_sigma = (planck_stat - mean_sim) / std_sim if std_sim > 0 else 0.0

    if p_value_two_sided <= 0.01:
        strength = "strong"
    elif p_value_two_sided <= 0.05:
        strength = "moderate"
    else:
        strength = "negligible"

    interp = (
        f"Mean saddle-point eccentricity = {planck_stat:.4f}. "
        f"Sim mean = {mean_sim:.4f} ± {std_sim:.4f} "
        f"({deviation_sigma:+.2f}σ). Percentile = {percentile:.1f}%, "
        f"two-sided p = {p_value_two_sided:.4f}. Strength: {strength}. "
    )

    if planck_stat > mean_sim + 2 * std_sim:
        interp += (
            "The Planck map's saddle points are anomalously elongated compared to "
            "Gaussian LCDM expectations, suggesting large-scale anisotropic features "
            "in the low-ℓ temperature field with pronounced directional curvature."
        )
    elif planck_stat < mean_sim - 2 * std_sim:
        interp += (
            "The Planck map's saddle points are anomalously circular (compressed) "
            "compared to Gaussian LCDM expectations, suggesting an unusually isotropic "
            "curvature geometry at saddle-type critical points."
        )
    else:
        interp += (
            "The mean eccentricity of saddle points is consistent with Gaussian "
            "LCDM expectations. No significant anomaly detected in the curvature "
            "geometry of saddle-type critical points at low multipoles."
        )

    caption = (
        "Histogram of mean saddle-point eccentricity e = 1 − |λ₂|/|λ₁| "
        "in the low-ℓ (ℓ≤30) band-limited CMB temperature map across "
        "LCDM simulations. Higher values indicate more elongated saddle points. "
        "The red line marks the Planck observed value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value_one_sided_upper": p_value_upper,
        "p_value_one_sided_lower": p_value_lower,
        "p_value_two_sided": p_value_two_sided,
        "rank_above": rank_above,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": deviation_sigma,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Saddle-Point Mean Eccentricity",
            "xlabel": "Mean eccentricity e = 1 − |λ₂|/|λ₁|",
            "ylabel": "Number of simulations",
            "bins": 50,
            "caption": caption,
        },
    }
