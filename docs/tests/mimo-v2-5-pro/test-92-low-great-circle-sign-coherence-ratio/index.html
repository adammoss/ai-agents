Hypothesis

The Planck CMB temperature map exhibits a statistically significant higher low-ℓ great circle sign-coherence ratio than expected from a statistically isotropic, Gaussian ΛCDM model, indicating an anomalous large-scale spatial coherence.

Method

For a band-limited temperature map (ℓ_max = 32), 1000 great circles are randomly selected. Along each circle, 360 equidistant temperature samples are drawn and the proportion of consecutive sample pairs sharing the same sign (both positive or both negative) is computed. Averaging over all circles yields a single scalar measuring large-scale spatial coherence of temperature anomalies.

Result Summary

Planck sign-coherence ratio = 0.9312, compared to simulation mean ± std = 0.9359 ± 0.0037 (z = −1.29, two-sided p = 0.176). The Planck value ranks at the 8.8th percentile of simulations—below the interquartile range but in the *opposite* direction from the hypothesis, indicating slightly *lower* coherence than typical ΛCDM realizations rather than higher.

The Planck low-ℓ temperature map shows no excess sign-coherence along great circles. If anything, it exhibits marginally *more* rapid sign changes than typical simulations, though not at a statistically significant level. The result provides no support for a coherent, dipole-like large-scale structure at low multipoles.

Agent Justification

The test description explicitly states that "a significantly high value would indicate a coherent, possibly dipole-like, large-scale structure," establishing a clear directional expectation. The test's purpose is specifically to detect excessive spatial coherence along great circles, which manifests as a ratio significantly above the baseline expectation of ~0.5 for an isotropic field. Therefore, the analysis should focus on the upper tail of the simulated distribution to identify whether the observed Planck statistic is unusually high.

Comparison To Literature

No prior studies specifically measuring great-circle sign-coherence ratios were found in the literature. Related probes of low-ℓ anomalies include multipole alignment, planarity, and hemispherical asymmetry (e.g., Gurzadyan et al. 2003), but none employ this sign-persistence statistic directly. The search yielded no directly comparable results.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Great Circle Sign-Coherence Ratio. "
    "For a band-limited temperature map (ℓ_max=32), randomly select 1000 great circles. "
    "For each great circle, sample the temperature values along its perimeter at 360 "
    "equidistant points. Compute the proportion of consecutive sample pairs (along the "
    "circle) that share the same sign (both positive or both negative relative to the "
    "zero mean of the full map). Average this proportion over all great circles to yield "
    "a single scalar. This statistic quantifies the large-scale spatial coherence of "
    "temperature anomalies by measuring how consistently the sign of the fluctuation "
    "persists along random great circles, as opposed to rapid sign changes expected in "
    "a statistically isotropic Gaussian field. A significantly high value would indicate "
    "a coherent, possibly dipole-like, large-scale structure."
)


def analyze_map(m):
    """Return the average great-circle sign-coherence ratio at low ℓ."""
    nside = hp.npix2nside(len(m))
    lmax = 32
    n_circles = 1000
    n_samples = 360

    # Replace masked pixels with 0.0 for spherical harmonic transforms
    valid = np.isfinite(m)
    m_filled = np.where(valid, m, 0.0)

    # Band-limit to ℓ_max=32 via forward and inverse SHT
    alm = hp.map2alm(m_filled, lmax=lmax)
    m_band = hp.alm2map(alm, nside, lmax=lmax)

    # Subtract the mean of valid pixels (monopole removal)
    valid_mean = np.mean(m_band[valid])
    m_band = m_band - valid_mean

    # Set up random state with fixed seed for reproducibility
    rng = np.random.RandomState(12345)

    # Precompute sampling angles along the great circle
    angles = np.linspace(0, 2 * np.pi, n_samples, endpoint=False)

    # Generate random normal vectors uniformly on the sphere
    # (defines the plane of each great circle)
    phi_normal = rng.uniform(0, 2 * np.pi, n_circles)
    cos_theta_normal = rng.uniform(-1, 1, n_circles)
    sin_theta_normal = np.sqrt(1 - cos_theta_normal**2)

    # Normal vectors in Cartesian
    nx = sin_theta_normal * np.cos(phi_normal)
    ny = sin_theta_normal * np.sin(phi_normal)
    nz = cos_theta_normal

    # Build orthonormal basis (u, v) in the plane perpendicular to n
    # Choose initial vector that's not parallel to n
    abs_n = np.abs(np.column_stack([nx, ny, nz]))
    # Use (0,0,1) or (1,0,0) whichever is less parallel
    use_x = abs_n[:, 2] > abs_n[:, 0]  # True if |nz| > |nx|, use (1,0,0)
    e1_x = np.where(use_x, 1.0, 0.0)
    e1_z = np.where(use_x, 0.0, 1.0)
    e1_y = np.zeros(n_circles)

    # u = n × e1 (cross product)
    ux = ny * e1_z - nz * e1_y
    uy = nz * e1_x - nx * e1_z
    uz = nx * e1_y - ny * e1_x

    # Normalize u
    u_norm = np.sqrt(ux**2 + uy**2 + uz**2)
    ux /= u_norm
    uy /= u_norm
    uz /= u_norm

    # v = n × u
    vx = ny * uz - nz * uy
    vy = nz * ux - nx * uz
    vz = nx * uy - ny * ux

    # Compute coherent ratios for all circles
    coherence_ratios = np.zeros(n_circles)

    for i in range(n_circles):
        # Points along the great circle: p(θ) = cos(θ)*u + sin(θ)*v
        cos_a = np.cos(angles)
        sin_a = np.sin(angles)

        px = cos_a * ux[i] + sin_a * vx[i]
        py = cos_a * uy[i] + sin_a * vy[i]
        pz = cos_a * uz[i] + sin_a * vz[i]

        # Convert Cartesian to spherical coordinates for healpy
        theta = np.arccos(np.clip(pz, -1.0, 1.0))
        phi = np.arctan2(py, px) % (2 * np.pi)

        # Get pixel indices and sample the band-limited map
        pix = hp.ang2pix(nside, theta, phi)
        samples = m_band[pix]

        # Compute sign coherence: proportion of consecutive pairs with same sign
        # Use circular adjacency: pair (i, i+1) and wrap (n-1, 0)
        s = np.sign(samples)
        # Consecutive product: positive if same sign, negative if different
        prod = s * np.roll(s, -1)
        # Count same-sign pairs (product > 0), ignoring zeros
        coherence_ratios[i] = np.mean(prod > 0)

    # Average coherence over all great circles
    result = float(np.mean(coherence_ratios))

    if not np.isfinite(result):
        return 0.0

    return result


def summarize_results(planck_stat, sim_results):
    """Interpret the Low-ℓ Great Circle Sign-Coherence Ratio."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)
    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    median_s = float(np.median(sims))
    rank = int(np.sum(sims <= planck_stat))
    p_upper = float(np.mean(sims >= planck_stat))
    p_lower = float(np.mean(sims <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else 0.0
    pct_rank = rank / n * 100 if n > 0 else 50.0

    if pct_rank < 2.5 or pct_rank > 97.5:
        sig_desc = "highly anomalous (<2.5% or >97.5% of sims)"
    elif pct_rank < 5 or pct_rank > 95:
        sig_desc = "marginally anomalous (<5% or >95% of sims)"
    elif pct_rank < 25 or pct_rank > 75:
        sig_desc = "noteworthy (outside IQR of sims)"
    else:
        sig_desc = "typical (within IQR of sims)"

    # For a Gaussian isotropic field, we expect coherence ~0.5
    # (equal probability of same or different sign for independent fluctuations)
    # Higher coherence indicates spatial correlation / large-scale structure
    if planck_stat > mean_s + std_s:
        coh_desc = "higher sign coherence than typical LCDM (possible large-scale coherent structure or dipole-like pattern)"
    elif planck_stat < mean_s - std_s:
        coh_desc = "lower sign coherence than typical LCDM (unusually rapid sign changes along great circles)"
    else:
        coh_desc = "sign coherence consistent with LCDM expectations"

    return {
        "planck_stat": planck_stat,
        "sign_coherence_ratio": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "sim_median": median_s,
        "z_score": z_score,
        "rank": rank,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "percentile_rank": pct_rank,
        "coherence_description": coh_desc,
        "significance_description": sig_desc,
        "interpretation": (
            f"Great circle sign-coherence ratio = {planck_stat:.6f}. {coh_desc}. "
            f"Sim mean ± std = {mean_s:.6f} ± {std_s:.6f}. "
            f"{sig_desc}. z = {z_score:.2f}, two-sided p = {p_two:.4f}."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Great Circle Sign-Coherence Ratio",
            "xlabel": "Sign-Coherence Ratio (proportion of same-sign consecutive pairs)",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck sign-coherence = {planck_stat:.6f}. "
                f"Sim mean ± std = {mean_s:.6f} ± {std_s:.6f}. "
                f"z = {z_score:.2f}, P(two-sided) = {p_two:.4f}."
            ),
        },
    }
