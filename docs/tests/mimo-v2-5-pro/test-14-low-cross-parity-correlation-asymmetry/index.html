Hypothesis

The Planck SMICA map exhibits unusually large coherent parity-odd phase correlations between adjacent low-ℓ multipoles (ℓ=2-10) compared to ΛCDM simulations, as measured by the mean |R_ℓ| statistic.

Method

Measures the mean of the absolute values of a dimensionless parity-odd cross-spectrum ratio, |R_ℓ|, computed for consecutive multipole pairs (ℓ, ℓ+1) from ℓ=2 to 9. This statistic quantifies coherent phase correlations between adjacent low-ℓ multipoles that would break parity symmetry.

Result Summary

The Planck SMICA map yielded a statistic T = 1.388. From 1000 ΛCDM simulations, the mean was 1.369 ± 0.379. The one-tailed p-value (upper) was 0.451, corresponding to a z-score of ~0.05σ.

The observed statistic is well within the range expected from ΛCDM simulations. There is no evidence for an excess of coherent parity-odd phase correlations between adjacent low-ℓ multipoles in the Planck data.

Agent Justification

The test statistic T is the mean of absolute values of R_ℓ, which is non-negative. The physical motivation is to detect an excess of parity-odd cross-correlations, which would manifest as T being significantly larger than expected from simulations. The test description explicitly states that non-zero T indicates coherent parity-odd phase correlations, implying interest in values larger than the null expectation. There is no physical reason to consider unusually small T as an anomaly; small T is expected under the null. Therefore, a one-tailed test focusing on the upper tail is appropriate.

Comparison To Literature

The test examines a distinct phase-correlation statistic. While related to known low-ℓ parity asymmetries found in power spectra and multipole alignments (e.g., Aluri et al. 2017, Hansen et al. 2011), this specific cross-parity correlation between adjacent ℓ has not been the primary focus of those studies.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Cross-Parity Correlation Asymmetry. Extracts spherical harmonic "
    "coefficients a_{ℓm} for ℓ = 2–10 from the Planck SMICA map (masked). "
    "For each consecutive multipole pair (ℓ, ℓ+1) with ℓ = 2–9, computes "
    "the parity-odd cross-correlation X_ℓ = Σ_{m≥0} Im[a_{ℓ,m} a*_{ℓ+1,m}] "
    "and forms the dimensionless ratio R_ℓ = X_ℓ / √(C_ℓ C_{ℓ+1}), where "
    "C_ℓ is the pseudo-power spectrum. The scalar statistic is the mean "
    "|R_ℓ| over the 8 pairs. Under ΛCDM with parity symmetry, Im[a_{ℓ,m} a*_{ℓ+1,m}] "
    "products are symmetrically distributed with zero mean, so T should be "
    "small. Non-zero T indicates coherent parity-odd phase correlations "
    "between adjacent multipoles — a signature not captured by power-based "
    "odd/even comparisons."
)


def analyze_map(m):
    m = np.asarray(m, dtype=np.float64)
    mask = np.isfinite(m)
    m_clean = np.where(mask, m, 0.0)

    lmin = 2
    lmax = 10
    n_pairs = lmax - lmin  # 8 pairs: (2,3) through (9,10)

    # Compute alm up to lmax
    alm = hp.map2alm(m_clean, lmax=lmax)

    # Compute pseudo-spectrum C_ell for ell = 2..10
    # hp.anafast returns pseudo-Cl with proper m>=0 weighting
    cl = hp.anafast(m_clean, lmax=lmax)

    # Compute the parity-odd cross-statistic
    R_values = []
    for l in range(lmin, lmax):  # l = 2..9
        # Sum Im[a_{l,m} * conj(a_{l+1,m})] for m = 0..l
        X_l = 0.0
        for mm in range(0, l + 1):
            idx_l = hp.Alm.getidx(lmax, l, mm)
            idx_lp1 = hp.Alm.getidx(lmax, l + 1, mm)
            a_lm = alm[idx_l]
            a_lp1_m = alm[idx_lp1]
            X_l += np.imag(a_lm * np.conj(a_lp1_m))

        C_l = cl[l]
        C_lp1 = cl[l + 1]

        denom = np.sqrt(C_l * C_lp1)
        if denom > 1e-30:
            R_l = X_l / denom
        else:
            R_l = 0.0

        R_values.append(np.abs(R_l))

    T = float(np.mean(R_values))
    return T


def summarize_results(planck_stat, sim_results):
    sim_arr = np.array(sim_results, dtype=np.float64)
    finite_mask = np.isfinite(sim_arr)
    n_finite = int(finite_mask.sum())

    if n_finite > 0:
        valid = sim_arr[finite_mask]
        sim_median = float(np.median(valid))
        p_upper = float(np.sum(valid >= planck_stat) / n_finite)
        p_lower = float(np.sum(valid <= planck_stat) / n_finite)
        p_value = min(p_upper, p_lower)
        rank_upper = int(np.sum(valid >= planck_stat))
        rank_lower = int(np.sum(valid <= planck_stat))
    else:
        p_value = float("nan")
        rank_upper = rank_lower = 0
        sim_median = float("nan")

    sim_mean = float(np.nanmean(sim_arr))
    sim_std = float(np.nanstd(sim_arr))

    if n_finite > 0 and not np.isnan(p_value):
        if p_upper <= p_lower:
            direction, rank = "high", rank_upper
        else:
            direction, rank = "low", rank_lower
    else:
        direction, rank = "unknown", 0

    # Build interpretation
    parts = [
        f"Mean |R_ℓ| (cross-parity correlation) = {planck_stat:.6f}. "
        f"Under ΛCDM with parity symmetry, Im[a_{{ℓ,m}} a*_{{ℓ+1,m}}] products "
        f"are symmetrically distributed with zero mean, so this statistic should be small.",
        f"Two-sided p = {p_value:.3f} ({rank}/{n_finite} simulations "
        f"at least as extreme in the {direction} direction).",
        f"Simulation mean = {sim_mean:.6f} ± {sim_std:.6f}, "
        f"median = {sim_median:.6f}.",
    ]

    if not np.isnan(p_value) and p_value < 0.05:
        parts.append(
            "Anomalous: the coherent parity-odd cross-correlation between adjacent "
            "low-ℓ multipoles is unusually large compared to ΛCDM simulations, "
            "suggesting unexpected phase correlations that break parity symmetry."
        )
    else:
        parts.append(
            "The low-ℓ cross-parity correlation is consistent with ΛCDM "
            "simulation expectations (parity symmetry preserved)."
        )

    result = {
        "test_description": test_description,
        "planck_stat": float(planck_stat),
        "p_value": float(p_value),
        "rank": rank,
        "n_simulations": len(sim_results),
        "n_valid": n_finite,
        "sim_mean": sim_mean,
        "sim_std": sim_std,
        "sim_median": sim_median,
        "exceeds_95th_percentile": bool(p_value < 0.05) if not np.isnan(p_value) else False,
        "anomaly_direction": direction,
        "interpretation": " ".join(parts),
        "ell_range": "2-10",
    }

    result["plot_spec"] = {
        "kind": "histogram",
        "title": "Low-ℓ Cross-Parity Correlation Asymmetry",
        "xlabel": "Mean |R_ℓ| (parity-odd cross-spectrum ratio)",
        "ylabel": "Number of simulations",
        "bins": 50,
        "caption": (
            f"Histogram of mean |R_ℓ| from {n_finite} ΛCDM simulations. "
            f"Planck value = {planck_stat:.6f} "
            f"(two-sided p = {p_value:.3f}, rank {rank}/{n_finite}, "
            f"{direction} tail). Simulation mean = {sim_mean:.6f} ± {sim_std:.6f}. "
            f"High values indicate coherent parity-odd phase correlations "
            f"between adjacent low-ℓ multipoles."
        ),
    }

    return result
