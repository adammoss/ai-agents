Hypothesis

The absolute ecliptic latitude |β| of the low-ℓ cold spot is anomalously extreme, either too close to or too far from the ecliptic plane compared to ΛCDM simulations.

Method

The temperature map is filtered to retain only multipoles ℓ ≤ 8, convolved with a 10° FWHM Gaussian beam, and masked with the Planck common mask. The pixel with the minimum temperature on the unmasked sky is identified, and its ecliptic latitude β is reported. This tests whether the location of the coldest large-scale feature is anomalously close to or far from the ecliptic plane under ΛCDM.

Result Summary

Planck SMICA β = −34.82° (|β| = 34.82°). Simulations yield mean |β| = 31.34° ± 21.64°. The observed value lies at the 63.3rd percentile of |β|. Two-sided p-value = 0.364; z-score = −0.95σ. Anomaly strength is negligible.

The ecliptic latitude of the low-ℓ cold spot is fully consistent with ΛCDM simulations. There is no evidence that the coldest large-scale feature is preferentially aligned with or displaced from the ecliptic plane. The Planck value sits near the centre of the simulated |β| distribution.

Agent Justification

The test is designed to detect anomalies in both directions (close to and far from the ecliptic plane), as evidenced by the calculation of two-sided p-values and the symmetric treatment of |β| in the analysis code. There is no clear directional prediction specified in the test description or code, making a two-tailed test appropriate for this exploratory anomaly check.

Comparison To Literature

No prior studies testing the ecliptic latitude of the low-ℓ cold spot specifically were identified. While the CMB cold spot (in Eridanus) and low-ℓ anomalies are extensively discussed—e.g., Planck 2018 VII (Isotropy and Statistics)—the ecliptic-latitude statistic of the bandpass-filtered cold pixel does not appear as a documented test. This makes a direct literature comparison difficult.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Cold Spot Ecliptic Latitude: Filter the temperature map to retain only "
    "multipoles ℓ ≤ 8, then convolve with a Gaussian beam of 10° FWHM to suppress "
    "pixel noise. Apply the Planck common mask. Identify the pixel with the minimum "
    "temperature on the unmasked sky and convert its coordinates from galactic to "
    "ecliptic. Report the ecliptic latitude β of this coldest pixel as the single "
    "scalar statistic. This tests whether the location of the lowest-temperature "
    "large-scale feature is anomalously close to or far from the ecliptic plane, "
    "which has no physical significance in the standard model. The statistic "
    "complements the existing 'Low-ℓ Cold Spot Depth' test (which measures the "
    "temperature magnitude) by probing the directional information instead. "
    "For nside=256, the 10° smoothing beam is well resolved (beam σ ≈ 4.2°, "
    "pixel size ≈ 0.23°), and the computation is dominated by the harmonic "
    "filtering and smoothing steps."
)


def analyze_map(m):
    """Compute the ecliptic latitude β of the coldest pixel after low-ℓ filtering and smoothing.

    Steps:
    1. Identify unmasked pixels; fill masked (NaN) with 0 for spherical-harmonic transforms.
    2. Decompose into alm up to lmax=8 (ℓ ≤ 8 bandpass).
    3. Smooth with a 10° FWHM Gaussian beam using healpy.smoothing.
    4. Find the pixel with the minimum smoothed temperature among unmasked pixels.
    5. Convert its galactic coordinates to ecliptic and return β (degrees).
    """
    nside = hp.npix2nside(len(m))

    # Identify valid (unmasked) pixels
    valid = np.isfinite(m)

    # Fill masked pixels with 0 for SH transforms
    m_filled = np.where(valid, m, 0.0).astype(np.float64)

    # --- Step 1: Bandpass filter to ℓ ≤ 8 via spherical harmonics ---
    lmax = 8
    alm = hp.map2alm(m_filled, lmax=lmax, iter=0)

    # --- Step 2: Smooth with 10° FWHM Gaussian beam ---
    # σ_beam = FWHM / (2√(2 ln 2)) ≈ 4.2466°, well resolved at nside=256
    fwhm_rad = np.radians(10.0)

    # Smooth via harmonic-space beam multiplication then map reconstruction
    # hp.smoothing with lmax=lmax ensures only ℓ ≤ 8 modes survive
    m_smooth = hp.smoothing(m_filled, fwhm=fwhm_rad, lmax=lmax, iter=0)

    # --- Step 3: Find the coldest unmasked pixel ---
    temps_valid = m_smooth[valid]
    idx_valid = np.where(valid)[0]

    # Vectorized argmin over unmasked pixels (avoids slow Python pixel scan)
    min_idx_local = np.argmin(temps_valid)
    cold_pix = idx_valid[min_idx_local]

    # --- Step 4: Convert galactic → ecliptic coordinates ---
    theta_gal, phi_gal = hp.pix2ang(nside, cold_pix)

    rot = hp.Rotator(coord=['G', 'E'])  # galactic to ecliptic
    theta_ecl, phi_ecl = rot(theta_gal, phi_gal)

    # Ecliptic latitude: β = 90° - colatitude
    beta_ecl = np.degrees(np.pi / 2.0 - theta_ecl)

    return float(beta_ecl)


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ℓ Cold Spot Ecliptic Latitude anomaly.

    The key question is whether |β| is anomalously small (near ecliptic plane)
    or anomalously large (near ecliptic poles). We treat |β| as the primary
    anomaly variable: a small |β| means the cold spot lies close to the
    ecliptic plane, which would be geometrically suspicious.
    """
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float('nan'),
            "p_value_low": float('nan'),
            "p_value_high": float('nan'),
            "rank": 0,
            "n_sims": 0,
            "percentile": float('nan'),
            "sim_mean_beta": float('nan'),
            "sim_std_beta": float('nan'),
            "planck_abs_beta": float('nan'),
            "sim_mean_abs_beta": float('nan'),
            "asymmetry_label": "unknown",
            "asymmetry_direction": "no simulations available",
            "interpretation": "No simulation results to compare.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Cold Spot Ecliptic Latitude",
                "xlabel": "Ecliptic latitude β (degrees)",
                "ylabel": "Number of simulations",
                "bins": 40,
                "caption": "No simulation data available.",
            },
        }

    # Signed β statistics
    mean_beta = float(np.mean(sim))
    std_beta = float(np.std(sim))

    # |β| statistics (anomaly proxy: smaller |β| = closer to ecliptic plane)
    abs_sim = np.abs(sim)
    abs_planck = abs(planck_stat)
    mean_abs_beta = float(np.mean(abs_sim))
    std_abs_beta = float(np.std(abs_sim))

    # One-sided p-value: fraction of simulations with |β| as small or smaller
    # (tests if Planck's cold spot is anomalously close to the ecliptic plane)
    p_value_close = float(np.mean(abs_sim <= abs_planck))

    # One-sided p-value: fraction with |β| as large or larger (far from plane)
    p_value_far = float(np.mean(abs_sim >= abs_planck))

    # Two-sided p-value: is |β| extreme in either direction?
    p_value_two = 2.0 * min(p_value_close, p_value_far)
    p_value_two = min(p_value_two, 1.0)  # clamp

    # Rank by |β| (low rank = close to ecliptic plane)
    rank_close = int(np.sum(abs_sim <= abs_planck))
    percentile_close = 100.0 * rank_close / n_sims

    # Signed rank (for reference)
    rank_signed = int(np.sum(sim < planck_stat))
    percentile_signed = 100.0 * rank_signed / n_sims

    # Anomaly strength based on proximity to ecliptic plane
    if percentile_close <= 5:
        strength = "strong"
        direction = (
            f"Planck's cold spot ecliptic latitude β = {planck_stat:.2f}° "
            f"(|β| = {abs_planck:.2f}°) is anomalously close to the ecliptic plane, "
            f"placing it at the {percentile_close:.1f}th percentile of |β| "
            "among ΛCDM simulations."
        )
    elif percentile_close <= 10:
        strength = "moderate"
        direction = (
            f"Planck's cold spot ecliptic latitude β = {planck_stat:.2f}° "
            f"(|β| = {abs_planck:.2f}°) is moderately close to the ecliptic plane, "
            f"placing it at the {percentile_close:.1f}th percentile of |β|."
        )
    elif p_value_far <= 0.05:
        strength = "strong"
        direction = (
            f"Planck's cold spot ecliptic latitude β = {planck_stat:.2f}° "
            f"(|β| = {abs_planck:.2f}°) is anomalously far from the ecliptic plane, "
            f"placing it at the {100 - percentile_close:.1f}th percentile of |β|."
        )
    elif p_value_far <= 0.10:
        strength = "moderate"
        direction = (
            f"Planck's cold spot ecliptic latitude β = {planck_stat:.2f}° "
            f"(|β| = {abs_planck:.2f}°) is moderately far from the ecliptic plane."
        )
    else:
        strength = "negligible"
        direction = (
            f"Planck's cold spot ecliptic latitude β = {planck_stat:.2f}° "
            f"(|β| = {abs_planck:.2f}°) is consistent with ΛCDM simulations "
            f"at the {percentile_close:.1f}th percentile of |β|."
        )

    interp = (
        "Low-ℓ (ℓ ≤ 8) cold spot ecliptic latitude β = {beta:.2f}°, "
        "corresponding to |β| = {abs_beta:.2f}° from the ecliptic plane. "
        "{direction} "
        "Simulated mean β = {mean_beta:.2f}° ± {std_beta:.2f}°; "
        "mean |β| = {mean_abs:.2f}° ± {std_abs:.2f}°. "
        "Two-sided p-value on |β| = {pval:.4f}. "
        "Anomaly strength: {strength}."
    ).format(
        beta=planck_stat,
        abs_beta=abs_planck,
        direction=direction,
        mean_beta=mean_beta,
        std_beta=std_beta,
        mean_abs=mean_abs_beta,
        std_abs=std_abs_beta,
        pval=p_value_two,
        strength=strength,
    )

    caption = (
        "Histogram of the ecliptic latitude β of the coldest pixel in the low-ℓ "
        "(ℓ ≤ 8, 10° FWHM smoothed) temperature map. Under ΛCDM the cold spot "
        "location is uniformly distributed over the sky (cos β weighting), so "
        "its ecliptic latitude carries no special information. An anomalously "
        "small |β| (close to the ecliptic plane) would suggest a foreground or "
        "scanning-strategy artifact. The red line marks the Planck SMICA value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value_two,
        "p_value_close_to_plane": p_value_close,
        "p_value_far_from_plane": p_value_far,
        "rank_close_to_plane": rank_close,
        "rank_signed": rank_signed,
        "n_sims": n_sims,
        "percentile_close_to_plane": percentile_close,
        "percentile_signed": percentile_signed,
        "planck_abs_beta": abs_planck,
        "sim_mean_beta": mean_beta,
        "sim_std_beta": std_beta,
        "sim_mean_abs_beta": mean_abs_beta,
        "sim_std_abs_beta": std_abs_beta,
        "asymmetry_label": strength,
        "asymmetry_direction": direction,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Cold Spot Ecliptic Latitude",
            "xlabel": "Ecliptic latitude β (degrees)",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
