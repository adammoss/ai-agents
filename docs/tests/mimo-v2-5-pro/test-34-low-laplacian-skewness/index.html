Hypothesis

The low-ℓ (ℓ=2–10) Laplacian skewness of the observed CMB temperature map deviates from zero, indicating a non-Gaussian asymmetry in the spatial curvature distribution between hot and cold regions.

Method

The angular Laplacian ∇²T is computed in harmonic space by multiplying each spherical harmonic coefficient T_{ℓm} (ℓ = 2–10) by −ℓ(ℓ+1), then synthesised to real space at nside=256. The skewness γ = ⟨(∇²T)³⟩ / ⟨(∇²T)²⟩^{3/2} measures asymmetry in the curvature distribution of the large-scale temperature pattern—e.g., whether cold spots have sharper curvature peaks than hot spots. This is distinct from raw temperature skewness, temperature–Laplacian cross-correlations, or gradient-magnitude statistics.

Result Summary

Planck skewness = −0.142; simulation mean = −0.002 ± 0.144. The observed value lies at −0.97σ from the simulation mean (percentile 16.2%), yielding a two-tailed p-value of 0.324. Anomaly strength is negligible.

The low-ℓ Laplacian skewness is fully consistent with isotropic Gaussian ΛCDM expectations. There is no evidence for asymmetric curvature structure between hot and cold regions at large angular scales. The result is unremarkable compared to the simulated ensemble.

Agent Justification

The test is exploratory and designed to detect any significant deviation from Gaussian expectations (γ=0) without a strong prior directional prediction. The code implements a two-sided empirical p-value and interprets both negative and positive skewness as potential anomalies, indicating a two-tailed framework is appropriate.

Comparison To Literature

CMB non-Gaussianity at large scales has been probed extensively using other statistics—temperature skewness, Minkowski functionals, bispectrum estimators, and multipole vectors—but the specific statistic of Laplacian-field skewness does not appear to have been previously studied. The present search returned no relevant prior work on this quantity. The result is consistent with the general finding that Planck large-scale data show no strong evidence for primordial non-Gaussianity of generic type.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Laplacian Skewness: "
    "Compute the angular Laplacian of the band-limited temperature map (ℓ = 2–10) "
    "and measure its skewness as a single scalar. The Laplacian is constructed in "
    "harmonic space by multiplying each T_{ℓm} coefficient by −ℓ(ℓ+1), then synthesized "
    "to real space at nside=256. The skewness γ = ⟨(∇²T)³⟩ / ⟨(∇²T)²⟩^{3/2} is "
    "computed over sky-area-weighted unmasked pixels. This statistic probes non-Gaussian "
    "spatial structure — specifically asymmetry in the curvature distribution of the "
    "large-scale temperature pattern (e.g., whether cold spots have sharper curvature "
    "than hot spots) — which is distinct from raw temperature skewness, temperature–"
    "Laplacian cross-correlation, and gradient-magnitude spatial variability."
)


def analyze_map(m):
    """Compute the skewness of the angular Laplacian of the low-ℓ temperature map.

    Steps:
      1. Identify unmasked (finite) pixels; bail out if too few.
      2. Fill masked pixels with 0 for healpy compatibility.
      3. Extract alm coefficients for ℓ = 2 through 10.
      4. Multiply each coefficient by −ℓ(ℓ+1) to get the Laplacian alm.
      5. Synthesize the real-space Laplacian map.
      6. Compute skewness γ = ⟨(∇²T)³⟩ / ⟨(∇²T)²⟩^{3/2} over unmasked pixels.

    Returns
    -------
    float
        Skewness of the Laplacian map (dimensionless scalar).
    """
    mask = np.isfinite(m)
    valid_idx = np.where(mask)[0]
    n_valid = len(valid_idx)

    if n_valid < 100:
        return 0.0

    nside = hp.npix2nside(len(m))

    # Fill masked pixels with 0 for healpy transforms
    m_filled = np.where(mask, m, 0.0)

    # Extract alm up to ℓ_max = 10
    lmax = 10
    alm = hp.map2alm(m_filled, lmax=lmax)

    # Apply −ℓ(ℓ+1) Laplacian operator to each coefficient
    l_arr, m_arr = hp.Alm.getlm(lmax)
    lap_alm = alm.copy()
    lap_alm *= -(l_arr * (l_arr + 1)).astype(np.float64)

    # Zero out ℓ = 0 and ℓ = 1 explicitly (they contribute no Laplacian)
    lap_alm[l_arr < 2] = 0.0

    # Synthesize real-space Laplacian map
    lap_map = hp.alm2map(lap_alm, nside, lmax=lmax, verbose=False)

    # Extract values at unmasked pixels
    vals = lap_map[valid_idx]

    # Compute area-weighted moments (unmasked pixels at fixed nside have equal area)
    mean_v = np.mean(vals)
    centered = vals - mean_v

    m2 = np.mean(centered ** 2)
    if m2 <= 0.0:
        return 0.0

    m3 = np.mean(centered ** 3)

    skewness = m3 / (m2 ** 1.5)

    return float(skewness)


def summarize_results(planck_stat, sim_results):
    """Summarise the Low-ℓ Laplacian Skewness test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available for comparison.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Laplacian Skewness",
                "xlabel": "Laplacian skewness γ",
                "ylabel": "Number of simulations",
                "bins": 40,
                "caption": "No simulation data available.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    # Two-sided empirical p-value
    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    # Direction interpretation
    if std_sim > 0:
        deviation_sigma = (planck_stat - mean_sim) / std_sim
    else:
        deviation_sigma = 0.0

    if planck_stat < mean_sim - 2 * std_sim:
        direction = (
            "Planck's low-ℓ Laplacian is significantly negatively skewed, meaning "
            "cold spots exhibit sharper curvature peaks than hot spots. This asymmetry "
            "in the curvature distribution could reflect non-Gaussian features such as "
            "the Cold Spot or other large-scale anomalies in the temperature field."
        )
    elif planck_stat > mean_sim + 2 * std_sim:
        direction = (
            "Planck's low-ℓ Laplacian is significantly positively skewed, meaning "
            "hot spots exhibit sharper curvature peaks than cold spots — an atypical "
            "configuration that may indicate asymmetric large-scale temperature structure."
        )
    else:
        direction = (
            "The Laplacian skewness is consistent with isotropic Gaussian ΛCDM "
            "expectations; the curvature distribution of the large-scale temperature "
            "pattern shows no significant asymmetry between hot and cold regions."
        )

    interp = (
        f"Laplacian skewness = {planck_stat:.6f}. "
        f"Simulation mean = {mean_sim:.6f} ± {std_sim:.6f} "
        f"({deviation_sigma:+.2f}σ from mean). "
        f"Percentile = {percentile:.1f}%, two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of the Laplacian skewness γ = ⟨(∇²T)³⟩ / ⟨(∇²T)²⟩^{3/2} of the "
        "ℓ = 2–10 band-limited temperature map from isotropic ΛCDM simulations. "
        "The angular Laplacian ∇²T is computed by multiplying each spherical harmonic "
        "coefficient T_{ℓm} by −ℓ(ℓ+1), then transforming to real space. "
        "For a statistically isotropic Gaussian field, γ ≈ 0 by symmetry. "
        "Significantly negative γ indicates cold spots have sharper curvature than hot "
        "spots (or vice versa for positive γ). "
        "The red line marks the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": deviation_sigma,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Laplacian Skewness",
            "xlabel": "Laplacian skewness γ",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
