Hypothesis

The hottest spot in the low-ℓ (ℓ≤10) smoothed CMB temperature map is located closer to the Galactic plane than expected from Gaussian simulations.

Method

The test identifies the Galactic latitude of the pixel with the maximum temperature in the CMB map smoothed to ℓ_max ≤ 10. It checks if this hottest spot is preferentially located closer to the Galactic plane than expected in ΛCDM simulations.

Result Summary

The hottest spot in the Planck data is at a Galactic latitude of -55.1°. From 1000 Gaussian simulations, the mean latitude is 0.5° ± 44.5°. The observed value is 1.25σ from the mean. The one-tailed p-value (for a spot closer to the plane) is 0.783, indicating no significant concentration toward the Galactic plane.

The observed hottest spot is at a mid-latitude in the southern Galactic hemisphere. Its location is statistically consistent with the distribution expected from Gaussian, isotropic realizations of the ΛCDM model. There is no evidence for an anomalous Galactic plane alignment.

Agent Justification

The test is specifically designed to check for Galactic plane concentration, as indicated by the description focusing on proximity to the Galactic plane and the code computing p-value for smaller absolute Galactic latitude (p(|b| ≤ observed)). The interpretation in the code explicitly mentions that a small p-value suggests the hottest spot is unusually close to the Galactic plane, indicating a directional prediction that the observed absolute latitude will be smaller than expected from simulations.

Comparison To Literature

This specific test (hottest spot latitude in low-ℓ) is not commonly reported, but related analyses of low-ℓ multipole alignment with the Galactic plane or "axis of evil" are standard in CMB anomaly studies (e.g., Land & Magueijo, 2005). This null result is consistent with some studies finding no strong alignment for the hottest spot.

import numpy as np
import healpy as hp

test_description = (
    "Smooth the CMB temperature map to a maximum multipole of ℓ_max=10 using "
    "spherical harmonic decomposition. Identify the unmasked pixel with the "
    "highest temperature value in the smoothed map. The statistic is the "
    "Galactic latitude (in degrees) of that pixel. Values near 0° indicate "
    "the hottest spot lies close to the Galactic plane; values near ±90° "
    "indicate alignment with the Galactic poles."
)


def analyze_map(m):
    """Return Galactic latitude (degrees) of the hottest pixel in the low-ℓ (ℓ≤10) smoothed map."""
    nside = hp.npix2nside(len(m))
    mask = np.isfinite(m)
    n_valid = int(np.sum(mask))
    if n_valid < 500:
        return 0.0  # Neutral value for severely masked maps

    # Fill masked pixels with 0 for healpy compatibility
    m_filled = np.where(mask, m, 0.0)

    # Decompose into spherical harmonics up to lmax=10
    alm = hp.map2alm(m_filled, lmax=10, iter=0)

    # Reconstruct the smoothed low-ℓ map
    m_smooth = hp.alm2map(alm, nside, lmax=10, verbose=False)

    # Set masked pixels to -inf so they cannot be selected as the maximum
    m_smooth_valid = np.where(mask, m_smooth, -np.inf)

    # Find the pixel with the highest temperature
    max_pix = np.argmax(m_smooth_valid)
    max_temp = m_smooth[max_pix]

    if not np.isfinite(max_temp):
        return 0.0

    # Convert pixel index to angular coordinates (theta=colatitude, phi=longitude)
    theta, phi = hp.pix2ang(nside, max_pix)

    # Convert colatitude to latitude: Galactic lat = pi/2 - theta
    gal_lat_rad = np.pi / 2.0 - theta
    gal_lat_deg = float(np.degrees(gal_lat_rad))

    if not np.isfinite(gal_lat_deg):
        return 0.0

    return gal_lat_deg


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ℓ Hottest Spot Galactic Latitude test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Hottest Spot Galactic Latitude",
                "xlabel": "Galactic latitude (degrees)",
                "ylabel": "Count",
                "bins": 50,
                "caption": "No simulation data.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    # Absolute latitude (distance from Galactic plane)
    abs_planck = abs(planck_stat)
    abs_sim = np.abs(sim)

    # Deviation from simulation mean in terms of latitude
    deviation_sigma = (planck_stat - mean_sim) / std_sim if std_sim > 0 else 0.0

    # P-value: probability of finding a hotter spot closer to the Galactic plane
    # (smaller |latitude|) — tests for Galactic plane concentration
    p_value_plane = float(np.mean(abs_sim <= abs_planck))

    # Two-sided p-value: probability of being as extreme or more so
    p_value_two_sided = 2.0 * min(p_value_plane, 1.0 - p_value_plane)
    p_value_two_sided = min(p_value_two_sided, 1.0)

    # Rank and percentile
    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims

    if p_value_two_sided <= 0.01:
        strength = "strong"
    elif p_value_two_sided <= 0.05:
        strength = "moderate"
    else:
        strength = "negligible"

    # Interpret the location
    if abs_planck < 10:
        location_desc = "very close to the Galactic plane"
    elif abs_planck < 30:
        location_desc = "at low Galactic latitude"
    elif abs_planck < 60:
        location_desc = "at mid-Galactic latitude"
    else:
        location_desc = "near a Galactic pole"

    if planck_stat >= 0:
        hemisphere = "northern"
    else:
        hemisphere = "southern"

    direction = (
        f"The hottest spot in the low-ℓ (ℓ≤10) smoothed map is {location_desc} "
        f"in the {hemisphere} Galactic hemisphere (latitude ≈ {planck_stat:.1f}°). "
    )

    if p_value_plane <= 0.05:
        direction += (
            "This proximity to the Galactic plane is statistically unusual compared "
            "to Gaussian simulations, potentially suggesting residual foreground "
            "contamination or an anomalous large-scale temperature structure aligned "
            "with Galactic coordinates."
        )
    else:
        direction += (
            "This location is consistent with expectations from Gaussian simulations. "
            "No significant Galactic latitude preference is detected in the hottest spot."
        )

    interp = (
        f"Galactic latitude = {planck_stat:.2f}°. "
        f"Sim mean = {mean_sim:.2f}° ± {std_sim:.2f}° "
        f"({deviation_sigma:+.2f}σ). Percentile = {percentile:.1f}%, "
        f"two-sided p = {p_value_two_sided:.4f}, "
        f"p(|b|≤observed) = {p_value_plane:.4f}. "
        f"Strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of Galactic latitude (degrees) of the hottest pixel in the "
        "low-ℓ (ℓ≤10) smoothed CMB temperature map across FFP simulations. "
        "Values near 0° indicate concentration near the Galactic plane; values "
        "near ±90° indicate alignment with Galactic poles. Red line marks the "
        "Planck observed value."
    )

    return {
        "planck_stat": planck_stat,
        "planck_abs_latitude": abs_planck,
        "p_value_two_sided": p_value_two_sided,
        "p_value_galactic_plane": p_value_plane,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": deviation_sigma,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Hottest Spot Galactic Latitude",
            "xlabel": "Galactic latitude (degrees)",
            "ylabel": "Number of simulations",
            "bins": 50,
            "caption": caption,
        },
    }
