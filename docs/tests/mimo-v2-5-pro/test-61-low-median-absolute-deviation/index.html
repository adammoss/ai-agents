Hypothesis

The median absolute deviation (MAD) of the low-ℓ (ℓ ≤ 30) CMB temperature fluctuations from Planck is different from the distribution expected under the ΛCDM model.

Method

The median absolute deviation (MAD) of the low-ℓ (ℓ ≤ 30) SMICA CMB temperature map is computed. For each valid pixel, the absolute deviation from the median temperature is calculated; the statistic is the median of these deviations. This provides a robust, outlier-insensitive measure of typical large-scale temperature fluctuation amplitude.

Result Summary

Planck MAD = 28.67 μK, compared to simulations with mean 32.68 ± 2.64 μK. The observed value lies at z = −1.52σ (percentile 3.4%, two-tailed p = 0.068). The anomaly strength is classified as negligible.

The MAD of large-scale temperature fluctuations is consistent with ΛCDM predictions. The typical amplitude of temperature scatter around the median is neither unusually low nor high, indicating no anomalous uniformity or variability at the largest angular scales. While the Planck value is modestly lower than the simulation mean, it does not reach statistical significance.

Agent Justification

The test is designed to detect both unusually low MAD (indicating smoother-than-expected large-scale temperature patterns) and unusually high MAD (indicating more variable-than-expected patterns). The interpretation in the code considers both directions, and there is no clear prior expectation for a specific direction. Therefore, a two-tailed test is appropriate.

Comparison To Literature

No prior studies specifically testing the median absolute deviation of low-ℓ CMB temperature maps were found in the literature or through searches. Related work on low-multipole anomalies typically examines angular power spectra or specific multipole moments rather than robust pixel-space dispersion measures.

import numpy as np
import healpy as hp

test_description = (
    "For the low-ℓ SMICA map (ℓ ≤ 30), compute the median temperature T_med across valid "
    "pixels. Then for each valid pixel, compute the absolute deviation |T - T_med|. The "
    "statistic is the median of these absolute deviations (MAD). This provides a robust "
    "measure of typical temperature fluctuation amplitude at large angular scales, with "
    "low sensitivity to extreme outliers such as the CMB cold spot. Anomalous MAD values "
    "could indicate unexpected uniformity or variability in large-scale temperature patterns."
)


def analyze_map(m):
    """Return the median absolute deviation of temperature in the ℓ ≤ 30 filtered map."""
    mask = np.isfinite(m)
    n_valid = int(mask.sum())
    if n_valid < 10:
        return 0.0

    # Replace NaN with 0 for healpy transforms
    m_filled = np.where(mask, m, 0.0)

    # Low-pass filter to ℓ ≤ 30
    alm = hp.map2alm(m_filled, lmax=30)
    m_filtered = hp.alm2map(alm, nside=hp.npix2nside(len(m)), lmax=30, verbose=False)

    # Extract valid (unmasked) pixels from the filtered map
    valid_temps = m_filtered[mask]

    # Compute median temperature
    T_med = np.median(valid_temps)

    # Compute median absolute deviation
    mad = np.median(np.abs(valid_temps - T_med))

    return float(mad)


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ℓ Median Absolute Deviation test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value_two_sided": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Median Absolute Deviation",
                "xlabel": "MAD (μK)",
                "ylabel": "Count",
                "bins": 50,
                "caption": "No simulation data.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))
    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    rank_below = int(np.sum(sim <= planck_stat))
    percentile = 100.0 * rank_below / n_sims
    dev_sigma = (planck_stat - mean_sim) / std_sim if std_sim > 0 else 0.0

    strength = (
        "strong" if p_two <= 0.01
        else "moderate" if p_two <= 0.05
        else "negligible"
    )

    interp = (
        f"Median absolute deviation (MAD) = {planck_stat:.4f} μK. "
        f"Simulation mean = {mean_sim:.4f} ± {std_sim:.4f} μK ({dev_sigma:+.2f}σ). "
        f"Percentile = {percentile:.1f}%, two-sided p = {p_two:.4f}. "
        f"Strength: {strength}. "
    )

    if planck_stat < mean_sim - 2 * std_sim:
        interp += (
            "Planck has UNUSUALLY UNIFORM large-scale temperature fluctuations (lower MAD), "
            "suggesting the ℓ ≤ 30 sky is smoother than expected from ΛCDM. This could be "
            "related to the anomalously low large-scale power observed in Planck, or to a "
            "suppression of variance that makes the CMB appear more homogeneous at the "
            "largest angular scales."
        )
    elif planck_stat > mean_sim + 2 * std_sim:
        interp += (
            "Planck has UNUSUALLY VARIABLE large-scale temperature fluctuations (higher MAD), "
            "suggesting enhanced scatter in the ℓ ≤ 30 map relative to ΛCDM expectations. "
            "This robust (outlier-insensitive) measure implies the excess variability is not "
            "driven solely by extreme features like the cold spot, but reflects a broader "
            "pattern of elevated temperature contrasts across the sky."
        )
    else:
        interp += (
            "The median absolute deviation of large-scale (ℓ ≤ 30) temperature fluctuations "
            "is consistent with ΛCDM predictions. The typical amplitude of temperature "
            "scatter around the median is neither unusually low nor high, indicating no "
            "anomalous uniformity or variability at the largest angular scales."
        )

    return {
        "planck_stat": planck_stat,
        "p_value_one_sided_upper": p_upper,
        "p_value_one_sided_lower": p_lower,
        "p_value_two_sided": p_two,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": dev_sigma,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Median Absolute Deviation",
            "xlabel": "MAD (μK)",
            "ylabel": "Number of simulations",
            "bins": 50,
            "caption": (
                "Histogram of the median absolute deviation (MAD) of temperature across "
                "ΛCDM simulations for the ℓ ≤ 30 filtered map. Lower values indicate "
                "more uniform large-scale temperature patterns; higher values indicate "
                "greater typical fluctuation amplitude. The red line marks the Planck "
                "observed value."
            ),
        },
    }
