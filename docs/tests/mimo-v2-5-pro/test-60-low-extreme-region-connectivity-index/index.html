Hypothesis

The Planck CMB temperature map exhibits an anomalous number of connected components in extreme temperature regions (|T| > 1.5 × T_rms) at large angular scales (ℓ ≤ 30) compared to the distribution predicted by ΛCDM simulations, indicating a deviation in morphological fragmentation.

Method

For a CMB temperature map band-limited to ℓ ≤ 30, extreme regions are defined as connected pixel sets where |T| > 1.5 × T_rms, using HEALPix 8-neighbor adjacency. The statistic is ln(N_components + 1), where N is the total count of distinct connected components (hot and cold). This measures morphological fragmentation of large-scale temperature extremes.

Result Summary

Planck observed value = 4.174, simulation mean = 4.093 ± 0.155. The Planck map lies at +0.53σ from the ΛCDM simulation mean, corresponding to the 70.8th percentile. Two-sided p-value = 0.666. Anomaly strength is classified as negligible.

The morphological fragmentation of large-scale temperature extremes (ℓ ≤ 30) in the Planck map is fully consistent with ΛCDM predictions. No anomalous coherence or fragmentation pattern is detected — the number of distinct connected components in extreme regions is typical for a Gaussian random field.

Agent Justification

The test is designed to detect any deviation from Gaussian random field predictions in the morphology of large-scale temperature extremes, which could manifest as either more coherent extremes (fewer connected components, lower statistic) or more fragmented extremes (more connected components, higher statistic). The code calculates both one-sided p-values and uses a two-sided p-value as the primary metric for anomaly strength, with interpretations provided for both directions. There is no strong prior or clear directional preference in the test description; it explores general anomalies such as the CMB cold spot or hemispherical asymmetry, which could affect fragmentation in either way, thus warranting a two-tailed test.

Comparison To Literature

Morphological analyses of the CMB have been conducted using Minkowski functionals and genus statistics, which are topologically related to connected component counting (e.g., Planck 2018 results VII; Gott et al. 1990; Ducout et al. 2013). However, the specific metric of counting connected components in thresholded extreme regions at low multipoles does not appear to have a direct prior analogue in published studies. Online searches returned no closely matching literature for this particular statistic.

import numpy as np
import healpy as hp

test_description = (
    "For a CMB temperature map band-limited to ℓ ≤ 30, compute the RMS temperature T_rms. "
    "Define 'extreme regions' as connected sets of pixels where |T| > 1.5 × T_rms, with "
    "connectivity defined using the HEALPix 8-neighbor adjacency rule. Count the total number "
    "of distinct connected components (both hot and cold) across the full map. The statistic is "
    "the natural logarithm of this count plus one. This measures the morphological fragmentation "
    "or coherence of large-scale temperature extremes, which can differ between Gaussian random "
    "fields and maps with specific anomaly patterns."
)


def analyze_map(m):
    """Return ln(N_components + 1) for extreme regions in the ℓ ≤ 30 filtered map."""
    mask = np.isfinite(m)
    n_valid = int(mask.sum())
    if n_valid < 10:
        return 0.0

    nside = hp.npix2nside(len(m))
    work_nside = 32
    npix_work = hp.nside2npix(work_nside)

    # Replace NaN with 0 for healpy transforms
    m_filled = np.where(mask, m, 0.0)

    # Low-pass filter to ℓ ≤ 30 and downgrade to work nside
    alm = hp.map2alm(m_filled, lmax=30)
    m_filtered = hp.alm2map(alm, nside=work_nside, lmax=30, verbose=False)

    # Build validity mask at work nside
    if nside != work_nside:
        mask_lo = hp.ud_grade(mask.astype(float), nside_out=work_nside)
        work_mask = mask_lo > 0.5
    else:
        work_mask = mask

    # Compute T_rms from valid pixels only
    valid_temps = m_filtered[work_mask]
    T_rms = float(np.sqrt(np.mean(valid_temps ** 2)))

    if T_rms < 1e-30:
        return 0.0

    # Identify extreme pixels: |T| > 1.5 * T_rms
    threshold = 1.5 * T_rms
    extreme_mask = (np.abs(m_filtered) > threshold) & work_mask
    extreme_pixels = np.where(extreme_mask)[0].astype(np.int64)
    n_extreme = len(extreme_pixels)

    if n_extreme == 0:
        return 0.0

    # Vectorized neighbor computation for ALL pixels at once
    all_pixels = np.arange(npix_work, dtype=np.int64)
    neighbors = hp.get_all_neighbours(work_nside, all_pixels)  # shape (8, npix_work)

    # Build set for fast lookup
    extreme_set = set(extreme_pixels.tolist())

    # Union-Find with path compression
    parent = np.arange(npix_work, dtype=np.int64)

    def find(x):
        while parent[x] != x:
            parent[x] = parent[parent[x]]
            x = parent[x]
        return x

    def union(a, b):
        ra, rb = find(a), find(b)
        if ra != rb:
            parent[ra] = rb

    # For each extreme pixel, union with extreme neighbors
    # Restrict neighbor lookup to extreme pixels only
    nbs_of_extreme = neighbors[:, extreme_pixels]  # shape (8, n_extreme)
    for idx in range(n_extreme):
        pix = extreme_pixels[idx]
        for nb in nbs_of_extreme[:, idx]:
            if nb >= 0 and nb in extreme_set:
                union(pix, nb)

    # Count distinct roots among extreme pixels
    roots = set()
    for pix in extreme_pixels:
        roots.add(find(pix))
    n_components = len(roots)

    return float(np.log(n_components + 1.0))


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ℓ Extreme-Region Connectivity Index test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value_two_sided": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Extreme-Region Connectivity Index",
                "xlabel": "ln(N_components + 1)",
                "ylabel": "Count",
                "bins": 50,
                "caption": "No simulation data.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))
    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    rank_below = int(np.sum(sim <= planck_stat))
    percentile = 100.0 * rank_below / n_sims
    dev_sigma = (planck_stat - mean_sim) / std_sim if std_sim > 0 else 0.0

    strength = (
        "strong" if p_two <= 0.01
        else "moderate" if p_two <= 0.05
        else "negligible"
    )

    interp = (
        f"Extreme-region connectivity index = {planck_stat:.4f} (ln(N+1)). "
        f"Simulation mean = {mean_sim:.4f} ± {std_sim:.4f} ({dev_sigma:+.2f}σ). "
        f"Percentile = {percentile:.1f}%, two-sided p = {p_two:.4f}. "
        f"Strength: {strength}. "
    )

    if planck_stat < mean_sim - 2 * std_sim:
        interp += (
            "Planck has FEWER connected extreme regions than expected, indicating unusually "
            "coherent (less fragmented) large-scale temperature extremes. This could be "
            "linked to the CMB cold spot or hemispherical power asymmetry, where a small "
            "number of large, dominant extreme regions persist rather than many smaller ones."
        )
    elif planck_stat > mean_sim + 2 * std_sim:
        interp += (
            "Planck has MORE connected extreme regions than expected, indicating unusually "
            "fragmented large-scale temperature extremes. This could suggest patchy or "
            "anisotropic power distribution at the largest angular scales."
        )
    else:
        interp += (
            "The morphological fragmentation of large-scale temperature extremes (ℓ ≤ 30) "
            "is consistent with ΛCDM predictions. No anomalous coherence or fragmentation "
            "pattern is detected from this statistic."
        )

    return {
        "planck_stat": planck_stat,
        "p_value_one_sided_upper": p_upper,
        "p_value_one_sided_lower": p_lower,
        "p_value_two_sided": p_two,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": dev_sigma,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Extreme-Region Connectivity Index",
            "xlabel": "ln(N_components + 1)",
            "ylabel": "Number of simulations",
            "bins": 50,
            "caption": (
                "Histogram of the connectivity index ln(N+1) across ΛCDM simulations, "
                "where N is the number of distinct connected components among pixels with "
                "|T| > 1.5 T_rms in the ℓ ≤ 30 filtered map. Lower values indicate more "
                "coherent (fewer, larger) extreme regions; higher values indicate more "
                "fragmented extremes. The red line marks the Planck observed value."
            ),
        },
    }
