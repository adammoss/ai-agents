Hypothesis

The coefficient of variation of iso-temperature contour perimeters for the Planck CMB map differs from the distribution expected under the standard ΛCDM Gaussian random field model.

Method

A morphological statistic measuring the coefficient of variation (CV) of iso-temperature contour perimeter lengths across nine percentile thresholds (10th–90th) of the low-ℓ (ℓ=2–30) CMB temperature field. Boundary pixels are identified at each threshold as hot pixels with at least one cold or masked neighbor, and perimeters are scaled by average pixel diameter at nside=256. Higher CV indicates uneven distribution of morphological complexity across temperature levels, which can signal non-Gaussian features such as cold spots or dipole modulation.

Result Summary

The Planck SMICA map yields CV = 0.2447, compared to the simulation mean of 0.2646 ± 0.0176 (−1.13σ, percentile = 13.0%). The two-sided p-value is 0.26, indicating no significant departure from Gaussian LCDM expectations. Anomaly strength is classified as negligible.

The contour perimeter variability across temperature thresholds is fully consistent with the standard ΛCDM Gaussian random field model. The Planck value sits slightly below the simulation mean but well within the expected scatter, showing no evidence that morphological complexity is preferentially concentrated at particular temperature levels (e.g., due to a cold spot or modulation pattern).

Agent Justification

The test is designed to detect any deviation in morphological complexity distribution from Gaussian expectations. The anomaly could manifest as either higher CV (uneven complexity, e.g., from a cold spot) or lower CV (unusually uniform complexity). The summary function explicitly computes a two-sided p-value and interprets both higher and lower directions, indicating a general exploratory test for any significant change.

Comparison To Literature

No directly comparable studies using a coefficient of variation of iso-temperature contour perimeters were found. Related morphological analyses include Minkowski functionals, genus curves, and spot ellipticity studies (e.g., Gurzadyan et al. 2003), but this specific statistic appears unexplored in the CMB anomaly literature.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Threshold-Contour Perimeter Coefficient of Variation: "
    "This statistic measures the relative variability of iso-temperature "
    "contour perimeter lengths across temperature thresholds, capturing "
    "how morphological complexity distributes across the temperature field. "
    "In Gaussian random fields, contour complexity distributes relatively "
    "uniformly across temperature percentiles, while modulated or "
    "non-Gaussian fields produce uneven complexity (e.g., a cold spot "
    "concentrates perimeter at low thresholds). The statistic is "
    "independent of the one-point PDF and distinct from topological "
    "measures like genus. It belongs to the morphological/spatial-structure "
    "family of anomaly statistics."
)


def analyze_map(m):
    """Low-ℓ Threshold-Contour Perimeter Coefficient of Variation.

    Steps:
    1. Filter to ℓ = 2–30 via spherical harmonic decomposition (zero ℓ=0,1).
    2. Compute 9 percentile thresholds (10th–90th) on valid (unmasked) pixels.
    3. For each threshold, identify boundary pixels (hot pixel with at least
       one cold or masked neighbor) and compute contour perimeter.
    4. Return CV = std(perimeters) / mean(perimeters).
    """
    mask = np.isfinite(m)
    n_valid = int(np.sum(mask))
    if n_valid < 100:
        return 0.0

    nside = hp.get_nside(m)
    npix = 12 * nside ** 2

    # Step 1: Filter to ℓ = 2–30
    clean = m.copy()
    clean[~mask] = 0.0

    lmax = 30
    alm = hp.map2alm(clean, lmax=lmax)
    # Zero out ℓ=0 and ℓ=1 to retain only ℓ=2–30
    # alm index layout: (l, m) -> l*(l+1)//2 + m
    alm[0] = 0.0   # (0, 0)
    alm[1] = 0.0   # (1, 0)
    alm[2] = 0.0   # (1, 1)
    filtered = hp.alm2map(alm, nside, lmax=lmax, verbose=False)

    # Step 3: Compute 9 thresholds (10th through 90th percentiles)
    temps_valid = filtered[mask]
    thresholds = np.percentile(temps_valid, np.arange(10, 100, 10))  # 9 values

    # Average pixel diameter at nside=256 ≈ 0.23°, converted to radians
    pixel_diam_rad = 0.23 * np.pi / 180.0

    # Precompute neighbor indices once (shape: 8 x npix)
    neighbors = hp.get_all_neighbours(nside, np.arange(npix))
    # Replace -1 (missing neighbor at poles) with 0 for safe indexing
    safe_neighbors = np.where(neighbors >= 0, neighbors, 0).astype(np.intp)
    valid_neighbors = neighbors >= 0  # bool mask for real neighbors

    # Step 4: For each threshold, compute contour perimeter
    perimeters = np.zeros(9)
    for i, T_i in enumerate(thresholds):
        # Hot pixels: unmasked AND filtered temperature >= T_i
        hot = np.zeros(npix, dtype=bool)
        hot[mask] = filtered[mask] >= T_i

        # Cold pixels: masked OR filtered < T_i (complement of hot)
        cold = ~hot

        # Vectorized neighbor check: any neighbor that is cold?
        neighbor_cold = cold[safe_neighbors]        # (8, npix)
        neighbor_cold[~valid_neighbors] = False      # missing neighbors don't count
        any_neighbor_cold = np.any(neighbor_cold, axis=0)

        # Boundary pixels: hot AND at least one cold neighbor
        boundary = hot & any_neighbor_cold
        perimeters[i] = int(np.sum(boundary)) * pixel_diam_rad

    # Step 5: Coefficient of variation of perimeter values
    mean_L = np.mean(perimeters)
    if mean_L <= 0:
        return 0.0

    cv = float(np.std(perimeters) / mean_L)
    return cv if np.isfinite(cv) else 0.0


def summarize_results(planck_stat, sim_results):
    """Summarize the Threshold-Contour Perimeter CV test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Threshold-Contour Perimeter CV",
                "xlabel": "CV",
                "ylabel": "Count",
                "bins": 40,
                "caption": "No simulation data.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims
    deviation_sigma = (planck_stat - mean_sim) / std_sim if std_sim > 0 else 0.0

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    if planck_stat > mean_sim + 2 * std_sim:
        direction = (
            "Planck shows significantly higher contour perimeter variability "
            "than LCDM simulations, indicating non-uniform morphological "
            "complexity across temperature thresholds — consistent with "
            "modulated or non-Gaussian structure (e.g., a cold spot "
            "concentrating perimeter at low thresholds)."
        )
    elif planck_stat < mean_sim - 2 * std_sim:
        direction = (
            "Planck shows significantly lower contour perimeter variability "
            "than LCDM simulations, suggesting unusually uniform morphological "
            "complexity across temperature thresholds."
        )
    else:
        direction = (
            "The contour perimeter variability is consistent with Gaussian "
            "LCDM expectations."
        )

    interp = (
        f"Threshold-contour perimeter CV = {planck_stat:.4f}. "
        f"Simulation mean = {mean_sim:.4f} ± {std_sim:.4f} "
        f"({deviation_sigma:+.2f}σ). Percentile = {percentile:.1f}%, "
        f"two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of the coefficient of variation (CV) of iso-temperature "
        "contour perimeters computed at 9 percentile thresholds (10th–90th) "
        "of the low-ℓ (ℓ=2–30) CMB temperature field. Higher CV indicates "
        "that morphological complexity is unevenly distributed across "
        "temperature levels, which can signal non-Gaussian features such "
        "as cold spots or modulation. Red line marks the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": deviation_sigma,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Threshold-Contour Perimeter CV",
            "xlabel": "CV of contour perimeters",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
