Hypothesis

The mean wavelet entropy of the low-ℓ CMB temperature map differs from the distribution expected under the standard ΛCDM cosmological model.

Method

A discrete wavelet transform decomposes the low-ℓ (ℓ ≤ 32) masked CMB temperature map across J_max = log2(nside) scales. At each scale j, Shannon entropy H_j is computed from the histogram of |wavelet coefficients|. These entropies are averaged into a single scalar capturing multi-scale non-Gaussianity and information content in the large-scale wavelet domain.

Result Summary

Planck mean wavelet entropy = 5.985 bits. Simulation mean = 6.045 ± 0.054 bits. The Planck value lies at the 12.5th percentile, deviating −1.12σ from simulations. Two-sided p-value = 0.250, anomaly strength classified as negligible.

The mean wavelet entropy of the low-ℓ Planck CMB map is fully consistent with Gaussian ΛCDM expectations. The mild −1.12σ deficit suggests no significant departure in the distribution of multi-scale wavelet coefficient structure. There is no indication of concentrated or disordered large-scale wavelet patterns beyond what isotropic Gaussian fluctuations predict.

Agent Justification

The test is explicitly designed to capture "multi-scale non-Gaussianity and information content," with the analysis code calculating and interpreting anomalies in both directions (higher or lower entropy than simulations). The `summarize_results` function computes both an upper and lower p-value and selects the smaller for the two-sided p-value. The interpretation strings describe different physical scenarios for both significantly higher and lower entropy. This indicates a general exploration for any unexpected pattern in the wavelet entropy statistic, rather than a prediction of a specific directional change, making a two-tailed test appropriate.

Comparison To Literature

Wavelet-based CMB analyses are well established (e.g., Vielva et al. 2004; Planck 2018 VII), but a specific wavelet entropy coefficient restricted to low multipoles does not appear as a standard statistic in the anomaly literature. Searches returned no directly comparable studies combining Shannon entropy of wavelet coefficients with low-ℓ masking. This may represent a novel construction, though foundational wavelet techniques are common.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Wavelet Entropy Coefficient: "
    "Compute a discrete wavelet transform of the masked, low-ℓ limited "
    "temperature map (ℓ ≤ 32) up to a maximum wavelet scale "
    "J_max = log2(nside). The wavelet decomposition is achieved by "
    "successive spherical-harmonic filtering: at each scale j, the "
    "ℓ_max is halved and wavelet coefficients are obtained as the "
    "difference between consecutive filtered maps. For each scale, "
    "compute the Shannon entropy of the distribution of |c_j| using "
    "a histogram with the Freedman-Diaconis bin rule. Average these "
    "entropies over all J_max scales to obtain a single scalar. This "
    "statistic captures multi-scale non-Gaussianity and information "
    "content in the low-ℓ wavelet domain."
)


def analyze_map(m):
    """Low-ℓ wavelet entropy coefficient."""
    mask = np.isfinite(m)
    n_valid = int(np.sum(mask))
    if n_valid < 100:
        return 0.0

    nside = hp.get_nside(m)

    # Replace NaN with 0 for healpy operations
    clean = m.copy()
    clean[~mask] = 0.0

    # Band-limit to ell <= 32
    low_lmax = min(32, 3 * nside - 1)

    # J_max = log2(nside), minimum 2 scales
    J_max = max(int(np.log2(nside)), 2)

    # Build wavelet scales by successive halving of lmax
    lmax_values = []
    current_lmax = low_lmax
    for j in range(J_max):
        if current_lmax < 2:
            break
        lmax_values.append(current_lmax)
        current_lmax = max(current_lmax // 2, 1)

    # Need at least 2 filtered maps to form wavelet differences
    if len(lmax_values) < 2:
        return 0.0

    # Compute filtered maps at each lmax, recomputing alm for correct sizing
    filtered_maps = []
    for lmax in lmax_values:
        alm_j = hp.map2alm(clean, lmax=lmax)
        filtered = hp.alm2map(alm_j, nside, lmax=lmax, verbose=False)
        filtered[~mask] = 0.0
        filtered_maps.append(filtered)

    def shannon_entropy(values):
        """Compute Shannon entropy using Freedman-Diaconis bin rule."""
        if len(values) < 50:
            return np.nan
        q75, q25 = np.percentile(values, [75, 25])
        iqr = q75 - q25
        if iqr > 0:
            bin_width = 2.0 * iqr * (len(values) ** (-1.0 / 3.0))
            vmin, vmax = values.min(), values.max()
            if bin_width > 0 and vmax > vmin:
                n_bins = max(10, min(100, int(np.ceil((vmax - vmin) / bin_width))))
            else:
                n_bins = 50
        else:
            n_bins = 50

        hist, _ = np.histogram(values, bins=n_bins, density=False)
        total = hist.sum()
        if total == 0:
            return np.nan
        p = hist.astype(float) / total
        p_pos = p[p > 0]
        return -np.sum(p_pos * np.log2(p_pos))

    # Wavelet coefficients = differences between successive filtered maps
    # c_j = map(lmax_j) - map(lmax_{j+1})  captures detail between two scales
    entropies = []
    for j in range(len(filtered_maps) - 1):
        coeffs = filtered_maps[j] - filtered_maps[j + 1]
        coeff_mag = np.abs(coeffs[mask])
        h = shannon_entropy(coeff_mag)
        if np.isfinite(h):
            entropies.append(h)

    # Include entropy of the coarsest approximation
    coarsest = filtered_maps[-1]
    coarse_mag = np.abs(coarsest[mask])
    h_coarse = shannon_entropy(coarse_mag)
    if np.isfinite(h_coarse):
        entropies.append(h_coarse)

    if len(entropies) == 0:
        return 0.0

    mean_entropy = float(np.mean(entropies))
    return mean_entropy if np.isfinite(mean_entropy) else 0.0


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ℓ Wavelet Entropy Coefficient test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Wavelet Entropy Coefficient",
                "xlabel": "Mean wavelet entropy (bits)",
                "ylabel": "Count",
                "bins": 40,
                "caption": "No simulation data.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims
    deviation_sigma = (planck_stat - mean_sim) / std_sim if std_sim > 0 else 0.0

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    if planck_stat > mean_sim + 2 * std_sim:
        direction = (
            "Planck has significantly higher wavelet entropy than LCDM "
            "simulations, indicating more diffuse or disordered structure "
            "across multiple angular scales in the low-ℓ CMB than expected "
            "from Gaussian isotropy."
        )
    elif planck_stat < mean_sim - 2 * std_sim:
        direction = (
            "Planck has significantly lower wavelet entropy than LCDM "
            "simulations, suggesting more concentrated or coherent "
            "multi-scale structure in the low-ℓ CMB than expected from "
            "Gaussian isotropy, possibly indicating non-Gaussian "
            "features or anomalous large-scale power distribution."
        )
    else:
        direction = (
            "The multi-scale wavelet entropy is consistent with Gaussian "
            "LCDM expectations."
        )

    interp = (
        f"Mean wavelet entropy = {planck_stat:.4f} bits. "
        f"Simulation mean = {mean_sim:.4f} ± {std_sim:.4f} "
        f"({deviation_sigma:+.2f}σ). Percentile = {percentile:.1f}%, "
        f"two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of the mean Shannon entropy (in bits) of wavelet "
        "coefficient magnitudes across multi-resolution scales derived "
        "from the low-ℓ (ℓ ≤ 32) CMB temperature map. Higher entropy "
        "indicates more uniformly distributed coefficient magnitudes; "
        "lower entropy indicates more concentrated structure. "
        "Red line marks the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": deviation_sigma,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Wavelet Entropy Coefficient",
            "xlabel": "Mean wavelet entropy (bits)",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
