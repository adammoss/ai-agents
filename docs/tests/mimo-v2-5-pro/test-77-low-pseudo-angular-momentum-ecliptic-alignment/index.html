Hypothesis

The Planck CMB temperature map exhibits a statistically significant alignment of the low-ℓ pseudo-angular-momentum vector with the ecliptic north pole, as indicated by a higher value of |L̂ · ê| than expected from ΛCDM simulations under isotropy.

Method

A pseudo-angular-momentum vector is constructed from ℓ=2–5 spherical harmonic coefficients using L_x = Σ ℓ·Re(a_{ℓ,1}) and L_y = Σ ℓ·Im(a_{ℓ,1}). After normalization, the absolute cosine |L̂ · ê| with the ecliptic north pole (l,b)=(96.4°,29.8°) measures whether the coherent circulation pattern in low-ℓ modes preferentially aligns with the ecliptic direction.

Result Summary

Planck |L̂·ê| = 0.684. Simulation mean ± σ = 0.551 ± 0.265 (isotropic expectation ≈ 0.552). The Planck value corresponds to z ≈ 0.5σ with an upper-tail p-value of 0.415.

The Planck map's pseudo-angular-momentum vector shows a mild tendency toward ecliptic alignment, but the value lies well within the spread expected from isotropic ΛCDM realizations. The wide dispersion of the statistic (σ ≈ 0.265) reflects the small number of modes (ℓ=2–5, m=1 only), yielding low statistical power. No evidence for anomalous ecliptic alignment is found with this test.

Agent Justification

The test is explicitly designed to probe anomalous ecliptic alignment, where high values of the statistic (near 1) indicate alignment with the ecliptic pole. The analysis code calculates p_value_upper for this direction, and the interpretation emphasizes that a small p_value_upper suggests anomalous alignment. Since the hypothesis is directional, focusing on whether the Planck value is unusually high, a one-tailed upper test is appropriate.

Comparison To Literature

Low-ℓ directional anomalies and multipole-vector alignment with the ecliptic have been extensively studied (Copi, Huterer & Starkman 2004; Abramo et al. 2006). However, the specific pseudo-angular-momentum construction using ℓ-weighted imaginary parts of m=1 coefficients does not appear to have a direct precedent in the literature. Searches for this exact statistic returned no matching studies.

import numpy as np
import healpy as hp

test_description = (
    "Construct a pseudo-angular-momentum vector from the low-ℓ spherical harmonic "
    "coefficients of the masked, beam-deconvolved temperature map (ℓ=2–5). Define "
    "L_x = Σ ℓ·Re(a_{ℓ,1}) and L_y = Σ ℓ·Im(a_{ℓ,1}) for ℓ=2 through 5, with "
    "L_z = 0 (since Im(a_{ℓ,0})=0 for a real-valued map). These two components "
    "encode the orthogonal cos-φ and sin-φ directional structure of the m=1 modes. "
    "Normalize to unit length L̂ = L/|L|. The ecliptic north pole in Galactic "
    "coordinates is approximately (l,b) = (96.4°, 29.8°); convert to a Cartesian "
    "unit vector ê. The scalar statistic is |L̂ · ê|, yielding a value in [0, 1]. "
    "Values near 1 indicate that the coherent circulation pattern in the low-ℓ "
    "temperature field is preferentially aligned with the ecliptic pole; values near "
    "0 indicate orthogonality. This statistic probes the phase-coherent directional "
    "structure of low-ℓ modes through the m≠0 coefficients, capturing a distinct "
    "axis-of-alignment signal that is neither a pure power measurement nor a "
    "parity-even/odd decomposition."
)


def analyze_map(m):
    """Return |L̂ · ê|: alignment of pseudo-angular-momentum with ecliptic north pole."""
    nside = hp.npix2nside(len(m))
    lmax = 5

    # Replace NaNs with 0 for healpy compatibility
    m_clean = np.where(np.isfinite(m), m, 0.0)

    # Compute spherical harmonic coefficients up to lmax=5
    alm = hp.map2alm(m_clean, lmax=lmax, iter=10)

    # Build pseudo-angular-momentum vector from ℓ=2..5, m=1 coefficients.
    # L_x = Σ ℓ · Re(a_{ℓ,1})  — encodes the cos(φ) directional component
    # L_y = Σ ℓ · Im(a_{ℓ,1})  — encodes the sin(φ) directional component
    # L_z = 0  (Im(a_{ℓ,0}) = 0 for real-valued maps)
    L_x = 0.0
    L_y = 0.0
    for ell in range(2, 6):
        idx = hp.Alm.getidx(lmax, ell, 1)
        coeff = alm[idx]
        L_x += ell * coeff.real
        L_y += ell * coeff.imag

    # Normalize to unit length
    L_mag = np.sqrt(L_x ** 2 + L_y ** 2)
    if L_mag < 1e-30:
        return 0.0
    L_hat = np.array([L_x / L_mag, L_y / L_mag, 0.0])

    # Ecliptic north pole in Galactic coordinates: (l, b) = (96.4°, 29.8°)
    # Convert to Galactic Cartesian unit vector
    l_gal = np.radians(96.4)
    b_gal = np.radians(29.8)
    e_hat = np.array([
        np.cos(b_gal) * np.cos(l_gal),
        np.cos(b_gal) * np.sin(l_gal),
        np.sin(b_gal),
    ])

    # Absolute cosine of angle between L̂ and ê
    return float(abs(np.dot(L_hat, e_hat)))


def summarize_results(planck_stat, sim_results):
    """Interpret the Low-ℓ Pseudo-Angular-Momentum Ecliptic Alignment statistic."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)

    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    median_s = float(np.median(sims))

    # p-value: fraction of simulations with statistic >= Planck (anomalous alignment)
    p_upper = float(np.mean(sims >= planck_stat))
    # p-value: fraction of simulations with statistic <= Planck (anomalous orthogonality)
    p_lower = float(np.mean(sims <= planck_stat))
    # Two-sided p-value
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)

    # Under isotropy, L̂ is uniform on the unit circle in the x-y plane.
    # |L̂ · ê| = cos(b_ecl) × |cos(α - l_ecl)| where α ~ Uniform(0, 2π)
    # E[|cos θ|] = 2/π, so E[|L̂ · ê|] = cos(29.8°) × 2/π
    b_ecl = np.radians(29.8)
    expected_mean = float(np.cos(b_ecl) * 2.0 / np.pi)

    return {
        "planck_stat": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "sim_median": median_s,
        "expected_mean_isotropic": expected_mean,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "interpretation": (
            f"Absolute cosine of the angle between the pseudo-angular-momentum vector "
            f"(from ℓ=2–5, m=1 SH coefficients) and the ecliptic north pole. "
            f"A value near 1 indicates alignment of the low-ℓ circulation pattern with "
            f"the ecliptic pole; near 0 indicates orthogonality. Under isotropy, "
            f"E[|L̂·ê|] = cos(29.8°)×2/π ≈ {expected_mean:.3f}. "
            f"A high Planck value (small p_value_upper) suggests anomalous ecliptic alignment."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Pseudo-Angular-Momentum Ecliptic Alignment",
            "xlabel": "|L̂ · ê_ecliptic|",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck |L̂·ê| = {planck_stat:.4f}. "
                f"Sim mean ± σ = {mean_s:.4f} ± {std_s:.4f} "
                f"(isotropic expect. {expected_mean:.4f}). "
                f"P(|L̂·ê| ≥ Planck) = {p_upper:.3f}."
            ),
        },
    }
