Hypothesis

The excess kurtosis ratio between the ecliptic north and south hemispheres for the low-ℓ CMB temperature field is equal to 1 under isotropy, versus not equal to 1 under anisotropy.

Method

Quantifies whether heavy-tailed non-Gaussianity (excess kurtosis) in the CMB temperature field at multipoles ℓ = 2–10 differs between ecliptic north and south hemispheres. Each unmasked pixel is assigned by ecliptic latitude, sample excess kurtosis κ = μ₄/σ⁴ − 3 is computed per hemisphere, and the ratio κ_North/κ_South is the test statistic. Isotropy predicts a ratio near unity; the Planck value is ranked against 1000 FFP ΛCDM simulations.

Result Summary

Planck ratio = 8.115; simulation mean ± σ = 0.001 ± 3.501; z-score = 2.32; two-tailed p-value = 0.066.

The observed ratio is elevated, suggesting the ecliptic north hemisphere may exhibit heavier tails than the south at low multipoles. However, the departure falls short of the conventional 2σ (p < 0.05) threshold. The asymmetry in kurtosis could point to ecliptic-correlated systematics or a distinct signature of cosmological anisotropy beyond the well-known power asymmetry.

Agent Justification

The test aims to detect any asymmetry in heavy-tailedness between ecliptic hemispheres, with no directional preference specified. The description states "a value near unity is expected under isotropy," indicating the null hypothesis is ratio = 1, and departures in either direction are of interest. The code computes both upper and lower p-values and emphasizes a two-sided p-value in the summary, reinforcing a two-tailed approach.

Comparison To Literature

Ecliptic hemispherical anomalies at low ℓ—particularly the dipolar power asymmetry and parity oddities—have been reported extensively (e.g., Planck Collaboration 2016, Isotropy and Statistics, arXiv:1506.07135). However, no prior study was found that specifically tests an excess kurtosis ratio between ecliptic hemispheres, making direct comparison to published results difficult.

import numpy as np
import healpy as hp

test_description = (
    "Quantifies whether the non-Gaussian heavy-tailedness (excess kurtosis) of the "
    "low-ℓ (ℓ = 2–10) CMB temperature field differs between ecliptic hemispheres. "
    "Each unmasked pixel is assigned to the ecliptic north (β > 0°) or south (β < 0°). "
    "The sample excess kurtosis κ = μ₄/σ⁴ − 3 is computed per hemisphere (each centred "
    "on its own mean). The statistic is the ratio κ_North / κ_South; values near unity "
    "are expected under isotropy."
)


def analyze_map(m):
    """Return the ecliptic hemispherical excess kurtosis ratio for ℓ = 2–10."""
    nside = hp.npix2nside(len(m))

    # --- 1. Low-pass filter to ℓ = 2–10 ---
    mask_valid = np.isfinite(m)
    m_clean = m.copy()
    m_clean[~mask_valid] = 0.0

    lmax = 10
    alm = hp.map2alm(m_clean, lmax=lmax)

    # Zero out ℓ = 0 and ℓ = 1 modes
    for ell in range(2):
        for m_idx in range(ell + 1):
            alm[hp.Alm.getidx(lmax, ell, m_idx)] = 0.0

    m_low = hp.alm2map(alm, nside, lmax=lmax, verbose=False)
    m_low[~mask_valid] = np.nan

    # --- 2. Ecliptic hemisphere assignment ---
    npix = hp.nside2npix(nside)
    pix = np.arange(npix)
    theta_gal, phi_gal = hp.pix2ang(nside, pix)

    R = hp.Rotator(coord=['G', 'E'])
    theta_ecl, _ = R(theta_gal, phi_gal)       # ecliptic colatitude

    north = (theta_ecl < np.pi / 2) & mask_valid
    south = (theta_ecl > np.pi / 2) & mask_valid

    vals_n = m_low[north]
    vals_s = m_low[south]

    # --- 3. Excess kurtosis per hemisphere ---
    def excess_kurtosis(vals):
        if len(vals) < 4:
            return 0.0
        c = vals - np.mean(vals)
        var = np.mean(c * c)
        if var < 1e-30:
            return 0.0
        mu4 = np.mean(c ** 4)
        return mu4 / (var ** 2) - 3.0

    kN = excess_kurtosis(vals_n)
    kS = excess_kurtosis(vals_s)

    # --- 4. Ratio with guard for near-zero denominator ---
    eps = 1e-8
    if abs(kS) < eps:
        return 1.0 if abs(kN) < eps else 10.0

    return float(np.clip(kN / kS, -10.0, 10.0))


def summarize_results(planck_stat, sim_results):
    """Interpret the ecliptic hemispherical excess kurtosis ratio."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)

    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    p_upper = float(np.mean(sims >= planck_stat))
    p_lower = float(np.mean(sims <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else np.nan

    return {
        "planck_stat": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "z_score": z_score,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "interpretation": (
            "This statistic probes the low-ℓ CMB for asymmetric non-Gaussian "
            "heavy-tailedness between the ecliptic hemispheres. Under isotropy the "
            "excess kurtosis ratio should be near unity. A significant departure "
            "could point to ecliptic-correlated systematics or intrinsic cosmological "
            "anisotropy in the non-Gaussian tail behaviour of the temperature field."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Ecliptic Hemispherical Excess Kurtosis Ratio",
            "xlabel": "κ_North / κ_South",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck ratio = {planck_stat:.3f}. "
                f"Sim mean ± σ = {mean_s:.3f} ± {std_s:.3f}. "
                f"z = {z_score:.2f}, p(two-sided) = {p_two:.3f}."
            ),
        },
    }
