Hypothesis

The low-ℓ power in the CMB temperature map is isotropic with respect to the galactic plane, meaning the hemispherical power ratio R = C̄_N / C̄_S equals 1.

Method

The angular power spectrum C_ℓ is computed separately for the northern (b > 0) and southern (b < 0) galactic hemispheres over multipoles 2 ≤ ℓ ≤ 10. The mean power C̄ is calculated for each hemisphere, and the statistic R = C̄_N / C̄_S quantifies low-ℓ power asymmetry relative to the galactic plane. R = 1 indicates isotropy; deviations signal hemispherical asymmetry.

Result Summary

The Planck data yield R = 0.394, meaning the southern galactic hemisphere contains roughly 2.5 times more low-ℓ power than the northern hemisphere. Comparing to 1000 lensed-ΛCDM simulations (mean R = 1.10, median R = 0.99, std = 0.53), the two-sided p-value is 0.042 (z ≈ 2.03σ). The asymmetry is 60.6% stronger in the southern hemisphere.

The observed ratio R = 0.394 represents a substantial deficit of low-ℓ power in the northern galactic hemisphere relative to the south. The p-value of 0.042 just reaches the 5% significance threshold, and the z-score of 2.03σ is marginally above the 2σ level. While technically significant at these conventional thresholds, the result sits right at the boundary, and with only 1000 simulations the p-value estimation has limited precision. This is consistent with the well-known hemispherical power asymmetry anomaly previously reported in WMAP and Planck data.

Agent Justification

The test description and code check for deviations from R=1 in either direction (north or south stronger) using a two-sided p-value based on absolute deviations from 1. No directional preference is specified in the test setup, and the anomaly is generally sought as a departure from isotropy, making a two-tailed test appropriate.

Comparison To Literature

The hemispherical power asymmetry at low ℓ is a well-established CMB anomaly, first reported by Eriksen et al. (2004) and Hansen et al. (2004) using WMAP data, and confirmed by Planck. However, my web search did not return a study using this exact galactic-plane-split statistic R = C̄_N/C̄_S over ℓ = 2–10, so I cannot confirm a direct literature match for this specific implementation.

import numpy as np
import healpy as hp

test_description = (
    "Galactic Hemispherical Power Ratio. Computes the angular power spectrum C_ℓ "
    "of the masked CMB temperature map for multipoles 2 ≤ ℓ ≤ 10. The sky is "
    "split into northern (galactic latitude b > 0) and southern (b < 0) hemispheres. "
    "For each hemisphere, a half-sky map is constructed by zeroing the opposite "
    "hemisphere, and the power spectrum is computed via spherical harmonic transform. "
    "The mean power C̄ = (1/9) Σ_{ℓ=2}^{10} C_ℓ is calculated for each hemisphere, "
    "and the statistic R = C̄_N / C̄_S is returned. A value of R = 1 indicates "
    "isotropic low-ℓ power with respect to the galactic plane, while R significantly "
    "different from 1 reveals a hemispherical power asymmetry — one of the most "
    "prominent large-scale CMB anomalies."
)


def analyze_map(m):
    m = np.asarray(m, dtype=np.float64)
    nside = hp.npix2nside(len(m))
    npix = hp.nside2npix(nside)

    # Replace NaN (masked) pixels with 0.0 for healpy transforms
    m_clean = np.where(np.isfinite(m), m, 0.0)

    # Get pixel colatitudes; in galactic coordinates, b > 0 corresponds to θ < π/2
    theta, _ = hp.pix2ang(nside, np.arange(npix))
    north_mask = theta < (np.pi / 2)   # b > 0
    south_mask = ~north_mask            # b < 0

    # Build hemispheric maps by zeroing the opposite hemisphere
    m_north = np.where(north_mask, m_clean, 0.0)
    m_south = np.where(south_mask, m_clean, 0.0)

    # Compute power spectra up to ℓ = 10
    lmax = 10
    cl_north = hp.anafast(m_north, lmax=lmax, use_pixel_weights=True)
    cl_south = hp.anafast(m_south, lmax=lmax, use_pixel_weights=True)

    # Extract ℓ = 2..10 and compute mean power in each hemisphere
    ell_range = slice(2, 11)  # indices 2 through 10 inclusive
    mean_n = np.mean(cl_north[ell_range])
    mean_s = np.mean(cl_south[ell_range])

    # Return hemispherical power ratio; guard against zero denominator
    if mean_s > 0:
        ratio = float(mean_n / mean_s)
    else:
        ratio = float('inf')

    return float(ratio)


def summarize_results(planck_stat, sim_results):
    sim_arr = np.array(sim_results, dtype=np.float64)
    finite_mask = np.isfinite(sim_arr)
    n_finite = int(finite_mask.sum())

    if n_finite > 0:
        valid = sim_arr[finite_mask]
        # Two-sided: rank by distance from 1 (perfect isotropy)
        deviations = np.abs(valid - 1.0)
        planck_deviation = abs(planck_stat - 1.0)
        rank = int(np.sum(deviations >= planck_deviation))
        p_value = rank / n_finite
    else:
        rank = 0
        p_value = float('nan')

    sim_median = float(np.nanmedian(sim_arr))
    sim_mean = float(np.nanmean(sim_arr))
    sim_std = float(np.nanstd(sim_arr))

    # Determine which hemisphere is stronger
    if planck_stat > 1.0:
        hemi_stronger = "northern"
        asymmetry_pct = (planck_stat - 1.0) * 100
    else:
        hemi_stronger = "southern"
        asymmetry_pct = (1.0 - planck_stat) * 100

    interpretation = (
        f"Anomalous: low-ℓ power is {asymmetry_pct:.1f}% stronger in the "
        f"{hemi_stronger} galactic hemisphere (R = {planck_stat:.4f}), "
        f"two-sided p = {p_value:.3f}"
        if p_value < 0.05
        else f"Hemispherical power ratio R = {planck_stat:.4f} is consistent "
             f"with ΛCDM expectations (two-sided p = {p_value:.3f})"
    )

    result = {
        "test_description": test_description,
        "planck_stat": float(planck_stat),
        "p_value": float(p_value),
        "rank": rank,
        "n_simulations": len(sim_results),
        "n_valid": n_finite,
        "sim_mean": sim_mean,
        "sim_std": sim_std,
        "sim_median": sim_median,
        "exceeds_95th_percentile": bool(p_value < 0.05),
        "interpretation": interpretation,
        "asymmetry_direction": hemi_stronger,
        "asymmetry_percent": float(asymmetry_pct),
        "expected_random_ratio": "≈1 for isotropic low-ℓ power; R>1 means north has more power",
    }

    result["plot_spec"] = {
        "kind": "histogram",
        "title": "Galactic Hemispherical Power Ratio (ℓ = 2–10)",
        "xlabel": "R = C̄_N / C̄_S  (northern / southern mean power)",
        "ylabel": "Number of simulations",
        "bins": 50,
        "caption": (
            f"Histogram of galactic hemispherical power ratio R from {n_finite} "
            f"lensed-ΛCDM simulations. Planck value = {planck_stat:.4f} "
            f"(two-sided p = {p_value:.3f}, rank {rank}/{n_finite}). "
            f"A ratio of 1 indicates equal low-ℓ power in both hemispheres; "
            f"deviations indicate hemispherical power asymmetry relative to "
            f"the galactic plane."
        ),
    }

    return result
