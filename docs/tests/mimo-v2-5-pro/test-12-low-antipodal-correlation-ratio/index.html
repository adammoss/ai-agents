Hypothesis

The low-ℓ antipodal correlation ratio R from the CMB temperature map is inconsistent with the distribution expected under ΛCDM simulations, indicating an unexpected even-odd power imbalance.

Method

Computes R = [Σ(2ℓ+1)C_ℓ(-1)^ℓ] / [Σ(2ℓ+1)C_ℓ] for multipoles 2 ≤ ℓ ≤ 10. R quantifies the normalized even–odd power balance: R > 0 indicates antipodal symmetry (even-ℓ dominance), R < 0 indicates antipodal antisymmetry (odd-ℓ dominance). This statistic captures the correlation between the low-ℓ temperature map and its antipodal reflection.

Result Summary

Planck R = −0.135 (odd-ℓ dominant), while 1000 ΛCDM simulations yield mean R = 0.179 ± 0.201. The Planck value sits 1.51σ below the simulation mean (two-tailed p = 0.132), corresponding to rank ~66/1000 in the lower tail.

The observed CMB shows modest odd-ℓ power excess at low multipoles, whereas ΛCDM simulations mildly favor even-ℓ dominance on average. However, at ~1.5σ significance, this deviation does not reach the conventional 2σ anomaly threshold. The even–odd power balance is broadly consistent with ΛCDM expectations, though the Planck value lies on the low end of the simulated distribution.

Agent Justification

The test is exploratory, aiming to detect any deviation in the even-odd ℓ power balance without a prior directional preference. The analysis code computes a two-sided p-value by considering both high and low extremes of R, and the description highlights potential anomalies in either direction, confirming a two-tailed approach.

Comparison To Literature

The even–odd multipole parity asymmetry has been extensively studied, with prior work finding odd-ℓ power excess at ~2σ significance (Hansen et al. 2011; Kim & Naselsky 2010; Aluri et al. 2017). The present antipodal correlation ratio R is a distinct but related statistic to the C_ℓ odd/even ratio used in earlier analyses. No prior studies using this specific R formulation were found, though the underlying phenomenon is well-documented.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Antipodal Correlation Ratio. Computes the angular power spectrum C_ℓ "
    "of the temperature map restricted to multipoles 2 ≤ ℓ ≤ 10, then forms the "
    "statistic R = [Σ_{ℓ=2}^{10} (2ℓ+1) C_ℓ (-1)^ℓ] / [Σ_{ℓ=2}^{10} (2ℓ+1) C_ℓ]. "
    "Because P_ℓ(cos π) = (-1)^ℓ, the numerator equals the sky-averaged correlation "
    "between the map and its antipodal (θ = 180°) version in the low-ℓ regime, while "
    "the denominator is the total weighted power. R ∈ [-1, 1]: R = +1 means all power "
    "resides in even ℓ (strong antipodal symmetry), R = -1 means all power in odd ℓ "
    "(strong antipodal antisymmetry), R ≈ 0 means no net antipodal preference. "
    "An anomalous R indicates an unexpected even–odd power imbalance at large angular "
    "scales, potentially related to the hemispherical power asymmetry or parity "
    "violation signatures reported in the CMB."
)


def analyze_map(m):
    m = np.asarray(m, dtype=np.float64)

    # Identify unmasked pixels (finite values)
    mask = np.isfinite(m)

    # Replace NaN with 0.0 for healpy spherical harmonic transforms
    m_clean = np.where(mask, m, 0.0)

    # Compute the angular power spectrum up to ℓ = 10
    lmax = 10
    cl = hp.anafast(m_clean, lmax=lmax)

    # Compute the antipodal correlation ratio R for 2 ≤ ℓ ≤ 10
    ells = np.arange(2, 11)  # ℓ = 2, 3, ..., 10
    weights = 2.0 * ells + 1.0
    cl_slice = cl[2:11]  # C_2, C_3, ..., C_10
    signs = (-1.0) ** ells  # +1 for even ℓ, -1 for odd ℓ

    numerator = float(np.sum(weights * cl_slice * signs))
    denominator = float(np.sum(weights * cl_slice))

    # Guard against division by zero
    if not np.isfinite(denominator) or denominator < 1e-30:
        return 0.0

    R = numerator / denominator

    if not np.isfinite(R):
        return 0.0

    return float(R)


def summarize_results(planck_stat, sim_results):
    sim_arr = np.array(sim_results, dtype=np.float64)
    finite_mask = np.isfinite(sim_arr)
    n_finite = int(finite_mask.sum())

    if n_finite > 0:
        valid = sim_arr[finite_mask]
        sim_median = float(np.median(valid))
        p_upper = float(np.sum(valid >= planck_stat) / n_finite)
        p_lower = float(np.sum(valid <= planck_stat) / n_finite)
        p_value = min(p_upper, p_lower)
        rank_upper = int(np.sum(valid >= planck_stat))
        rank_lower = int(np.sum(valid <= planck_stat))
    else:
        p_value = float('nan')
        rank_upper = 0
        rank_lower = 0
        sim_median = float('nan')

    sim_mean = float(np.nanmean(sim_arr))
    sim_std = float(np.nanstd(sim_arr))

    if n_finite > 0 and not np.isnan(p_value):
        if p_upper <= p_lower:
            direction = "high"
            rank = rank_upper
        else:
            direction = "low"
            rank = rank_lower
    else:
        direction = "unknown"
        rank = 0

    # Even-ℓ dominance label
    if planck_stat > 0:
        parity_label = "even-ℓ dominant (antipodal symmetry)"
    elif planck_stat < 0:
        parity_label = "odd-ℓ dominant (antipodal antisymmetry)"
    else:
        parity_label = "balanced even/odd ℓ power"

    interpretation_parts = []
    interpretation_parts.append(
        f"Low-ℓ antipodal correlation ratio R = {planck_stat:.4f}. "
        f"This means the low-ℓ power spectrum is {parity_label}."
    )
    interpretation_parts.append(
        f"Two-sided p = {p_value:.3f} ({rank}/{n_finite} simulations "
        f"at least as extreme in the {direction} direction)."
    )
    interpretation_parts.append(
        f"Simulation mean = {sim_mean:.4f}, std = {sim_std:.4f}, "
        f"median = {sim_median:.4f}."
    )

    if not np.isnan(p_value) and p_value < 0.05:
        if direction == "high":
            interpretation_parts.append(
                "Anomalous: the antipodal correlation ratio is unusually HIGH, "
                "indicating excess even-ℓ power relative to simulations — "
                "the large-scale temperature pattern has stronger antipodal "
                "symmetry than expected from ΛCDM, potentially linked to "
                "hemispherical power asymmetry or parity-violation signatures."
            )
        else:
            interpretation_parts.append(
                "Anomalous: the antipodal correlation ratio is unusually LOW, "
                "indicating excess odd-ℓ power relative to simulations — "
                "the large-scale temperature pattern has stronger antipodal "
                "antisymmetry than expected from ΛCDM, suggesting an unexpected "
                "odd-parity preference at large angular scales."
            )
    else:
        interpretation_parts.append(
            "The even–odd ℓ power balance is consistent with ΛCDM simulation expectations."
        )

    result = {
        "test_description": test_description,
        "planck_stat": float(planck_stat),
        "p_value": float(p_value),
        "rank": rank,
        "n_simulations": len(sim_results),
        "n_valid": n_finite,
        "sim_mean": sim_mean,
        "sim_std": sim_std,
        "sim_median": sim_median,
        "exceeds_95th_percentile": bool(p_value < 0.05) if not np.isnan(p_value) else False,
        "anomaly_direction": direction,
        "interpretation": " ".join(interpretation_parts),
        "ell_range": "2-10",
    }

    result["plot_spec"] = {
        "kind": "histogram",
        "title": "Low-ℓ Antipodal Correlation Ratio R",
        "xlabel": "R = [Σ(2ℓ+1)C_ℓ(-1)^ℓ] / [Σ(2ℓ+1)C_ℓ]",
        "ylabel": "Number of simulations",
        "bins": 50,
        "caption": (
            f"Histogram of the low-ℓ (2 ≤ ℓ ≤ 10) antipodal correlation ratio R "
            f"from {n_finite} FFP ΛCDM simulations. Planck value = {planck_stat:.4f} "
            f"(two-sided p = {p_value:.3f}, rank {rank}/{n_finite}, "
            f"{direction} direction). Simulation mean = {sim_mean:.4f} ± {sim_std:.4f}. "
            f"R > 0 indicates even-ℓ (antipodal symmetric) dominance; "
            f"R < 0 indicates odd-ℓ (antipodal antisymmetric) dominance."
        ),
    }

    return result
