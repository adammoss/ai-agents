Hypothesis

The boundary-to-interior ratio of the largest hot region in the ℓ=2–20 band-limited Planck CMB temperature map deviates from the distribution predicted by simulated ΛCDM maps.

Method

Measures the geometric complexity of the largest connected hot region in the ℓ=2–20 band-limited Planck CMB temperature map, using the ratio of boundary to interior pixels after thresholding at +1σ.

Result Summary

Observed ratio = 0.869. Simulated ΛCDM distribution mean = 0.597 ± 0.174. The observed value lies at the 92.8th percentile, corresponding to a deviation of +1.46σ with a two-tailed p-value of 0.144.

The dominant low-ℓ hot region in the Planck data shows a moderately higher boundary complexity than the ΛCDM simulation average, suggesting a slightly more elongated or filamentary morphology. However, this deviation is not statistically significant at the 2σ level.

Agent Justification

The test is designed as a general exploratory check for geometric anomalies in the low-ℓ temperature field's morphology. The description and code indicate deviations in either direction are of interest: a higher ratio signals complex, filamentary structure, while a lower ratio signals unusually compact, blob-like structure. The code explicitly computes both upper and lower one-sided p-values and combines them into a two-sided p-value for significance assessment. There is no prior theoretical expectation that Planck's data should show a specific directional deviation, making a two-tailed test appropriate for detecting any departure from the Gaussian ΛCDM expectation.

Comparison To Literature

No prior studies were found that specifically test this boundary-to-interior complexity ratio for CMB hot regions. Related work has analyzed the shapes and isotropy of CMB spots (e.g., Khan & Saha 2022) and perimeter estimation in random fields (Cotsakis et al. 2023), but not this exact metric.

import numpy as np
import healpy as hp

test_description = (
    "Band-limit the temperature map to ell = 2-20 using spherical harmonic "
    "decomposition. Threshold the resulting map at +1 sigma (where sigma is the rms "
    "temperature of the band-limited map) to define the hot excursion set. Identify "
    "all connected hot regions using 8-connectivity on the HEALPix grid and select "
    "the largest one by pixel count. Compute the number of boundary pixels of this "
    "largest connected hot region (pixels that are inside the hot region but have at "
    "least one neighbor outside the region or in the masked area). Define the "
    "statistic as the ratio of the number of boundary pixels to the total number of "
    "interior pixels in the largest hot region. This boundary-to-interior ratio "
    "measures the geometric complexity of the dominant hot structure independent of "
    "its absolute size: compact, blob-like regions yield low values, while elongated, "
    "filamentary, or fractally rough regions yield high values. For isotropic "
    "Gaussian random fields the dominant excursion-set component at 1 sigma tends "
    "toward compact shapes with predictable boundary-to-interior ratios determined "
    "by the field's spectral properties. Deviations signal either non-Gaussian "
    "texture (e.g., unusual spot elongation or filamentary morphology) or "
    "anisotropic structure in the temperature field. Compare the observed value "
    "against the distribution from 1000 simulated LCDM maps to assess significance."
)


def analyze_map(m):
    """Return the boundary-to-interior ratio of the largest hot region in the
    ell=2-20 band-limited temperature map."""
    mask = np.isfinite(m)
    n_valid = int(np.sum(mask))
    if n_valid < 500:
        return 0.0

    m_filled = np.where(mask, m, 0.0)

    lmax = 20
    alm = hp.map2alm(m_filled, lmax=lmax, iter=0)

    # Zero out monopole (ell=0) and dipole (ell=1)
    alm[0] = 0.0
    alm[1] = 0.0
    alm[2] = 0.0

    nside_work = 32
    npix = hp.nside2npix(nside_work)

    m_smooth = hp.alm2map(alm, nside_work, lmax=lmax, verbose=False)

    sigma = np.std(m_smooth)
    if sigma < 1e-15:
        return 0.0

    hot = m_smooth > sigma
    n_hot = int(np.sum(hot))

    if n_hot < 20:
        return 0.0

    all_neighbors = hp.get_all_neighbours(nside_work, np.arange(npix))
    n_dirs = all_neighbors.shape[0]
    safe_nbrs = np.where(all_neighbors >= 0, all_neighbors, np.arange(npix))

    # BFS to find largest connected component (list-based queue, no deque needed)
    visited = np.zeros(npix, dtype=bool)
    hot_idx = np.where(hot)[0]

    largest_size = 0
    best_comp_pixels = None

    for seed in hot_idx:
        if visited[seed]:
            continue
        # BFS using list as queue (front pointer for O(1) amortized pop)
        q = [seed]
        front = 0
        visited[seed] = True
        comp = [seed]
        while front < len(q):
            px = q[front]
            front += 1
            nbrs = safe_nbrs[:, px]
            for nb in nbrs:
                if hot[nb] and not visited[nb]:
                    visited[nb] = True
                    q.append(nb)
                    comp.append(nb)
        if len(comp) > largest_size:
            largest_size = len(comp)
            best_comp_pixels = comp

    if largest_size < 10 or best_comp_pixels is None:
        return 0.0

    # Count boundary vs interior pixels (vectorized)
    best_comp = np.array(best_comp_pixels, dtype=np.int64)

    in_comp = np.zeros(npix, dtype=bool)
    in_comp[best_comp] = True

    nbrs_of_comp = safe_nbrs[:, best_comp]  # (n_dirs, largest_size)
    all_in_comp = np.ones(largest_size, dtype=bool)
    for d in range(n_dirs):
        all_in_comp &= in_comp[nbrs_of_comp[d, :]]

    n_interior = int(np.sum(all_in_comp))
    n_boundary = largest_size - n_interior

    if n_interior == 0:
        return 0.0

    return float(n_boundary / n_interior)


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ell Hot-Region Boundary Complexity Index test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Hot-Region Boundary Complexity Index",
                "xlabel": "Boundary / interior ratio",
                "ylabel": "Count",
                "bins": 50,
                "caption": "No simulation data.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    p_value_upper = float(np.mean(sim >= planck_stat))
    p_value_lower = float(np.mean(sim <= planck_stat))
    p_value_two_sided = min(2.0 * min(p_value_upper, p_value_lower), 1.0)

    rank_above = int(np.sum(sim >= planck_stat))
    rank_below = int(np.sum(sim <= planck_stat))
    percentile = 100.0 * rank_below / n_sims
    deviation_sigma = (planck_stat - mean_sim) / std_sim if std_sim > 0 else 0.0

    if p_value_two_sided <= 0.01:
        strength = "strong"
    elif p_value_two_sided <= 0.05:
        strength = "moderate"
    else:
        strength = "negligible"

    interp = (
        f"Boundary-to-interior ratio of largest hot region = {planck_stat:.4f}. "
        f"Sim mean = {mean_sim:.4f} ± {std_sim:.4f} "
        f"({deviation_sigma:+.2f}σ). Percentile = {percentile:.1f}%, "
        f"two-sided p = {p_value_two_sided:.4f}. Strength: {strength}. "
    )

    if planck_stat > mean_sim + 2 * std_sim:
        interp += (
            "The Planck map's dominant hot region has anomalously high boundary "
            "complexity compared to Gaussian LCDM expectations, suggesting "
            "elongated, filamentary, or fractally rough morphology in the low-ℓ "
            "temperature hot excursion set. This could indicate non-Gaussian "
            "texture or anisotropic structure in the large-scale temperature field."
        )
    elif planck_stat < mean_sim - 2 * std_sim:
        interp += (
            "The Planck map's dominant hot region has anomalously low boundary "
            "complexity compared to Gaussian LCDM expectations, suggesting an "
            "unusually compact, blob-like morphology. This could indicate "
            "non-Gaussian temperature features with unusually smooth boundaries."
        )
    else:
        interp += (
            "The boundary complexity of the dominant hot region is consistent "
            "with Gaussian LCDM expectations. No significant anomaly detected "
            "in the geometric morphology of the low-ℓ hot excursion set."
        )

    caption = (
        "Histogram of the boundary-to-interior pixel ratio for the largest "
        "connected hot region (above +1σ) in the ℓ=2–20 band-limited "
        "CMB temperature map across LCDM simulations. Higher values indicate "
        "more geometrically complex (elongated/filamentary) hot structures. "
        "The red line marks the Planck observed value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value_one_sided_upper": p_value_upper,
        "p_value_one_sided_lower": p_value_lower,
        "p_value_two_sided": p_value_two_sided,
        "rank_above": rank_above,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": deviation_sigma,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Hot-Region Boundary Complexity Index",
            "xlabel": "Boundary / interior ratio (largest hot region)",
            "ylabel": "Number of simulations",
            "bins": 50,
            "caption": caption,
        },
    }
