Hypothesis

The low-ℓ (ℓ=2–20) temperature excursion set lacunarity for the Planck CMB map is significantly higher than typical ΛCDM realizations, indicating a more heterogeneous (gappy) pattern of hot regions at large angular scales.

Method

The CMB map is bandpass-filtered to retain only ℓ = 2–20, and a binary excursion-set mask is constructed for pixels exceeding +1σ of the filtered map. Lacunarity is computed as the squared coefficient of variation (CV²) of the smoothed local density (5° FWHM) across valid pixels, quantifying spatial heterogeneity ("gappiness") in the hot regions at large angular scales. The statistic is the mean CV² over the valid sky.

Result Summary

Planck lacunarity = 3.234. Simulation mean ± σ = 3.759 ± 0.279. Anomaly ratio = 0.860. The Planck value lies *below* the simulation mean (z = −1.88). One-tailed upper p-value = 0.983 (no evidence for excess); lower-tail p-value = 0.017, two-sided p = 0.034.

The hypothesis predicted excessive gappiness in Planck's low-ℓ hot pattern. The data show the opposite — Planck's excursion set is *less* heterogeneous (more homogeneous) than typical ΛCDM realisations, at approximately 2.1σ below the mean. The directional upper-tail test yields no anomaly; the signal is in the lower tail, not the one being tested.

Agent Justification

The test statistic is defined such that higher lacunarity values correspond to greater spatial heterogeneity (gappiness). The analysis code explicitly computes an upper-tail p-value (p_value_upper) and states that "a significantly high value would indicate the hot low-ℓ pattern in the real sky is more heterogeneous (patchier) than typical ΛCDM realizations." This reveals a clear directional expectation that anomalies manifest as excessive gappiness, making a one-tailed upper test appropriate.

Comparison To Literature

A literature search for "CMB lacunarity excursion set morphology" returned no directly comparable studies. Barreiro et al. (1997) analysed excursion-set correlations for non-Gaussianity but did not use lacunarity. No prior work applying lacunarity to low-ℓ CMB excursion sets was found.

import numpy as np
import healpy as hp

test_description = (
    "Compute the low-ℓ (ℓ=2–20) temperature excursion set by filtering the CMB map with a "
    "bandpass in multipole space. Define a binary mask for pixels where the filtered temperature "
    "exceeds +1σ of its own standard deviation. Spatial heterogeneity ('lacunarity') of this "
    "binary mask is measured by smoothing it with a 5° FWHM Gaussian beam to estimate local "
    "density, then computing the squared coefficient of variation (CV² = Var/Mean²) of the "
    "smoothed density over valid pixels. Higher values indicate a more 'gappy' or clustered "
    "pattern of hot regions at large angular scales."
)


def analyze_map(m):
    """Return low-ℓ excursion set lacunarity (CV² of smoothed binary map)."""
    # --- 1. Identify valid (non-masked) pixels ---
    valid = np.isfinite(m)
    if not np.any(valid):
        return 0.0
    m_clean = np.where(valid, m, 0.0)

    nside = hp.npix2nside(len(m_clean))

    # --- 2. Bandpass filter in ℓ space: keep only ℓ = 2–20 ---
    alm = hp.map2alm(m_clean, lmax=20)
    ell, _ = hp.Alm.getlm(lmax=20)
    bandpass_mask = (ell >= 2) & (ell <= 20)
    alm_filtered = np.where(bandpass_mask, alm, 0.0)
    m_filtered = hp.alm2map(alm_filtered, nside, lmax=20, verbose=False)

    # --- 3. Build binary excursion set: pixels > +1σ of the filtered map ---
    filt_valid = m_filtered[valid]
    sigma = np.std(filt_valid)
    if sigma == 0:
        return 0.0
    threshold = sigma  # +1σ above mean (mean ≈ 0 for bandpassed map)

    binary_map = np.where(m_filtered > threshold, 1.0, 0.0)
    binary_map[~valid] = 0.0  # zero out masked pixels for smoothing

    # --- 4. Smooth binary map with ~5° FWHM beam to estimate local density ---
    # This is a fully vectorized approximation to counting on-pixels in test discs.
    fwhm_rad = np.radians(5.0)
    smoothed = hp.smoothing(binary_map, fwhm=fwhm_rad, verbose=False)

    # --- 5. Compute lacunarity = CV² of smoothed density over valid pixels ---
    s_valid = smoothed[valid]
    mean_s = np.mean(s_valid)
    var_s = np.var(s_valid, ddof=0)

    if mean_s == 0:
        return 0.0

    lacunarity = var_s / (mean_s ** 2)

    return float(lacunarity)


def summarize_results(planck_stat, sim_results):
    """Interpret the Low-ℓ Excursion Set Lacunarity statistic."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)

    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    median_s = float(np.median(sims))

    # Fraction of simulations with lacunarity >= Planck (upper tail = more gappy)
    p_value_upper = float(np.mean(sims >= planck_stat))
    p_value_lower = float(np.mean(sims <= planck_stat))
    p_two = min(2.0 * min(p_value_upper, p_value_lower), 1.0)

    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else 0.0
    anomaly_ratio = planck_stat / mean_s if mean_s > 0 else float("inf")

    return {
        "planck_stat": planck_stat,
        "planck_lacunarity": planck_stat,
        "anomaly_ratio": anomaly_ratio,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "sim_median": median_s,
        "z_score": z_score,
        "p_value_upper": p_value_upper,
        "p_value_lower": p_value_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "interpretation": (
            f"Lacunarity CV² measures spatial heterogeneity in the low-ℓ (2–20) hot "
            f"excursion set (+1σ threshold), estimated via 5°-smoothed density variance. "
            f"Higher values indicate a more 'gappy' or clustered pattern of hot regions. "
            f"Planck lacunarity = {planck_stat:.4f}. "
            f"Sim mean ± σ = {mean_s:.4f} ± {std_s:.4f}. "
            f"Anomaly ratio (Planck/sim_mean) = {anomaly_ratio:.4f}. "
            f"z = {z_score:.2f}, two-sided p = {p_two:.4f}. "
            f"A significantly high value would indicate the hot low-ℓ pattern in the real "
            f"sky is more heterogeneous (patchier) than typical ΛCDM realizations."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Excursion Set Lacunarity",
            "xlabel": "Lacunarity (CV² of smoothed density)",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck lacunarity = {planck_stat:.4f}. "
                f"Sim mean ± σ = {mean_s:.4f} ± {std_s:.4f}. "
                f"Anomaly ratio = {anomaly_ratio:.4f}. "
                f"z = {z_score:.2f}, P(two-sided) = {p_two:.4f}."
            ),
        },
    }
