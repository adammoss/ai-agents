Hypothesis

The observed power-weighted mean multipole ⟨ℓ⟩_w for ℓ=2–10 in the Planck CMB map is different from the distribution predicted by ΛCDM simulations.

Method

Computes the power-weighted mean multipole ⟨ℓ⟩_w = Σ ℓ(2ℓ+1)C_ℓ / Σ (2ℓ+1)C_ℓ over ℓ=2–10 for the Planck SMICA map (nside=256), characterising where within the low-ℓ range power preferentially accumulates. This single scalar summarises the cumulative power distribution shape, analogous to a Kolmogorov–Smirnov style test on cumulative angular power. Compared against 1000 ΛCDM simulations processed identically (two-tailed).

Result Summary

Planck ⟨ℓ⟩_w = 5.504; simulation mean ± σ = 4.795 ± 0.482; z = 1.47; two-tailed p = 0.152. The Planck value is shifted toward higher effective multipoles relative to the simulation mean, indicating power at ℓ=2–10 is slightly biased toward ℓ≈10 rather than concentrated at the lowest multipoles, but the departure is modest.

The observed ⟨ℓ⟩_w lies 0.71 above the simulation mean—about 1.47σ—falling well within normal statistical fluctuations. No evidence that power accumulates anomalously quickly (which would signal a low-ℓ deficit) or anomalously slowly. The result is consistent with ΛCDM expectations at this precision.

Agent Justification

The test is designed to detect general anomalies in the low-ℓ cumulative power spectrum shape without a specific directional prediction, as indicated by the analysis code computing both upper and lower p-values and interpreting them as a two-tailed test. The statistic ⟨ℓ⟩_w could be higher or lower than expected, reflecting power accumulating too quickly or too slowly at low multipoles, so a two-tailed approach is appropriate for exploratory analysis.

Comparison To Literature

The low-ℓ power deficit and quadrupole–octupole anomalies are extensively documented (e.g., Planck 2015 results XVI). However, this specific cumulative-deviation scalar—using a power-weighted mean multipole over ℓ=2–10 as a summary statistic—does not appear as a standard test in published Planck anomaly analyses. The negative finding is broadly consistent with prior studies finding ambiguous or marginal low-ℓ anomalies depending on estimator and masking choices.

import numpy as np
import healpy as hp

test_description = (
    "For the masked Planck SMICA map, compute the angular power spectrum C_ℓ for ℓ=2–10. "
    "Compute the cumulative power S(ℓ) = Σ_{ℓ'=2}^{ℓ} (2ℓ'+1)C_ℓ' and normalize to get "
    "F(ℓ) = S(ℓ)/S(10). The per-map statistic is the power-weighted mean multipole "
    "⟨ℓ⟩_w = Σ ℓ(2ℓ+1)C_ℓ / Σ (2ℓ+1)C_ℓ over ℓ=2–10, characterizing where within the "
    "low-ℓ range power preferentially accumulates. Comparing Planck to ΛCDM simulations "
    "tests whether the observed cumulative power buildup shape is anomalous, analogous to "
    "a Kolmogorov–Smirnov style test on the cumulative angular power spectrum. A lower "
    "⟨ℓ⟩_w indicates power concentrated at the lowest multipoles (ℓ=2,3); a higher value "
    "indicates power biased toward ℓ~10."
)


def analyze_map(m):
    """Return power-weighted mean multipole ⟨ℓ⟩_w for ℓ=2..10.

    Computes C_ℓ via anafast on the (masked→0-filled) map, then returns
    Σ ℓ(2ℓ+1)C_ℓ / Σ (2ℓ+1)C_ℓ as a single scalar summarizing the
    cumulative power distribution shape at low ℓ.
    """
    nside = hp.npix2nside(len(m))

    # Replace NaN masked pixels with 0 for healpy compatibility
    valid = np.isfinite(m)
    m_clean = np.where(valid, m, 0.0)

    # Compute pseudo-C_ℓ up to ℓ=10
    cl = hp.anafast(m_clean, lmax=10)

    # Multipole range ℓ=2..10
    ells = np.arange(2, 11)
    weights = (2 * ells + 1) * cl[2:11]

    total_weight = np.sum(weights)
    if total_weight <= 0:
        # Fallback: midpoint of range if no power detected
        return 6.0

    weighted_mean_ell = np.sum(ells * weights) / total_weight
    return float(weighted_mean_ell)


def summarize_results(planck_stat, sim_results):
    """Interpret the Low-ℓ Power Spectrum Cumulative Deviation Index."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)

    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    median_s = float(np.median(sims))

    # p-value upper: fraction of sims with ⟨ℓ⟩_w as extreme or more than Planck (toward low ℓ)
    p_upper = float(np.mean(sims >= planck_stat))
    # p-value lower: fraction of sims with ⟨ℓ⟩_w as extreme or more toward high ℓ
    p_lower = float(np.mean(sims <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)

    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else 0.0

    return {
        "planck_stat": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "sim_median": median_s,
        "z_score": z_score,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "interpretation": (
            f"The statistic is the power-weighted mean multipole ⟨ℓ⟩_w in the range ℓ=2–10. "
            f"Planck ⟨ℓ⟩_w = {planck_stat:.4f}. Sim mean ± σ = {mean_s:.4f} ± {std_s:.4f}. "
            f"A lower ⟨ℓ⟩_w means power accumulates faster at the lowest multipoles (ℓ=2,3), "
            f"consistent with a low-ℓ power deficit or anomalous quadrupole–octupole structure. "
            f"A higher ⟨ℓ⟩_w means power is biased toward ℓ~10. "
            f"For a flat C_ℓ spectrum, ⟨ℓ⟩_w ≈ 7.03. "
            f"z = {z_score:.2f}, two-sided p = {p_two:.4f}."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Power Spectrum Cumulative Deviation Index",
            "xlabel": "Power-weighted mean multipole ⟨ℓ⟩_w (ℓ=2–10)",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck ⟨ℓ⟩_w = {planck_stat:.4f}. "
                f"Sim mean ± σ = {mean_s:.4f} ± {std_s:.4f}. "
                f"z = {z_score:.2f}, P(two-sided) = {p_two:.4f}."
            ),
        },
    }
