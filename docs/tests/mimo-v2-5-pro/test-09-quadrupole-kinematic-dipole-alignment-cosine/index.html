Hypothesis

The observed CMB quadrupole is abnormally aligned with the kinematic dipole direction, resulting in a |cos θ| statistic significantly higher than the isotropic expectation of ~0.46.

Method

The ℓ = 2 spherical harmonic coefficients are extracted from the Planck low-ℓ SMICA map, and the quadrupole's principal axis direction is computed. The absolute cosine of the angle between this axis and the CMB kinematic dipole direction (l, b) = (264°, 48°) is formed as |cos θ| = |Σ_m a_{2,m} d_{2,m}| / (Σ_m |a_{2,m}|²)^{1/2}. Under isotropy, the expected value is ≈ 0.46. The statistic is evaluated against 1000 FFP ΛCDM simulations at nside = 256.

Result Summary

Planck |cos θ| = 0.889, compared to simulation mean = 0.482 and standard deviation = 0.278. The one-sided p-value is 0.079 (rank 79/1000), corresponding to ~1.41σ. The quadrupole is moderately aligned with the kinematic dipole, roughly 1.5× the isotropic expectation.

The observed alignment exceeds the isotropic mean by a notable margin but does not reach the conventional 2σ (p = 0.05) significance threshold. The quadrupole plane appears more perpendicular to the dipole direction than typical ΛCDM realizations, but this level of tension could arise by chance in ~8% of random skies.

Agent Justification

The test statistic is |cos θ|, measuring the alignment between the quadrupole and dipole. Under isotropy, this value should be around 0.46, but the anomaly hypothesis posits an unusually strong alignment. The analysis code explicitly computes a one-sided p-value as the fraction of simulations with |cos θ| ≥ Planck value, focusing on the upper tail of the distribution. This indicates a clear directional preference for higher alignment values, justifying an upper one-tailed test.

Comparison To Literature

Quadrupole-dipole alignment has been previously investigated using multipole vector methods by Abramo et al. (2006) and in the context of dipole straylight contamination by Gruppuso et al. (2007). The present cosine-alignment statistic is a specific scalar projection; broadly consistent directional alignments have been noted but generally with low significance. The current result is consistent with the marginal, non-significant tensions reported in earlier work.

import numpy as np
import healpy as hp

test_description = (
    "Quadrupole-Kinematic Dipole Alignment Cosine. Extracts the ℓ=2 spherical "
    "harmonic coefficients a_{2,m} from the low-ℓ SMICA map and computes the "
    "direction of the quadrupole's principal (normal) axis. The signed dot product "
    "with the CMB kinematic dipole direction n̂_dip = (cos b cos l, cos b sin l, sin b) "
    "evaluated at (l, b) = (264°, 48°) in galactic coordinates is formed, and the "
    "statistic is the absolute cosine of the angle between them: "
    "|cos θ| = |Σ_m a_{2,m} d_{2,m}| / (Σ_m |a_{2,m}|²)^{1/2}, where "
    "d_{2,m} = {sin b cos b, −sin l cos b, cos l cos b, −cos 2l cos b, sin 2l cos b} "
    "for m = −2,…,2. This single scalar measures whether the quadrupole plane is "
    "abnormally aligned with the observer's motion direction; in isotropy it should "
    "be ≈ 0.46 (the expected |cos θ| for a random direction). "
    "Evaluated by comparing the Planck value against 1000 FFP ΛCDM simulations."
)


def analyze_map(m):
    m = np.asarray(m, dtype=np.float64)
    
    # Replace NaN (masked) pixels with 0.0 for healpy
    m_clean = np.where(np.isfinite(m), m, 0.0)
    
    # Compute spherical harmonic coefficients up to l=2
    alm = hp.map2alm(m_clean, lmax=2)
    
    # Extract real a_{2,m} coefficients for m = -2, -1, 0, 1, 2
    # For lmax=2, alm array has 6 elements: [a00, a10, a11, a20, a21, a22]
    # Indices: 0=ℓ0,m0; 1=ℓ1,m0; 2=ℓ1,m1; 3=ℓ2,m0; 4=ℓ2,m1; 5=ℓ2,m2
    
    # Extract complex a_{2,m} coefficients
    a20_complex = alm[3]  # m=0 (real)
    a21_complex = alm[4]  # m=1 (complex)
    a22_complex = alm[5]  # m=2 (complex)
    
    # Convert to real coefficients in the standard basis
    # For m=0: a_{2,0} = Re(a_{2,0})
    # For m>0: a_{2,m} = Re(a_{2,m}), a_{2,-m} = Im(a_{2,m})
    a2m = np.array([
        a22_complex.imag,       # m = -2: a_{2,-2} = Im(a_{2,2})
        -a21_complex.imag,      # m = -1: a_{2,-1} = -Im(a_{2,1})
        a20_complex.real,       # m =  0: a_{2,0} = Re(a_{2,0})
        a21_complex.real,       # m = +1: a_{2,1} = Re(a_{2,1})
        a22_complex.real,       # m = +2: a_{2,2} = Re(a_{2,2})
    ], dtype=np.float64)
    
    # CMB kinematic dipole direction in galactic coordinates
    l_deg = 264.0
    b_deg = 48.0
    l_rad = np.radians(l_deg)
    b_rad = np.radians(b_deg)
    
    # Compute d_{2,m} coefficients at the dipole direction
    sin_b = np.sin(b_rad)
    cos_b = np.cos(b_rad)
    sin_l = np.sin(l_rad)
    cos_l = np.cos(l_rad)
    
    d2m = np.array([
        sin_b * cos_b,                  # m = -2
        -sin_l * cos_b,                 # m = -1
        cos_l * cos_b,                  # m =  0
        -np.cos(2 * l_rad) * cos_b,     # m = +1
        np.sin(2 * l_rad) * cos_b,      # m = +2
    ], dtype=np.float64)
    
    # Compute |cos θ| = |Σ_m a_{2,m} d_{2,m}| / (Σ_m |a_{2,m}|²)^{1/2}
    norm_a = np.sqrt(np.sum(a2m ** 2))
    if norm_a == 0.0:
        return 0.0
    
    dot_product = np.dot(a2m, d2m)
    abs_cos_theta = np.abs(dot_product) / norm_a
    
    return float(abs_cos_theta)


def summarize_results(planck_stat, sim_results):
    sim_arr = np.array(sim_results, dtype=np.float64)
    finite_mask = np.isfinite(sim_arr)
    n_finite = int(finite_mask.sum())
    
    if n_finite > 0:
        valid = sim_arr[finite_mask]
        # One-sided p-value: fraction of simulations with alignment >= Planck
        rank = int(np.sum(valid >= planck_stat))
        p_value = rank / n_finite
    else:
        rank = 0
        p_value = float('nan')
    
    sim_median = float(np.nanmedian(sim_arr))
    sim_mean = float(np.nanmean(sim_arr))
    sim_std = float(np.nanstd(sim_arr))
    
    # Expected |cos θ| for random direction in 5D space = sqrt(π/16) ≈ 0.443
    # Or for a unit vector dot a fixed unit vector: E[|cos θ|] ≈ 0.46
    expected_iso = 0.46
    
    interpretation = (
        f"Anomalous: quadrupole-dipole alignment |cos θ| = {planck_stat:.4f} "
        f"(one-sided p = {p_value:.3f}, {rank}/{n_finite}). "
        f"This is significantly above the isotropic expectation of ≈ {expected_iso:.2f}, "
        f"indicating the quadrupole plane is abnormally aligned with the observer's "
        f"motion direction."
        if p_value < 0.05
        else f"Quadrupole-dipole alignment |cos θ| = {planck_stat:.4f} "
             f"(one-sided p = {p_value:.3f}) is consistent with isotropic "
             f"ΛCDM expectations (E[|cos θ|] ≈ {expected_iso:.2f})."
    )
    
    result = {
        "test_description": test_description,
        "planck_stat": float(planck_stat),
        "p_value": float(p_value),
        "rank": rank,
        "n_simulations": len(sim_results),
        "n_valid": n_finite,
        "sim_mean": sim_mean,
        "sim_std": sim_std,
        "sim_median": sim_median,
        "exceeds_95th_percentile": bool(p_value < 0.05),
        "interpretation": interpretation,
        "dipole_direction_galactic": {"l_deg": 264.0, "b_deg": 48.0},
        "expected_isotropic_value": expected_iso,
        "alignment_enhancement": float(planck_stat / expected_iso) if expected_iso > 0 else float('nan'),
    }
    
    result["plot_spec"] = {
        "kind": "histogram",
        "title": "Quadrupole–Kinematic Dipole Alignment (ℓ = 2)",
        "xlabel": "|cos θ| (alignment statistic)",
        "ylabel": "Number of simulations",
        "bins": 50,
        "caption": (
            f"Histogram of |cos θ| = |Σ a_{{2,m}} d_{{2,m}}| / ||a_{{2,m}}|| "
            f"from {n_finite} FFP ΛCDM simulations. "
            f"Planck value = {planck_stat:.4f} (one-sided p = {p_value:.3f}, "
            f"rank {rank}/{n_finite}). "
            f"The dipole direction is (l, b) = (264°, 48°). "
            f"The isotropic expectation is ≈ {expected_iso:.2f}; "
            f"a value near 1 indicates the quadrupole plane is nearly perpendicular "
            f"to (or aligned with) the dipole direction."
        ),
    }
    
    return result
