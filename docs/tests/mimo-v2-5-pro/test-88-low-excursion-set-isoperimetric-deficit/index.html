Hypothesis

The isoperimetric deficit of the low-ℓ hot excursion set in the Planck CMB temperature map deviates significantly from the distribution expected under ΛCDM simulations, indicating an anomaly in the compactness or boundary structure of the hot region.

Method

For the Planck CMB temperature map band-limited to ℓ ≤ 20, an excursion set is defined as pixels exceeding the mean (0 µK). The isoperimetric deficit D = 1 − (4πA/P²) is computed from the set's area A and boundary perimeter P, quantifying shape compactness: D = 0 for a perfect hemisphere, approaching 1 for fractal-like boundaries.

Result Summary

Planck deficit D = 0.9968, indicating an extremely complex, near-fractal boundary structure for the hot excursion region. Simulation mean ± std = 0.9960 ± 0.0007 from 1000 ΛCDM realisations. The Planck value ranks 933/1000. z-score = 1.23, two-sided p-value = 0.134.

The hot excursion region in the low-ℓ Planck map exhibits highly convoluted boundary morphology, but this is expected: both Planck data and simulations yield deficits near 0.996, reflecting that the ℓ ≤ 20 excursion set on a sphere inherently has a complex, fragmented boundary. The slight excess (z = 1.23) is not statistically significant. The boundary complexity is a generic geometric property of low-ℓ excursion sets rather than an anomaly.

Agent Justification

The test measures compactness, and anomalies could appear as either unusually compact (low deficit) or highly convoluted (high deficit) boundaries. The analysis code computes both upper and lower p-values and a two-sided p-value, showing no inherent directional bias. Since this is an exploratory anomaly test without a prior expectation for a specific change, a two-tailed test is appropriate for detecting deviations in either direction.

Comparison To Literature

No prior studies were found applying the isoperimetric deficit specifically to CMB excursion sets. Related work includes excursion set correlation analyses for non-Gaussianity testing (Barreiro et al. 1997) and isoperimetric deficit theory in differential geometry (Röger & Schätzle 2011), but neither addresses this particular statistic on CMB maps.

import numpy as np
import healpy as hp

test_description = (
    "For the low-ℓ filtered temperature map (ℓ ≤ 20), define an excursion set "
    "as all pixels where the temperature exceeds 0 µK (the mean). Compute the "
    "set's total area A in square degrees and the total perimeter P in degrees "
    "of the set's boundary (counting the edges between pixels where one is inside "
    "the set and the other is outside). The statistic is the isoperimetric deficit: "
    "1 - (4πA / P²). This measures the shape compactness of the hot excursion region, "
    "where a perfect hemisphere on a sphere gives a deficit of 0 and more convoluted, "
    "fractal-like boundaries yield values closer to 1."
)


def analyze_map(m):
    """Return isoperimetric deficit of the low-ℓ excursion set boundary.

    The deficit is 1 - (4π n_pixels / n_edges²), which measures how convoluted
    the hot excursion-set boundary is relative to a compact (hemisphere-like) shape.
    0 = perfectly compact; approaches 1 for fractal-like boundaries.
    """
    valid = np.isfinite(m)
    n_valid = int(np.sum(valid))

    if n_valid < 50:
        return 0.0

    # Replace NaN with 0.0 for healpy compatibility
    m_clean = np.where(valid, m, 0.0)
    nside = hp.npix2nside(len(m_clean))

    # Band-limit to ℓ ≤ 20
    lmax = 20
    alm = hp.map2alm(m_clean, lmax=lmax)
    m_low = hp.alm2map(alm, nside=nside, lmax=lmax, verbose=False)

    # Excursion set: valid pixels where T > 0 (mean after filtering)
    excursion = (m_low > 0.0) & valid
    n_pixels = int(np.sum(excursion))

    if n_pixels == 0:
        return 0.0

    # Count boundary edges between excursion and non-excursion pixels
    npix = len(m)
    all_pix = np.arange(npix)
    neighbors = hp.get_all_neighbours(nside, all_pix)  # (8, npix)

    # For each neighbor direction, count boundary edges
    # Each boundary edge is counted twice (once from each side), so divide by 2
    n_boundary_edges = 0
    for d in range(8):
        nb = neighbors[d]  # (npix,)
        valid_nb = (nb >= 0) & valid[nb]  # valid neighbor exists
        # Boundary edge: exactly one of (pixel, neighbor) is in excursion set
        is_boundary = valid_nb & (excursion != excursion[nb])
        n_boundary_edges += int(np.sum(is_boundary))

    # Divide by 2 since each edge is counted from both sides
    n_boundary_edges = n_boundary_edges // 2

    if n_boundary_edges == 0:
        return 0.0

    # Isoperimetric deficit: 1 - (4π n_pixels / n_edges²)
    # This is the discrete analogue of 1 - (4πA / P²) where
    # the pixel edge length L cancels: A = n_pixels * L², P = n_edges * L
    ratio = 4.0 * np.pi * n_pixels / (n_boundary_edges ** 2)
    deficit = 1.0 - ratio

    return float(np.clip(deficit, 0.0, 1.0))


def summarize_results(planck_stat, sim_results):
    """Interpret the Low-ℓ Excursion Set Isoperimetric Deficit."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)

    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    median_s = float(np.median(sims))
    rank = int(np.sum(sims <= planck_stat))
    p_upper = float(np.mean(sims >= planck_stat))
    p_lower = float(np.mean(sims <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else 0.0

    # Interpret the compactness of the hot excursion region
    if planck_stat < 0.2:
        compactness = "very compact, hemisphere-like hot region"
    elif planck_stat < 0.4:
        compactness = "moderately compact with some boundary irregularity"
    elif planck_stat < 0.6:
        compactness = "moderately convoluted boundary"
    elif planck_stat < 0.8:
        compactness = "highly convoluted, filamentary boundary"
    else:
        compactness = "extremely complex, near-fractal boundary structure"

    return {
        "planck_stat": planck_stat,
        "isoperimetric_deficit": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "sim_median": median_s,
        "z_score": z_score,
        "rank": rank,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "excursion_compactness": compactness,
        "interpretation": (
            f"The isoperimetric deficit of the low-ℓ (ℓ≤20) hot excursion set is "
            f"D = {planck_stat:.4f}. Values near 0 indicate a compact, hemisphere-like "
            f"hot region; values approaching 1 indicate highly convoluted, fractal-like "
            f"boundaries. Planck deficit = {planck_stat:.4f} ({compactness}). "
            f"Simulation mean ± std = {mean_s:.4f} ± {std_s:.4f}. "
            f"z = {z_score:.2f}, two-sided p = {p_two:.4f}."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Excursion Set Isoperimetric Deficit",
            "xlabel": "Isoperimetric deficit (1 − 4πA/P²)",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck deficit = {planck_stat:.4f}. "
                f"Sim mean ± std = {mean_s:.4f} ± {std_s:.4f}. "
                f"z = {z_score:.2f}, P(two-sided) = {p_two:.4f}."
            ),
        },
    }
