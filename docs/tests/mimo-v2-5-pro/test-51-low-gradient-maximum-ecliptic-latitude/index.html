Hypothesis

The maximum gradient in the low-ℓ (ℓ=2–10) CMB temperature map is significantly aligned with the ecliptic plane, resulting in a lower-than-expected |sin(ecliptic latitude)| statistic.

Method

Computes gradient magnitude of the low-ℓ (ℓ=2–10) CMB temperature map via spherical harmonic derivatives. Identifies the pixel with maximum gradient and converts its position to ecliptic coordinates. The statistic is |sin(ecliptic latitude)|, designed to be small if the steepest temperature change aligns with the ecliptic plane.

Result Summary

Planck yields |sin(ecl_lat)| = 0.9184 (ecliptic lat ≈ 66.7°), placing the gradient maximum near the ecliptic pole rather than the plane. Simulations give mean = 0.9175 ± 0.0009. The Planck value is +0.96σ above the simulation mean (percentile = 76.7%), with two-sided p = 0.466. One-sided p-value toward the ecliptic plane = 1.0, meaning zero simulations produced a smaller value. Anomaly strength is negligible.

The maximum gradient in the low-ℓ temperature field lies close to the ecliptic pole, not the ecliptic plane. This result is fully consistent with Gaussian random realizations. No evidence is found for ecliptic-plane alignment of the large-scale temperature gradient structure.

Agent Justification

The test description explicitly focuses on measuring how close the steepest temperature change is to the ecliptic plane, with the statistic |sin(ecliptic latitude)| designed to be small when aligned with the plane. The code interpretation section highlights that small values indicate alignment with the ecliptic plane and the analysis computes a specific one-sided p-value for the ecliptic plane alignment. This represents a clear directional hypothesis about an anomaly, rather than a general deviation in any direction.

Comparison To Literature

No directly comparable study using gradient-based statistics was found in the search results. Related work by Weeks (2004) and Abramo et al. (2006) examined multipole vector alignment with the ecliptic plane at low ℓ, finding some evidence for quadrupole–octopole–ecliptic alignment at >2σ. The present gradient-based approach probes a different observable and finds no such anomaly.

import numpy as np
import healpy as hp

test_description = (
    "Compute the gradient magnitude of the low-ℓ (ℓ=2–10) temperature map by "
    "taking spherical harmonic derivatives. Find the pixel with the maximum "
    "gradient magnitude and convert its coordinates to ecliptic coordinates. "
    "The statistic is the absolute value of the sine of the ecliptic latitude "
    "of that pixel. A value near 0 indicates the steepest temperature change "
    "is aligned with the ecliptic plane; a value near 1 indicates alignment "
    "with the ecliptic poles."
)


def analyze_map(m):
    """Compute |sin(ecliptic latitude)| of the pixel with maximum gradient magnitude in low-ℓ map."""
    nside = hp.npix2nside(len(m))
    mask = np.isfinite(m)
    n_valid = int(np.sum(mask))
    if n_valid < 500:
        return 0.5  # Return neutral value for severely masked maps

    # Fill masked pixels with 0 for healpy compatibility
    m_filled = np.where(mask, m, 0.0)

    # Get spherical harmonic coefficients
    alm = hp.map2alm(m_filled, lmax=10, iter=0)

    # Zero out ℓ=0 and ℓ=1 to isolate ℓ=2-10
    alm_low = alm.copy()
    # Zero out ℓ=0 (index 0) and ℓ=1 (indices 1, 2)
    alm_low[0] = 0.0
    alm_low[1] = 0.0
    alm_low[2] = 0.0

    # Compute map and its derivatives from alm using alm2map_der1
    # alm2map_der1 returns (map, dT/dtheta, dT/dphi)
    # where dT/dphi is the raw phi derivative (NOT divided by sin(theta))
    maps = hp.alm2map_der1(alm_low, nside, lmax=10)
    
    # maps[0] = temperature map (not needed)
    # maps[1] = dT/dtheta (theta derivative)
    # maps[2] = dT/dphi (raw phi derivative)
    d_theta = maps[1]
    d_phi = maps[2]

    # For gradient magnitude on sphere, we need:
    # |grad T| = sqrt( (dT/dtheta)^2 + (dT/dphi / sin(theta))^2 )
    # Get theta for each pixel
    theta_all = hp.pix2ang(nside, np.arange(hp.nside2npix(nside)))[0]
    sin_theta = np.sin(theta_all)
    # Avoid division by zero at poles
    sin_theta = np.where(sin_theta > 1e-10, sin_theta, 1e-10)

    grad_mag = np.sqrt(d_theta**2 + (d_phi / sin_theta)**2)

    # Only consider valid (unmasked) pixels
    grad_mag_valid = np.where(mask, grad_mag, -np.inf)
    
    # Find pixel with maximum gradient
    max_pix = np.argmax(grad_mag_valid)
    max_grad = grad_mag[max_pix]
    
    if not np.isfinite(max_grad) or max_grad <= 0:
        return 0.5

    # Convert pixel to equatorial (theta, phi)
    theta, phi = hp.pix2ang(nside, max_pix)

    # Convert from equatorial (celestial) to ecliptic coordinates
    rot = hp.Rotator(coord=['C', 'E'])
    ecl_theta, ecl_phi = rot([theta, phi], r=True)

    # Ecliptic latitude = pi/2 - ecliptic colatitude
    ecl_lat = np.pi / 2 - ecl_theta

    # Statistic: absolute value of sine of ecliptic latitude
    result = float(np.abs(np.sin(ecl_lat)))

    if not np.isfinite(result):
        return 0.5

    return result


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ℓ Gradient Maximum Ecliptic Latitude test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Gradient Maximum Ecliptic Latitude",
                "xlabel": "|sin(ecliptic latitude)|",
                "ylabel": "Count",
                "bins": 40,
                "caption": "No simulation data.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    # Deviation from simulation mean
    deviation_sigma = (planck_stat - mean_sim) / std_sim if std_sim > 0 else 0.0

    # Rank and percentile
    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims

    # Two-sided p-value
    p_value_lower = float(np.mean(sim <= planck_stat))
    p_value_upper = float(np.mean(sim >= planck_stat))
    p_value_two_sided = 2.0 * min(p_value_lower, p_value_upper)
    p_value_two_sided = min(p_value_two_sided, 1.0)

    # One-sided toward ecliptic plane (small values)
    p_value_ecliptic = float(np.mean(sim <= planck_stat))

    if p_value_two_sided <= 0.01:
        strength = "strong"
    elif p_value_two_sided <= 0.05:
        strength = "moderate"
    else:
        strength = "negligible"

    # Interpret the alignment
    ecl_lat_deg = np.degrees(np.arcsin(planck_stat))
    
    if planck_stat < 0.26:
        alignment_desc = "very close to the ecliptic plane"
    elif planck_stat < 0.5:
        alignment_desc = "moderately close to the ecliptic plane"
    elif planck_stat < 0.71:
        alignment_desc = "at intermediate ecliptic latitude"
    elif planck_stat < 0.87:
        alignment_desc = "toward the ecliptic pole"
    else:
        alignment_desc = "very close to the ecliptic pole"

    direction = (
        f"The maximum gradient in the low-ℓ temperature field is {alignment_desc} "
        f"(ecliptic latitude ≈ {ecl_lat_deg:.1f}°). "
    )
    
    if p_value_two_sided <= 0.05:
        direction += (
            "This alignment is statistically unusual compared to Gaussian simulations, "
            "suggesting a potential connection between the CMB's large-scale temperature "
            "structure and the ecliptic coordinate system."
        )
    else:
        direction += (
            "This alignment is consistent with random Gaussian realizations. "
            "No significant ecliptic preference is detected in the gradient structure."
        )

    interp = (
        f"|sin(ecl_lat)| = {planck_stat:.4f} (ecliptic lat ≈ {ecl_lat_deg:.1f}°). "
        f"Sim mean = {mean_sim:.4f} ± {std_sim:.4f} "
        f"({deviation_sigma:+.2f}σ). Percentile = {percentile:.1f}%, "
        f"two-sided p = {p_value_two_sided:.4f}. "
        f"Strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of |sin(ecliptic latitude)| for the pixel with maximum gradient "
        "magnitude in the low-ℓ (ℓ=2–10) temperature map across FFP simulations. "
        "Values near 0 indicate alignment with the ecliptic plane; values near 1 "
        "indicate alignment with the ecliptic poles. Red line marks the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value_two_sided": p_value_two_sided,
        "p_value_ecliptic_plane": p_value_ecliptic,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": deviation_sigma,
        "ecliptic_latitude_deg": ecl_lat_deg,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Gradient Maximum Ecliptic Latitude",
            "xlabel": "|sin(ecliptic latitude)|",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
