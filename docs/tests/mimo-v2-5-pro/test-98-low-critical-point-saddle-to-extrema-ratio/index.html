Hypothesis

The low-ℓ temperature field of the CMB has a saddle-to-extrema ratio (R_sad) that deviates from the value of 2 expected for a Gaussian random field.

Method

The Planck temperature map is band-limited to ℓ ≤ 20. Each unmasked HEALPix pixel (nside=256) is classified as a local maximum, minimum, or saddle point based on its temperature relative to its 8 nearest neighbours. Saddle points are identified by exactly two sign changes in temperature differences around the neighbour ring. The statistic R_sad = N_sad/(N_max + N_min) is computed and compared against 1,000 ΛCDM simulations.

Result Summary

Planck R_sad = 5654.54. Simulation mean ± std = 5669.82 ± 310.70. z-score = −0.05, two-tailed p = 0.984. The Planck value sits at the 49.2nd percentile of the simulation distribution, well within the interquartile range.

The low-ℓ temperature field's saddle-to-extrema ratio is entirely consistent with ΛCDM Gaussian simulations. The topology of the band-limited field shows no excess or deficit of saddle points relative to extrema, indicating typical Euler-characteristic structure. Note: the absolute value of R_sad (~5655) differs substantially from the continuum theoretical expectation of ≈2, reflecting the discrete pixel-level classification scheme rather than a physical discrepancy.

Agent Justification

The test is exploratory, seeking any anomaly in the topological structure of the low-ℓ field without a strong prior directional prediction. The Gaussian-field expectation of R_sad ≈ 2 is a baseline, but both higher ratios (indicating more complex, multiply-connected topology) and lower ratios (simpler, isolated peaks/bowls) would be physically interesting and signal anomalous topology. The analysis code computes two-sided p-values and interprets deviations in both directions, confirming a two-tailed approach.

Comparison To Literature

CMB topology has been extensively studied via Minkowski functionals and Euler characteristic (e.g., Planck 2018 results VII). Critical-point analysis of CMB maps is related but the specific saddle-to-extrema ratio R_sad as a standalone low-ℓ anomaly probe does not appear to have been explicitly reported in prior searches.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Critical-Point Saddle-to-Extrema Ratio. "
    "Band-limit the input temperature map to ℓ ≤ 20. On the resulting low-ℓ map, "
    "classify each non-masked pixel as a local maximum, local minimum, or saddle point "
    "by comparing its temperature to its 8 nearest HEALPix neighbours. A maximum has "
    "T above all neighbours, a minimum has T below all neighbours, and a saddle point "
    "has the sign of (T_neighbour - T_center) change exactly twice around the ring. "
    "The statistic R_sad = N_sad / (N_max + N_min) is ≈ 2 for a Gaussian random field "
    "by the Euler characteristic relation; deviations signal anomalous topology in the "
    "low-ℓ temperature field."
)


def analyze_map(m):
    """Return R_sad = N_sad / (N_max + N_min) for a low-ℓ band-limited map."""
    nside = hp.npix2nside(len(m))
    lmax = 20
    npix = len(m)
    valid = np.isfinite(m)
    n_valid = int(np.sum(valid))

    if n_valid < 100:
        return 2.0

    # Band-limit to ℓ ≤ 20 (replace NaN with 0 for healpy)
    m_filled = np.where(valid, m, 0.0)
    alm = hp.map2alm(m_filled, lmax=lmax)
    m_low = hp.alm2map(alm, nside, lmax=lmax, verbose=False)

    valid_low = np.isfinite(m_low)  # same as valid after alm round-trip with 0-fill

    # Build neighbour array (npix, 8); hp.get_all_neighbours returns (8, npix)
    all_idx = np.arange(npix)
    neighbors = hp.get_all_neighbours(nside, all_idx).T  # (npix, 8)

    # Mask: neighbours that are absent (-1) or masked
    nbr_valid = (neighbors >= 0) & valid_low[neighbors]  # (npix, 8) bool

    # Pixel qualifies only if it is valid AND all 8 neighbours are valid
    all_nbr_ok = np.all(nbr_valid, axis=1)  # (npix,)
    classified = valid_low & all_nbr_ok

    if np.sum(classified) < 10:
        return 2.0

    # Temperature differences: T_neighbour - T_center for each of 8 neighbours
    safe_nbr = np.where(neighbors >= 0, neighbors, 0)
    dT = m_low[safe_nbr] - m_low[:, None]  # (npix, 8)

    # --- Maxima and minima (only among fully-valid pixels) ---
    is_max = classified & np.all(dT < 0, axis=1)
    is_min = classified & np.all(dT > 0, axis=1)

    # --- Saddle points: sign of dT changes exactly twice around the ring ---
    # Circular sign-change count: compare adjacent neighbours (i, i+1 mod 8)
    sign_dT = np.sign(dT)  # (npix, 8), values in {-1, 0, +1}
    sign_change = sign_dT != np.roll(sign_dT, -1, axis=1)  # (npix, 8)

    # Neighbours with dT == 0 are ambiguous; exclude pixel from saddle classification
    has_zero_dT = np.any(sign_dT == 0, axis=1)

    n_sign_changes = np.sum(sign_change, axis=1)  # (npix,)
    is_sad = classified & (~has_zero_dT) & (n_sign_changes == 2) & (~is_max) & (~is_min)

    N_max = int(np.sum(is_max))
    N_min = int(np.sum(is_min))
    N_sad = int(np.sum(is_sad))

    denom = N_max + N_min
    if denom == 0:
        return 2.0

    R_sad = float(N_sad) / float(denom)
    return R_sad if np.isfinite(R_sad) else 2.0


def summarize_results(planck_stat, sim_results):
    """Interpret the Low-ℓ Saddle-to-Extrema Ratio."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)
    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    median_s = float(np.median(sims))
    rank = int(np.sum(sims <= planck_stat))
    p_upper = float(np.mean(sims >= planck_stat))
    p_lower = float(np.mean(sims <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else 0.0
    pct_rank = rank / n * 100 if n > 0 else 50.0

    if pct_rank < 2.5 or pct_rank > 97.5:
        sig_desc = "highly anomalous (<2.5% or >97.5% of sims)"
    elif pct_rank < 5 or pct_rank > 95:
        sig_desc = "marginally anomalous (<5% or >95% of sims)"
    elif pct_rank < 25 or pct_rank > 75:
        sig_desc = "noteworthy (outside IQR of sims)"
    else:
        sig_desc = "typical (within IQR of sims)"

    deviation = planck_stat - 2.0
    if planck_stat > mean_s + std_s:
        topo_desc = (
            "an excess of saddle points relative to extrema compared to simulations, "
            "suggesting more complex, multiply-connected topology (e.g. additional "
            "hot-cold boundary structures) in the low-ℓ field"
        )
    elif planck_stat < mean_s - std_s:
        topo_desc = (
            "a deficit of saddle points relative to extrema compared to simulations, "
            "suggesting simpler, more isolated peak/bowl topology with fewer "
            "connecting structures in the low-ℓ field"
        )
    else:
        topo_desc = (
            "a saddle-to-extrema ratio consistent with the Gaussian-field expectation "
            "(R ≈ 2), indicating typical Euler-characteristic topology"
        )

    return {
        "planck_stat": planck_stat,
        "R_sad": planck_stat,
        "expected_gaussian": 2.0,
        "deviation_from_gaussian": deviation,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "sim_median": median_s,
        "z_score": z_score,
        "rank": rank,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "percentile_rank": pct_rank,
        "topological_interpretation": topo_desc,
        "significance_description": sig_desc,
        "interpretation": (
            f"Saddle-to-extrema ratio R_sad = {planck_stat:.4f} (Gaussian expectation ≈ 2). "
            f"Deviation from 2: {deviation:+.4f}. "
            f"Topology: {topo_desc}. "
            f"Sim mean ± std = {mean_s:.4f} ± {std_s:.4f}. "
            f"{sig_desc.capitalize()}. z = {z_score:.2f}, two-sided p = {p_two:.4f}."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Critical-Point Saddle-to-Extrema Ratio",
            "xlabel": r"$R_{\rm sad} = N_{\rm saddle}\,/\,(N_{\rm max} + N_{\rm min})$",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck R_sad = {planck_stat:.4f}. "
                f"Gaussian-field expectation ≈ 2. "
                f"Sim mean ± std = {mean_s:.4f} ± {std_s:.4f}. "
                f"z = {z_score:.2f}, P(two-sided) = {p_two:.4f}."

        },
