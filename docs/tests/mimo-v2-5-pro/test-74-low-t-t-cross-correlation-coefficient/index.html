Hypothesis

The CMB temperature map is consistent with an isotropic Gaussian random field, implying the cross-correlation coefficient r between T² and (∇²T)² should be zero.

Method

This test computes the normalized cross-correlation coefficient *r* between the squared temperature field T² and the squared Laplacian (curvature) field (∇²T)² over multipoles ℓ = 2–20. It probes whether the CMB's most intense temperature features (hot/cold spots) systematically coincide with regions of large spatial curvature—a coupling absent in isotropic Gaussian fields where T² and (∇²T)² respond independently to temperature structure. Non-Gaussian sources such as topological defects, foreground residuals, or primordial non-Gaussianity could inflate |r| relative to ΛCDM expectations.

Result Summary

The Planck observed value is r = 0.4917, while 1000 ΛCDM simulations yield a mean ± σ of 0.4795 ± 0.0716. The z-score is 0.17 and the two-tailed p-value is 0.902, indicating excellent agreement with the isotropic Gaussian hypothesis.

The observed cross-correlation falls comfortably within the simulation distribution, showing no evidence of anomalous spatial coupling between temperature intensity and curvature extremes at low multipoles. The intrinsic non-zero mean of *r* in simulations arises from the nonlinear mapping between the underlying Gaussian field and its squared quantities, not from non-Gaussianity.

Agent Justification

The test is designed to detect any significant deviation from zero, as the description states that a "significantly non-zero r" indicates spatial coupling absent in isotropic Gaussian fields. The analysis code computes both upper and lower p-values and uses a two-sided p-value for interpretation, showing no directional preference. Non-Gaussian sources could produce either positive or negative correlation, making a two-tailed test appropriate for general exploration.

Comparison To Literature

No prior studies examining this specific T²–(∇²T)² cross-correlation statistic on the CMB were identified in available literature or online searches. Related work exists on hot/cold spot statistics and Minkowski functionals, but this particular multipole-space cross-correlation of squared fields does not appear to have been previously documented.

import numpy as np
import healpy as hp

test_description = (
    "Tests whether the spatial distribution of extreme temperatures (hot and cold "
    "spots) correlates with the distribution of large temperature curvature across "
    "the low-ℓ (ℓ = 2–20) sky. Computes the normalized cross-correlation coefficient "
    "between the squared temperature field T² and the squared Laplacian field (∇²T)² "
    "in multipole space. For isotropic Gaussian fields, T² and (∇²T)² respond "
    "independently to temperature structure, so the expected correlation is zero. "
    "Non-Gaussian sources such as topological defects, foreground residuals, or "
    "primordial non-Gaussianity could produce correlated structure between intensity "
    "and curvature extremes that inflates |r| relative to ΛCDM simulations."
)


def analyze_map(m):
    """Return the T²–(∇²T)² cross-correlation coefficient r for ℓ = 2–20."""
    nside = hp.npix2nside(len(m))
    lmax = 20

    # --- 1. Clean masked pixels for healpy compatibility ---
    mask_valid = np.isfinite(m)
    m_clean = np.where(mask_valid, m, 0.0)

    # --- 2. Get a_ℓm and zero out ℓ = 0, 1 (discard monopole and dipole) ---
    alm = hp.map2alm(m_clean, lmax=lmax)
    for ell in range(2):
        for m_idx in range(ell + 1):
            alm[hp.Alm.getidx(lmax, ell, m_idx)] = 0.0

    # --- 3. Compute Laplacian a_ℓm: multiply by -ℓ(ℓ+1) ---
    # (Ignoring 1/R²_CMB since it cancels in the normalized correlation)
    l_arr, _ = hp.Alm.getlm(lmax)
    factors = -l_arr.astype(float) * (l_arr.astype(float) + 1.0)
    alm_lap = alm * factors

    # --- 4. Reconstruct band-limited T and ∇²T maps in pixel space ---
    T_map = hp.alm2map(alm, nside, lmax=lmax)
    lap_map = hp.alm2map(alm_lap, nside, lmax=lmax)

    # --- 5. Construct squared-field maps ---
    T2_map = T_map ** 2
    lap2_map = lap_map ** 2

    # --- 6. Transform squared maps to spherical harmonics ---
    alm_T2 = hp.map2alm(T2_map, lmax=lmax)
    alm_lap2 = hp.map2alm(lap2_map, lmax=lmax)

    # Zero out ℓ = 0, 1 in both squared-field harmonic arrays
    for ell in range(2):
        for m_idx in range(ell + 1):
            idx = hp.Alm.getidx(lmax, ell, m_idx)
            alm_T2[idx] = 0.0
            alm_lap2[idx] = 0.0

    # --- 7. Compute cross-spectra across multipole space ---
    # r = Σ_{ℓ=2}^{20} (2ℓ+1) C_ℓ^{T²,(∇²T)²} / sqrt[ Σ (2ℓ+1) C_ℓ^{T²,T²} · Σ (2ℓ+1) C_ℓ^{(∇²T)²,(∇²T)²} ]
    # where (2ℓ+1) C_ℓ^{A,B} = Σ_m a_ℓm^A a_ℓm^{B*}
    # So r = Σ_{ℓ,m} a_ℓm^{T²} a_ℓm^{(∇²T)²*} / sqrt[ Σ |a_ℓm^{T²}|² · Σ |a_ℓm^{(∇²T)²}|² ]
    # Since ℓ=0,1 are zeroed, summing over all indices gives ℓ=2..20 only.

    cross_num = np.real(np.sum(alm_T2 * np.conj(alm_lap2)))
    cross_den1 = np.real(np.sum(alm_T2 * np.conj(alm_T2)))
    cross_den2 = np.real(np.sum(alm_lap2 * np.conj(alm_lap2)))

    # --- 8. Compute normalized cross-correlation coefficient ---
    denom = np.sqrt(cross_den1 * cross_den2)
    if denom < 1e-30:
        return 0.0

    r = cross_num / denom
    return float(np.clip(r, -1.0, 1.0))


def summarize_results(planck_stat, sim_results):
    """Interpret the T²–(∇²T)² cross-correlation coefficient."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)

    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    p_upper = float(np.mean(sims >= planck_stat))
    p_lower = float(np.mean(sims <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else np.nan

    return {
        "planck_stat": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "z_score": z_score,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "interpretation": (
            "This statistic measures the multipole-space cross-correlation between the "
            "squared temperature field T² and the squared Laplacian field (∇²T)² over "
            "low multipoles ℓ = 2–20. For a statistically isotropic Gaussian random "
            "field, T² and (∇²T)² respond independently to temperature structure, "
            "yielding an expected correlation of zero. A significantly non-zero r "
            "indicates that the CMB's most intense temperature features (hot and cold "
            "spots) systematically coincide with its most curved regions—a spatial "
            "coupling absent in isotropic Gaussian fields. Such coupling could arise "
            "from topological defects, foreground residuals, or primordial "
            "non-Gaussianity."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ T²–(∇²T)² Cross-Correlation Coefficient",
            "xlabel": "r = corr(T², (∇²T)²)",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck r = {planck_stat:.4f}. "
                f"Sim mean ± σ = {mean_s:.4f} ± {std_s:.4f}. "
                f"z = {z_score:.2f}, p(two-sided) = {p_two:.3f}."
            ),
        },
    }
