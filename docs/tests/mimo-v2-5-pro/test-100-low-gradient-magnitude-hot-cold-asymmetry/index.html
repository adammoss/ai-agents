Hypothesis

The gradient magnitude ratio for hot to cold pixels is different from 1.0, indicating asymmetry in local slopes around temperature extrema.

Method

Measures the ratio of mean gradient magnitude (|∇T|) around hot pixels (T > +1σ) versus cold pixels (T < −1σ) in the low-ℓ (ℓ ≤ 32) SMICA temperature map smoothed at 5° FWHM. This tests whether local slopes around temperature maxima are systematically steeper or shallower than around minima, probing potential asymmetries in small-scale structure around extrema.

Result Summary

Hot-cold gradient ratio = 1.183 (symmetric expectation = 1.0). Simulation mean ± std = 1.049 ± 0.294. The Planck value ranks at the 70.3rd percentile of simulations, yielding z = 0.46 and two-sided p = 0.594. Hot spots exhibit moderately steeper gradients than cold spots, but this falls well within the interquartile range of ΛCDM realizations.

The observed ratio is entirely consistent with Gaussian simulations. No significant asymmetry is detected between gradient magnitudes around temperature maxima and minima. The slight preference for steeper hot-spot gradients is a statistical fluctuation with no evidential weight.

Agent Justification

The test statistic is a ratio with a symmetric expectation of 1.0, and the analysis code computes a two-sided p-value, interpreting deviations in both directions (ratio > 1.05 for steeper hot spots, ratio < 0.95 for steeper cold spots). There is no prior directional preference mentioned in the test description; it is framed as a general exploration for asymmetry between hot and cold regions.

Comparison To Literature

Prior CMB anomaly studies at low multipoles have focused on power deficit, quadrupole–octupole alignment, parity asymmetry (e.g., Creswell & Naselsky 2021, arXiv:2102.13166), and the Cold Spot. No directly comparable gradient-magnitude hot-cold asymmetry test was identified in the literature; existing work examines related but distinct asymmetry measures.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Gradient Magnitude Hot-Cold Asymmetry. "
    "For the low-ℓ (ℓ ≤ 32) SMICA temperature map smoothed with a 5° FWHM beam, "
    "compute the magnitude of the spatial gradient (|∇T|) at each pixel. "
    "Define 'hot' pixels as those with temperature T > +1σ (σ is the rms of the map) "
    "and 'cold' pixels as T < -1σ. Calculate the mean gradient magnitude for each set, "
    "then compute the statistic as the ratio: (mean |∇T| for hot pixels) / "
    "(mean |∇T| for cold pixels). This single scalar measures whether the local slope "
    "around hot spots is systematically steeper or shallower than around cold spots, "
    "testing a potential asymmetry in small-scale structure around extrema."
)


def analyze_map(m):
    """Return ratio of mean gradient magnitude for hot vs cold pixels at low-ℓ."""
    nside = hp.npix2nside(len(m))
    lmax = 32
    fwhm_rad = np.radians(5.0)

    valid = np.isfinite(m)
    n_valid = int(np.sum(valid))

    if n_valid < 100:
        return 1.0

    # Fill NaN with 0 for healpy compatibility
    m_filled = np.where(valid, m, 0.0)

    # Band-limit to ℓ ≤ 32
    alm = hp.map2alm(m_filled, lmax=lmax)

    # Apply 5° FWHM Gaussian beam to alm
    bl = hp.gauss_beam(fwhm_rad, lmax=lmax, pol=False)
    l_idx, _ = hp.Alm.getlm(lmax)
    alm_smoothed = alm * bl[l_idx]

    # alm2map_der1 returns (map, dT/dtheta, dT/dphi)
    m_smooth, dT_dtheta, dT_dphi = hp.alm2map_der1(
        alm_smoothed, nside, lmax=lmax
    )

    # Get colatitude theta for each pixel
    theta, _ = hp.pix2ang(nside, np.arange(hp.nside2npix(nside)))
    sin_theta = np.sin(theta)

    # Avoid division by zero at poles: clamp sin_theta
    sin_theta = np.clip(sin_theta, 1e-10, None)

    # Gradient magnitude on the sphere: |∇T| = sqrt((dT/dθ)² + ((1/sinθ)·dT/dφ)²)
    grad_mag = np.sqrt(dT_dtheta ** 2 + (dT_dphi / sin_theta) ** 2)

    # Compute rms (sigma) from valid pixels in the smoothed map
    temps_valid = m_smooth[valid]
    sigma = np.std(temps_valid, ddof=0)

    if sigma == 0.0:
        return 1.0

    # Define hot pixels (T > +1σ) and cold pixels (T < -1σ)
    hot_mask = valid & (m_smooth > sigma)
    cold_mask = valid & (m_smooth < -sigma)

    n_hot = int(np.sum(hot_mask))
    n_cold = int(np.sum(cold_mask))

    if n_hot < 10 or n_cold < 10:
        return 1.0

    # Mean gradient magnitude for each set
    mean_grad_hot = np.mean(grad_mag[hot_mask])
    mean_grad_cold = np.mean(grad_mag[cold_mask])

    if mean_grad_cold == 0.0:
        return 1.0

    ratio = float(mean_grad_hot / mean_grad_cold)

    return ratio if np.isfinite(ratio) else 1.0


def summarize_results(planck_stat, sim_results):
    """Interpret the Low-ℓ Gradient Magnitude Hot-Cold Asymmetry."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)
    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    median_s = float(np.median(sims))
    rank = int(np.sum(sims <= planck_stat))
    p_upper = float(np.mean(sims >= planck_stat))
    p_lower = float(np.mean(sims <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else 0.0
    pct_rank = rank / n * 100 if n > 0 else 50.0

    if pct_rank < 2.5 or pct_rank > 97.5:
        sig_desc = "highly anomalous (<2.5% or >97.5% of sims)"
    elif pct_rank < 5 or pct_rank > 95:
        sig_desc = "marginally anomalous (<5% or >95% of sims)"
    elif pct_rank < 25 or pct_rank > 75:
        sig_desc = "noteworthy (outside IQR of sims)"
    else:
        sig_desc = "typical (within IQR of sims)"

    if planck_stat > 1.05:
        asym_desc = (
            "hot spots have systematically steeper gradients than cold spots, "
            "suggesting enhanced small-scale structure around temperature maxima"
        )
    elif planck_stat < 0.95:
        asym_desc = (
            "cold spots have systematically steeper gradients than hot spots, "
            "suggesting enhanced small-scale structure around temperature minima "
            "(e.g., possible signatures of non-Gaussian features like the Cold Spot)"
        )
    else:
        asym_desc = (
            "hot and cold spots have comparable gradient magnitudes, "
            "consistent with a statistically symmetric temperature field"
        )

    return {
        "planck_stat": planck_stat,
        "hot_cold_gradient_ratio": planck_stat,
        "symmetric_expectation": 1.0,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "sim_median": median_s,
        "z_score": z_score,
        "rank": rank,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "percentile_rank": pct_rank,
        "asymmetry_description": asym_desc,
        "significance_description": sig_desc,
        "interpretation": (
            f"Hot-cold gradient magnitude ratio = {planck_stat:.4f} "
            f"(symmetric expectation = 1.0). "
            f"Asymmetry: {asym_desc}. "
            f"Sim mean ± std = {mean_s:.4f} ± {std_s:.4f}. "
            f"{sig_desc.capitalize()}. z = {z_score:.2f}, two-sided p = {p_two:.4f}."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Gradient Magnitude Hot-Cold Asymmetry",
            "xlabel": r"Mean $|\nabla T|$ (hot) / Mean $|\nabla T|$ (cold)",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck ratio = {planck_stat:.4f}. "
                f"Symmetric expectation = 1.0. "
                f"Sim mean ± std = {mean_s:.4f} ± {std_s:.4f}. "
                f"z = {z_score:.2f}, P(two-sided) = {p_two:.4f}."
            ),
        },
    }
