Hypothesis

The observed Planck CMB temperature map has non-zero excess kurtosis in its low-ℓ Laplacian field, indicating non-Gaussian features compared to Gaussian LCDM simulations.

Method

Band-limit the Planck temperature map to ℓ ≤ 20, construct the Laplacian field ∇²T by multiplying spherical harmonic coefficients by −ℓ(ℓ+1), and compute the excess kurtosis of pixel-space values over unmasked pixels. For a Gaussian CMB, the Laplacian remains Gaussian with zero excess kurtosis; localized features like the Cold Spot produce heavier tails and nonzero kurtosis. The observed value is compared against 1000 ΛCDM simulations.

Result Summary

Observed excess kurtosis of ∇²T = −0.0711. Simulation mean ± std = 0.0216 ± 0.1573. z-score = −0.59, two-tailed p-value = 0.608. The Planck value ranks at the 30.4th percentile of simulations, well within the interquartile range.

The observed kurtosis is entirely consistent with Gaussian ΛCDM expectations. The small negative value indicates very slightly lighter tails than the simulation mean, but at negligible statistical significance. There is no evidence of anomalous curvature from localized non-Gaussian features at low multipoles.

Agent Justification

The test is designed to detect any deviation from Gaussianity, as the test description mentions "nonzero excess kurtosis" without specifying a direction (could be heavier or lighter tails), and the code computes both one-sided p-values and a two-sided p-value for comparison, implying a two-tailed approach for exploring anomalies in either direction.

Comparison To Literature

Monteserin et al. (2006) studied the Laplacian among other scalar quantities as a non-Gaussianity detector on CMB maps, including its kurtosis response. However, they used Edgeworth-expanded non-Gaussian simulations rather than testing excess kurtosis of the low-ℓ Laplacian field specifically against Planck data. No directly comparable published result for this exact test was found.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Laplacian Kurtosis. "
    "Band-limit the temperature map to ℓ ≤ 20. Synthesize the Laplacian field "
    "∇²T by multiplying each spherical harmonic coefficient a_{ℓm} by −ℓ(ℓ+1) "
    "and performing an inverse spherical harmonic transform to obtain pixel-space "
    "values. Compute the excess kurtosis (fourth central moment divided by the "
    "squared standard deviation, minus 3) of the ∇²T pixel values over all "
    "unmasked pixels. For a Gaussian CMB, the Laplacian is also Gaussian and the "
    "excess kurtosis is zero. Localized non-Gaussian features—such as the CMB "
    "cold spot—produce regions of anomalous curvature, yielding heavier tails "
    "and nonzero excess kurtosis. Compare the observed value against the "
    "distribution from 1000 identically processed simulated LCDM maps."
)


def analyze_map(m):
    """Return the excess kurtosis of the Laplacian ∇²T of the low-ℓ CMB map."""
    nside = hp.npix2nside(len(m))
    lmax = 20

    # --- Replace masked pixels with 0.0 for SH transforms ---
    valid = np.isfinite(m)
    m_filled = np.where(valid, m, 0.0)

    # --- Forward transform to get a_{ℓm} ---
    alm = hp.map2alm(m_filled, lmax=lmax)

    # --- Multiply by -ℓ(ℓ+1) to get Laplacian coefficients ---
    fl = np.array([-ell * (ell + 1.0) for ell in range(lmax + 1)])
    alm_lap = hp.almxfl(alm, fl)

    # --- Inverse transform to get ∇²T in pixel space ---
    lap_map = hp.alm2map(alm_lap, nside, lmax=lmax)

    # --- Extract valid (unmasked) pixel values ---
    lap_valid = lap_map[valid]

    n = len(lap_valid)
    if n < 4:
        return 0.0

    # --- Compute excess kurtosis ---
    mean = np.mean(lap_valid)
    diff = lap_valid - mean
    var = np.mean(diff ** 2)
    if var <= 0 or not np.isfinite(var):
        return 0.0

    m4 = np.mean(diff ** 4)
    kurtosis = m4 / (var ** 2) - 3.0

    return float(kurtosis)


def summarize_results(planck_stat, sim_results):
    """Interpret the Low-ℓ Laplacian Kurtosis statistic."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)
    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    median_s = float(np.median(sims))
    rank = int(np.sum(sims <= planck_stat))
    p_upper = float(np.mean(sims >= planck_stat))
    p_lower = float(np.mean(sims <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else 0.0
    pct_rank = rank / n * 100 if n > 0 else 50.0

    if pct_rank < 2.5 or pct_rank > 97.5:
        sig_desc = "highly anomalous (<2.5% or >97.5% of sims)"
    elif pct_rank < 5 or pct_rank > 95:
        sig_desc = "marginally anomalous (<5% or >95% of sims)"
    elif pct_rank < 25 or pct_rank > 75:
        sig_desc = "noteworthy (outside IQR of sims)"
    else:
        sig_desc = "typical (within IQR of sims)"

    if planck_stat > 1.0:
        kurt_desc = "positive excess kurtosis (heavier tails than Gaussian, possible localized features)"
    elif planck_stat < -1.0:
        kurt_desc = "negative excess kurtosis (lighter tails than Gaussian)"
    else:
        kurt_desc = "excess kurtosis near zero (consistent with Gaussian)"

    return {
        "planck_stat": planck_stat,
        "excess_kurtosis": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "sim_median": median_s,
        "z_score": z_score,
        "rank": rank,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "percentile_rank": pct_rank,
        "kurtosis_description": kurt_desc,
        "significance_description": sig_desc,
        "interpretation": (
            f"Excess kurtosis of ∇²T = {planck_stat:.4f}. {kurt_desc}. "
            f"Sim mean ± std = {mean_s:.4f} ± {std_s:.4f}. "
            f"{sig_desc}. z = {z_score:.2f}, two-sided p = {p_two:.4f}."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Laplacian Kurtosis",
            "xlabel": "Excess kurtosis of ∇²T",
            "ylabel": "Count",
            "bins": 30,

            ),
        },
