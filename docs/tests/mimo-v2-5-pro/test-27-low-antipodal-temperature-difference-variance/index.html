Hypothesis

The CMB temperature map is consistent with isotropic ΛCDM simulations in terms of antipodal temperature difference variance at low multipoles.

Method

The CMB temperature map is filtered to retain only multipoles ℓ=2–5, and the temperature difference ΔT(θ,φ) = T(θ,φ) − T(π−θ, φ+π) is computed between each unmasked pixel and its antipode. The variance of ΔT over all valid pixels is taken as the test statistic. Since spherical harmonics satisfy Y_ℓm(π−θ,φ+π) = (−1)^ℓ Y_ℓm(θ,φ), even-ℓ modes contribute correlated signals while odd-ℓ modes contribute anti-correlated signals; the variance therefore probes the even–odd multipole balance.

Result Summary

Planck Var(ΔT) = 1223.9 μK², compared to simulation mean 1480.8 ± 626.0 μK² (1000 ΛCDM realisations). The two-tailed p-value is 0.806 (z = −0.41, 40.3rd percentile). Anomaly strength is negligible.

The observed antipodal temperature difference variance at low multipoles is fully consistent with isotropic ΛCDM simulations. There is no evidence of anomalous hemispherical power asymmetry or odd–even multipole imbalance in this pixel-space statistic.

Agent Justification

The test is designed to detect any departure from isotropy at large angular scales, as evidenced by both high and low variance being of interest. The analysis code computes a two-sided p-value and interprets anomalies in both directions, indicating no prior directional preference.

Comparison To Literature

Previous literature has studied hemispherical asymmetry and odd–even multipole imbalances using spherical-harmonic-space diagnostics (e.g., odd–even ℓ power ratios, dipole modulation). However, no directly comparable pixel-space variance-of-antipodal-differences statistic was identified in the search results or in prior work known to the author.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Antipodal Temperature Difference Variance: For the low-ℓ temperature "
    "map (filtered to retain multipoles ℓ=2–5), compute the temperature difference "
    "between each unmasked pixel and its antipodal point (ΔT(θ,φ) = T(θ,φ) − "
    "T(π−θ, φ+π)). Then calculate the variance of these ΔT values over all valid "
    "pixels. This single scalar quantifies large-scale antipodal asymmetry; a higher "
    "variance indicates stronger anti-correlation compared to the distribution from "
    "simulated LCDM maps."
)


def analyze_map(m):
    """Compute variance of antipodal temperature differences in the ℓ=2–5 filtered map.

    Steps:
      1. Replace masked (NaN) pixels with 0.0 for healpy compatibility.
      2. Compute alm up to lmax=5, zero out ℓ=0 and ℓ=1, reconstruct the
         band-limited map containing only ℓ=2–5.
      3. Determine antipodal pixel indices for every pixel via pix→ang→ang→pix.
      4. Identify pixels where both the pixel and its antipode are unmasked
         in the original map.
      5. Compute ΔT = T_filtered(p) − T_filtered(antipode(p)) for all valid
         pixels and return np.var(ΔT).

    Returns
    -------
    float
        The population variance of antipodal temperature differences.
    """
    nside = hp.npix2nside(len(m))
    m_filled = np.where(np.isfinite(m), m, 0.0)

    # --- Band-limit to ℓ = 2–5 ---
    lmax = 5
    alm = hp.map2alm(m_filled, lmax=lmax)
    # Zero out ℓ = 0 and ℓ = 1
    for ell in range(2):
        for em in range(ell + 1):
            alm[hp.Alm.getidx(lmax, ell, em)] = 0.0
    m_filtered = hp.alm2map(alm, nside, verbose=False)

    # --- Antipodal pixel mapping (vectorised) ---
    npix = hp.nside2npix(nside)
    pix = np.arange(npix)
    theta, phi = hp.pix2ang(nside, pix)
    antipix = hp.ang2pix(nside, np.pi - theta, phi + np.pi)

    # --- Valid pairs: both pixel and antipode unmasked in original map ---
    valid = np.isfinite(m) & np.isfinite(m[antipix])

    # --- Antipodal temperature difference ---
    delta_T = m_filtered[valid] - m_filtered[antipix[valid]]

    return float(np.var(delta_T))


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ℓ Antipodal Temperature Difference Variance test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results to compare.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Antipodal Temperature Difference Variance",
                "xlabel": "Var(ΔT), ℓ = 2–5 [μK²]",
                "ylabel": "Number of simulations",
                "bins": 40,
                "caption": "No simulation data available.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    direction = ""
    if planck_stat > mean_sim + 2 * std_sim:
        direction = (
            "Planck shows significantly high antipodal temperature difference "
            "variance, indicating strong large-scale antipodal asymmetry. "
            "This could reflect a hemispherical power asymmetry, an odd–even "
            "multipole imbalance, or other departure from isotropy at the "
            "largest angular scales."
        )
    elif planck_stat < mean_sim - 2 * std_sim:
        direction = (
            "Planck shows significantly low antipodal temperature difference "
            "variance, indicating unusually strong symmetry between antipodal "
            "regions at large angular scales — potentially reflecting "
            "suppressed odd-ℓ power or enhanced even-ℓ coherence."
        )
    else:
        direction = (
            "The antipodal temperature difference variance is consistent "
            "with isotropic ΛCDM expectations, showing no anomalous "
            "large-scale antipodal asymmetry."
        )

    interp = (
        f"Var(ΔT) = {planck_stat:.4f} μK². "
        f"Simulation mean = {mean_sim:.4f} ± {std_sim:.4f} μK². "
        f"Percentile = {percentile:.1f}%, two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of the variance of antipodal temperature differences "
        "ΔT(θ,φ) = T(θ,φ) − T(π−θ, φ+π) in the ℓ=2–5 filtered map from "
        "isotropic ΛCDM simulations. Spherical harmonics satisfy "
        "Y_ℓm(π−θ,φ+π) = (−1)^ℓ Y_ℓm(θ,φ), so even-ℓ modes are "
        "correlated and odd-ℓ modes are anti-correlated between antipodal "
        "points. The variance of ΔT therefore probes the balance between "
        "even and odd multipoles. Under ΛCDM this balance is statistically "
        "fixed; significantly elevated variance may signal hemispherical "
        "asymmetry or odd–even multipole anomalies. The red line marks "
        "the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Antipodal Temperature Difference Variance",
            "xlabel": "Var(ΔT), ℓ = 2–5 [μK²]",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
