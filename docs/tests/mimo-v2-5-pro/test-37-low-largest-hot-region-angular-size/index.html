Hypothesis

The angular diameter of the largest connected hot region in the low-ℓ CMB temperature map is consistent with the distribution predicted by the isotropic Gaussian ΛCDM model.

Method

Threshold the band-limited (ℓ=2–10) CMB temperature map at its mean to identify hot pixels. Using 4-connectivity on the nside=256 HEALPix grid, find connected components of hot pixels and compute the area of each. The statistic is the angular diameter (= 2√(area/π)) of the largest connected hot region.

Result Summary

Planck's largest hot region diameter = 102.50°, compared to a simulation mean of 100.12° ± 8.80° (z = +0.27σ). Two-tailed empirical p-value = 0.902, placing Planck at the 54.9th percentile. Anomaly strength is negligible.

The angular extent of the largest connected hot region in the low-ℓ CMB temperature field is fully consistent with isotropic Gaussian ΛCDM expectations. The observed value sits comfortably within the bulk of the simulated distribution, showing no tendency toward unusually large or fragmented hot structures at large angular scales.

Agent Justification

The test description and code do not specify a directional preference; the summarize function explicitly computes a two-sided empirical p-value by considering both upper and lower tails, and the interpretation examines deviations in both directions (unusually large or small). This approach is standard for exploratory anomaly tests where no prior directional hypothesis exists, allowing detection of any departure from the ΛCDM expectations.

Comparison To Literature

Studies of CMB spot morphology at large scales have examined ellipticity (Gurzadyan et al. 2003) and hot/cold spot spatial uniformity via orientation matrices (Khan & Saha 2022). Abramo et al. (2006) discussed extended foregrounds linked to low-ℓ anomalies. However, none of these specifically quantify the angular diameter of the largest connected hot component using graph-based connected-component analysis. The specific statistic tested here does not appear to have a direct precedent in the literature.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Largest Hot Region Angular Size: "
    "Compute the mean temperature of the low-ℓ (ℓ = 2–10) temperature map. "
    "Threshold the map at this mean to create a binary mask where pixels with "
    "temperature above the mean are labeled as hot. On the nside=256 HEALPix grid, "
    "identify connected components of hot pixels using 4-connectivity (neighboring "
    "pixels sharing an edge). For each connected component, calculate its area in "
    "square degrees by summing the solid angles of its constituent pixels. The final "
    "statistic is the angular diameter (in degrees) of the largest connected hot region, "
    "defined as 2 × √(area/π)."
)


def analyze_map(m):
    """Compute the angular diameter of the largest connected hot region in the low-ell map.

    Steps:
      1. Identify unmasked pixels; bail out if too few.
      2. Fill masked pixels with 0 for healpy compatibility.
      3. Band-limit the map to ell = 2-10 via spherical harmonic transform,
         zeroing out ell=0 (monopole) and ell=1 (dipole) coefficients.
      4. Compute mean temperature over unmasked pixels of the band-limited map.
      5. Threshold: hot pixels are those strictly above the mean AND unmasked.
      6. Identify 4-connected neighbors (edge-sharing) using HEALPix neighbor
         indices [0, 2, 4, 6] corresponding to S, E, N, W cardinal directions.
      7. Build sparse adjacency matrix for hot pixels and find connected components
         using scipy.sparse.csgraph.connected_components.
      8. Compute area of each component (n_pixels x pixel_area) and angular
         diameter = 2*sqrt(area/pi) of the largest.

    Returns
    -------
    float
        Angular diameter in degrees of the largest connected hot region.
    """
    from scipy import sparse

    nside = hp.get_nside(m)
    npix = hp.nside2npix(nside)
    mask = np.isfinite(m)
    n_valid = int(np.sum(mask))

    if n_valid < 100:
        return 0.0

    # Fill masked pixels with 0 for healpy transforms
    m_filled = np.where(mask, m, 0.0)

    # Band-limit to ell = 2-10
    lmax = 10
    alm = hp.map2alm(m_filled, lmax=lmax)
    # Zero out ell=0 (monopole) and ell=1 (dipole): indices 0, 1, 2
    alm[:3] = 0.0
    m_low = hp.alm2map(alm, nside, lmax=lmax, verbose=False)

    # Compute mean temperature over unmasked pixels
    mean_temp = np.mean(m_low[mask])

    # Binary hot mask: strictly above mean AND originally unmasked
    hot = (m_low > mean_temp) & mask

    # Get all 8 neighbors for every pixel (shape: 8 x npix)
    all_neighbors = hp.get_all_neighbours(nside, np.arange(npix))
    # 4 edge-sharing (cardinal) neighbors: S, E, N, W at indices 0, 2, 4, 6
    edge_neighbors = all_neighbors[[0, 2, 4, 6], :]  # shape: (4, npix)

    # Identify hot pixel indices
    hot_indices = np.where(hot)[0]
    n_hot = len(hot_indices)
    if n_hot == 0:
        return 0.0

    # Map from pixel index to hot-pixel ordinal index
    hot_map = -np.ones(npix, dtype=np.int64)
    hot_map[hot_indices] = np.arange(n_hot)

    # Get edge-neighbor pixel indices for hot pixels only
    hot_neighbors = edge_neighbors[:, hot_indices]  # shape: (4, n_hot)

    # Determine which neighbors are also hot
    neighbor_hot_idx = np.full(hot_neighbors.shape, -1, dtype=np.int64)
    valid_nb = hot_neighbors >= 0
    neighbor_hot_idx[valid_nb] = hot_map[hot_neighbors[valid_nb]]
    valid_edge = valid_nb & (neighbor_hot_idx >= 0)

    # Build edge list (directed edges; each undirected edge appears twice)
    src_dir, dst_hot = np.where(valid_edge)
    rows = dst_hot
    cols = neighbor_hot_idx[src_dir, dst_hot]

    # Build sparse adjacency matrix and find connected components
    data = np.ones(len(rows), dtype=np.int8)
    adj = sparse.csr_matrix((data, (rows, cols)), shape=(n_hot, n_hot))
    n_components, labels = sparse.csgraph.connected_components(adj, directed=False)

    # Count pixels per component
    component_sizes = np.bincount(labels, minlength=n_components)

    # Angular diameter of the largest component
    pixel_area_sq_deg = hp.nside2pixarea(nside, degrees=True)
    max_area = float(np.max(component_sizes)) * pixel_area_sq_deg
    angular_diameter = 2.0 * np.sqrt(max_area / np.pi)

    return float(angular_diameter)


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ell Largest Hot Region Angular Size test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available for comparison.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ell Largest Hot Region Angular Size",
                "xlabel": "Angular diameter (degrees)",
                "ylabel": "Number of simulations",
                "bins": 40,
                "caption": "No simulation data available.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    # Two-sided empirical p-value
    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims

    if std_sim > 0:
        deviation_sigma = (planck_stat - mean_sim) / std_sim
    else:
        deviation_sigma = 0.0

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    if planck_stat > mean_sim + 2 * std_sim:
        direction = (
            "Planck's largest connected hot region at low multipoles is unusually large "
            "compared to typical LCDM realizations. This could indicate an anomalous "
            "concentration of hot power in the large-scale temperature field, potentially "
            "related to large-scale CMB anomalies such as the cold spot's counterpart "
            "or a preferred direction in the temperature pattern."
        )
    elif planck_stat < mean_sim - 2 * std_sim:
        direction = (
            "Planck's largest connected hot region at low multipoles is unusually small "
            "compared to typical LCDM realizations, indicating that hot regions at large "
            "angular scales are more fragmented than expected."
        )
    else:
        direction = (
            "The angular size of the largest hot region at low multipoles is consistent "
            "with isotropic Gaussian LCDM expectations."
        )

    interp = (
        f"Largest hot region angular diameter = {planck_stat:.2f} deg. "
        f"Simulation mean = {mean_sim:.2f} deg +/- {std_sim:.2f} deg "
        f"({deviation_sigma:+.2f} sigma from mean). "
        f"Percentile = {percentile:.1f}%, two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of the angular diameter (degrees) of the largest connected hot "
        "region in the low-ell (ell = 2-10) band-limited temperature map, computed from "
        "isotropic LCDM simulations. Hot pixels are those strictly above the mean "
        "temperature; connected components use 4-connectivity (edge-sharing neighbors) "
        "on the HEALPix grid. Angular diameter = 2*sqrt(area/pi). "
        "The red line marks the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": deviation_sigma,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ell Largest Hot Region Angular Size",
            "xlabel": "Angular diameter (degrees)",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
