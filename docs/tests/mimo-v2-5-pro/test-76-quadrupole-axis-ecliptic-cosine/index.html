Hypothesis

The absolute value of the cosine of the angle between the quadrupole preferred axis and the ecliptic north pole is anomalously small, indicating the quadrupole axis lies close to the ecliptic plane.

Method

The preferred axis of the CMB quadrupole (ℓ=2) is extracted by diagonalizing the quadrupole moment tensor built from a₂ₘ coefficients. The eigenvector of the largest eigenvalue is expressed in ecliptic Cartesian coordinates, and the cosine of its angle with the ecliptic north pole (0,0,1) is reported as the anomaly statistic.

Result Summary

The Planck quadrupole axis yields cos(θ) = −0.673 (|cos(θ)| = 0.673), corresponding to ~48° from the ecliptic pole (~42° from the ecliptic plane). The z-score is −1.24 with one-tailed lower p-value = 0.154. The simulation-based p-value for the absolute cosine test is P(|cos| ≥ 0.673) = 0.389, meaning the Planck value sits comfortably within the simulated distribution and is not in the lower tail.

The hypothesis predicted an anomalously small |cos(θ)|, indicating alignment with the ecliptic plane. Instead, |cos(θ)| = 0.673 is entirely consistent with statistical isotropy — roughly 61% of isotropic realizations have |cos| ≤ 0.673. The quadrupole axis is moderately inclined relative to the ecliptic pole but shows no statistically significant preference for the ecliptic plane.

Agent Justification

The test description explicitly states that a small absolute cosine is anomalous because the CMB quadrupole is known to have a preferred direction near the ecliptic plane. This is a directional hypothesis: under statistical isotropy, the absolute cosine should be uniformly distributed, but the alternative hypothesis predicts it to be concentrated near zero (lower tail). Thus, the test is one-tailed, focusing on small values of the absolute cosine.

Comparison To Literature

The ecliptic alignment of the CMB quadrupole has been investigated extensively (e.g., Tegmark et al. 2003; Copi et al. 2004; Planck 2013 XVI; Planck 2020). Most prior studies focus on quadrupole–octupole alignment or multipole axis correlations rather than isolating this single cosine statistic, though related directional tests exist. The search results did not return a directly comparable published analysis using this exact statistic.

import numpy as np
import healpy as hp

test_description = (
    "For the low-ℓ temperature map (ℓ=2 only), compute the quadrupole moment tensor "
    "T_{ij} from the spherical harmonic coefficients a_{2m} by reconstructing the ℓ=2 map "
    "and numerically summing T_2(p) * n_i(p) * n_j(p) over all pixels. Diagonalize the "
    "3×3 tensor to find the eigenvector corresponding to the largest eigenvalue, representing "
    "the preferred axis of the quadrupole. Express this eigenvector as a unit vector in "
    "Cartesian coordinates aligned with the ecliptic system (z-axis = ecliptic north pole). "
    "The anomaly statistic is the cosine of the angle between this quadrupole axis and the "
    "ecliptic north pole (0,0,1), which equals simply the z-component of the eigenvector. "
    "This cosine ranges from -1 (anti-aligned) to +1 (aligned), with values near 0 indicating "
    "an axis perpendicular to the ecliptic pole. The CMB quadrupole is known to have a "
    "preferred direction near the ecliptic plane, so a small absolute cosine would be "
    "anomalous. The computation is efficient as it only requires low-ℓ harmonics and a 3×3 "
    "matrix diagonalization."
)


def analyze_map(m):
    """Return cos(angle) between the quadrupole preferred axis and ecliptic north pole."""
    nside = hp.npix2nside(len(m))
    npix = hp.nside2npix(nside)
    lmax = 2

    # --- 1. Replace NaNs with 0 for healpy compatibility ---
    mask_valid = np.isfinite(m)
    m_clean = np.where(mask_valid, m, 0.0)

    # --- 2. Compute a_lm and keep only ℓ=2 ---
    alm = hp.map2alm(m_clean, lmax=lmax, iter=10)
    # Zero out ℓ=0 and ℓ=1
    for ell in range(2):
        for m_idx in range(ell + 1):
            alm[hp.Alm.getidx(lmax, ell, m_idx)] = 0.0

    # --- 3. Reconstruct the ℓ=2 only map ---
    T2 = hp.alm2map(alm, nside, lmax=lmax, verbose=False)

    # --- 4. Compute pixel unit vectors ---
    vecs = np.array(hp.pix2vec(nside, np.arange(npix)))  # shape (3, npix)
    # vecs[0] = x, vecs[1] = y, vecs[2] = z (ecliptic coords)

    # --- 5. Build the quadrupole moment tensor Q_{ij} = Σ_p T2(p) * n_i(p) * n_j(p) ---
    # This is a symmetric 3x3 tensor
    Q = np.zeros((3, 3))
    for i in range(3):
        for j in range(i, 3):
            Q[i, j] = np.sum(T2 * vecs[i] * vecs[j])
            Q[j, i] = Q[i, j]

    # --- 6. Diagonalize to find preferred axis ---
    eigenvalues, eigenvectors = np.linalg.eigh(Q)
    # eigh returns eigenvalues in ascending order
    # The preferred axis corresponds to the largest eigenvalue (index 2)
    preferred_axis = eigenvectors[:, 2]  # unit eigenvector

    # --- 7. Cosine of angle with ecliptic north pole (0, 0, 1) ---
    cos_angle = preferred_axis[2]  # z-component of the eigenvector

    return float(cos_angle)


def summarize_results(planck_stat, sim_results):
    """Interpret the Quadrupole Axis Ecliptic Cosine statistic."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)

    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    p_upper = float(np.mean(sims >= planck_stat))
    p_lower = float(np.mean(sims <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else np.nan

    abs_planck = abs(planck_stat)
    p_abs_upper = float(np.mean(np.abs(sims) >= abs_planck))

    return {
        "planck_stat": planck_stat,
        "planck_stat_abs": abs_planck,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "z_score": z_score,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "p_value_abs_upper": p_abs_upper,
        "n_sims": n,
        "interpretation": (
            "The cosine of the angle between the quadrupole preferred axis and the ecliptic "
            "north pole. A value near ±1 means the quadrupole axis is aligned/anti-aligned with "
            "the ecliptic pole, while a value near 0 means it lies in or near the ecliptic "
            "plane. The CMB quadrupole is known to have its axis close to the ecliptic plane, "
            "so a small |cos(angle)| (p_value_abs_upper small) would indicate an anomalous "
            "alignment with the ecliptic. Under statistical isotropy, the cosine should be "
            "uniformly distributed in [-1, 1]."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Quadrupole Axis Ecliptic Cosine",
            "xlabel": "cos(angle) with ecliptic north pole",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck cos(θ) = {planck_stat:.4f} (|cos(θ)| = {abs_planck:.4f}). "
                f"Sim mean ± σ = {mean_s:.4f} ± {std_s:.4f}. "
                f"z = {z_score:.2f}, p(two-sided) = {p_two:.3f}. "
                f"P(|cos| ≥ |cos_planck|) = {p_abs_upper:.3f}."
            ),
        },
