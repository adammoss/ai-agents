Hypothesis

The skewness of the gradient magnitude of the low-ℓ CMB temperature map is consistent with the distribution of skewness values from ΛCDM simulations.

Method

The spatial gradient magnitude field of the low-ℓ (ℓ ≤ 32) CMB temperature map is computed using spherical harmonic derivatives. The skewness (third standardized moment) of the gradient magnitude values across all unmasked pixels is then calculated, capturing asymmetries in the sharpness and distribution of temperature gradients and potentially revealing non-Gaussian features in the large-scale temperature topology.

Result Summary

The observed skewness of |∇T| is 0.5885, compared to a simulation mean of 0.6453 ± 0.0634. This places Planck at the 19.0th percentile, corresponding to a deviation of −0.88σ. The two-sided p-value is 0.38, well above the 0.05 threshold, indicating no significant tension with ΛCDM predictions. The anomaly strength is classified as negligible.

The gradient magnitude skewness is fully consistent with ΛCDM Gaussian expectations. The distribution of temperature gradients shows no significant asymmetry beyond what is expected from a statistically isotropic random field. There is no indication of excess localized sharp features, anomalous structures, or hemispherical asymmetries in the large-scale CMB gradient topology. The mild 19th-percentile positioning is well within normal statistical fluctuation.

Agent Justification

The test is designed to explore non-Gaussian features in the low-ℓ temperature field by measuring the skewness of the gradient magnitude. There is no a priori reason to expect the skewness to be specifically higher or lower than the ΛCDM expectation; the anomaly could manifest as either too high or too low skewness. The code computes a two-sided p-value, confirming that deviations in both directions are of interest.

Comparison To Literature

No prior studies specifically measuring the skewness of the CMB gradient magnitude field at low multipoles were identified in the search. Related low-ℓ anomaly tests exist—such as multipole vector correlations (Copi et al. 2004) and spot ellipticity measurements (Gurzadyan et al. 2003)—but these use fundamentally different statistics. The gradient magnitude skewness probes a distinct aspect of the temperature field topology.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Gradient Magnitude Skewness: "
    "Compute the spatial gradient magnitude field of the low-ℓ (ℓ ≤ 32) CMB "
    "temperature map using spherical harmonic derivatives. Calculate the "
    "skewness (third standardized moment) of the gradient magnitude values "
    "across all unmasked pixels. This single scalar captures asymmetries in "
    "the sharpness and distribution of temperature gradients, potentially "
    "revealing non-Gaussian features in the topology of the low-ℓ "
    "temperature field."
)


def analyze_map(m):
    """Compute skewness of the low-ℓ gradient magnitude field."""
    mask = np.isfinite(m)
    n_valid = int(np.sum(mask))
    if n_valid < 100:
        return 0.0

    lmax = 32
    clean = np.where(mask, m, 0.0)

    # Spherical harmonic transform
    alm = hp.map2alm(clean, lmax=lmax, iter=0)

    # Get gradient components via spin-1 reconstruction:
    #   dtheta ≈ -∂T/∂θ  (sign irrelevant since we square it)
    #   dphi   ≈ -(1/sinθ) ∂T/∂φ  (covariant φ-component; sign irrelevant)
    nside = hp.npix2nside(len(m))
    _tmap, dtheta, dphi = hp.alm2map_der1(alm, nside)

    # Gradient magnitude: |∇T|² = (∂T/∂θ)² + (1/sinθ ∂T/∂φ)²
    grad_mag = np.sqrt(dtheta ** 2 + dphi ** 2)

    # Apply mask
    grad_mag = grad_mag[mask]
    grad_mag = grad_mag[np.isfinite(grad_mag)]

    if len(grad_mag) < 50:
        return 0.0

    std = np.std(grad_mag)
    if std < 1e-30:
        return 0.0

    skew = float(np.mean(((grad_mag - np.mean(grad_mag)) / std) ** 3))
    if not np.isfinite(skew):
        return 0.0
    return skew


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ℓ Gradient Magnitude Skewness test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Low-ℓ Gradient Magnitude Skewness",
                "xlabel": "Skewness of |∇T|",
                "ylabel": "Count",
                "bins": 40,
                "caption": "No simulation data.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims
    deviation_sigma = (planck_stat - mean_sim) / std_sim if std_sim > 0 else 0.0

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    if planck_stat > mean_sim + 2 * std_sim:
        direction = (
            "Planck shows significantly more positively skewed gradient "
            "magnitudes than ΛCDM simulations, indicating an excess of "
            "localized sharp features or outliers in the low-ℓ temperature "
            "gradient field—possibly related to anomalous structures such "
            "as the CMB cold spot or non-standard topology."
        )
    elif planck_stat < mean_sim - 2 * std_sim:
        direction = (
            "Planck shows significantly less positively skewed gradient "
            "magnitudes than ΛCDM simulations, suggesting an unusually "
            "smooth and symmetric distribution of temperature gradients "
            "at large angular scales."
        )
    else:
        direction = (
            "The gradient magnitude skewness is consistent with "
            "ΛCDM Gaussian expectations. The distribution of temperature "
            "gradients shows no significant asymmetry beyond what is "
            "expected from a statistically isotropic random field."
        )

    interp = (
        f"Skewness of |∇T| = {planck_stat:.6f}. "
        f"Sim mean = {mean_sim:.6f} ± {std_sim:.6f} "
        f"({deviation_sigma:+.2f}σ). Percentile = {percentile:.1f}%, "
        f"two-sided p = {p_value:.4f}. "
        f"Strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of the skewness of the gradient magnitude field |∇T| "
        "computed from the low-ℓ (ℓ ≤ 32) CMB temperature map via "
        "spherical harmonic derivatives. The gradient magnitude is always "
        "non-negative, so its distribution is naturally right-skewed. "
        "Deviations from the ΛCDM expectation may indicate non-Gaussian "
        "features, localized anomalies, or hemispherical asymmetries in "
        "the large-scale temperature topology. Red line marks the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "deviation_sigma": deviation_sigma,
        "anomaly_strength": strength,
        "interpretation": interp,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Gradient Magnitude Skewness",
            "xlabel": "Skewness of |∇T|",
            "ylabel": "Number of simulations",
            "bins": 40,
            "caption": caption,
        },
    }
