Hypothesis

The dipole amplitude of the low-ℓ parity difference map D = T_odd − T_even is inconsistent with predictions from ΛCDM simulations, indicating an unexpected directional preference in the parity asymmetry.

Method

Constructs a parity difference map D = T_odd − T_even from CMB temperature multipoles ℓ=2–20 (odd ℓ: 3,5,... vs even ℓ: 2,4,...), then fits a dipole-plus-monopole to unmasked pixels. The statistic is the dipole amplitude √(A_x²+A_y²+A_z²), quantifying whether the low-ℓ parity asymmetry has a directional preference.

Result Summary

Planck dipole amplitude = 1.94 µK, compared to ΛCDM simulation mean of 2.16 ± 1.11 µK (median = 2.04 µK). Two-sided p = 0.936 (z = −0.19σ). The observed value sits near the center of the simulation distribution with no excess in either tail.

The dipole amplitude of the odd−even parity difference map is fully consistent with ΛCDM expectations. There is no evidence that the low-ℓ parity power imbalance (odd > even) is concentrated along any preferred spatial direction. The parity asymmetry, if present at the power-spectrum level, does not manifest as a directional signal in pixel space.

Agent Justification

The test statistic is a non-negative scalar (dipole amplitude), and the anomaly could manifest as either an unusually large or an unusually small value. While a large amplitude directly indicates a directional preference, an unusually small amplitude could also be anomalous if it suggests excessive isotropy or an improbable cancellation. The provided code explicitly calculates a two-sided p-value (taking the minimum of the upper and lower tail probabilities) and the description notes that "a large amplitude is anomalous," but the analysis framework treats both extremes as noteworthy. Therefore, as a general exploratory test for any deviation, it is appropriately two-tailed.

Comparison To Literature

Previous work by Zhao (2017) and Creswell & Naselsky (2021) examined directional dependence of parity asymmetry using power/alignment tensors and found ~2–3σ alignments with kinematic dipole axes. Aluri et al. (2017) used alignment statistics for even/odd multipoles up to ℓ=61. The present pixel-space dipole-fit approach differs from these tensor-based methods, though it probes a related question. No directly analogous statistic was found in the literature.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Parity Difference Dipole Amplitude. Extracts a_{ℓm} up to ℓ=20 "
    "from the masked temperature map and constructs two maps: one using only "
    "odd-ℓ coefficients (ℓ=3,5,7,...) and another using only even-ℓ coefficients "
    "(ℓ=2,4,6,...). Computes the pixel-wise difference D = T_odd − T_even, then "
    "fits a dipole (ℓ=1 pattern) plus monopole to the unmasked pixels of D via "
    "least squares. The statistic is the dipole amplitude √(A_x² + A_y² + A_z²), "
    "quantifying whether the low-ℓ parity asymmetry (odd−even power imbalance) "
    "has a directional preference. Under ΛCDM parity symmetry, D should be "
    "isotropic and not favour a dipole direction, so a large amplitude is anomalous."
)


def analyze_map(m):
    m = np.asarray(m, dtype=np.float64)
    mask = np.isfinite(m)
    m_clean = np.where(mask, m, 0.0)

    lmax = 20

    # Extract alm up to lmax
    alm = hp.map2alm(m_clean, lmax=lmax)

    # Build odd-l alm (l = 3, 5, 7, ...)
    alm_odd = np.zeros_like(alm)
    for ell in range(3, lmax + 1, 2):
        for mm in range(ell + 1):
            idx = hp.Alm.getidx(lmax, ell, mm)
            alm_odd[idx] = alm[idx]

    # Build even-l alm (l = 2, 4, 6, ...)
    alm_even = np.zeros_like(alm)
    for ell in range(2, lmax + 1, 2):
        for mm in range(ell + 1):
            idx = hp.Alm.getidx(lmax, ell, mm)
            alm_even[idx] = alm[idx]

    # Reconstruct maps from partial alm
    nside = hp.npix2nside(len(m))
    T_odd = hp.alm2map(alm_odd, nside, lmax=lmax, verbose=False)
    T_even = hp.alm2map(alm_even, nside, lmax=lmax, verbose=False)

    # Difference map
    D = T_odd - T_even

    # Fit monopole + dipole to unmasked pixels of D
    # Use pixel coordinates (x, y, z) for the dipole pattern
    valid = mask  # pixels with real data
    if valid.sum() < 4:
        return 0.0

    theta, phi = hp.pix2ang(nside, np.arange(hp.nside2npix(nside)))
    x = np.sin(theta) * np.cos(phi)
    y = np.sin(theta) * np.sin(phi)
    z = np.cos(theta)

    # Design matrix: [1, x, y, z] for valid pixels
    b = D[valid]
    A = np.column_stack([
        np.ones(valid.sum()),
        x[valid],
        y[valid],
        z[valid]
    ])

    # Least-squares fit
    coeffs, _, _, _ = np.linalg.lstsq(A, b, rcond=None)
    # coeffs = [A_mono, A_x, A_y, A_z]
    A_x, A_y, A_z = coeffs[1], coeffs[2], coeffs[3]

    dipole_amplitude = np.sqrt(A_x**2 + A_y**2 + A_z**2)
    return float(dipole_amplitude)


def summarize_results(planck_stat, sim_results):
    sim_arr = np.array(sim_results, dtype=np.float64)
    finite_mask = np.isfinite(sim_arr)
    n_finite = int(finite_mask.sum())

    if n_finite > 0:
        valid = sim_arr[finite_mask]
        sim_median = float(np.median(valid))
        p_upper = float(np.sum(valid >= planck_stat) / n_finite)
        p_lower = float(np.sum(valid <= planck_stat) / n_finite)
        p_value = min(p_upper, p_lower)
        rank_upper = int(np.sum(valid >= planck_stat))
        rank_lower = int(np.sum(valid <= planck_stat))
    else:
        p_value = float("nan")
        rank_upper = rank_lower = 0
        sim_median = float("nan")

    sim_mean = float(np.nanmean(sim_arr))
    sim_std = float(np.nanstd(sim_arr))

    if n_finite > 0 and not np.isnan(p_value):
        if p_upper <= p_lower:
            direction, rank = "high", rank_upper
        else:
            direction, rank = "low", rank_lower
    else:
        direction, rank = "unknown", 0

    parts = [
        f"Planck dipole amplitude in parity difference map D = T_odd − T_even "
        f"at low ℓ is {planck_stat:.4f} µK.",
        f"Two-sided p = {p_value:.3f} ({rank}/{n_finite} simulations "
        f"at least as extreme in the {direction} direction).",
        f"Simulation mean = {sim_mean:.4f} ± {sim_std:.4f} µK, "
        f"median = {sim_median:.4f} µK.",
    ]

    if not np.isnan(p_value) and p_value < 0.05:
        parts.append(
            "Anomalous: the dipole amplitude of the odd−even parity difference "
            "map is unusually large, suggesting the low-ℓ parity asymmetry "
            "has a preferred spatial direction rather than being isotropic."
        )
    else:
        parts.append(
            "The dipole amplitude of the parity difference map is consistent "
            "with ΛCDM simulation expectations — no evidence for a directional "
            "preference in the low-ℓ parity asymmetry."
        )

    result = {
        "test_description": test_description,
        "planck_stat": float(planck_stat),
        "p_value": float(p_value),
        "rank": rank,
        "n_simulations": len(sim_results),
        "n_valid": n_finite,
        "sim_mean": sim_mean,
        "sim_std": sim_std,
        "sim_median": sim_median,
        "exceeds_95th_percentile": bool(p_value < 0.05) if not np.isnan(p_value) else False,
        "anomaly_direction": direction,
        "interpretation": " ".join(parts),
        "ell_range": "odd: 3,5,... vs even: 2,4,... (up to lmax=20)",
    }

    result["plot_spec"] = {
        "kind": "histogram",
        "title": "Low-ℓ Parity Difference Dipole Amplitude",
        "xlabel": "Dipole amplitude in D = T_odd − T_even (µK)",
        "ylabel": "Number of simulations",
        "bins": 50,
        "caption": (
            f"Histogram of dipole amplitudes from {n_finite} ΛCDM simulations. "
            f"Planck value = {planck_stat:.4f} µK "
            f"(two-sided p = {p_value:.3f}, rank {rank}/{n_finite}, "
            f"{direction} tail). Simulation mean = {sim_mean:.4f} ± {sim_std:.4f} µK. "
            f"A large dipole indicates the odd−even parity imbalance has a "
            f"preferred spatial direction."
        ),
    }

    return result
