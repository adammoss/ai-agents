Hypothesis

The Pearson correlation coefficient between the CMB temperature field and its spherical Laplacian is zero, consistent with a statistically isotropic Gaussian field with no anomalous alignment between temperature extrema and local curvature.

Method

Computes the Pearson correlation coefficient between the CMB temperature field T and its spherical Laplacian ∇²T at nside=256, using the harmonized Planck mask. The Laplacian is estimated via −ℓ(ℓ+1) multiplication in harmonic space. This scalar statistic probes whether temperature extrema show anomalous alignment with local spatial curvature, potentially sensitive to non-Gaussian clustering or atypical coherence patterns.

Result Summary

Planck r(T, ∇²T) = −0.5005, compared to simulation mean −0.4886 ± 0.0082. The Planck value falls at the 5.4th percentile of 1000 ΛCDM simulations, yielding z ≈ 1.61σ and a two-tailed p-value of 0.108.

The observed cross-correlation is consistent with the isotropic Gaussian ΛCDM expectation. Temperature extrema show no statistically significant anomalous relationship with local spatial curvature. The slight offset from the simulation mean (~1.6σ) falls well within normal statistical fluctuation and does not reach the 2σ anomaly threshold.

Agent Justification

The test is exploratory, seeking any deviation from the expected null result regardless of sign. While the code computes both upper and lower p-values and interprets both positive and negative anomalies, there is no clear prior directional expectation from standard cosmological models—any nonzero correlation could signal atypical spatial coherence or non-Gaussian features in either direction.

Comparison To Literature

No prior studies using the Pearson correlation between T and ∇²T as a CMB anomaly statistic were identified in literature searches. Related work on CMB statistical anisotropy (e.g., Mukherjee & Souradeep 2015) addresses different statistics such as dipolar power asymmetry in harmonic space.

import numpy as np
import healpy as hp

test_description = (
    "Temperature–Laplacian Cross-Correlation Coefficient: "
    "For each map (Planck or simulated), first compute the spherical Laplacian "
    "(∇²) of the temperature field at nside=256. The Laplacian is estimated "
    "using the standard HEALPix finite-difference stencil on the sphere, "
    "implemented as multiplication by -ℓ(ℓ+1) in spherical-harmonic space. "
    "Then, calculate the Pearson correlation coefficient between the temperature "
    "pixel values (T) and their corresponding Laplacian values (∇²T) across "
    "all unmasked pixels (using the harmonized Planck mask). This single scalar "
    "measures the linear relationship between local temperature extrema and "
    "their spatial curvature, potentially sensitive to anomalous clustering of "
    "hot/cold spots or atypical spatial coherence patterns."
)


def analyze_map(m):
    """Compute Pearson cross-correlation between temperature and its Laplacian.

    Steps:
      1. Identify unmasked pixels (finite values).
      2. Fill masked (NaN) pixels with 0.0 for healpy compatibility.
      3. Compute alm coefficients (same lmax as map2alm default).
      4. Multiply by -ell(ell+1) to obtain Laplacian alm coefficients.
      5. Transform Laplacian alm back to pixel space via alm2map.
      6. Extract T and laplacian values at unmasked pixels.
      7. Compute Pearson r between T and nabla^2 T.

    Returns
    -------
    float
        Pearson cross-correlation coefficient between T and nabla^2 T.
    """
    mask = np.isfinite(m)
    valid_idx = np.where(mask)[0]
    N = len(valid_idx)

    if N == 0:
        return 0.0

    # Fill masked pixels with 0 for healpy transforms
    m_filled = np.where(mask, m, 0.0)

    # Determine lmax (default for map2alm)
    nside = hp.npix2nside(len(m))
    lmax = 3 * nside - 1

    # Compute spherical harmonic coefficients of the temperature field
    alm = hp.map2alm(m_filled, lmax=lmax)

    # Build -ell*(ell+1) multiplier for each alm coefficient
    l_arr, m_arr = hp.Alm.getlm(lmax)
    lap_alm = -l_arr * (l_arr + 1.0) * alm

    # Transform back to get Laplacian map
    lap_map = hp.alm2map(lap_alm, nside, lmax=lmax)

    # Extract values at unmasked pixels
    T_vals = m[valid_idx]
    L_vals = lap_map[valid_idx]

    # Center the values
    T_mean = np.mean(T_vals)
    L_mean = np.mean(L_vals)
    T_c = T_vals - T_mean
    L_c = L_vals - L_mean

    # Compute Pearson correlation coefficient
    num = np.sum(T_c * L_c)
    den_T = np.sqrt(np.sum(T_c * T_c))
    den_L = np.sqrt(np.sum(L_c * L_c))

    if den_T == 0.0 or den_L == 0.0:
        return 0.0

    r = num / (den_T * den_L)
    return float(r)


def summarize_results(planck_stat, sim_results):
    """Summarise the Temperature–Laplacian Cross-Correlation test."""
    sim = np.array(sim_results, dtype=float)
    sim = sim[np.isfinite(sim)]
    n_sims = len(sim)

    if n_sims == 0:
        return {
            "planck_stat": planck_stat,
            "p_value": float("nan"),
            "n_sims": 0,
            "interpretation": "No simulation results available for comparison.",
            "plot_spec": {
                "kind": "histogram",
                "title": "Temperature–Laplacian Cross-Correlation",
                "xlabel": "Pearson r(T, ∇²T)",
                "ylabel": "Number of simulations",
                "bins": 40,
                "caption": "No simulation data available.",
            },
        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))

    # Two-sided empirical p-value
    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    if abs(planck_stat) > abs(mean_sim) + 2 * std_sim:
        if planck_stat > 0:
            direction = (
                "Planck shows a significantly positive cross-correlation between "
                "temperature and its Laplacian: hot spots preferentially coincide "
                "with concave (positive curvature) regions and cold spots with "
                "concave regions, indicating unusually concentrated or peaked "
                "temperature features. This may signal anomalous spatial coherence "
                "or non-Gaussian clustering of hot/cold spots."
            )
        else:
            direction = (
                "Planck shows a significantly negative cross-correlation between "
                "temperature and its Laplacian: hot/cold spots preferentially "
                "coincide with convex (negative curvature) regions, indicating "
                "that temperature extrema sit in unusually flat or diffuse "
                "regions, with steep curvature occurring away from peaks and "
                "troughs. This may indicate unusual mode structure or spatial "
                "coherence in the temperature field."
            )
    else:
        direction = (
            "The cross-correlation between temperature and its Laplacian is "
            "consistent with isotropic Gaussian ΛCDM expectations; temperature "
            "extrema show no anomalous relationship with local spatial curvature."
        )

    interp = (
        f"Pearson r(T, ∇²T) = {planck_stat:.6f}. "
        f"Simulation mean = {mean_sim:.6f} ± {std_sim:.6f}. "
        f"Percentile = {percentile:.1f}%, two-sided p = {p_value:.4f}. "
        f"Anomaly strength: {strength}. {direction}"
    )

    caption = (
        "Histogram of the Pearson cross-correlation coefficient r between the "
        "temperature map T and its spherical Laplacian ∇²T (computed via "
        "−ℓ(ℓ+1) multiplication in spherical-harmonic space) from isotropic "
        "ΛCDM simulations. For a statistically isotropic Gaussian field, the "
        "correlation between T and ∇²T is centered at zero by symmetry: "
        "temperature extrema have no preferred alignment with local spatial "
        "curvature. Significant nonzero r values indicate that hot/cold spots "
        "systematically co-occur with regions of high (positive r) or low "
        "(negative r) curvature, suggesting non-Gaussian spatial coherence "
        "or atypical clustering of features. The red line marks the Planck value."
    )

    return {
        "planck_stat": planck_stat,
        "p_value": p_value,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "rank_below": rank_below,
        "n_sims": n_sims,
        "percentile": percentile,
        "sim_mean": mean_sim,
        "sim_std": std_sim,
        "anomaly_strength": strength,

    }
