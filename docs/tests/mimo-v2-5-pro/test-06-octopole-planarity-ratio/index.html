Hypothesis

The octopole planarity ratio (λ₂/λ₃) observed in the Planck CMB map is anomalously high, indicating the octopole is more planar than predicted by ΛCDM simulations.

Method

Computes the 3×3 power tensor of the ℓ=3 octopole field on the sphere, whose eigenvalues λ₁ ≥ λ₂ ≥ λ₃ quantify how octopole energy is distributed across three orthogonal spatial directions. The statistic λ₂/λ₃ measures "planarity": values near 1 indicate isotropic octopole energy, while significantly larger values indicate power concentrated in a single plane. Compared against 1000 lensed-ΛCDM simulations (one-tailed upper test).

Result Summary

Planck yields λ₂/λ₃ = 2.785 versus a simulation mean of 2.068 ± 0.649 (median 1.943). The one-sided p-value is 0.162, corresponding to a tension of approximately 1.0σ. The Planck value lies comfortably within the simulated distribution and does not exceed the 95th percentile.

The observed octopole planarity ratio is elevated relative to the simulation mean but remains well within the expected ΛCDM scatter. Roughly 16% of simulated universes produce an equal or greater planarity ratio, indicating no statistically significant departure from the standard model. The octopole's internal structure is not anomalously planar.

Agent Justification

The test is designed to detect if the octopole is "planar" with power concentrated in a single plane, which corresponds to a high λ₂/λ₃ ratio. The analysis code computes a one-sided p-value by ranking the Planck statistic against simulations only for values greater than or equal to it, and the interpretation explicitly flags anomalously high ratios. This directional preference makes it a one-tailed upper test.

Comparison To Literature

No prior studies using this specific λ₂/λ₃ eigenvalue-ratio statistic for the octopole alone were identified in the literature. Related work includes analyses of multipole planarity via multipole vectors (Copi et al. 2010), the quadrupole–octopole alignment (de Oliveira-Costa et al. 2004; Planck 2018 VII), and general low-multipole geometry studies, but the internal planarity of the ℓ=3 multipole as a standalone scalar test does not appear to have been previously published.

import numpy as np
import healpy as hp

test_description = ("Octopole Planarity Ratio. Extracts the ℓ=3 spherical harmonic "
    "coefficients a₃ₘ from the temperature map, reconstructs the octopole field, "
    "and computes a 3×3 power tensor M_ij = Σ_pixels n_i n_j T²(θ,φ) whose "
    "eigenvalues λ₁≥λ₂≥λ₃ measure how the octopole energy is distributed across "
    "three orthogonal spatial directions. The statistic λ₂/λ₃ is returned: a value "
    "near 1 indicates roughly isotropic octopole energy, while a value significantly "
    "greater than 1 indicates a 'planar' octopole whose power is concentrated in a "
    "single plane with the third axis suppressed.")


def analyze_map(m):
    m = np.asarray(m, dtype=np.float64)
    nside = hp.npix2nside(len(m))
    npix = hp.nside2npix(nside)

    # Replace NaN (masked) pixels with 0.0 for healpy transforms
    m_clean = np.where(np.isfinite(m), m, 0.0)

    # Extract spherical harmonic coefficients up to ℓ=3
    lmax = 3
    alm = hp.map2alm(m_clean, lmax=lmax)

    # Zero out all coefficients except ℓ=3 (indices for ℓ=3, m=-3..3)
    # In healpy's triangular indexing, a₃ₘ starts at index 6 (ℓ(ℓ+1)/2 + m)
    alm_new = np.zeros_like(alm)
    for m_idx in range(-3, 4):
        idx = hp.Alm.getidx(lmax, 3, abs(m_idx))
        alm_new[idx] = alm[idx]

    # Reconstruct the octopole-only map
    octo_map = hp.alm2map(alm_new, nside, lmax=3)

    # Get Cartesian pixel coordinates (unit vectors on the sphere)
    theta, phi = hp.pix2ang(nside, np.arange(npix))
    nx = np.sin(theta) * np.cos(phi)
    ny = np.sin(theta) * np.sin(phi)
    nz = np.cos(theta)

    # Compute the 3×3 power tensor: M_ij = Σ_pixels n_i * n_j * T²(θ,φ)
    # This captures the directional distribution of octopole energy
    T2 = octo_map ** 2
    coords = np.array([nx, ny, nz])  # shape (3, npix)

    M = np.zeros((3, 3))
    for i in range(3):
        for j in range(i, 3):
            M[i, j] = np.sum(coords[i] * coords[j] * T2)
            M[j, i] = M[i, j]

    # Normalize so eigenvalues represent relative power fractions
    trace = np.trace(M)
    if trace > 0:
        M /= trace

    # Diagonalize and sort eigenvalues descending
    eigenvalues = np.linalg.eigvalsh(M)
    eigenvalues = np.sort(eigenvalues)[::-1]  # λ₁ ≥ λ₂ ≥ λ₃

    lam1, lam2, lam3 = eigenvalues

    # Return the planarity ratio λ₂/λ₃
    # Guard against division by zero (would indicate a perfectly linear octopole)
    if lam3 > 0:
        ratio = float(lam2 / lam3)
    else:
        ratio = float('inf')

    return float(ratio)


def summarize_results(planck_stat, sim_results):
    sim_arr = np.array(sim_results, dtype=np.float64)
    finite_mask = np.isfinite(sim_arr)
    n_finite = int(finite_mask.sum())

    if n_finite > 0:
        valid = sim_arr[finite_mask]
        # Count simulations with ratio >= Planck (one-sided: is Planck more planar?)
        rank = int(np.sum(valid >= planck_stat))
        p_value = rank / n_finite
    else:
        rank = 0
        p_value = float('nan')

    sim_median = float(np.nanmedian(sim_arr))
    sim_mean = float(np.nanmean(sim_arr))
    sim_std = float(np.nanstd(sim_arr))

    interpretation = (
        "Anomalous: octopole is unusually planar compared to ΛCDM simulations"
        if p_value < 0.05
        else "Octopole planarity ratio consistent with ΛCDM expectations"
    )

    result = {
        "test_description": test_description,
        "planck_stat": float(planck_stat),
        "p_value": float(p_value),
        "rank": rank,
        "n_simulations": len(sim_results),
        "n_valid": n_finite,
        "sim_mean": sim_mean,
        "sim_std": sim_std,
        "sim_median": sim_median,
        "exceeds_95th_percentile": bool(p_value < 0.05),
        "interpretation": interpretation,
        "expected_random_ratio": "≈1 for isotropic octopole; >1 indicates planar concentration",
    }

    result["plot_spec"] = {
        "kind": "histogram",
        "title": "Octopole Planarity Ratio (λ₂/λ₃)",
        "xlabel": "λ₂ / λ₃  (octopole planarity ratio)",
        "ylabel": "Number of simulations",
        "bins": 50,
        "caption": (
            f"Histogram of octopole planarity ratio λ₂/λ₃ from {n_finite} "
            f"lensed-ΛCDM simulations. Planck value = {planck_stat:.4f} "
            f"(one-sided p = {p_value:.3f}, rank {rank}/{n_finite}). "
            f"A ratio of 1 indicates isotropic octopole energy; values "
            f"significantly exceeding 1 indicate a planar octopole whose "
            f"power is concentrated in a single plane."
        ),
    }

    return result
