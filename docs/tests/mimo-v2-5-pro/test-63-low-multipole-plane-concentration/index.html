Hypothesis

The observed alignment of low-multipole planes (ℓ=2-5) in the CMB is unexpectedly strong, resulting in a significantly higher concentration parameter κ compared to standard ΛCDM simulations.

Method

For each multipole ℓ=2,3,4,5, compute the quadrupolar tensor of the temperature field to find principal axes. Extract the normal vector to the plane of the two largest axes per ℓ. Fit a von Mises-Fisher distribution to the four normal vectors; the statistic is the concentration parameter κ (higher κ implies a common preferred plane).

Result Summary

Observed Planck κ = 3.980, compared to simulation mean ± σ = 5.323 ± 3.420. The z-score is −0.39, indicating the observed value lies slightly *below* the simulation mean. The one-tailed upper p-value is 0.61, meaning 61% of ΛCDM simulations produce κ values ≥ the Planck value.

The low-multipole planes in Planck data show no unusual degree of mutual alignment. The observed concentration is entirely consistent with — and actually marginally less than — what ΛCDM simulations predict. There is no evidence for a preferred plane in the low-ℓ multipoles using this statistic.

Agent Justification

The anomaly test specifically seeks evidence for an *excess* alignment of the planes defined by low multipoles. The code defines the p-value as the fraction of simulations with κ *greater than or equal to* the observed Planck value (`p_value = float(np.mean(sims_finite >= planck_stat))`). This indicates the primary interest is in the upper tail of the simulation distribution, where the observed κ is abnormally large. The scientific context of "preferred plane" anomalies also suggests a directional hypothesis: if there is an anomaly, the planes should be *more* aligned than expected by chance, not less. Therefore, a one-tailed upper test is appropriate.

Comparison To Literature

Low-ℓ alignment anomalies have been extensively studied, e.g., Abramo et al. (2006) who tested multipole vector correlations and normal-plane alignments for ℓ=2–5 using WMAP data, finding quadrupole–octopole alignment at the ~10% level. The present test uses a similar conceptual framework but applies a von Mises-Fisher concentration parameter rather than pairwise angular statistics, making it methodologically distinct.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Multipole Plane Concentration: "
    "For each multipole ℓ=2,3,4,5, compute the quadrupolar tensor of T_ℓ² "
    "on the sphere to find principal axes. Extract the normal to the plane "
    "of the two largest axes for each ℓ. Fit a von Mises-Fisher distribution "
    "to the set of normal vectors; the statistic is the concentration "
    "parameter κ (higher κ means the low-ℓ multipole planes share a common "
    "preferred plane)."
)


def analyze_map(m):
    """Return von Mises-Fisher κ for the alignment of low-ℓ multipole planes."""

    masked = ~np.isfinite(m)
    m_clean = m.copy()
    m_clean[masked] = 0.0

    nside = hp.npix2nside(len(m))
    lmax = 5

    alm = hp.map2alm(m_clean, lmax=lmax)

    # Pixel directions in Cartesian
    npix = hp.nside2npix(nside)
    theta, phi = hp.pix2ang(nside, np.arange(npix))
    n_vec = np.column_stack([
        np.sin(theta) * np.cos(phi),
        np.sin(theta) * np.sin(phi),
        np.cos(theta),
    ])  # (npix, 3)

    normals = []

    for ell in [2, 3, 4, 5]:
        # Zero out all alm except this ℓ
        alm_ell = np.zeros_like(alm)
        for m_ in range(ell + 1):
            idx = hp.Alm.getidx(lmax, ell, m_)
            alm_ell[idx] = alm[idx]

        # Reconstruct single-ℓ map
        T_ell = hp.alm2map(alm_ell, nside, lmax=lmax, verbose=False)

        # Quadrupolar tensor: M_ij = Σ_p T_ℓ²(p) n_i(p) n_j(p)
        T2 = T_ell ** 2
        w_n = T2[:, None] * n_vec          # (npix, 3)
        M = n_vec.T @ w_n                   # (3, 3)

        # Eigenvalues in ascending order; smallest → normal to the plane
        evals, evecs = np.linalg.eigh(M)
        normal = evecs[:, 0]
        normals.append(normal)

    normals = np.array(normals)  # (4, 3)

    # Handle antipodal symmetry: normals define planes, so n and -n are
    # equivalent.  Find orientations that maximise the resultant length.
    n_vecs = len(normals)
    best_R2 = -1.0

    for signs in range(1 << n_vecs):
        s = np.array([1 if (signs >> i) & 1 else -1 for i in range(n_vecs)])
        oriented = normals * s[:, None]
        s_sum = oriented.sum(axis=0)
        R2 = s_sum @ s_sum
        if R2 > best_R2:
            best_R2 = R2

    R = np.sqrt(best_R2)
    R_bar = R / n_vecs

    # MLE for κ: solve  coth(κ) − 1/κ = R_bar  via Newton's method
    if R_bar >= 1.0 - 1e-12:
        kappa = 1e4
    elif R_bar < 1e-10:
        kappa = 0.0
    else:
        # Initial guess (Banerjee et al. 2005 approximation)
        kappa = R_bar * (3.0 - R_bar ** 2) / max(1.0 - R_bar ** 2, 1e-10)
        kappa = max(kappa, 1e-6)

        for _ in range(100):
            if kappa > 500:
                f = 1.0 - 1.0 / kappa - R_bar
                fp = 1.0 / kappa ** 2
            else:
                ck = 1.0 / np.tanh(kappa)
                f = ck - 1.0 / kappa - R_bar
                fp = -1.0 / np.sinh(kappa) ** 2 + 1.0 / kappa ** 2

            if abs(fp) < 1e-30:
                break
            delta = f / fp
            kappa -= delta
            kappa = max(kappa, 1e-10)
            if abs(delta) < 1e-12 * max(1.0, kappa):
                break

    return float(kappa)


def summarize_results(planck_stat, sim_results):
    """Interpret the von Mises-Fisher κ relative to simulation ensemble."""
    sims = np.array(sim_results, dtype=float)
    sims_finite = sims[np.isfinite(sims)]
    n_sims = len(sims_finite)

    if n_sims == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims_finite))
    std_s = float(np.std(sims_finite, ddof=1)) if n_sims > 1 else 0.0
    p_value = float(np.mean(sims_finite >= planck_stat)) if n_sims > 0 else np.nan
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else np.nan

    return {
        "planck_stat": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "z_score": z_score,
        "p_value": p_value,
        "n_sims": n_sims,
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Multipole Plane Concentration",
            "xlabel": r"von Mises–Fisher $\kappa$",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck κ = {planck_stat:.3f}.  "
                f"Sim mean ± σ = {mean_s:.3f} ± {std_s:.3f}.  "
                f"z = {z_score:.2f}, p = {p_value:.3f}  "
                f"(fraction of sims ≥ Planck)."
            ),
        },
    }
