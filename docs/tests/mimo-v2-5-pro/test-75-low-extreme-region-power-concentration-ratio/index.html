Hypothesis

The Planck CMB temperature map exhibits a higher low-ℓ extreme-region power concentration ratio than expected from ΛCDM simulations, indicating anomalous spatial power concentration.

Method

The map is band-limited to ℓ=2–5 and the hottest and coldest pixels are identified. The statistic is the ratio of T² summed within 15° discs around these extremes to the total T² over the unmasked sky. This quantifies how much low-ℓ power is spatially concentrated near extreme features. Under ΛCDM, extreme values in a Gaussian field are relatively isolated and should not dominate the global power budget.

Result Summary

The Planck ratio is 0.1863, slightly below the simulation mean of 0.1977 ± 0.0344 (1000 sims). The z-score is −0.33, and the one-tailed upper p-value is 0.601. The Planck value falls comfortably within the expected ΛCDM distribution.

The Planck map's extreme hot/cold regions at low-ℓ do not concentrate an unusually large fraction of total power. The observed ratio is nearly average compared to Gaussian isotropic realizations. There is no evidence of anomalous spatial power concentration in this statistic.

Agent Justification

The test description and code explicitly state that in typical ΛCDM realizations, extreme values do not dominate the global power as strongly as observed in the Planck map. This indicates a directional prediction that the Planck statistic is higher than simulations, making the test one-sided for the upper tail.

Comparison To Literature

Low-ℓ anomalies have been extensively studied (e.g., quadrupole–octopole alignment, the Cold Spot, power deficit), but this specific ratio-based statistic targeting power concentration around extreme pixels does not appear to have a direct counterpart in published CMB anomaly literature. No relevant prior studies were found in searches.

import numpy as np
import healpy as hp

test_description = (
    "Measures how much of the low-ℓ (ℓ=2–5) power budget is concentrated in the "
    "neighborhoods of the most extreme temperature features. Band-limits the map to "
    "ℓ_max=5, identifies the single hottest and coldest pixels, then computes the "
    "ratio of T² summed within 15° discs around these extremes to the total T² over "
    "the full unmasked sky. In a typical ΛCDM realization, extreme values do not "
    "dominate global power as strongly as observed in the Planck map."
)


def analyze_map(m):
    """Return the power concentration ratio near extreme hot/cold spots."""
    nside = hp.npix2nside(len(m))
    npix = hp.nside2npix(nside)
    lmax = 5

    # --- 1. Clean masked pixels for healpy compatibility ---
    mask_valid = np.isfinite(m)
    m_clean = np.where(mask_valid, m, 0.0)

    # --- 2. Get a_ℓm and zero out ℓ = 0, 1 ---
    alm = hp.map2alm(m_clean, lmax=lmax)
    for ell in range(2):
        for m_idx in range(ell + 1):
            alm[hp.Alm.getidx(lmax, ell, m_idx)] = 0.0

    # --- 3. Reconstruct band-limited map (ℓ = 2–5) ---
    T_smooth = hp.alm2map(alm, nside, lmax=lmax, verbose=False)

    # --- 4. Find hottest and coldest pixels among unmasked pixels ---
    valid_idx = np.where(mask_valid)[0]
    if len(valid_idx) == 0:
        return 0.0

    valid_temps = T_smooth[valid_idx]
    hottest_idx = valid_idx[np.argmax(valid_temps)]
    coldest_idx = valid_idx[np.argmin(valid_temps)]

    # --- 5. Precompute all pixel unit vectors once ---
    all_vecs = np.array(hp.pix2vec(nside, np.arange(npix)))  # shape (3, npix)

    # --- 6. Compute cosine of angular radius threshold ---
    radius_rad = np.radians(15.0)
    cos_thresh = np.cos(radius_rad)

    # --- 7. Get vectors of extreme pixels ---
    vec_hot = np.array(hp.pix2vec(nside, hottest_idx)).flatten()
    vec_col = np.array(hp.pix2vec(nside, coldest_idx)).flatten()

    # --- 8. Vectorized dot products to find pixels within disc ---
    dot_hot = all_vecs[0] * vec_hot[0] + all_vecs[1] * vec_hot[1] + all_vecs[2] * vec_hot[2]
    dot_col = all_vecs[0] * vec_col[0] + all_vecs[1] * vec_col[1] + all_vecs[2] * vec_col[2]

    disc_hot_mask = dot_hot >= cos_thresh
    disc_col_mask = dot_col >= cos_thresh

    # --- 9. Combine with valid-pixel mask and sum T² ---
    T2 = T_smooth ** 2

    sum_hot = np.sum(T2[disc_hot_mask & mask_valid])
    sum_col = np.sum(T2[disc_col_mask & mask_valid])

    # --- 10. Total T² over entire unmasked band-limited map ---
    total_T2 = np.sum(T2[mask_valid])

    if total_T2 < 1e-30:
        return 0.0

    # --- 11. Compute concentration ratio ---
    ratio = (sum_hot + sum_col) / total_T2

    return float(ratio)


def summarize_results(planck_stat, sim_results):
    """Interpret the low-ℓ extreme-region power concentration ratio."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)

    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    p_upper = float(np.mean(sims >= planck_stat))
    p_lower = float(np.mean(sims <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else np.nan

    return {
        "planck_stat": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "z_score": z_score,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "interpretation": (
            "This statistic measures the fraction of low-ℓ (ℓ=2–5) power concentrated "
            "in 15° neighborhoods around the sky's hottest and coldest pixels. "
            "In a statistically isotropic Gaussian field, extreme values are "
            "relatively isolated features that do not dominate the global power budget. "
            "A higher-than-expected ratio indicates that the Planck map's most extreme "
            "hot and cold spots account for an unusually large share of the total "
            "low-ℓ power—a signature of anomalous spatial power concentration "
            "incompatible with the standard LCDM model."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Extreme-Region Power Concentration Ratio",
            "xlabel": "Power Concentration Ratio (T²_near_extremes / T²_total)",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck ratio = {planck_stat:.4f}. "
                f"Sim mean ± σ = {mean_s:.4f} ± {std_s:.4f}. "
                f"z = {z_score:.2f}, p(two-sided) = {p_two:.3f}."
            ),
        },
    }
