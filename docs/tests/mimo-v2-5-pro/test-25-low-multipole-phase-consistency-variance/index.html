Hypothesis

The circular variance V of low-ℓ (ℓ=2–5) m=1 phase angles is significantly lower than expected under isotropic ΛCDM, indicating anomalous phase alignment (coherence) among these modes.

Method

For multipoles ℓ=2 to 5, the phase θ_ℓ = arg(a_{ℓ,1} + i a_{ℓ,-1}) is extracted from the CMB temperature harmonic coefficients, and the circular variance V = 1 − |mean(exp(iθ_ℓ))| is computed. Low V indicates anomalous phase coherence among low-ℓ modes, potentially related to the quadrupole-octupole alignment.

Result Summary

Planck SMICA yields V = 0.4495. From 1000 isotropic ΛCDM simulations, the mean V = 0.5073 ± 0.2321 (theoretical isotropic expectation ≈ 0.2348). The Planck value falls at the 38.8th percentile of simulations (z = −0.249). The one-tailed lower p-value is 0.388; the two-sided p-value is 0.776. Anomaly strength is classified as negligible.

The observed circular variance is fully consistent with isotropic ΛCDM predictions. The Planck value is slightly below the simulation mean but well within one standard deviation, showing no evidence for anomalous phase alignment or coherence among the m=1 components of multipoles ℓ=2–5. There is no indication of the phase coherence that would signal a quadrupole-octupole alignment anomaly in this specific formulation.

Agent Justification

The test is specifically motivated by the well-known quadrupole-octupole alignment anomaly, which manifests as an unexpected coherence among the low-ℓ multipole phases. This anomaly corresponds to a circular variance V that is lower than the isotropic expectation, since coherent phases yield a high resultant length |mean(exp(iθ))|, thus low V. The test description, analysis code, and interpretive output all consistently emphasize that low V indicates anomalous phase alignment. There is no physical motivation or known anomaly for V being significantly high (i.e., phases being more dispersed than random) in this context. Therefore, the directional hypothesis focuses on the lower tail.

Comparison To Literature

The quadrupole-octupole alignment is a well-studied anomaly (e.g., Copi et al. 2010; Planck 2013 XVI), though typically quantified via directional correlation statistics rather than circular variance of m=1 phases. Previous online searches for this specific circular-variance statistic did not return relevant results, so direct literature comparison is limited.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Multipole Phase Consistency Variance: For multipoles ℓ=2 to 5, "
    "compute the phase θ_ℓ = arg(a_{ℓ,1} + i a_{ℓ,-1}) from the spherical "
    "harmonic coefficients of the CMB temperature map. For a real map, this "
    "reduces to the polar angle of the (ℓ, m=1) coefficient. The circular "
    "variance V = 1 - |mean(exp(iθ_ℓ))| measures phase coherence across "
    "these low multipoles. Under isotropic ΛCDM, phases are independent and "
    "uniform, giving E[V] ≈ 1 - J₀(1) ≈ 0.235. Significantly low V "
    "indicates anomalous phase alignment, potentially related to the "
    "quadrupole-octupole alignment anomaly."
)


def analyze_map(m):
    """Compute the circular variance of low-ℓ (ℓ=2-5) m=1 phase angles.

    Extracts the phase θ_ℓ = atan2(Im(a_{ℓ,1}), Re(a_{ℓ,1})) for each
    multipole ℓ=2 to 5, then computes the circular variance
    V = 1 - |mean(exp(iθ))|. Under isotropic ΛCDM, the four phases are
    independent uniform on [0, 2π], giving E[V] ≈ 1 - J₀(1) ≈ 0.235.
    Low V indicates anomalous phase alignment across low multipoles.
    """
    # Replace masked pixels with 0.0 for healpy compatibility
    m_filled = np.where(np.isfinite(m), m, 0.0)
    
    # Compute alm up to lmax=5
    lmax = 5
    alm = hp.map2alm(m_filled, lmax=lmax)
    
    # Extract phases for (ℓ, m=1) coefficients for ℓ=2..5
    # healpy alm indexing: index = m*(2*lmax+1-m)//2 + l for m >= 0
    # For m=1, indices are: 1*(2*5+1-1)//2 + l = 5 + l
    # So ℓ=2 → index 7, ℓ=3 → index 8, ℓ=4 → index 9, ℓ=5 → index 10
    l_values = np.arange(2, lmax + 1)
    m = 1
    indices = m * (2 * lmax + 1 - m) // 2 + l_values
    a_lm1 = alm[indices]
    
    # Compute phases
    phases = np.arctan2(a_lm1.imag, a_lm1.real)
    
    # Compute circular variance: V = 1 - |mean(exp(iθ))|
    result_len = np.abs(np.mean(np.exp(1j * phases)))
    V = 1.0 - result_len
    
    return float(V)


def summarize_results(planck_stat, sim_results):
    """Summarize the Low-ℓ Multipole Phase Consistency Variance test."""
    sim = np.array(sim_results, dtype=float)

        }

    mean_sim = float(np.mean(sim))
    std_sim = float(np.std(sim))
    # Expected value under isotropy: 1 - J_0(1) ≈ 0.2348
    expected_V = 0.2348

    p_upper = float(np.mean(sim >= planck_stat))
    p_lower = float(np.mean(sim <= planck_stat))
    p_value = float(min(2.0 * min(p_upper, p_lower), 1.0))

    rank_below = int(np.sum(sim < planck_stat))
    percentile = 100.0 * rank_below / n_sims

    if p_value <= 0.02:
        strength = "strong"
    elif p_value <= 0.10:
        strength = "moderate"
    else:
        strength = "negligible"

    direction = ""
    if planck_stat < mean_sim - 2 * std_sim:

            "This is consistent with the well-known quadrupole-octupole "
            "alignment anomaly, where low-ℓ multipoles appear to share a "
            "preferred axis, violating the expected isotropy of ΛCDM."
