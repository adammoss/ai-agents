Hypothesis

The CMB temperature map, when low-pass filtered to ℓ ≤ 20, exhibits a significant asymmetry in the mean squared Laplacian (curvature) between extremely hot and extremely cold large-scale regions, indicating an anomaly in the distribution of sharp features relative to standard ΛCDM predictions.

Method

This test computes the spherical Laplacian (∇²T) of the CMB temperature map low-pass filtered to ℓ ≤ 20. Pixels with temperatures beyond ±2σ from the mean are classified as hot or cold extremes. The statistic is the natural log of the ratio of the mean squared Laplacian for hot extremes to that for cold extremes.

Result Summary

The Planck statistic (ln ratio) is 0.0049. Simulations (ΛCDM) yield a mean ± standard deviation of 0.0048 ± 0.3640. The z-score is 0.00, with a two-tailed p-value of 0.992.

The value is consistent with zero asymmetry between the Laplacian (curvature) sharpness of extremely hot and cold large-scale CMB regions, indicating no departure from the standard model prediction.

Agent Justification

The test is exploratory and designed to detect any deviation in the Laplacian asymmetry statistic from simulations, without a strong prior directional prediction. The code computes both upper and lower p-values and interprets the direction based on the statistic's value (e.g., positive or negative), confirming a two-tailed approach to assess differences in either direction.

Comparison To Literature

Tests examining directional or scale-dependent CMB anomalies often use statistics like the dipole or multipole vectors. While the Laplacian is a common local descriptor, the specific use of an extreme-temperature Laplacian asymmetry ratio is not a standard, well-documented statistic in the literature for anomaly detection.

import numpy as np
import healpy as hp

test_description = (
    "Compute the spherical Laplacian (nabla^2 T) of the CMB temperature map low-pass "
    "filtered to l <= 20. Define extreme pixels as those whose low-l temperature lies "
    "beyond +/-2 standard deviations from the mean of the masked sky. The statistic is "
    "the natural logarithm of the ratio of mean squared Laplacian for hot extremes "
    "(T > +2 sigma) to that for cold extremes (T < -2 sigma): "
    "ln(<(nabla^2 T)^2>_hot / <(nabla^2 T)^2>_cold). This tests whether the sharpness "
    "of local curvature features is asymmetrically associated with extremely hot versus "
    "cold large-scale regions."
)


def analyze_map(m):
    """Return ln(mean_laplacian_sq_hot / mean_laplacian_sq_cold) for l<=20 map."""
    valid = np.isfinite(m)
    if not np.any(valid):
        return 0.0

    m_clean = np.where(valid, m, 0.0)
    nside = hp.npix2nside(len(m_clean))
    lmax = 20

    # Band-limit to l <= 20
    alm = hp.map2alm(m_clean, lmax=lmax)
    m_low = hp.alm2map(alm, nside=nside, lmax=lmax, verbose=False)

    # Compute spherical Laplacian via harmonic relation: nabla^2 Y_lm = -l(l+1) Y_lm
    ell = np.arange(lmax + 1, dtype=float)
    lap_factors = -ell * (ell + 1.0)
    alm_lap = hp.almxfl(alm, lap_factors)
    lap_map = hp.alm2map(alm_lap, nside=nside, lmax=lmax, verbose=False)

    # Statistics on valid (unmasked) pixels only
    m_low_valid = m_low[valid]
    mean_low = np.mean(m_low_valid)
    std_low = np.std(m_low_valid)

    if std_low == 0:
        return 0.0

    # Define extreme pixels
    hot_mask = valid & (m_low > mean_low + 2.0 * std_low)
    cold_mask = valid & (m_low < mean_low - 2.0 * std_low)

    n_hot = int(np.sum(hot_mask))
    n_cold = int(np.sum(cold_mask))

    if n_hot < 5 or n_cold < 5:
        return 0.0

    # Mean squared Laplacian for hot and cold extremes
    lap_sq_hot = np.mean(lap_map[hot_mask] ** 2)
    lap_sq_cold = np.mean(lap_map[cold_mask] ** 2)

    if lap_sq_cold <= 0.0 or lap_sq_hot <= 0.0:
        return 0.0

    stat = np.log(lap_sq_hot / lap_sq_cold)

    if not np.isfinite(stat):
        return 0.0

    return float(stat)


def summarize_results(planck_stat, sim_results):
    """Interpret the Low-ℓ Extreme-Temperature Laplacian Asymmetry statistic."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)

    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    median_s = float(np.median(sims))

    rank = int(np.sum(sims <= planck_stat))
    p_upper = float(np.mean(sims >= planck_stat))
    p_lower = float(np.mean(sims <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)

    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else 0.0

    if abs(z_score) < 1.0:
        direction = "consistent with LCDM"
    elif planck_stat > mean_s:
        direction = (
            "hot extremes have sharper Laplacian features than cold extremes "
            "(positive asymmetry)"
        )
    else:
        direction = (
            "cold extremes have sharper Laplacian features than hot extremes "
            "(negative asymmetry)"
        )

    return {
        "planck_stat": planck_stat,
        "planck_log_ratio": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "sim_median": median_s,
        "z_score": z_score,
        "rank": rank,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "direction": direction,
        "interpretation": (
            f"This statistic measures the asymmetry in Laplacian (curvature) power "
            f"between extremely hot and extremely cold large-scale (l<=20) CMB regions. "
            f"A value of 0 means hot and cold extremes have equal mean squared Laplacian. "
            f"Positive means hot regions have sharper features; negative means cold regions do. "
            f"Planck ln-ratio = {planck_stat:.4f}. "
            f"Sim mean ± std = {mean_s:.4f} ± {std_s:.4f}. "
            f"z = {z_score:.2f}, two-sided p = {p_two:.4f}. "
            f"Direction: {direction}."
        ),
