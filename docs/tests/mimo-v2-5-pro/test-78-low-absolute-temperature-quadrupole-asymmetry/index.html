Hypothesis

The Planck CMB temperature map exhibits a higher absolute-temperature quadrupole-to-monopole ratio (P₂/P₀) than expected from isotropic simulations, indicating that large temperature deviations are concentrated along preferred axes.

Method

Band-limit the Planck temperature map to ℓ ∈ [2, 10], compute |T_lowℓ| at each pixel, decompose into spherical harmonics, and form the ratio P₂/P₀ of quadrupole-to-monopole power in the absolute-temperature field. This probes whether large |T| deviations concentrate along preferred axes (elevated ratio) or distribute isotropically.

Result Summary

Planck P₂/P₀ = 0.0057, well below the simulation mean ± σ = 0.0417 ± 0.0252. z-score = −1.42; upper-tail p-value P(P₂/P₀ ≥ Planck) = 0.985 (1000 simulations). Lower-tail p-value = 0.015.

Contrary to the hypothesis, Planck's absolute-temperature quadrupole power is *suppressed* relative to simulations rather than enhanced. The large |T| deviations in the low-ℓ band do not exhibit quadrupolar concentration along preferred axes; instead, the |T_lowℓ| field is more isotropic—possibly more so—than typical ΛCDM realisations. The upper-tail p-value of 0.985 provides no support for the tested anomaly. The lower-tail signal (p ≈ 0.015, ~1.4σ) is mildly suggestive but does not reach the 2σ threshold.

Agent Justification

The test is specifically designed to measure directional anisotropy in the magnitude of temperature deviations, with an elevated P₂/P₀ ratio indicating concentration along preferred axes (e.g., elongation or dipole-like structures). The test description and code interpretation highlight that values significantly above the simulation mean signal such asymmetry, suggesting a clear directional preference for the upper tail in detecting anomalies. While the code computes both upper and lower p-values for exploratory purposes, the primary hypothesis is focused on the upper tail due to the physical interpretation of the statistic.

Comparison To Literature

Low-ℓ anomalies have been extensively studied (e.g., Planck Collaboration 2014, XVI; Schwarz et al. 2016, arXiv:1510.07929), including quadrupole–octopole alignment and hemispherical asymmetry. However, taking the absolute temperature field before extracting its quadrupolar structure is not a standard statistic in the literature, making direct comparison difficult. The suppression finding is consistent with the general pattern of low-ℓ power deficits rather than axis-aligned excesses.

import numpy as np
import healpy as hp

test_description = (
    "Band-limit the temperature map to ℓ ∈ [2, 10] using a sharp spherical-harmonic filter. "
    "At each unmasked pixel, compute the absolute temperature |T_lowℓ(p)|. Decompose |T_lowℓ| "
    "into spherical harmonics a_{ℓm}^{|T|} via map2alm (no further smoothing). Compute the "
    "monopole power P₀ = |a₀₀|² and quadrupole power P₂ = Σ_m |a_{2m}^{|T|}|² from the |T| alm. "
    "The statistic is the ratio P₂ / P₀. This measures the degree of directional anisotropy in "
    "the magnitude of temperature deviations—whether large |T| values are concentrated along "
    "preferred axes (high ratio) or distributed isotropically across the sky (low ratio). A "
    "cold–hot symmetric, isotropically distributed map yields P₂/P₀ dominated by sampling noise; "
    "a map with preferred elongation or dipole-like concentration of strong temperature deviations "
    "yields an elevated ratio. The statistic is distinct from gradient-based or raw-power tests "
    "because it probes the spatial organization of |T|, which folds together both positive and "
    "negative temperature extremes into a single field before measuring its quadrupolar structure."
)


def analyze_map(m):
    """Return P₂/P₀: quadrupole-to-monopole power ratio of the absolute band-limited temperature field."""
    nside = hp.npix2nside(len(m))
    lmax_filter = 10

    # Replace NaNs with 0 for healpy compatibility
    m_clean = np.where(np.isfinite(m), m, 0.0)

    # Step 1: Decompose into spherical harmonics up to lmax=10
    alm_low = hp.map2alm(m_clean, lmax=lmax_filter, iter=10)

    # Step 2: Zero out all coefficients with ℓ < 2 (keep ℓ ∈ [2, 10])
    for ell in range(0, 2):
        for em in range(0, ell + 1):
            idx = hp.Alm.getidx(lmax_filter, ell, em)
            alm_low[idx] = 0.0

    # Step 3: Reconstruct the band-limited map
    T_low = hp.alm2map(alm_low, nside=nside, lmax=lmax_filter)

    # Step 4: Compute absolute temperature |T_lowℓ| at each pixel
    abs_T = np.abs(T_low)

    # Step 5: Decompose |T_lowℓ| into spherical harmonics (lmax=2 is sufficient
    # since we only need ℓ=0 and ℓ=2 coefficients)
    alm_abs = hp.map2alm(abs_T, lmax=2, iter=0)

    # Step 6: Compute monopole power P₀ = |a₀₀|²
    # a₀₀ is at index 0 in the alm array
    a00 = alm_abs[0]
    P0 = np.abs(a00) ** 2

    # Step 7: Compute quadrupole power P₂ = Σ_m |a_{2m}|²
    # For ℓ=2, m=0,1,2: indices are 3, 4, 5 in standard alm ordering
    P2 = 0.0
    for em in range(0, 3):
        idx = hp.Alm.getidx(2, 2, em)
        P2 += np.abs(alm_abs[idx]) ** 2

    # Step 8: Return the ratio P₂/P₀
    if P0 < 1e-60:
        return 0.0

    return float(P2 / P0)


def summarize_results(planck_stat, sim_results):
    """Interpret the Low-ℓ Absolute-Temperature Quadrupole Asymmetry statistic."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)

    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    median_s = float(np.median(sims))

    # p-value: fraction of simulations with statistic >= Planck (anomalously high quadrupole)
    p_upper = float(np.mean(sims >= planck_stat))
    # p-value: fraction of simulations with statistic <= Planck (anomalously low quadrupole)
    p_lower = float(np.mean(sims <= planck_stat))
    # Two-sided p-value
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)

    # z-score relative to simulation ensemble
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else 0.0

    return {
        "planck_stat": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "sim_median": median_s,
        "z_score": z_score,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "interpretation": (
            f"P₂/P₀ = quadrupole-to-monopole power ratio of |T_lowℓ|. "
            f"Values significantly above the simulation mean indicate that large |T| "
            f"deviations are concentrated along preferred axes (quadrupolar structure), "
            f"while values near or below the mean suggest isotropic distribution of "
            f"temperature magnitudes. "
            f"Planck z-score = {z_score:.2f} relative to simulations. "
            f"P(P₂/P₀ ≥ Planck) = {p_upper:.4f}."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Absolute-Temperature Quadrupole Asymmetry",
            "xlabel": "P₂ / P₀ (Quadrupole / Monopole power of |T_lowℓ|)",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck P₂/P₀ = {planck_stat:.4f}. "
                f"Sim mean ± σ = {mean_s:.4f} ± {std_s:.4f}. "
                f"z = {z_score:.2f}, P(≥ Planck) = {p_upper:.4f}."
            ),
        },
    }
