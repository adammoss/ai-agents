Hypothesis

The slope of pixel skewness versus ℓ_max for low multipoles is different from zero, indicating scale-dependent non-Gaussianity in the CMB temperature map compared to the LCDM prediction.

Method

For each ℓ_max ∈ {5, 8, 10, 13, 15, 18, 20}, the CMB temperature map is band-limited and pixel skewness γ(ℓ_max) = ⟨T³⟩/⟨T²⟩^(3/2) computed over unmasked pixels. A linear fit of γ versus ℓ_max yields the slope Δγ/Δℓ_max, capturing whether low-ℓ asymmetry strengthens, weakens, or reverses across the large-scale multipole range. The statistic is compared against 1000 ΛCDM simulations.

Result Summary

Planck slope = −0.00163. Simulation mean ± std = −0.00013 ± 0.01510. The z-score is −0.10, yielding a two-tailed p-value of 0.91. The Planck value lies well within the simulation distribution, near the median.

The pixel skewness shows no significant dependence on ℓ_max in the Planck data. The measured slope is entirely consistent with zero, indicating no detectable scale-dependent non-Gaussianity in the temperature distribution across the low-ℓ range. There is no evidence for residual foreground contamination, ISW-lensing coupling, or exotic sources preferentially affecting particular angular scales.

Agent Justification

The test is exploratory, designed to detect any deviation from the expected zero slope under a Gaussian LCDM model, without a prior directional prediction. The description and code allow for anomalies that could be positive or negative, reflecting various physical sources (e.g., foregrounds or exotic effects), and the summary function explicitly computes a two-sided p-value. Thus, a two-tailed test is appropriate for assessing whether the slope is statistically different from zero in either direction.

Comparison To Literature

No directly comparable study was found. Existing low-ℓ non-Gaussianity analyses (e.g., Copi et al. 2004 on multipole vectors; Abramo et al. 2006 on alignment tests) focus on phase correlations and multipole geometry rather than pixel-domain skewness as a function of ℓ_max. The search returned no prior work using this specific statistic.

import numpy as np
import healpy as hp

test_description = (
    "For each l_max in {5, 8, 10, 13, 15, 18, 20}, reconstruct a band-limited temperature "
    "map by retaining only spherical harmonic coefficients a_{lm} with l <= l_max. Compute the "
    "pixel skewness gamma(l_max) = <T^3>_pix / <T^2>_pix^{3/2} over all unmasked pixels in "
    "each band-limited map. Perform an ordinary least-squares linear fit of gamma versus l_max "
    "across the seven scale values and report the fitted slope Delta-gamma/Delta-l_max as the "
    "single scalar anomaly statistic. This captures whether low-l non-Gaussianity, as measured "
    "by the asymmetry of the temperature distribution, strengthens, weakens, or reverses across "
    "the large-scale multipole range. For a statistically isotropic Gaussian LCDM field the "
    "skewness is consistent with zero at every l_max, so the slope is consistent with zero; "
    "an anomalous slope would indicate scale-dependent non-Gaussianity from residual foregrounds, "
    "ISW-lensing coupling, or exotic sources concentrated at particular angular scales."
)


def analyze_map(m):
    """Return the slope of pixel skewness vs l_max across low multipoles.

    For each l_max in {5, 8, 10, 13, 15, 18, 20}, band-limit the map to l <= l_max,
    compute the skewness of the temperature distribution over unmasked pixels, then
    fit a line gamma = slope * l_max + intercept and return the slope.
    """
    valid = np.isfinite(m)
    if not np.any(valid):
        return 0.0

    m_clean = np.where(valid, m, 0.0)
    nside = hp.npix2nside(len(m_clean))

    lmax_values = [5, 8, 10, 13, 15, 18, 20]
    lmax_max = max(lmax_values)  # 20

    # Compute spherical harmonic coefficients once at highest lmax needed
    alm = hp.map2alm(m_clean, lmax=lmax_max)
    ell_arr, _ = hp.Alm.getlm(lmax=lmax_max)

    skewnesses = []
    for lmax in lmax_values:
        # Zero out coefficients with l > lmax
        alm_f = alm.copy()
        alm_f[ell_arr > lmax] = 0.0

        # Reconstruct band-limited map
        m_f = hp.alm2map(alm_f, nside=nside, lmax=lmax_max, verbose=False)

        # Compute skewness on original valid (unmasked) pixels
        T = m_f[valid]
        T_c = T - T.mean()
        var = (T_c ** 2).mean()
        if var > 0:
            skew = (T_c ** 3).mean() / (var ** 1.5)
        else:
            skew = 0.0
        skewnesses.append(skew)

    # Ordinary least-squares linear fit: gamma = slope * l_max + intercept
    x = np.array(lmax_values, dtype=float)
    y = np.array(skewnesses, dtype=float)
    slope, intercept = np.polyfit(x, y, 1)

    return float(slope)


def summarize_results(planck_stat, sim_results):
    """Interpret the Low-l Skewness Scale Dependence Slope statistic."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)

    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    median_s = float(np.median(sims))

    # Fraction of simulations with slope as extreme or more extreme than Planck
    p_upper = float(np.mean(sims >= planck_stat))
    p_lower = float(np.mean(sims <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)

    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else 0.0

    # Determine direction of anomaly
    if abs(z_score) < 1.0:
        direction = "consistent with LCDM"
    elif planck_stat > mean_s:
        direction = "positive skewness slope (increasing asymmetry with l_max)"
    else:
        direction = "negative skewness slope (decreasing asymmetry with l_max)"

    return {
        "planck_stat": planck_stat,
        "planck_slope": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "sim_median": median_s,
        "z_score": z_score,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "direction": direction,
        "interpretation": (
            f"The skewness slope measures how the pixel temperature asymmetry "
            f"(skewness) changes across the low-l range l_max in {{5,...,20}}. "
            f"A positive slope means the temperature distribution becomes more "
            f"positively skewed at finer angular scales; a negative slope means "
            f"the asymmetry decreases or reverses. "
            f"Planck slope = {planck_stat:.6f}. "
            f"Sim mean +/- std = {mean_s:.6f} +/- {std_s:.6f}. "
            f"z = {z_score:.2f}, two-sided p = {p_two:.4f}. "
            f"Direction: {direction}. "
            f"A significantly non-zero slope would indicate scale-dependent "
            f"non-Gaussianity, e.g. from residual foregrounds, ISW-lensing "
            f"coupling, or exotic sources at particular angular scales."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-l Skewness Scale Dependence Slope",
            "xlabel": "Slope Δγ / Δℓ_max",
            "ylabel": "Count",
            "bins": 30,

                f"Planck slope = {planck_stat:.6f}. "

                f"z = {z_score:.2f}, P(two-sided) = {p_two:.4f}."
