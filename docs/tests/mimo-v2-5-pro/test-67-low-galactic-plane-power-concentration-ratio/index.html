Hypothesis

The Planck SMICA map exhibits a different average fraction of low-ℓ power (ℓ=2–20) concentrated within ±10° of the Galactic plane compared to the ΛCDM simulation distribution, indicating a potential anomaly in large-scale power distribution.

Method

For multipoles ℓ = 2–20, the ratio of angular power within |b| ≤ 10° of the Galactic plane to total-sky power is computed at each ℓ and averaged to produce a single scalar. The Planck SMICA value (0.0631) is compared against a distribution from 1000 ΛCDM simulations (mean ± σ = 0.0484 ± 0.0123). This tests whether an anomalous fraction of large-scale power is concentrated near the Galactic plane.

Result Summary

z = 1.19, two-tailed p = 0.248. Planck shows more low-ℓ power near the Galactic plane than the simulation median, but the deviation is well within the 2σ range (thresholds at ~0.029 and ~0.075).

The higher Planck ratio (0.0631 vs. simulation mean 0.0484) indicates somewhat more large-scale power near the Galactic plane than ΛCDM predicts, possibly hinting at residual foreground contamination or a genuine cosmological feature. However, at only 1.19σ significance, this does not constitute a statistically significant anomaly. The result is fully consistent with ΛCDM at standard thresholds.

Comparison To Literature

Natale et al. (2020) reported that the low-ℓ lack-of-power anomaly is correlated with Galactic latitude, finding ~3σ evidence that large-scale power is preferentially concentrated near the Galactic plane. Our test probes a related phenomenon using a direct power-fraction ratio, but the specific multipole-by-multipole concentration statistic averaged over ℓ = 2–20 does not appear to have been studied in a directly comparable form. The lack of anomalous signal here may reflect differences in statistic design or latitude-band choice.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Galactic Plane Power Concentration Ratio: "
    "Compute the angular power spectrum of the temperature map for multipoles ℓ = 2 to 20. "
    "For each ℓ, calculate the ratio of the power within a galactic latitude band of ±10° "
    "from the Galactic plane to the total sky power at that ℓ. Average these ratios over "
    "ℓ = 2–20 to obtain a single scalar value. This statistic measures the fraction of low-ℓ "
    "power concentrated near the Galactic plane. Compare the Planck SMICA value to the "
    "distribution from 1000 ΛCDM simulations."
)


def analyze_map(m):
    """Return the average fraction of low-ℓ power in the Galactic plane band (|b| ≤ 10°)."""
    lmax = 20
    lmin = 2

    finite_mask = np.isfinite(m)
    if not np.any(finite_mask):
        return 0.0

    nside = hp.npix2nside(len(m))
    npix = hp.nside2npix(nside)

    # Replace NaN with 0 for healpy transforms
    m_clean = np.where(finite_mask, m, 0.0)

    # Compute total sky angular power spectrum
    cl_full = hp.anafast(m_clean, lmax=lmax)

    # Build Galactic plane mask: |b| <= 10°
    # Convert Galactic coordinates to Healpy convention (colatitude = pi/2 - latitude)
    theta, phi = hp.pix2ang(nside, np.arange(npix))
    gal_lat = np.pi / 2.0 - theta  # Galactic latitude in radians
    gal_band_mask = np.abs(gal_lat) <= np.deg2rad(10.0)

    # Combine with valid-pixel mask (exclude originally masked pixels)
    band_mask = gal_band_mask & finite_mask

    # Apply band mask to the cleaned map
    m_band = np.where(band_mask, m_clean, 0.0)

    # Compute power spectrum of the Galactic-band map
    cl_band = hp.anafast(m_band, lmax=lmax)

    # Extract ℓ = 2 through 20
    ell = np.arange(lmin, lmax + 1)
    cl_total = cl_full[lmin:lmax + 1]
    cl_plane = cl_band[lmin:lmax + 1]

    # Compute ratio for each ℓ, avoiding division by zero
    # Use a small floor to stabilize
    power_floor = np.max(cl_total) * 1e-30 if np.max(cl_total) > 0 else 1e-30
    ratios = cl_plane / np.maximum(cl_total, power_floor)

    # Average ratio over ℓ = 2–20
    avg_ratio = float(np.mean(ratios))

    # Clamp to [0, 1] for robustness
    avg_ratio = float(np.clip(avg_ratio, 0.0, 1.0))

    return avg_ratio


def summarize_results(planck_stat, sim_results):
    """Interpret Galactic plane power concentration ratio relative to simulations."""
    sims = np.array(sim_results, dtype=float)
    sims_finite = sims[np.isfinite(sims)]
    n_sims = len(sims_finite)

    if n_sims == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims_finite))
    std_s = float(np.std(sims_finite, ddof=1)) if n_sims > 1 else 0.0
    p_value_upper = float(np.mean(sims_finite >= planck_stat))
    p_value_lower = float(np.mean(sims_finite <= planck_stat))
    p_value_two_sided = 2.0 * min(p_value_upper, p_value_lower)
    p_value_two_sided = min(p_value_two_sided, 1.0)
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else np.nan

    if planck_stat > mean_s:
        tail_note = "Planck has more low-ℓ power concentrated near the Galactic plane than simulations."
    else:
        tail_note = "Planck has less low-ℓ power concentrated near the Galactic plane than simulations."

    return {
        "planck_stat": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "z_score": z_score,
        "p_value_upper": p_value_upper,
        "p_value_lower": p_value_lower,
        "p_value_two_sided": p_value_two_sided,
        "n_sims": n_sims,
        "tail_note": tail_note,
        "interpretation": (
            "This statistic measures the average fraction of low-ℓ (ℓ=2–20) angular power "
            "concentrated within ±10° of the Galactic plane. The ratio is computed as the power "
            "spectrum of the masked Galactic-band map divided by the full-sky power spectrum at "
            "each multipole, then averaged. A higher value indicates that a disproportionate "
            "share of large-scale power originates from near the Galactic plane, which could "
            "suggest residual Galactic contamination or a genuine cosmological anomaly. "
            "In ΛCDM, CMB fluctuations are isotropic, so the expected ratio should be roughly "
            "equal to the fractional sky area of the band (~2sin(10°) ≈ 0.347 for the cosine-"
            "weighted band). Significant excess could indicate foreground leakage into the CMB map."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Galactic Plane Power Concentration Ratio",
            "xlabel": "Average fraction of power in |b| ≤ 10°",
            "ylabel": "Count",

                f"Planck ratio = {planck_stat:.4f}. "

                f"Higher values indicate more low-ℓ power near the Galactic plane."

        },
