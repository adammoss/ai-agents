Hypothesis

The low-ℓ (ℓ=2-10) gradient magnitude interquartile range (IQR) of the CMB temperature map deviates significantly from the distribution expected based on simulations, indicating an anomaly in the spatial variation of large-scale temperature patterns.

Method

This test measures the statistical dispersion of spatial variation in the large-scale (ℓ=2-10) CMB temperature pattern by computing the interquartile range (IQR) of the gradient magnitudes. The gradient is calculated from the low-ℓ spherical harmonic coefficients, providing a robust, non-parametric summary of the map's spatial gradient structure.

Result Summary

The Planck IQR value is 113.82 µK/rad. This is lower than the simulation mean of 135.47 ± 12.31 µK/rad, corresponding to a z-score of -1.76 and a two-tailed p-value of 0.074.

The large-scale CMB temperature map exhibits notably lower gradient dispersion than expected from ΛCDM simulations, indicating a smoother-than-anticipated pattern at the largest angular scales. However, this deviation does not reach the conventional 2σ significance threshold (p < 0.05).

Agent Justification

The test is exploratory and aims to detect any significant deviation from the simulated distribution, whether the gradient dispersion is unusually low (indicating a smoother low-ℓ pattern) or unusually high (indicating sharper features). This is evident from the code, which computes both upper and lower p-values and uses a two-sided p-value for interpretation, with no directional preference specified in the test description.

Comparison To Literature

While low-ℓ anomalies like the quadrupole-octupole alignment and hemispherical power asymmetry are well-studied, the gradient magnitude IQR as a statistic appears to be a new probe of large-scale spatial variation. A direct literature comparison for this specific metric was not found (see note below).

import numpy as np
import healpy as hp

test_description = (
    "Compute the gradient magnitude of the low-ℓ (ℓ=2-10) temperature map. "
    "The gradient is calculated in spherical harmonic space up to ℓ=10 and then "
    "sampled at the nside=256 pixels. The interquartile range (IQR) of the gradient "
    "magnitudes over the unmasked pixels is then calculated as the difference between "
    "the 75th and 25th percentiles. This scalar measures the statistical dispersion "
    "of the gradient magnitudes in the low-ℓ map, providing a robust summary of the "
    "spatial variation without relying on parametric assumptions."
)


def analyze_map(m):
    """Return the IQR of gradient magnitudes for the low-ℓ (ℓ=2-10) temperature map.

    Steps:
    1. Replace NaN pixels with 0.0 for healpy compatibility.
    2. Compute spherical harmonic coefficients a_lm up to ℓ=10.
    3. Zero out ℓ=0 and ℓ=1 to isolate ℓ=2-10.
    4. Reconstruct the gradient components (dθ, dφ) via alm2map_der1.
    5. Compute gradient magnitude G = sqrt(dθ² + dφ²).
    6. Return IQR = Q75(G) - Q25(G) over unmasked pixels.
    """
    valid = np.isfinite(m)
    n_valid = int(np.sum(valid))

    if n_valid < 50:
        return 0.0

    # Replace NaN with 0.0 for healpy transforms
    m_clean = np.where(valid, m, 0.0)
    nside = hp.npix2nside(len(m_clean))

    # Band-limit to ℓ_max = 10
    lmax = 10
    alm = hp.map2alm(m_clean, lmax=lmax, pol=False)

    # Zero out ℓ=0 and ℓ=1 to isolate low-ℓ features in ℓ=2-10
    for ell in range(0, 2):
        for m_val in range(0, ell + 1):
            idx = hp.Alm.getidx(lmax, ell, m_val)
            alm[idx] = 0.0

    # Compute gradient: alm2map_der1 returns (map, dtheta, dphi)
    _, grad_theta, grad_phi = hp.alm2map_der1(alm, nside, lmax=lmax)

    # Gradient magnitude
    grad_mag = np.sqrt(grad_theta ** 2 + grad_phi ** 2)

    # Select only unmasked (valid) pixels
    grad_valid = grad_mag[valid]

    if len(grad_valid) < 50:
        return 0.0

    # Interquartile range: Q75 - Q25
    q25 = np.percentile(grad_valid, 25)
    q75 = np.percentile(grad_valid, 75)
    iqr = q75 - q25

    return float(iqr)


def summarize_results(planck_stat, sim_results):
    """Interpret the Low-ℓ Gradient Magnitude IQR statistic."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)

    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    median_s = float(np.median(sims))
    rank = int(np.sum(sims <= planck_stat))
    p_upper = float(np.mean(sims >= planck_stat))
    p_lower = float(np.mean(sims <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else 0.0

    if std_s > 0:
        pct_rank = rank / n * 100
    else:
        pct_rank = 50.0

    if pct_rank < 5:
        dispersion_desc = "much lower gradient dispersion than expected (very smooth low-ℓ pattern)"
    elif pct_rank < 25:
        dispersion_desc = "notably low gradient dispersion (relatively smooth large-scale features)"
    elif pct_rank < 75:
        dispersion_desc = "typical gradient dispersion consistent with simulations"
    elif pct_rank < 95:
        dispersion_desc = "notably high gradient dispersion (sharper large-scale features)"
    else:
        dispersion_desc = "much higher gradient dispersion than expected (anomalously sharp low-ℓ pattern)"

    return {
        "planck_stat": planck_stat,
        "gradient_iqr": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "sim_median": median_s,
        "z_score": z_score,
        "rank": rank,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "percentile_rank": pct_rank,
        "dispersion_description": dispersion_desc,
        "interpretation": (
            f"The interquartile range (IQR) of the low-ℓ (ℓ=2-10) gradient magnitude "
            f"is {planck_stat:.6f} µK/rad. This measures the spread of spatial variation "
            f"in the large-scale temperature pattern. "
            f"Planck IQR = {planck_stat:.6f} ({dispersion_desc}). "
            f"Simulation mean ± std = {mean_s:.6f} ± {std_s:.6f}. "
            f"z = {z_score:.2f}, two-sided p = {p_two:.4f}."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Gradient Magnitude Interquartile Range",
            "xlabel": "IQR of gradient magnitude (µK/rad)",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck IQR = {planck_stat:.6f}. "
                f"Sim mean ± std = {mean_s:.6f} ± {std_s:.6f}. "
                f"z = {z_score:.2f}, P(two-sided) = {p_two:.4f}."
            ),
        },
