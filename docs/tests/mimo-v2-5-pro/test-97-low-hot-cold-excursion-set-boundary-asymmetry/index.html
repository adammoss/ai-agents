Hypothesis

The observed Planck CMB map shows a significant asymmetry in the boundary complexity (perimeter-to-area ratio) between hot and cold excursion sets at low multipoles, deviating from the symmetry expected in a Gaussian random field.

Method

For a band-limited CMB temperature map (ℓ ≤ 32), excursion sets are computed at the ±1σ thresholds. The perimeter-to-area ratio (boundary edge count / pixel count) is calculated for each set on the HEALPix grid. The statistic is ln(hot P/A ratio / cold P/A ratio), measuring relative boundary complexity between hot and cold extreme-temperature structures.

Result Summary

The Planck value of ln(hot/cold P/A) = 0.1020, indicating slightly rougher hot-region boundaries. Simulation mean ± std = 0.0004 ± 0.1164, yielding z = 0.87 and a two-tailed p-value = 0.386. The observed value lies well within the simulation distribution (80.7th percentile).

The hot and cold excursion set boundaries are comparably complex, fully consistent with the up-down symmetry expected of a Gaussian random field. There is no evidence of non-Gaussian asymmetry between hot and cold low-ℓ structures. The result is unremarkable and carries no tension with ΛCDM expectations.

Agent Justification

The test statistic is a log ratio that can be positive (hot rougher) or negative (cold rougher). The analysis code explicitly computes both upper and lower p-values and combines them into a two-sided p-value. The test is exploratory, looking for any asymmetry without a prior directional preference, making a two-tailed approach appropriate.

Comparison To Literature

Prior work has studied excursion set perimeter estimation (Cotsakis et al. 2023, arXiv:2307.12963) and hot/cold spot isotropy separately (Khan & Saha 2022, arXiv:2211.02929). Direct searches for the specific perimeter-to-area hot-cold log-ratio statistic did not return prior studies; related analyses of hot-cold morphological asymmetry and boundary complexity on the CMB sphere appear sparse.

import numpy as np
import healpy as hp

test_description = (
    "Low-ℓ Hot-Cold Excursion Set Boundary Asymmetry. "
    "For a band-limited temperature map (ℓ ≤ 32), compute the excursion set "
    "for pixels with temperature > +1σ (hot region) and < -1σ (cold region), "
    "where σ is the map RMS over unmasked pixels. For each excursion set, "
    "calculate the perimeter-to-area ratio as (boundary edge count) / (pixel count), "
    "where a boundary edge connects a set pixel to a valid non-set neighbor. "
    "The statistic is the natural log of the ratio of hot to cold perimeter-to-area ratios, "
    "measuring relative boundary complexity or roughness between hot and cold "
    "extreme-temperature structures."
)


def analyze_map(m):
    """Return ln(hot_P/A  /  cold_P/A) for low-ℓ excursion sets.

    Hot set  : filtered T > +1σ   (σ = RMS of filtered map over unmasked pixels)
    Cold set : filtered T < -1σ
    P/A      : perimeter-to-area ratio on the HEALPix grid.
    """
    nside = hp.npix2nside(len(m))
    lmax = 32
    npix = len(m)
    valid = np.isfinite(m)
    n_valid = int(np.sum(valid))

    if n_valid < 100:
        return 0.0

    # Band-limit to ℓ ≤ 32
    m_filled = np.where(valid, m, 0.0)
    alm = hp.map2alm(m_filled, lmax=lmax)
    m_low = hp.alm2map(alm, nside, lmax=lmax, verbose=False)

    # Map RMS (std of valid pixels) as the 1σ threshold
    valid_vals = m_low[valid]
    sigma = np.std(valid_vals)

    if sigma < 1e-30:
        return 0.0

    # Excursion sets
    hot = valid & (m_low > sigma)
    cold = valid & (m_low < -sigma)
    n_hot = int(np.sum(hot))
    n_cold = int(np.sum(cold))

    if n_hot < 5 or n_cold < 5:
        return 0.0

    # Build full-sky neighbour array  (npix, 8), -1 for absent
    # hp.get_all_neighbours returns shape (8, npix), so transpose
    all_idx = np.arange(npix)
    neighbors_full = hp.get_all_neighbours(nside, all_idx).T  # now (npix, 8)

    # Helper: perimeter-to-area ratio
    def boundary_fraction(set_mask, valid_mask, nbrs_arr):
        idx = np.where(set_mask)[0]
        n_set = len(idx)
        if n_set == 0:
            return 0.0
        nbrs = nbrs_arr[idx]                       # (n_set, 8)
        ok = nbrs >= 0
        nbrs_safe = np.where(ok, nbrs, 0)          # safe index for gather
        nbr_in_set = set_mask[nbrs_safe]
        nbr_valid = valid_mask[nbrs_safe]
        is_boundary_edge = ok & nbr_valid & (~nbr_in_set)
        total_boundary_edges = int(np.sum(is_boundary_edge))
        return total_boundary_edges / n_set

    # Perimeter-to-area ratios
    hot_pa = boundary_fraction(hot, valid, neighbors_full)
    cold_pa = boundary_fraction(cold, valid, neighbors_full)

    if cold_pa < 1e-15 or hot_pa < 1e-15:
        return 0.0

    stat = float(np.log(hot_pa / cold_pa))
    return stat if np.isfinite(stat) else 0.0


def summarize_results(planck_stat, sim_results):
    """Interpret the Low-ℓ Hot-Cold Excursion Set Boundary Asymmetry."""
    sims = np.asarray(sim_results, dtype=float)
    sims = sims[np.isfinite(sims)]
    n = len(sims)
    if n == 0:
        return {"planck_stat": planck_stat, "note": "No finite simulation results."}

    mean_s = float(np.mean(sims))
    std_s = float(np.std(sims, ddof=1)) if n > 1 else 0.0
    median_s = float(np.median(sims))
    rank = int(np.sum(sims <= planck_stat))
    p_upper = float(np.mean(sims >= planck_stat))
    p_lower = float(np.mean(sims <= planck_stat))
    p_two = min(2.0 * min(p_upper, p_lower), 1.0)
    z_score = (planck_stat - mean_s) / std_s if std_s > 0 else 0.0
    pct_rank = rank / n * 100 if n > 0 else 50.0

    if pct_rank < 2.5 or pct_rank > 97.5:
        sig_desc = "highly anomalous (<2.5% or >97.5% of sims)"
    elif pct_rank < 5 or pct_rank > 95:
        sig_desc = "marginally anomalous (<5% or >95% of sims)"
    elif pct_rank < 25 or pct_rank > 75:
        sig_desc = "noteworthy (outside IQR of sims)"
    else:
        sig_desc = "typical (within IQR of sims)"

    if planck_stat > mean_s + std_s:
        asym_desc = (
            "hot regions have significantly rougher/more complex boundaries than "
            "cold regions, beyond what Gaussian-field symmetry predicts—possible "
            "indication of non-Gaussian asymmetry between hot and cold structures"
        )
    elif planck_stat < mean_s - std_s:
        asym_desc = (
            "cold regions have significantly rougher/more complex boundaries than "
            "hot regions, beyond what Gaussian-field symmetry predicts—possible "
            "indication of non-Gaussian asymmetry between hot and cold structures"
        )
    else:
        asym_desc = (
            "hot and cold excursion set boundaries are comparably complex, "
            "consistent with the up-down symmetry expected of a Gaussian random field"
        )

    return {
        "planck_stat": planck_stat,
        "log_hot_cold_pa_ratio": planck_stat,
        "sim_mean": mean_s,
        "sim_std": std_s,
        "sim_median": median_s,
        "z_score": z_score,
        "rank": rank,
        "p_value_upper": p_upper,
        "p_value_lower": p_lower,
        "p_value_two_sided": p_two,
        "n_sims": n,
        "percentile_rank": pct_rank,
        "boundary_asymmetry": asym_desc,
        "significance_description": sig_desc,
        "interpretation": (
            f"Log hot/cold perimeter-to-area ratio = {planck_stat:.4f}. "
            f"A value of 0 means hot and cold extreme regions have equal boundary "
            f"complexity. Positive means hot regions are rougher; negative means "
            f"cold regions are rougher. "
            f"Asymmetry: {asym_desc}. "
            f"Sim mean ± std = {mean_s:.4f} ± {std_s:.4f}. "
            f"{sig_desc.capitalize()}. z = {z_score:.2f}, two-sided p = {p_two:.4f}."
        ),
        "plot_spec": {
            "kind": "histogram",
            "title": "Low-ℓ Hot-Cold Excursion Set Boundary Asymmetry",
            "xlabel": "ln(hot P/A / cold P/A)",
            "ylabel": "Count",
            "bins": 30,
            "caption": (
                f"Planck ln(hot P/A / cold P/A) = {planck_stat:.4f}. "
                f"Sim mean ± std = {mean_s:.4f} ± {std_s:.4f}. "
                f"z = {z_score:.2f}, P(two-sided) = {p_two:.4f}."
            ),
        },
    }
